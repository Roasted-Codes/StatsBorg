Compiler Error C2477

'member' : static data member cannot be initialized via derived class

A static data member of a template class was initialized incorrectly. Versions of the Visual C++ compiler earlier than Visual Studio .NET 2003 did not enforce this error. However, this error is now enforced to enable Visual Studio to conform more fully to the ISO C++ standard.

Example

// C2477.cpp
// compile with: /Za /LD
template <class T>
struct S
{
   static int n;
};

struct X
{
};

struct A: S<X>
{
};

int A::n = 0;   // C2477
// try the following line instead
// int S<X>::n = 0;
