Image Builder Error IM1033

invalid cluster size specified with /UDCLUSTER

The cluster size you specify with the /UDCLUSTER option must be either 16K, 32K, or 64K. This means you can pass the values 16, 32, or 64 with the /UDCLUSTER option. You can also specify the cluster size in bytes as 16384, 32768, or 65536. Image Builder does not recognize any other values.
