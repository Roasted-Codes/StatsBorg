Compiler Error Messages

This section is a reference for the errors generated by the build tools. To get help on a particular error message, either click on an error number in the Output window (in the Visual C++ IDE) and press F1, or type the error number in index tab of this Help file.

Note   The topics that follow are not a comprehensive collection of all the errors you may encounter when building Xbox games. The errors documented here have particular relevance to Xbox developers. Other errors you may encounter are documented in the MSDN online help.
