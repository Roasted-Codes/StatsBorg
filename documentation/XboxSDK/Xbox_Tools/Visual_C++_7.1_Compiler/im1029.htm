Image Builder Warning IM1029

library 'library' is unapproved

The library your game links to is not approved for release. At certification, your game must link only with approved libraries. Approved libraries are defined on a month-by-month basis and will be documented in each month's XDK. In general, release versions of the Xbox Title Libraries are approved. Debug versions, special debug libraries, and beta or preview libraries of new functionality are unapproved.

To disable warnings about unapproved libraries, use the /NOLIBWARN property.
