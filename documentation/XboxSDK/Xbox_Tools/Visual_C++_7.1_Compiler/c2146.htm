Compiler Error C2146

syntax error : missing 'token' before identifier 'identifier'

The compiler expected token and found identifier instead.

Probable cause

  • A typographical error. Compiler rrror C2065 usually precedes this error.

The following sample generates C2146:

void main(void)
{
   intt x; //Syntax error: missing semicolon before 'x'
}

You might also see this error because explicit specializations no longer find template parameters from the primary template.

The use of T from the primary template is not allowed in an explicit specialization. For your code to be valid, replace all instances of the template parameter in the specialization with the explicitly specialized type.

The following sample compiles in Visual Studio .NET but fails in Visual Studio .NET 2003:

// C2146b.cpp
// compile with: /LD
template<class T>
struct S;

template <> 
struct S<int> 
{
   T m_t;   // C2146
   // Try the following line instead:
   // int m_t;
};
