API Monitor

The Xbox Development Kit (XDK) provides a tool called the API Monitor for verifying that functions in an application program interface (API) are used in an Xbox application and for monitoring calls to those functions. API Monitor can mimic or spoof calls to API functions and send back programmer-selected values to the calling function. This provides a wide range of testing capabilities.
