Test Suite Tool (API Monitor SE)

When using the API Monitor test suite execution program (API Monitor SE), you begin by selecting and loading a test suite. As you execute your Xbox application, you can monitor the API functions specified by the test suite. The API Monitor SE program automatically logs the results of the test. You can save the log to a file.

Selecting Test Suites

To selecte a test suite, start the API Monitor SE program and click the Load button. API Monitor SE displays a dialog box asking you to specify a file to load. All test suites reside in XML files. Use the Open Suite File dialog box navigate to the XML file containing the test suite. After you select the file, click the Open button. Upon loading the test suite, API Monitor SE displays the API functions being monitored in the Selected APIs box.

Note    API Monitor SE monitors the application running on the default Xbox.

Monitoring API Functions

After you've loaded a test suite, click the API Monitor SE program's Start button to begin monitoring API function calls. Next, start your Xbox program.

API Monitor SE counts the calls to each API function, as well as the duration of the calls. By clicking the Save As button in the Selected APIs group, you can save this information in a file of comma-separated values, also known as a CSV file. The CSV file format is recognized by a wide variety of programs. For example, you could load the resulting CSV file into Microsoft Excel.

Logging Results

The API Monitor SE program can log the values that the test suite functions return to the Xbox application. It also logs the time and duration of each call. To enable logging, click the Enable log check box in the lower left of the API Monitor SE program's window. You can save this log as a CSV file by clicking the Save As button in the Log group.
