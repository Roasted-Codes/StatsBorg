Remote DebugString Monitor Tool (XbDBSMon)

The Remote DebugString Monitor tool (XbDBSMon) prints strings passed to OutputDebugString when the Xbox application printing the strings is not currently being debugged in the Visual C++ IDE.

USAGE: xbdbsmon [-x xboxname]
OptionDescription
-x Xbox name or Xbox IP. Specify a particular target machine.
-? Display a complete list of available options.

See the (Remote) Target System Tools topic for more information about this and other remote target system tools.
