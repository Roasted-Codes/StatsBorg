Direct3D State Viewer

The Direct3D State Viewer is a Visual Studio .NET plug-in that allows you to examine Direct3D states as you debug a game title. The following image shows Direct3D State Viewer in action:

Like any other tool window in Visual Studio .NET, you can do the following: 1) reposition the Direct3D State Viewer by dragging its title bar, 2) resize the tool by dragging the window's lower-right corner, or 3) dock the tool by dragging its title bar to the edge of the screen where you wish to dock it (or by double-clicking the tool window's title bar).

Using Direct3D State Viewer

To start the Direct3D State Viewer, from the Tools menu, select Direct3D State Viewer. The tool is available only while debugging a project, not during design time.

The Direct3D State Viewer displays a list of render states and their values. Values that have changed since the last breakpoint are displayed in red text, and render states that are inactive are grayed out. To view an explanation for why a particular render state is inactive, hover the mouse cursor over the state's name in the list. A tool tip that contains the explanation will appear.

To hide unchanged values entirely, from the Options menu, select Hide Unchanged. To hide inactive render states entirely, from the Options menu, select Hide Inactive.

To maximize effective display space, the tool shortens the names of render states and value constants by removing the prefixes that are common to many render state variable names and constants. For example, the D3DRS_ALPHABLENDENABLE render state is shortened to ALPHABLENDENABLE.

To display the full names for render states, in the Options menu, deselect Shorten Names. To display the full names for values, in the Options menu, deselect Shorten Values. You may also temporarily display the full name for a render state or value by hovering the mouse cursor over the name you wish to expand. A tool tip that contains the full name will appear.

The Direct3D State Viewer allows you to copy values to the clipboard. To copy a value from the Direct3D State Viewer tool, select the value you are interested in, then choose Copy from the Options menu. You can select more than one value at a time by holding down Ctrl as you select values.

In addition to copying to the clipboard, you may also drag a value from the Direct3D State Viewer window and drop it into the Watch window.
