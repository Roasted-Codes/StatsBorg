Xbox Watson Crash Log Analysis Tool (XbDumpLog)

The Xbox Watson Crash Log Analysis Tool (XbDumpLog) generates a post-mortem stack trace, based on crash logging information produced by the XbWatson tool.

Usage

USAGE: xbdumplog -f crashlog [-p symbolpath]
Switch Description
-f crashlog Specifies the path and file name of a crash log that was generated by XbWatson.
-p symbolpath Specifies a symbol path that contains the .exe and .pdb files for the game title. If you omot the -p switch, XbDumpLog searches for these files in the current path.

Remarks

XbDumpLog produces a decorated call stack with values for all of a function call's parameters, as in the following example:

USER32! 77e148fc()
WinMain(HINSTANCE__ * 0x00400000, HINSTANCE__ * 0x00000000, char * 0x0013237c, int 1) line 75 + 18 bytes
WinMainCRTStartup() line 198 + 54 bytes
KERNEL32! 77e992a6()
