XPerfBeginCapture

Begins capturing performance data.

HRESULT XPerfBeginCapture(
    CONST CHAR *FileName,
    DWORD Flags
);

Parameters

FileName
[in] Location of the file to dump captured data to.
Flags
[in] Specifies which profiling methods will be used to capture data. Flags can be one or more of the following values:
Value Description
XPERF_USECPX Capture CPX data.
XPERF_USEFASTCAP Capture data using FASTCAP instrumentation.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

To stop capturing performance data, the title should call XPerfEndCapture.

Requirements

Header: Declared in Xperf.h.

Import Library: Use Xperf.lib.
