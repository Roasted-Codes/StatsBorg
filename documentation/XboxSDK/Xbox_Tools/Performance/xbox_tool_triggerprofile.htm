TriggerProfile Tool

The TriggerProfile tool remotely triggers a FastCAP profile by sending a DebugManager command to the Xbox application that is currently running. The application must be using XBProfiling_IsFastCAPRequested or some equivalent to listen for the command. The FastCAP profile should typically capture an entire frame of data. All Xbox samples are set up, in their Profile_FastCap configurations, to have profiling triggered by this class. For more information, see the Holistic Guide to Xbox Profilers.

Usage: TRIGGERPROFILE

This tool requires no command-line options.

Source: (XDK Root)\Tools\Win32\Profiling\TriggerProfile

Executable: (XDK Root)\xbox\bin\TriggerProfile.exe
