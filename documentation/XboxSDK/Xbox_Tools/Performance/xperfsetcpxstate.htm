XPerfSetCPXState

Sets the sampling methods used in CPX.

HRESULT XPerfSetCPXState(
    CONST XPERFCPXSTATE *pState
);

Parameters

pState
[in, optional] Pointer to an XPERFCPXSTATE structure that describes both the sampling method and threshold. If pState is NULL, CPX will use the Timer for sampling with a threshold of two million.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

XPerfSetCPXState should be called before calling XPerfBeginCapture with the XPERF_USECPX flag. If XPerfSetCPXState is called while a capture is already in progress, the settings will not take affect until the next capture.

Requirements

Header: Declared in Xperf.h.

Import Library: Use Xperf.lib.
