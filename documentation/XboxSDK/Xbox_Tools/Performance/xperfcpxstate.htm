XPERFCPXSTATE

Used by XPerfSetCPXState to set the sampling method and threshold used by CPX.

typedef struct _XPERFCPXSTATE{
    DWORD SamplingMethod;
    DWORD Threshold;
} XPERFCPXSTATE, *PXPERFCPXSTATE;

Members

SamplingMethod
Must be one of the following flags:
Value Description
XPERF_SAMPLING_TIMER Sample using the Timer. Threshold should be at least two million.
XPERF_SAMPLING_L2MISSES Sample when an L2 cache miss occurs. Threshold should be at least 40,960.
Threshold
Sets the sampling frequency. The lower the value, the more sampling will occur.

Requirements

Header: Declared in Xperf.h.
