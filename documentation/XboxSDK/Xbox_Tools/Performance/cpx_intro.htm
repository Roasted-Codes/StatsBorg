CPU Profiler for Xbox (CPX)

The CPU Profiler for Xbox (CPX) enables profiling of Xbox executables. CPX gives both call stack information and the relative time spent in each function. Keying off events such as the system clock or L2 cache misses, CPX uses a Monte Carlo method to randomly sample the title. CPX will interrupt and record the sampled function, quickly walk and record the call stack, and then return control to the interrupted function.
