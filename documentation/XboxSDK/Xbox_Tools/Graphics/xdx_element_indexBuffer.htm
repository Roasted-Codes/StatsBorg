indexBuffer Element

This element is used to specify an array of vertex indices.

<indexBuffer
   id = string
/>

Required Attributes

id

Identifier used for referencing from <draw> elements. For more details, see XDX id Attribute.

Optional Attributes

None.

Element Information

Parent Elements XDX
Child Elements None.

Remarks

Index buffers are always made up of short unsigned integer data.

Examples

Setting up an index buffer

The following example sets up an index buffer that says to use the zeroth, first, and second vertices.

<indexBuffer id="indexBuffer1">0 1 2</indexBuffer>
