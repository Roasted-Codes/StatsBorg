XBR::ModulatePixelShaderConstants (XbView)

HRESULT ModulatePixelShaderConstant( D3DPIXELSHADERDEF *pPsd, 
                                     DWORD Register, 
                                     Float *fMultiplier );

This function is called by XBR::SetPixelShaderResource. It multiplies in the directional light color or the ambient light color from the application into the pixel shader constants used to do the per-pixel lighting. This color is passed in as fMultiplier. The function returns the result of IDirect3DDevice8::SetPixelShaderConstant.
