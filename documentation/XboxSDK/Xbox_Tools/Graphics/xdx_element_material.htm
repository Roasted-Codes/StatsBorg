material Element

The <material> element identifies the material name that an artist assigned to the pass in the authoring package.

<material
   id = string
/>

Required Attributes

id

Identifier used by Deep Exploration. Identifiers must be 256 characters or less.

Optional Attributes

None.

Element Information

Parent Elements pass
Child Elements None.

Remarks

The <material> element is used at import time in Deep Exploration to prevent the loading of multiple instances of the same material. This element is ignored by both XbRC and XbView.
