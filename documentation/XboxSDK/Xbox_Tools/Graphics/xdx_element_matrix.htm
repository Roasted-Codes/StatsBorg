matrix Element

The <matrix> element groups animation data for a given frame.

<matrix
   id = string
/>

Required Attributes

id

Identifier used to associate animation data to a given frame node. The id for the <matrix> element must match the id of the associated <frame> element; otherwise, the animation data is ignored at run-time. Identifiers must be 64 characters or less. Longer identifiers will be truncated and generate linker warnings.

Optional Attributes

None.

Element Information

Parent Elements frame
Child Elements animOutput, rotate, scale, translate

Remarks

Animation data that is unassociated to a frame (because the id attributes don't match up) will still consume memory.
