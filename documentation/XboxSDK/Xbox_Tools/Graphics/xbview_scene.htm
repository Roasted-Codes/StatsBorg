Scene (XbView)

The Scene class and its helper classes manage the scene. They are constructed right after the resource file finishes loading. The scene constructor takes a pointer to the XBR class, which is the class containing the raw Direct3D resources loaded from the .XBR file. The constructor scans through the list of resources and identifies each resource that is a skeleton. Each skeleton found causes a BaseSkeletonNode instance to be constructed with a pointer to the skeleton found and the pointer to the XBR class. This instance of the BaseSkeletonNode class is added to the list of skeletons for the scene. The resources are scanned again in order to find any models that are not attached to a skeleton. Because the skeletons control the scene, any model that is not attached to a skeleton would not be drawn. The "free" models are attached to their own skeletons and added to the list of skeletons stored in the scene. The resources are scanned one more time so that any passes setting the AlphaBlend render state to TRUE can be identified. If any such passes are identified, there will need to be some alpha sorting.
