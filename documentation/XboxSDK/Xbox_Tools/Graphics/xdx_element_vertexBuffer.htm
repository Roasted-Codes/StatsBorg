vertexBuffer Element

The <vertexBuffer> element specifies a vertex buffer. Vertex buffers describe an array of vertex data, and how that data is to be formatted.

<vertexBuffer
   id = string
/>

Required Attributes

id

Identifier used for referencing from <vertexShader> elements. For more details, see XDX id Attribute.

Optional Attributes

None.

Element Information

Parent Elements XDX
Child Elements declVB, v

Examples

Setting up a vertex buffer

Vertex buffers need to have both a declaration block (enclosed by <declVB>) and vertex data (enclosed by <v>). The vertex buffer below describes a triangle with information about the position and normals for each vertex.

<vertexBuffer id="vertexBuffer1" >
   <declVB>
      <vregVB name="POSITION" format="FLOAT3" />
      <vregVB name="NORMAL" format="FLOAT3" />
   </declVB>

   <v>0 -0.5  0.866025  1 0 0</v>
   <v>0 -0.5 -0.866025  1 0 0</v>
   <v>0  1    0         1 0 0</v>
</vertexBuffer>
