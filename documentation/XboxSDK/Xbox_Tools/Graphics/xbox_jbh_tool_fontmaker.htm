FontMaker Tool (FontMaker)

FontMaker is a tool that converts Microsoft® Windows® fonts to the Xbox® bit font file format (.abc) that is used by the CXBFont class. Any ANSI or Unicode Windows font can be converted to an .abc file. The source code for FontMaker is provided with the Xbox SDK.

The output from FontMaker is a .tga image of the character set and an .abc file. The .abc file contains the location and spacing information for each of the characters in the .tga image. This information is used to place text correctly on the screen. These files must be bundled together (using the Bundler tool) into an Xbox packed resource (.xpr) file, which is used by the CXBFont class.

USAGE: FontMaker

When you launch FontMaker, the tool displays a standard font selection dialog box where you can select your options. Click Cancel to exit the tool without generating a bit font file, or click OK to generate the bit-font file based on your current settings.
