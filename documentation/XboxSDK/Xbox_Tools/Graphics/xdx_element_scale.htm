scale Element

The <scale> element groups scaling data that scales the frame in local space using three scalars.

<scale
   value = string
/>

Required Attributes

None.

Optional Attributes

value

If specified, the value attribute should contain three space-delimited floating-point values comprising the x, y, and z scaling factors for the frame. In most cases, value will be "1 1 1"—no scaling. If there is animation associated to the frame, this attribute should be left undefined. See the examples below.

Element Information

Parent Elements matrix
Child Elements animate, keyTimes, values

Examples

A. Simple Scaling Without Animation

In this example, the frame is scaled by a factor of 2 along the x-axis. Since this is a simple scaling operation, the <scale> element has no children.

<scale value="2 1 1"/>
B. Scaling With Animation

This scaling involves an animation that starts at 0 seconds and lasts for 5 seconds. It is broken up into multiple key times (ranging from 0 to 1.) Each key time corresponds to a single scaling operation. For example, at key time 0.75, the frame is scaled by a factor of 1.1 along the x-axis. Because there is animation with multiple rotations, the value attribute of the <scale> element is undefined.

<scale>
   <animate begin="0s" dur="5s"/>
   <values>
      1 1 1
      1.1 1 1
      1.2 1 1
   </values>
   <keyTimes>
      0
      0.75
      1
   </keyTimes>
</scale>
