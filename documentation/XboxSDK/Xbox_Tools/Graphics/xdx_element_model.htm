model Element

The <model> element is a container for all of the states required to draw an object on the Xbox.

<model
   id = string
/>

Required Attributes

id

Identifier used by Deep Exploration for reimportation of the XDX file. The Xbox Resource Compiler ignores this attribute.

Optional Attributes

None.

Element Information

Parent Elements frame
Child Elements pass

Remarks

A <model> element is always associated to a given <frame>; however, that relationship is not determined by inclusion. The identifier of the <frame> node that the <model> is attached to is determined by the parameter attribute of the <constantVS> tag that:

  1. Is a descendant of the <model>.
  2. Has a matrix attribute "WorldViewProjection".

To see how this works, see the example below.

Examples

Determining The Frame

In this example, the <model> is assocated to the <frame> with id "BAR" since that is the parameter of the <constantVS> descendant whose matrix is "WorldViewProjection".

<model id="FOO">
   <pass>
      <vertexShaderPass ...>
         ...
         <constantVS index="0" count="4" parameter="BAR" matrix="WorldViewProjection"/>
         ...
      </vertexShaderPass>
      ...
   </pass>
</model>
