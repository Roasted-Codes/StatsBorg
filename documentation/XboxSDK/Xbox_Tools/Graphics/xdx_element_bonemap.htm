boneMap Element

The <boneMap> element contains the data describing how bones are mapped to frames. It is used by Deep Exploration for the reimportation of XDX files but is ignored by the Xbox Resource Compiler.

<boneMap
   index = integer
   node = string
/>

Required Attributes

index

Denotes the order of the bones. The index must be an integer ranging from zero to the number of bones that are influencing the model or mesh.

node

Identifier for the <frame> influenced by the bone.

Optional Attributes

None.

Element Information

Parent Elements matrixMapping
Child Elements None.
