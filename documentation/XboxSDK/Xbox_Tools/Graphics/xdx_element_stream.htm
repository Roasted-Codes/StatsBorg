stream Element

Defines the stream mappings for a vertex shader.

<stream
   index = integer
   vertexBuffer = string
/>

Required Attributes

index

Specifies the data stream to use with the vertex shader. The index attribute must be between 0 and 15.

Optional Attributes

vertexBuffer

Specifies a vertex buffer to associate to the vertex shader, and needs to correspond with the id attribute of that <vertexBuffer>. This attribute is ignored within the context of a <vertexShader> because the vertex buffer associated to the shader is determined within the context of the <vertexShaderPass>. It is provided within a <vertexShader> context in case an end-user needs to fix a vertex buffer to a shader without using a <vertexShaderPass> to build up the data. In order to take advantage of this case, the end-user will need to customize the Xbox Preview Pipeline.
