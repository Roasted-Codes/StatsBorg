FrameConstant (XbView)

The FrameConstant structure stores the information necessary to attach skin and bones to the correct vertex shader constants. It is used by the XBR::ApplyVertexShaderConstants. 

struct FrameConstant
{
    CHAR m_strFrameId[FRAME_IDENTIFIER_SIZE];
    CHAR m_strSkinId[FRAME_IDENTIFIER_SIZE];
    CHAR m_bConstantRegister;
    CHAR m_bConstantCount;
};

The m_strFrameId member is the name of the frame node containing the bone matrix transformation for this vertex shader constant. The m_strSkinId member is the name of the frame node containing the skin object-to-world matrix. The m_bConstantRegister member is a variable that stores the vertex shader constant address for this bone. The m_bConstantCount member specifies how many constants will be set, including the constant specified. Since each resulting bone matrix is a 4×3 matrix and each vertex shader constant can hold four numbers, this variable is usually 3.
