Writing an Optimize Function (XbRC)

The next step to extend XbRC is to write the optimize function for the class derived from XBRCRawData to optimize the data, if desired. This function would take the compiled data and optimize it in some way. For example, you could generate triangle strips from triangle lists at this point. Any optimization you want to do can be included in this function. For cube maps, there isn't much to optimize so we have left this blank.
