draw Element

The <draw> element exectutes a single render pass of its <model> ancestor.

<draw
   indexBuffer = string
   primitive = string
   start = integer
   count = integer
/>

Required Attributes

indexBuffer

Name of the index buffer to use for the current pass. This value must correspond to the id attribute of a previously defined <indexBuffer>.

primitive

Type of primitive to render. This must be one of the values below:

Value Description
POINTLIST

Renders the vertices as a collection of isolated points.
LINELIST

Renders the vertices as a list of isolated line segments. The count must be even and greater than or equal to two.
LINELOOP

Renders the vertices as a single polyline with a line segment connecting the last vertex to the first vertex.
LINESTRIP

Renders the vertices as a single polyline. The count must be greater than or equal to two.
TRIANGLELIST

Renders the specified vertices as a sequence of isolated triangles. Each group of three vertices defines a separate triangle. Backface culling is affected by the current winding-order render state.
TRIANGLESTRIP Renders the vertices as a triangle strip. The backface-culling flag is automatically flipped on even-numbered triangles.
TRIANGLEFAN Renders the vertices as a triangle fan.
QUADLIST Each set of four vertices is used to construct a quadrilateral.
QUADSTRIP Renders the vertices as a quadrilateral strip.
POLYGON Renders the vertices as a polygon. This value can be used only with convex objects. The last vertex is joined with the first vertex.
count

Number of vertices to draw.

Optional Attributes

start

Starting index. This value must be between 0 and 65535. If start is not defined, the starting index will be zero.

Element Information
