animOutput Element

The <animOutput> element gives the output format for the animation data in the XBR file. There is only one output format supported by the Xbox Resource Compiler (XbRC), but using customization, this tag could be modified to support new formats.

<animOutput
   sampleIntervalHz = integer
   compression = string
/>

Required Attributes

sampleIntervalHz

Sampling frequency for the animation data. The default sampling interval for the Xbox Preview Pipeline is 30 Hz.

compression

The only valid value for this attribute is "none", causing XbRC to do no compression on the animation data. Support for different compression types could be added by customizing the Xbox Resource Compiler.

Optional Attributes

None.
