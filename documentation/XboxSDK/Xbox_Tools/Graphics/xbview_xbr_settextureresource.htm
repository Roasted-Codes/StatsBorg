XBR::SetTextureResource (XbView)

HRESULT SetTextureResource( DWORD iTextureStage, DWORD TextureResourceIndex );

This function uses the TextureResourceIndex to determine which texture resource should be set. It gets the appropriate texture and sets it at texture stage iTextureStage using IDirect3DDevice8::SetTexture. If the textureResourceIndex is out of range, or the resource at the index is not a texture, the function returns a failure result. Otherwise, the function returns the result of the IDirect3DDevice8::SetTexture call.
