vertexShader Element

The <vertexShader> element is used to group together the following: 1) a declaration which describes how vertex buffers are mapped to vertex registers, 2) a function in vertex shader assembly language, and 3) a block of constants.

<vertexShader
   id = string
/>

Required Attributes

id

Identifier used for referencing inside a <model> element. For more details, see XDX id Attribute.

Optional Attributes

None.

Element Information

Parent Elements XDX
Child Elements asm, constantVS, declVS

Examples

Vertex Shader with Diffuse Lighting

This examples demonstrates how to setup a vertex shader with its declaration, assembly code, and constants. A vertex shader does not need to have any constants defined.

<vertexShader id="shader1">
   <declVS>
      <stream index="0">
         <vreg index="0" name="POSITION" format="FLOAT3/>
         <vreg index="1" name="NORMAL" format="FLOAT3/>
      </stream/>
   </declVS>
   <asm>
      ;c0 = world * view * projection matrix
      ;c4 = light position
	  
      vs.1.0
	  
      ;Compute position
      m4x4 oPos v0, c0
	  
      ;Diffuse lighting
      add r0,c4,-v0
      dp3 r0.w,r0,r0
      rsq r1.x,r0.w
      mul r0,r0,r1.x
      dp3 oD0,v1,r0
   </asm>
   <constantVS index="4" comment="light position">0 0 -1 0</constantVS>
</vertexShader>
