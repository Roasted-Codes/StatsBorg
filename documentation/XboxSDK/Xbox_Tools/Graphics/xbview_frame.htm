Frame (XbView)

The Frame structure specifies a single node in the hierarchy needed to render this scene.

struct Frame 
{
    XGMATRIX   m_Matrix;                         // local transformation
    CHAR       m_strName[FRAME_IDENTIFIER_SIZE]; // name of this frame
    Frame *    m_pChild;                         // child frame
    Frame *    m_pNext;                          // sibling frame
};

The m_Matrix member is the local transformation matrix that will be concatenated, along with any animation data that might apply for this node, into the Local-to-World matrix for this node. The m_strName member is the identifier for this frame. When this identifier matches the Effect Identifier member, the Effect is "attached" to this node and the concatenation of all matrices for this node is used to transform the geometry. The m_pChild pointer points to a Frame that is considered a child of this node. The m_pNext pointer points to a Frame that is considered a sibling of this node.

The Skeleton structure stores a pointer to the root Frame and the rest of hierarchy can be traversed from there.
