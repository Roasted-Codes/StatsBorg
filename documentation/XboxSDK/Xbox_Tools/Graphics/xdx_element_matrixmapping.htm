matrixMapping Element

The <matrixMapping> element describes how bones are mapped to frames. It is used by Deep Exploration for the reimportation of XDX files but is ignored by the Xbox Resource Compiler.

<matrixMapping
   count = integer
/>

Required Attributes

None.

Optional Attributes

count

The number of <boneMap> children. This value is used only for error checking.

Element Information

Parent Elements pass
Child Elements boneMap

Remarks

There is only one <matrixMapping> tag that is specified in the first <pass> of the <model>.
