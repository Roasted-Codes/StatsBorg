include Element

The <include> element allows groups of related items to be put into different files. For example, a set of game-specific shaders could be put into a standard include. Mesh and vertex buffer files can be unwieldy and large and can be put into separate files for ease of browsing the model or scene file. Also, incremental compilation similar to pre-compiled headers can occur on include boundaries.

<include
   href = string
/>

Required Attributes

href

URL path to the .xdx file to include.

Optional Attributes

None.

Element Information

Parent Elements XDX
Child Elements None.
