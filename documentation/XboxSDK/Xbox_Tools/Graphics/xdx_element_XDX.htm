XDX Element

This is the main element of the .xdx file that contains all of the others.

<XDX
   version = string
/>

Required Attributes

version

Version string. Must be 1.5.

Optional Attributes

None.

Element Information

Parent Elements None.
Child Elements cubeMap, frame, include, indexBuffer, pixelShader, source, texture, vertexBuffer, vertexShader, volumeTexture
