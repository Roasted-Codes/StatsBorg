Scene::Update (XbView)

Scene::Update is called to update the scene, updating both the timing and animation code. The function takes two parameters. The first is a FLOAT that tells the update function how much time, in seconds, has elapsed since the last update. The second parameter is a DWORD, that acts as an index into which skeleton(s) to update. If this second parameter is 0xffffffff, the update function updates all of the skeletons. If the second parameter is a number other than 0xffffffff, update treats this number as an index into the list of skeletons and only updates that skeleton. The function insures that passing in an index outside of valid range does nothing.

Based on the index (the second parameter) passed in, Scene::Update will make either a single call to BaseSkeletonNode::Update or if the index passed in is 0xffffffff, Scene::Update calls all of the BaseSkeletonNode::Update functions in the list. Scene::Update returns the result of the last BaseSkeletonNode::Update that was called.

This function is instrumented to show up in PIX timing runs.
