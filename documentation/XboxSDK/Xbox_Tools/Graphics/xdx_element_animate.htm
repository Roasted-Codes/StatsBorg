animate Element

The <animate> element is used to indicate the begining and duration of an animation.

<animate
   begin = string
   dur = string
/>

Required Attributes

begin

Specifies the beginning time (in seconds) for the animation of the parent translation, rotation, or scaling.

dur

Specifies the duration (in seconds) for the animation of the parent translation, rotation, or scaling.

Optional Attributes

None.

Element Information

Parent Elements rotate, scale, translate
Child Elements None.

Remarks

Times in both the begin and dur attributes are floating point numbers which may be followed by the letter "s". For example, if an animation starts at 4 seconds, the begin attribute would be "4s". Since the "s" is optional, the attribute in this example could also be represented by just "4".
