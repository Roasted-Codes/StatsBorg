Convert X File Tool (ConvX)

The Convert X File tool reads an X file and converts it to either the binary or text X file format with or without templates.

USAGE: convx [options] file
SwitchDescription
-t Generates text format X files. The default is binary format.
-s Includes template definitions in X files (stand-alone).
-o "filename" Names the output file.

See Also
