DrawData (XbView)

The DrawData structure describes the index buffer and how it should be used to render an effect.

struct DrawData 
{
	DWORD IndexBufferResource;
	DWORD Primitive;
	DWORD Start;
	DWORD Count;
};

IndexBufferResource is an index back into the list of resources pointing to the index buffer to use for this pass. This allows you to use the same index buffer in multiple passes, if desired. Primitive specifies the Direct3D primitive type of this index buffer. This lets Direct3D know how many indices there are per primitive. Start and Count specify the starting index in the index buffer and how many indices should be used from that point to draw the effect.
