Xbox Resource Compiler (XbRC)

The Xbox Resource Compiler (XbRC) is a stand-alone executable that compiles .XDX files into Xbox DirectX Resources. The resulting binary file, which has the extension .XBR, can be loaded in a single step on the Xbox, dramatically reducing load times, and can be displayed using the XbView. It is a part of the Xbox Preview Pipeline.

Information on XbRC is divided into the following sections:

  • XbRC Command-Line Commands
  • XbRC Architecture
  • Extending XbRC

XbRC can be customized and extended to take advantage of new data types. For example, developers may want to extend the compiler by adding XML tags to represent data such as bounding boxes, DirectX 9 resources, or data describing how to convert the resources to a format useful for other game platforms.
