pixelShader Element

The <pixelShader> element specifies a pixel shader program and a block of constants.

<pixelShader
   id = string
/>

Required Attributes

id

Identifier used for referencing inside a <model> element. For more details, see XDX id Attribute.

Optional Attributes

None.

Element Information

Parent Elements XDX
Child Elements asm, constantPS

Examples

A. Applying a simple texture

This pixel shader program uses a simple texture as a decal: 

<pixelShader id="PixelShader1">
   <asm>
      ps.1.1
      tex t0
      mov r0, t0
   </asm>
</pixelShader>

B. Using a pixel shader constant

This examples uses a pixel shader constant to modulate the output color: 

<pixelShader id="PixelShader2">
   <asm>
      ps.1.1
      mul r0, v0, c0
   </asm>
   <constantPS index="0" count="1">
      1.0 1.0 0.5 1.0
   </constantPS>
</pixelShader>
