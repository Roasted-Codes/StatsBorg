BaseFrameNode::Draw (XbView)

BaseFrameNode::Draw sends the geometry to the GPU. It iterates across each Pass stored in the Effect and sets the vertex shader and the vertex shader constants. It also sets the transformation matrix in vertex shader constant 0, the local-space light direction in vertex shader constant 90, and the local-space eye-point direction in vertex shader constant 91. It saves off the current vertex shader constant 0 and restores it after iterating through all of the passes. It then sets the pixel shader and its constants. Next, the render states are set, followed by the texture stage states and the textures themselves. If a texture stage sets its texture to 0, the texture stage states for that stage are ignored. The function then iterates over the number of draws specified, sets the index buffer, and calls DrawIndexedVertices if there is an index buffer resource in the pass. If not, it calls DrawVertices instead. After iterating through all of the passes, the function returns S_OK.
