source Element

The <source> element explicitly states which file was the original source of the .xdx file. This is necessary because many .xdx files are converted from other art packages, such as Maya, 3DS/Max, Lightwave, or SoftImage.

<source
   filename = string
/>

Required Attributes

filename

URL path to the original art package.

Optional Attributes

None.

Element Information

Parent Elements XDX
Child Elements None.
