CXBView::Render (XbView)

CXBView::Render draws the current screen. It has a number of different draw paths through the code, depending on the current view mode. It also draws all of the text, handles clearing the screen and does the Swap or Present functions needed to render the scene. In the Model and Animation views, it calls Scene::Sort and CXBView::DrawModels. CXBView::DrawModels saves off the current Direct3D States so that you can restore them when the model(s) are finished drawing and then calls Scene::Draw. This is important because the resource file can change any number of these states and you will need to restore to a default state when you are done drawing the model(s) to allow the fonts and/or help screens to draw.
