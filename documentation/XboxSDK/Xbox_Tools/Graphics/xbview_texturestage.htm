TextureStage (XbView)

The TextureStage structure stores the information needed to set the texture and D3DTEXTURESTAGESTATETYPE for a given texture stage. The Xbox can set four textures per pass. As such, there are a maximum of four of these structures in the Pass context specifying the texture information for each stage.

struct TextureStage 
{
    DWORD TextureResource;
    DWORD TextureStateCount;
    TextureState *rTextureState;
    DWORD PaletteResource;
};

TextureResource is an index back into the list of resources specifying the texture map for this stage. To minimize storage of the resources on the Xbox, we only store each texture once and use references to them here rather then storing the texture map multiple times. TextureStateCount specifies the number of texture stage states there for this texture pass and the rTextureState pointer points to a list of simple structs that describes each TextureStage state constant and the value assigned to it before rendering.

Just as with render states, texture stage states are reset by XBR::SetDefaultState every pass to the defaults required by XbView.
