Convert XOF File Tool (ConvXOF)

The Convert XOF File tool reads an XOF file and converts it to either the binary or text X file format with or without templates.

USAGE: convxof [options] file
SwitchDescription
-t Generates text format X files. The default is binary format.
-s Includes template definitions in X files (standalone).
-o "filename" Names the output file.
