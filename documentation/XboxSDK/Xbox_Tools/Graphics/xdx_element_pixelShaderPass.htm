pixelShaderPass Element

Sets the pixel shader for current render pass.

<pixelShaderPass
   pixelShader = string
/>

Required Attributes

pixelShader

Name of the pixel shader to use for the render pass. This value must correspond to the id attribute of a previously defined <pixelShader>.

Optional Attributes

None.

Element Information

Parent Elements pass
Child Elements constantPS

Remarks

Any constants made using <constantVS> inside of a <pixelShaderPass> will add to or overwrite those made inside the shader referenced by the pixelShader attribute.
