What's New in NetSim 2

NetSim 2 is a significant upgrade of the network simulation tool included with previous releases of the XDK. In particular, NetSim 2 provides the following new features.

Setup Automation
NetSim 2 automates most of its installation. You'll find that it is much simpler to get NetSim 2 installed and working than NetSim.
New User Interface
NetSim 2 provides a simplified and far more straightforward user interface. Developers should find this new user interface much more intuitive and flexible than the user interface provided with NetSim.
TCR Testing
Using NetSim 2, you can create configuration files for specific simulations. This enables you to create and save simulations that are targeted toward particular TCRs.
Advanced Network Testing
The increased flexibility of NetSim 2 enables you to test your games against conditions that are not described in any of the Xbox Live TCRs.
