Matchmaking Simulator (MatchSim)

The Matchmaking Simulator tool (MatchSim) assists in creating matchmaking queries, which produces a query definition file that may be submitted to Microsoft for inclusion on a matchmaking server. The MatchSim tool also generates C++ code that wraps the XOnlineMatchSearch function, making it easier to call matchmaking queries from your game title. In addition, MatchSim facilitates local testing of matchmaking queries.

USAGE: matchsim
