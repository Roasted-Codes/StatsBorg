Creating, Saving, and Loading a Simulation

NetSim 2 enables you to save a group of settings in an XML file. With this capability, you can create groups of settings that simulate particular network conditions, and then save those settings as individual simulations. The simulations can be targeted toward particular TCRs, or toward specific network conditions that you want to test for.

To make an new simulation, select New from the File menu. Next, assign values for the bandwidth, latency, packet loss, and authentication settings that are appropriate for the simulation.

To save a simulation, choose Save or Save As from the File menu. Use Load Simulation to load the XML file containing a simulation's settings.
