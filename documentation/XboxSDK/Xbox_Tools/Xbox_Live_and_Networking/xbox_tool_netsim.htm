Network Simulator 2 (NetSim 2)

NetSim 2 is an updated version of the network simulation tool provided with previous releases of the XDK.

The XDK fully supports having both versions installed simultaneously on the same computer. Upgrade installations of NetSim from previous XDK releases are also fully supported.

The two versions of NetSim share the same ms_vlan.sys kernel mode driver. As a result, if both versions of NetSim are installed on the same computer, and one of the versions is uninstalled, the remaining version will not work properly.

Also, if one of the versions is uninstalled and subsequently re-installed—for example, to correct the situation described in the preceding paragraph—the user must reboot the computer between the uninstallation and re-installation.

Using NetSim 2, you can simulate real-world network conditions such as network latencies, limited bandwidth, and packet loss. You can test your Xbox game under these conditions to ensure that it handles them gracefully.
