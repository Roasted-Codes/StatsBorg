Xbox Set Recommended Message (XSetRecMsg)

The XSetRecMsg tool sets a recommended message for one or more users on an Xbox. A recommended message is a message that users should read when they log on to Xbox Live. This is different than required messages, which users must read.

The primary use of this tool is to verify compliance with Xbox Live TCRs.

Usage

USAGE: xsetrecmsg [options] [portmask]
Option Description
/H[elp] or /? Display the help and usage message.
/N[oLogo] Suppress the copyright notice.
/V[erbose] Display extended troubleshooting information.
/X[box] <xbox> Use alternate Xbox. If this option is not specified, XSetRecMsg uses the default Xbox for the development PC.
portmask Bitmask that specifies which ports to enable (using the value 1) or disable (using the value 0). Ports are specified in a bit order that proceeds from most significant bit to least significant bit. Therefore, a portmask of 1000 enables port 1. Using a portmask of 1011 enables ports 1, 3, and 4.

Common Error Messages

The following list explains the most common error messages developers see when using XSetRecMsg.

Error Message Description
HRESULT=XBDM_UNDEFINED XSetRecMsg was unable to set the recommended message. Most likely your Xbox was configured using a Recovery disk that was released before November 2003. Recover the Xbox with a Recovery disk that was released with the November 2003 or later XDK.
HRESULT=XBDM_NOXBOXNAME No default Xbox has been specified on the development PC and none were specified on the command line.
HRESULT=XBDM_CANNOTCONNECT XSetRecMsg cannot communicate with the Xbox. The Xbox is probably turned off, or there was a network error.
