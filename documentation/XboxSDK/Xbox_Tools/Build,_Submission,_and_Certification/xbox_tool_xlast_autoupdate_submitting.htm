Submitting an Autoupdate Package

After you have tested and built your autoupdate package, you can submit it to certification. Use the following URL to submit your package securely via the Web.

https://xds.xbox.com/BPProgInfo.asp?Page=scripts/psUpload/Upload.asp

You must provide your username and password to access the upload page. When the upload page appears, select Autoupdate from the Upload Type drop-down list. Use the Autoupdate Details edit box to specify the destination data center, such as PartnerNet or CertNet.
