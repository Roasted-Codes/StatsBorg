Xbox Live Authoring and Submission Tool (XLAST)

The Xbox Live Authoring and Submission Tool (XLAST) provides an interactive method of authoring submissions for Xbox Live. All submissions to Xbox Live must be authored with XLAST.

XLAST is actually a framework for a set of plug-in tools. New plug-in tools are added periodically to XLAST. For example, XLAST contains two plug-ins for stats configurations. One plug-in creates leaderboards for stats configurations version 3 or later. The other creates leaderboards for stats configurations before Stats version 3.

Note    Stats configurations before version 3 are called "legacy" stats configurations.

The XLAST plug-in tools do not support UNC paths. If you need to work with UNC paths, it is recommended that you map them to drive letters.

Submissions created with XLAST are not backward compatible with XDK releases before June 2003. If you build content with XLAST and your game is linked to an XTL that is older than June 2003, your game will not be able to find or verify its signature when it runs.
