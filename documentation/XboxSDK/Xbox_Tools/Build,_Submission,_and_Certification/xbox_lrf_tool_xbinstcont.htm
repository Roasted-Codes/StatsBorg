Xbox Install Content Tool (XbInstCont)

The Xbox Install Content Tool (XbInstCont) enables a game developer to install online content packages onto a connected Xbox Development Kit.

Usage

USAGE: xbinstcont [/x xboxname] [/t titleid] [/f] [/u] dir ...
Option Description
/x xboxname Xbox name or Xbox IP. Specify a particular target machine.
/t titleid Title ID of the game title to which the content belongs.
/f Force copied files to overwrite files on the Xbox console.
/u Install an auto-update to the Xbox Console.

Testing Online Content

Testing content packages during development requires the following steps:

  1. Create content (new levels, characters, or whatever your game title requires).
  2. Use XBldSigs to build the final package metadata file (ContentMeta.xbx).
  3. Use XbInstCont to copy the content and metadata to the correct locations on your Xbox Development Kit's hard disk.
  4. Test the new content. If the new content has a problem, repeat steps 1–3.

Example

The directory C:\GameContentPack directory contains the files SampleImage.bmp, SampleAudio.wav, and ContentMeta.xbx (created by running XBldSigs). The title ID for the sample game is 0xAABBCCDD. The following command line copies everything in C:\GameContentPack to the appropriate locations on the Xbox Development Kit whose name is MyXboxName:

xbinstcont /x MyXboxName /t 0xAABBCCDD C:\GameContentPack

The copied content may now be managed in the Memory section of the development kit's Dashboard.
