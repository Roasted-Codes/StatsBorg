XBECopy Tool

Microsoft Visual Studio .NET uses the XBECopy tool to automatically copy a game title and its associated media to an Xbox console after you build the title or start a debugging session.

The XBECopy tool is not intended for command line use. Instead, the Xbox Deployment Tool, which is integrated into the property pages of Visual Studio .NET, uses XBECopy to perform its work. See the Xbox Deployment Tool documentation for more information about changing the way in which Visual Studio .NET deploys a title to the Xbox console.

The XDK does provide a tool for command line use, if you need it. For more information, see File Transfer Tool (XbCp).
