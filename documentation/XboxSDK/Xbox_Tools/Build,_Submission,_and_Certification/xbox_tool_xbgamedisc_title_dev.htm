Title Development Process

At a high level, the process for developing a game title looks something like this:

  1. Develop the .xbe. (See Building an XBE.)
  2. Copy the .xbe, data, and media to the hard disk on the Xbox console. (See File Transfer Tool (XbCp).)
  3. Run and debug the .xbe. (See Running and Debugging an XBE.)
  4. If necessary, repeat from step 1.
  5. Create the game disc layout with XbGameDisc, placing files in such a way as to maximize performance.
  6. Emulate the disc to test performance and collect emulation data. (See A Quick Walk-Through of the Xbox Game Disc Emulation Software.)
  7. Use the XbGameDisc tool's optimization feature to analyze your emulation data. This enables you to fine-tune the layout of your game disk.
  8. If necessary, repeat from step 5.
  9. Premaster to DVD-R to verify emulation results.
  10. If necessary, repeat from step 5.
  11. Premaster to a tape or electronic submission package. The game disc layout must be submitted on tape or as an electronic package file to be certified or published. Optical media will not be accepted. (See Premastering a Game Disc.)
