Adding Leaderboards to the Project

To add leaderboards to a project, first click the leaderboard type. For example, to add Competition Leaderboard Templates, click the Competition Leaderboard Templates node in the project tree.

After you've selected the leaderboard type, right-click the selected node and choose Add Leaderboards from the menu that appears. XLAST presents a dialog box asking you for the number of leaderboards to add. It also asks for the ID number of the first leaderboard in the group. When it creates the group of leaderboards, it sets the first leaderboard ID to the number you specify, and then increments the ID numbers of each successive leaderboard.

If you already have leaderboards of the same type in the project, you can use one of them to copy the characteristics to the group of leaderboards you are creating. Simply select the ID number of the existing leaderboard from the Copy settings from drop-down list. This enables you to start your project with a single leaderboard, set its properties the way you want, and then copy it as many times as you like.

After you click OK, XLAST automatically creates the group of leaderboards of the specified type.
