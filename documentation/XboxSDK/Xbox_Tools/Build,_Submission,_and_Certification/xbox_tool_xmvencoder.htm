XMVEncoder Tool (XMVEncoder)

Description

XMVEncoder provides a simple means of encoding AVIs to XMVs, making use of the WM8EUTIL encoder available at http://www.microsoft.com/windows/windowsmedia/WM8/encoding.aspx as well as XMVTool (included in the Xbox SDK).

Required files and media

WM8EUTIL must be installed. Available at the following URL: http://www.microsoft.com/windows/windowsmedia/WM8/encoding.aspx.

Usage

Knowledge of the WM8EUTIL AVI-to-WMV encoder is required to use this tool. Consult the help documents included with WM8EUTIL for more information on the encoding parameters.

Basic usage:

  1. Upon first use, go to preferences and make sure the path for WM8EUTIL is correct.
  2. Select a source AVI file. The intermediate and output files will be auto-populated. You may change these if desired.
  3. Enter your encoding parameters for the AVI-to-WMV portion and the WMV-to-XMV portion (entering −1 uses the defaults for each parameter).
  4. Click Encode. After encoding is complete, the results of the operation will be shown in the output window.

XMVCompare interface:

This tool has the ability to interface with the XMVCompare sample included in the XDK (XboxSDK\sample\xbox\video\xmvcompare). To make use of this:

  1. Compile and launch the XMVCompare sample.
  2. Start the XMVEncoder tool.
  3. Select your source/intermediate/output files.
  4. Before clicking Encode, be sure After encode, upload to is checked and select a side to upload the resulting XMV to (either left or right).
  5. Click Encode. If the encoding is successful, the XMV will be uploaded to the XDK and the XMVCompare sample will play the resulting file.
  6. You may press the play button to remotely restart the videos.
