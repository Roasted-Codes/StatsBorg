Building Packages for Xbox Live

After you have the content package organized the way you want it, you can build it for distribution via Xbox Live. After you build your content package, test it on your local Xbox development console before posting it to PartnerNet for testing.

To build your package for Xbox Live, click Action in the main menu, and then click Build for Xbox Live.

When building content packages, it is common to get the following error.

Error: Binary value too long for property DetailsData (Max is 65536 characters)

This usually indicates that your Dashboard Icon bitmap or your Download Manager Icon bitmaps are not the correct size. Dashboard Icon bitmaps must be 64×64 pixels, and Download Manager Icon bitmaps must be 168×128 pixels in size.

Note    XLAST may take a while and become unresponsive while it is building. If it is unresponsive longer than you think it should be, use the Windows Task Manager to check its status.

Note    If there are empty folders in your project, XLAST does not create them in the final build.
