Saving and Loading Layout Files

To save an authoring session in progress, click the Save button in the tool bar, click Save or Save As on the File menu, or press Ctrl+S. A standard file dialog box appears, which prompts you for the name and location of an .xlo file that will contain the saved layout information.

To open a saved authoring session, click the Open button on the tool bar, click Open on the File menu, or press Ctrl+O. A standard file dialog box appears, from which you may select a previously saved .xlo file.

To start a new, blank authoring session, click the New Layout button on the tool bar, click New on the File menu, or press Ctrl+N.

If you created a layout in the older Xbox Game Disc Layout Tool (XbLayout), you can open that file in XbGameDisc. XbGameDisc tool automatically converts the old layout file (.fld extension) to the new format (.xlo extension).

To open and convert an XbLayout file, click the Open button on the tool bar, click Open on the File menu, or press Ctrl+O. A standard file dialog box appears. Select XbLayout Layout Files (*.fld) from the Files of type drop-down list, then select the desired .fld file and click Open.
