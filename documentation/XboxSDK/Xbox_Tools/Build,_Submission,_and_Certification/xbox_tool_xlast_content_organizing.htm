Organizing Content

After you've completed the New Contents Package Wizard, you can add files to the project. To do so, look at the tree in the left pane of the Document View window. In the tree, you'll find a folder called Contents. Right-click the Contents folder and click Add File. Use the dialog box that appears to add content files to the project.

You can organize the files in the package into folders, if you prefer. Right-click the Contents folder and click New Folder. The folders you create can hold both files and additional folders.
