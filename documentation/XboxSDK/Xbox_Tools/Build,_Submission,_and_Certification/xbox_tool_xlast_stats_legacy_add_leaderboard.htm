Adding Leaderboards to the Project for Legacy Versions

There are multiple ways to add leaderboards to a project. The first is to select Add Leaderboard from the Action menu. This adds a single, clean leaderboard to the project.

Another way to add leaderboards is to click Add Leaderboards on the Action menu. This option presents a dialog box asking you for the number of leaderboards to add. It also asks for a leaderboard from which to copy properties. By default, the selection is Clean Leaderboard. However, you can use the drop-down list to select any leaderboard in the project. This enables you to start your project with a single leaderboard, set its properties the way you want, and then copy it as many times as you like.
