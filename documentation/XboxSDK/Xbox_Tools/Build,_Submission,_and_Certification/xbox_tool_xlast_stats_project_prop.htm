Setting Project Properties

After you've completed the New Stats Configuration Wizard, you can modify the properties of the entire project using the project tree in the Document View window. Clicking the root item lets you set the properties for the entire project. The properties you can set on this page are as follows.

Title ID of Game
Contains the 8-digit hexadecimal title identifier that you receive from Microsoft.
Unit Leaderboard Inactivity Decay Half-Life

Use for Unit Leaderboards only. This value determines the number of days until the entries in the project's leaderboards are removed. Set this to zero if the entries never expire. The unit activity value at any given time is calculated with the following formula.

An = Ai × exp(−T × ln(2) / h)

where An represents the new unit activity value, Ai is the initial unit activity value, T is the time elapsed since the unit activity value was last updated, and h is decay half-life.

Stats Available on the Web
Specfies that the stats are available on the Web when checked. This becomes the default for all leaderboards in the project. You should also use the locale list to specify the locales for which your title displays Web leaderboards.

Note    The locales en-HK, en-SG, and es-US were removed in XLAST Stats version 3.

Default Locale
Sets the locale (such as English–United States or French–Belgium) that the leaderboards use as the default.

Note    The locales specified in XLAST are browser locales, not game regions.
