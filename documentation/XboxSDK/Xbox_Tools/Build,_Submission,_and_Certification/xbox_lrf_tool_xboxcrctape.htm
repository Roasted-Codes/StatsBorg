Xbox Tape CRC Tool (XboxCRCTape)

The Xbox Tape CRC Tool (XboxCRCTape) verifies that a game title image on tape has not been changed or corrupted in any way since it was generated.

Usage

USAGE: xboxcrctape [-verbose] tape-device

In cases where two layers are stored on two separate tapes, the usage is as follows.

USAGE: xboxcrctape [-verbose] layer0-tape layer1-tape

Remarks

In typical usage, the computer on which this tool is being run has one tape drive, which is usually designated as tape0. In this case, the command to invoke XboxCRCTape would be as follows: 

xboxcrctape -verbose \\.\tape0
