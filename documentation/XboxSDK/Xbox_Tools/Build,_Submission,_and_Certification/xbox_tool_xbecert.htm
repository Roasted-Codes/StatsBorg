XBE Certificate Tool (XBECert)

The XBE Certificate Tool (XBECert) tool enables you to change the certificate settings of an .xbe file.

USAGE: XBECert [/IN: infile] [/NOALTID] [/OUT: outfile] 
  [/TESTALTID: number [,key] ] [/TESTID: number] 
  [/TESTLANKEY: key] [/TESTMEDIATYPES: number] 
  [/TESTNAME: titlename] [/TESTRATINGS: titlename] 
  [/TESTREGION: number] [/TESTSIGNKEY: key] [/TESTVERSION: number] 
  [/?] [infile]
Option Description

/IN: infile

Input XBE file to process. If this option is omitted, you must specify the input file name after the options.

/NOALTID

Remove Alt IDs from certificate.

/OUT: outfile

Output XBE image file (defaults to input file).

/TESTALTID: number [,key]

Test Alt ID for access by alternate images. Multiple Test Alt IDs are allowed.

/TESTID: number

Test Title ID for this image.

/TESTLANKEY: key

Test LAN key for this image.

/TESTMEDIATYPES: number

Test allowed media types for this image.

/TESTNAME: titlename

Test title name for this image.

/TESTRATINGS: titlename

Test ratings for this image.

/TESTREGION: number

Test allowed regions for this image.

/TESTSIGNKEY: key

Test signature key for this image.

/TESTVERSION: number

Test signature key for this image.

/?

Display usage information.

infile

Not needed if you use the /IN option.

Remarks

Many of the options for this tool have corresponding options in the Image Builder tool. For a more complete explanation of those options, see XBE Image Builder Tool (ImageBld).
