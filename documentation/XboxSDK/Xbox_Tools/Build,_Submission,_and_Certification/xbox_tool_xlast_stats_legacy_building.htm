Building Statistics Configuration Submissions for Xbox Live for Legacy Versions

To test your statistics configuration submission, you must build it for Xbox Live. To do so, click Action on the main menu, and then click Build for Xbox Live.

After XLAST builds your statistics configuration submission, you need to post it on PartnerNet for testing. To do this, your development PC must be connected to the Internet. If it is, click Action in the main menu, and then click Send to Partnernet. XLAST then displays a dialog box that requests information validating your authorization to post the package to PartnerNet.

Note    XLAST may take a while and become unresponsive while it is building. If it is unresponsive longer than you think it should be, use the Windows Task Manager to check its status.
