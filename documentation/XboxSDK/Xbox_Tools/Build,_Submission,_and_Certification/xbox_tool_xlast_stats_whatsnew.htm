What's New in XLAST Stats Version 3

Version 3 of the Xbox Live Authoring and Submission Tool Stats adds many new capabilities and more flexibility to the creation of stats packages. These new capabilities include new types of leaderboards, leaderboard templates, and an updated New Stats Configuration Wizard.

Note    This documentation uses the terms "leaderboard" and "scoreboard" interchangeably.

Note    XLAST can also create submission packages for "legacy" libraries. Games compiled with libraries from XDKs before the November 2003 release should use the legacy format. For more information, see The New Stats Configuration Wizard

  • New Leaderboard Types
  • ELO Ratings

New Leaderboard Types

Begining with version 3, XLAST Stats now offers Standard Leaderboards, Unit Leaderboards, and Competition Leaderboard Templates. All three of these leaderboard types can be used for teams as well. For information on these leaderboard types, see Leaderboard Types

ELO Ratings

XLAST and the Xbox Title Libraries now support player and team ratings using the ELO rating system. For more information, see The ELO Rating System.
