Xbox Game Disc Authoring Tool (XbGameDisc)

The Xbox Game Disc Authoring Tool provides a GUI that enables you to lay out the data for your game disc in order to minimize disc access times. After designing the disc layout, you can easily emulate the disc and use the XbGameDisc tool's optimization feature to tweak the layout (rather than waste media). When you're ready, you can premaster your title to tape or to an electronic package file to be submitted for certification and publishing. The XbGameDisc tool can also premaster a title for burning to a DVD-R disc.

This tool is available from the Start menu on your development PC.

Note    The Xbox Game Disc Authoring Tool supersedes three older tools: the Xbox Game Disc Layout Tool (XbLayout), an older version of the Xbox Game Disc Emulation system, and the Xbox Premastering Wizard (XbPremaster). The older tools are still available in the c:\Program Files\Microsoft Xbox SDK\xbox\bin\OldDVDTools folder for use with older projects, but you are strongly encouraged to use XbGameDisc for game disc layout, premastering, and emulation.

Note    Emulation logs created with versions of XbGameDisc released prior to June 20003 do not work with versions of XbGameDisc released in June 2003 or later.
