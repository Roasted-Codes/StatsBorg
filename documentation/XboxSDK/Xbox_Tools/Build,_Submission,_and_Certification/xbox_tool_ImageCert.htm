Image Certification Tool (ImageCert)

The Image Certification Tool validates that the XBE image containing an Xbox title passes certification.

usage: IMAGECERT [options] filename.xbe
Option Description
/TITLE or /TCR:C1-3 Display the Image title.
/LIBS or /TCR:C3-1 Test the libraries used.
/NETWORK or /TCR:C3-2 Test the usage of the network libraries.
/CRT or /TCR:C3-3 Test the CRT usage.
/XTL or /TCR:C3-3 Test the XTL usage.
/DEMOHDWRITE or /TCR:C14-1 Test if this demo mounts the hard disk as writeable.
/DEMOHDUTIL or /TCR:C14-2 Test if this demo mounts Utility region formatted.
/DEMONONET or /TCR:C14-3 Test if this demo is using network libraries.
/HELP or /? Display usage information.
/TYPE:GameTypeSpecifier The type of game being tested, where GameTypeSpecifier is either GAME, DEMO, SYSLINK, or ONLINE.
/SCAN[:path] Scan all files on XD:\ (or the specified path) to find XBE files to test.
/KEEP Don't delete files copied locally by /SCAN.
/X XBoxName Specifies the name of the Xbox when using /SCAN.
/NOLOGO Suppress display of copyright banner.

Note    If no test switches are specified, then all tests are performed.

Example


ImageCert.exe default.xbe

        INFO:
                  LIBCMT 1.0.4361.1 [expired]

        PASSED: Xbox CRT is being used.

        FAILED: Utility Drive is mounted without the format flag.

        PASSED : No network libraries used.
