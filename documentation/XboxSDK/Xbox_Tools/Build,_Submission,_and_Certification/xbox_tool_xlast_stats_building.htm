Building Stats Configuration Submissions for Xbox Live

To test your stats configuration submission, you must build it for Xbox Live. To do so, click Action on the main menu, and then click Build.

After XLAST builds your stats configuration submission, you need to post it on PartnerNet for testing. To do this, your development PC must be connected to the Internet. If it is, select Action from the main menu, and then choose Send to Partnernet. XLAST then starts Internet Explorer to display a dialog box that requests information validating your authorization to post the package to PartnerNet.

Note    XLAST may take a while and become unresponsive while it is building. If it is unresponsive longer than you think it should be, use the Windows Task Manager to check its status.
