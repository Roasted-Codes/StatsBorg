Creating Competitions Configurations

Competitions configurations provide templates for dynamically creating competitions and tournaments at run time. The templates in a competitions configuration specify the content of the competitions' databases. The databases each contain three tables, one for the competition itself, one that keeps information about the entrants, and one that stores information about the competition's events. XLAST competitions configurations help you define the content of these tables. They also help you make queries for updating and obtaining information from them.
