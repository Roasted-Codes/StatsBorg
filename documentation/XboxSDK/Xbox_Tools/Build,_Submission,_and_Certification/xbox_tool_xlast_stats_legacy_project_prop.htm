Setting Project Properties for Legacy Versions

After you've completed the New Stats Configuration Wizard, you can modify the properties of the entire project using the project tree in the Document View window. Clicking the root item lets you set the properties for the entire project. The properties you can set on this page are as follows.

Title ID of Game
Contains the 8-digit hexadecimal title identifier that you receive from Microsoft.
Default Locale
Sets the locale (such as English–United States or Chinese–Hong Kong) that the leaderboards use as the default.
Stats Available on the Web
When checked, specifies that the stats are available on the Web. This becomes the default for all leaderboards in the project.
Manage Locales
Invokes a dialog box that enables you to add support for more locales to the project.
