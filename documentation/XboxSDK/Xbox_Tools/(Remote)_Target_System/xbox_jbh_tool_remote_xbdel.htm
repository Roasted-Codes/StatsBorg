Remote Delete Tool (XbDel)

The Remote Delete tool (XbDel) deletes files on the Xbox target system's file system.

USAGE: xbdel [-x xboxname] [options] file [file] ...
OptionDescription
-r Include files found in subdirectories.

If file is a directory, delete file and its contents.

If file is a wildcard, delete all matching files and directories contained in file's directory or subdirectories.

-s Include files found in subdirectories.
-h Include hidden files.
-f Force delete of read-only files.
-x Xbox name or Xbox IP. Specify a particular target machine.
-? Display a complete list of available options.

See the (Remote) Target System Tools topic for more information about this and other remote target system tools.
