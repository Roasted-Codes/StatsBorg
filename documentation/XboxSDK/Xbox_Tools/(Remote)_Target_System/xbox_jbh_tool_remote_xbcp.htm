File Transfer Tool (XbCp)

The File Transfer tool (XbCP) copies files between the Xbox target system and the development system via a network connection.

USAGE: xbcp [-x xboxname] [options] srcfile ... destfile
OptionDescription
-r Include files found in subdirectories.
-s Include files found in subdirectories.
-h Include hidden files.
-q Suppress file list output.
-y Don't prompt to overwrite files.
-d Copy files only if newer.
-e Don't copy empty directories.
-t Create destination directory if it does not exist.
-f Force copy over read-only files.
-x Xbox name or Xbox IP. Specify a particular target machine.
-? Display a complete list of available options.

See the (Remote) Target System Tools topic for more information about this and other remote target system tools.

Known Issues

The Xbox kernel supports ANSI file names only. As a result, any attempt to copy a file or folder with non-ANSI characters will result in an error saying it's an invalid file name.
