Remote Rename Tool (XbRen)

The Remote Rename tool (XbRen) renames a file that resides on the Microsoft® Xbox® target system's file system. 

USAGE: xbren [-x xboxname] [options] oldname newname
OptionDescription
-x Xbox name or Xbox IP. Specify a particular target machine.
-s Search in subdirectories for files to rename. Performs a recursive search for files that match oldname and renames them to newname.
-? Display a complete list of available options.

Note   The newname parameter must not contain any path information. This tool cannot be used to move files.

See the (Remote) Target System Tools topic for more information about this and other remote target system tools.

Known Issues

The Xbox kernel supports ANSI file names only. As a result, any attempt to rename a file or folder with non-ANSI characters will result in an error saying it's an invalid file name.
