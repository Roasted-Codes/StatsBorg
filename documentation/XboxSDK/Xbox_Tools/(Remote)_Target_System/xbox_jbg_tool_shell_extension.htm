Shell Extension

The Microsoft® Windows® Explorer Shell Extension for Xbox® lets the user seamlessly browse the Xbox Development Kit (XDK) hard disk from the Windows desktop, just as you would browse the local file system, a network share, or the Internet. You can cut\copy\paste or drag and drop files, as well as view, and change, file attributes. The property page for the consoles also allows you to see the path of the currently executing title, reboot the console, and perform screen captures.

This extension is not installed automatically with the XDK. If you have installed the XDK, the extension may be installed by clicking the Start menu, selecting Programs, Microsoft Xbox SDK, and finally, Setup Microsoft Windows Explorer Shell Extension for Xbox.
