Remote Config Tool (XbSetCfg)

The XbSetCfg tool allows you to configure your Xbox target system remotely.

USAGE: xbsetcfg [-x xboxname] [options]

The Remote Config tool (XbSetCfg) provides the following options:

OptionDescription
-x xboxname Xbox name or Xbox IP. Specifies a particular target machine.
-l Sets the default title to be the XDK Launcher.
-d Sets the default title to be the Xbox Dashboard.
-v mode Selects video mode, where mode is one of the following:
  • NTSC-M – NTSC, non-Japanese
  • NTSC-J – NTSC, Japanese
  • PAL-I – PAL
-w Enables widescreen support.
-6 Enables PAL-60.
-7 Enables 720p (HDTV only).
-1 Enables 1080i (HDTV only).
-a mode Selects audio mode, where mode is one of the following:
  • D – Enables Dolby Digital output.
  • T> – Enables DTS output.
  • R – Sets two-channel output to Dolby Surround.
  • S – Sets two-channel output to Stereo (only if DD is not enabled).
  • M – Sets two-channel output to Mono (only if DD is not enabled).
-n language Selects a display language for the console, where language is one of the following:
  • 1 – English
  • 2 – Japanese
  • 3 – German
  • 4 – French
  • 5 – Spanish
  • 6 – Italian
-t Sets the console system time to match the host system time.
-? Displays a complete list of available options.

For more information about this and other remote target system tools, see (Remote) Target System Tools.
