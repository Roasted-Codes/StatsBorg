Remote Attribute Tool (XbAttrib)

The XbAttrib tool changes file attributes of one or more files on the development or target system.

USAGE: xbattrib [/x xboxname] [options] file [file] ...
OptionDescription
-r Recursive change.
-s Include files found in subdirectories.
-h Include hidden files.
+ro Make file read-only.
-ro Make file read-write.
+hid Make file hidden.
-hid Make file visible.
-x Xbox name or Xbox IP. Specify a particular target machine.
-? Display a complete list of available options.

See the (Remote) Target System Tools topic for more information about this and other remote target system tools.
