Remote Screen Capture Tool (XbCapture)

The XbCapture tool remotely captures the displayed image (front buffer) of the Xbox and saves it as a .bmp file on the Windows machine. This makes it much easier for developers, testers, marketing groups, and others to get accurate screen shots. The XbCapture tool is a fairly straightforward command-line application that wraps the DmScreenShot API.

Note   To programmatically dump the screen contents of the Xbox console to a file use XGWriteSurfaceToFile (to dump to the Xbox hard disk) or DmScreenShot (to dump to the Windows hard disk).

USAGE: xbcapture [-x Name] filename
OptionDescription
filename Specify the filename of the resulting .bmp file.
-x Xbox name or Xbox IP. Specify a particular target machine.
-? Display a complete list of available options.

See the (Remote) Target System Tools topic for more information about this and other remote target system tools.
