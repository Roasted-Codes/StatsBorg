Networking Tools

This section provides information regarding the networking tools in this version of the Xbox Development Kit (XDK). The tools described within this section are subject to change in later versions of the XDK.

The tools in this section include:

  • Matchmaking Simulator (MatchSim)
  • Network Simulator (NetSim)
  • Network Simulator 2 (NetSim 2)
  • Ping PartnerNet Tool (PINGPNet)
  • Xbox Set Recommended Message (XSetRecMsg)
