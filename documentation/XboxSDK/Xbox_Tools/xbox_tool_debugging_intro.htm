Debugging Tools

This section provides information regarding the debugging tools in this version of the Xbox Development Kit (XDK). The tools described in this section are subject to change in later versions of the XDK.

The tools in this section include:

  • API Monitor
  • Direct3D State Viewer
  • Remote DebugString Monitor Tool (XbDBSMon)
  • Memory Dump Tool (XbMemDump)
  • Xbox File Event Viewer
  • Xbox Surface Viewer
  • Xbox Watson Crash Log Analysis Tool (XbDumpLog)
  • Xbox Watson Tool (XbWatson)
  • Xbox Xray Debug Tool (XbXray)
