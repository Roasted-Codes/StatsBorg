DSP Builder (DSPBuilder) Command-Line Commands

USAGE: dspbuilder [options]
Option Description
/B filename Batch build file name. This is the name of the file to open or batch build.
/Ob filename Name of the binary image output file.
/Oh filename Name of the C header output file.
/Oi path Path to place the .ini output files.

DSPBuilder will run in batch mode to build .fx files if passed "/B filename" as a command-line argument. It will open the specified file, build it, then exit. If it encounters any errors or problems, it will return a nonzero return code. If it completes successfully, it will return 0. This allows DSPBuilder to be used in makefiles.

The /O options are only valid for batch mode and will be ignored for interactive mode.
