Xbox ADPCM Encode Tool (XbADPCMEncode) (XbADPCMEncode)

The Xbox ADPCM Encode tool (XbADPCMEncode) converts .wav files in PCM format to Xbox ADPCM format.

USAGE: XbADPCMEncode [/C] [/Of] [/Ob]
Option Description

/C

Copy all chunks to encoded file.

/Of

Optimized encoding over the whole file.

/Ob

Optimized encoding per block.

Remarks

This program serves as both an XDK tool and a sample. You'll find the source code for it in the following directory. 
Source: (XDK Root)\Tools\Win32\Sound\xbadpcmencode
