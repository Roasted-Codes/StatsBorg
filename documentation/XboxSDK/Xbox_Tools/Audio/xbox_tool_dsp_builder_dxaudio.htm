DSP Builder (DSPBuilder)

The DSP Builder tool enables users to quickly assemble a DSP effects image using a graphical user interface. It creates the DSP effects image, the .ini file (which could be used with XGPImage later), and the .h file, without the user having to manually create and/or edit the .ini file used by XGPImage.
