Xbox Audio Creation Tool (XACT)

The Xbox Audio Creation Tool (XACT) provides an easy way for audio content creators and developers to create and use sound effects. The XACT run-time engine (xacteng.lib), the audition engine (xactengi.lib, or for debug, xactengd.lib), and the authoring tool (Xact.exe) provide a wealth of features, including streaming, Microsoft® Windows Media® Audio (WMA) soundtracks, real-time auditioning, weighting of content, and tools for in-game mixing, equalization, and resource monitoring. These and other features are discussed in the following sections. (Sound designers may want to go straight to the authoring guide and review the overview and conceptual information as needed.)

For information about the XACT library functions, see the Xbox Audio Creation Tool Library section.

Note that DirectMusic will continue to be the primary recommended solution for those interested in interactive MIDI music. You can mix and match between these solutions and libraries as appropriate for your title, however, because both DirectMusic and XACT run on top of DirectSound. We recommend that you investigate XACT as a more optimized solution for basic streaming, cross-fading, pre-rendered multichannel, and WMA.

Note    As always, as new features are added to the authoring tool and engine each month, we recommend that you recompile your title (and regenerate your sound banks, as the file format changes) to take advantage of these features.

To use XACT on Microsoft® Windows® 2000, you need to install Windows Media Player version 7.1 (WMP7.1), which you can download from the following location:

http://www.microsoft.com/windows/windowsmedia/download

You do not need to install Windows Media Player to run XACT on Microsoft® Windows® XP.
