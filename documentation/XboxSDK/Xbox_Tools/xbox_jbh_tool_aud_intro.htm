Audio Tools

This section provides information regarding the audio tools in this version of the Xbox Development Kit (XDK). The tools described within this section are subject to change in later versions of the XDK.

The tools in this section include:

  • Code Scrambler Tool (XCodeScr)
  • DirectMusic Content Analyzer (AudAnalyzer)
  • DSP Builder (DSPBuilder)
  • GP DSP Image Builder (XGPImage)
  • Wave Merge Tool (WavMerge)
  • Xbox ADPCM Encode Tool (ADPCMEncode)
  • Xbox Audio Creation Tool (XACT)
