The Game Loop

Pulling this all together, here is a simple main loop that, combined with the code examples in the previous topics, initializes the peripherals, tracks insertions and deletions, and reads gamepad and debug keyboard input:

//
// Main loop for device example.
//
void main( void )
{
    // Initialize peripherals.
    DeviceInit();

    // Main loop.
    while( TRUE )
    {       
        CheckDeviceChanges( g_dsDevices );
        CheckKeyboard();
        ReadGamepad();
    }
}

Note that in order to get the code presented in this entire Using Xbox Peripherals section to compile and link successfully, you'll need to link with xkbd.lib and xvoice.lib (or their equivalent debug versions) because the debug keyboard and the voice devices are referenced by the example.
