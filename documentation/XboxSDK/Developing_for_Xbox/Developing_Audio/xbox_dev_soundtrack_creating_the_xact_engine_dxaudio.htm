Creating the XACT Engine

Before XACT can be used, the XACT engine must be created and initialized. Calling the XACTEngineCreate function returns a pointer to the engine that can then be used for other XACT functionality. We can control the maximum number of 2D voices, 3D voices, concurrent streams available, and notifications by filling out the XACT_RUNTIME_PARAMETERS structure as a parameter to the XACTEngineCreate function. The example code below shows some initial values used in the XACT_RUNTIME_PARAMETERS structure that may be changed depending upon the particular needs of a given game title.

Also listed in the example code below are the soundtrack structure and global variables that will be discussed throughout the examples to follow:

#include <xtl.h>
#include <assert.h>
#include <dsstdfx.h>
#include <xact.h>
#include <vector>

using namespace std;

struct SOUNDTRACK
{
    UINT  uSoundtrackID;                 // Soundtrack's ID
    UINT  uSongCount;                    // Number of songs in the soundtrack
    WCHAR strName[MAX_SOUNDTRACK_NAME];  // Name of the soundtrack
};

IXACTEngine*       g_pXACT = NULL;                 // Pointer to the XACT engine
BYTE*              g_pbSoundBankData = NULL;       // Pointer to the sound bank data buffer
PXACTSOUNDBANK     g_pSoundBank = NULL;            // Pointer to the XACT sound bank
DWORD              g_dwSoundCueIndex = 0;          // Index of the sound cue
PXACTWMAPLAYLIST   g_pWMAPlaylist = NULL;          // Pointer to the XACT WMA play list
vector<SOUNDTRACK> g_vSoundtracks;                 // Vector of soundtrack information
UINT               g_uCurrentSoundtrackIndex = 0;  // Index of the current soundtrack in the soundtrack vector
HANDLE             g_hGamePad = NULL;              // Handle of the game pad
XINPUT_STATE       g_XInputLastState = { 0 };      // Last input state of the game pad

void CreateXACTEngine()
{
    // Set up runtime parameters struct
    XACT_RUNTIME_PARAMETERS Params;
    memset( &Params, 0, sizeof( Params ) );
    Params.dwMax2DHwVoices = 64;
    Params.dwMax3DHwVoices = 32;
    Params.dwMaxConcurrentStreams = 32;
    Params.dwMaxNotifications = 32;

    // Create the XACT engine
    HRESULT hr = XACTEngineCreate( &Params, &g_pXACT );
    assert( SUCCEEDED( hr)  );
}
