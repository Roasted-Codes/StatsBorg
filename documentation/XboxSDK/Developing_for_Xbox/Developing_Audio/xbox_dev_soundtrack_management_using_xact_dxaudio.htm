Soundtrack Management Using XACT

One of the great features of the Xbox console is that users can copy their favorite audio CDs to the Xbox hard disk as soundtracks. These soundtracks represent collections of songs in Microsoft® Windows Media® Audio (WMA) format, and can be played back either through the Xbox Dashboard or by the game title. The Xbox Audio Creation Tool (XACT) provides simplified jukebox-style interfaces to play both user-supplied and game-supplied soundtracks while the game title is running.

This section presents and explains end-to-end sample code that covers the steps necessary to play WMA soundtracks using XACT. The following sections are included:

  • Creating the XACT Engine describes the essential first step required for every application that uses XACT. Before XACT can be used, the XACT engine must be created and initialized.
  • Creating an XACT Sound Bank provides information on how to create an XSB file using the graphical user interface portion of XACT. This file can then be loaded in your application and registered with the XACT engine.
  • Downloading a DSP Effects Image discusses how to download a digital signal processor effects image to the global processor in the Xbox Media Communications Processor (MCP).
  • Registering and Processing XACT Notifications provides two sample routines to help manage event notifications with XACT. These act as a jumping-off point for your application's needs.
  • Enumerating Soundtracks shows how you can easily build a list of the soundtracks currently residing on the user's Xbox console.
  • Creating and Playing a WMA Playlist covers an essential part of soundtrack playback through XACT. This section shows how to create, add songs to, and begin playback with a WMA playlist.
  • Input and Support Routines shows general routines used with the sample code.
  • The Game Loop demonstrates a simple game loop that pulls together audio and input interaction (provided in the previous code samples).

XACT has many other additional powerful features not addressed in this section. For more information on XACT, please see the Xbox Audio Creation Tool (XACT) section.
