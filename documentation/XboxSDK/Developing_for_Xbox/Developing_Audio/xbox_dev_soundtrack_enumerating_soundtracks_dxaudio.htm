Enumerating Soundtracks

The example routine below uses the XFindFirstSoundtrack and XFindNextSoundtrack functions to enumerate the soundtracks currently on the user's Xbox. For each soundtrack (ignoring empty soundtracks) we copy the information into our own SOUNDTRACK structure and add it to our vector. We'll use this information later as we browse for playback.

void EnumerateSoundtracks()
{
    // Add each user provided soundtrack to the soundtrack vector
    XSOUNDTRACK_DATA stData;
    HANDLE hSoundtrack = XFindFirstSoundtrack( &stData );
    if( INVALID_HANDLE_VALUE != hSoundtrack )
    {
        do
        {
            // Ignore empty soundtracks
            if( stData.uSongCount > 0 )
            {
                SOUNDTRACK sndtrk;

                sndtrk.uSoundtrackID = stData.uSoundtrackId;
                sndtrk.uSongCount    = stData.uSongCount;
                wcscpy( sndtrk.strName, stData.szName );

                g_vSoundtracks.push_back( sndtrk );
            }
        } while( XFindNextSoundtrack( hSoundtrack, &stData ) );
    }
}
