XBE Sections

An XBE section is a portion of an XBE image that can be treated as a discrete collection of functions and data. A title can load or free an individual section as needed. The functions and data in a section are available while the section is loaded, and not available while the section is freed. Using sections enables a title to contain more code and data than can be loaded simultaneously into memory on the Xbox.
