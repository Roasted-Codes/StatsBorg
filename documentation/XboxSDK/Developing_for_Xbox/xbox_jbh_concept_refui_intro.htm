Reference User Interface

Reference user interface elements are in-game, user interface elements that Xbox is providing to help you deal with issues that come up when creating games. These UI elements are not intended to hamper your creative process, nor will they be required in any way. Reference UI is mainly intended as a tool to provide you with quality user interface elements and user research that you can make use of within your title. Reference UI will be delivered primarily as generic black and white elements.

Each reference UI element has been usability tested and we are sharing the results of the user research with you, as well as the sample code. The advantage of sharing the research is that even if you choose not to adopt our reference UI, you may be planning similar UI for your game and can leverage the findings, hopefully avoiding some usability issues. Source code will be provided for reference UI elements. Reference UI will also be consistent with and demonstrate the certification requirements.

Each reference document will start with a list of the top three recommendations for designing reference UI. Following that will be a summary of the document and the take-home message.
