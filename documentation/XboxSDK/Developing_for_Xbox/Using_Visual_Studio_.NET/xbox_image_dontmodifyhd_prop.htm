/DONTMODIFYHD (Don't Modify Hard Disk)

/DONTMODIFYHD

Doesn't allow the title to write to the hard disk. The T: and U: drives are inaccessible. This switch is useful for demo discs and utility titles that should not leave anything on the hard drive. The /DONTMODIFYHD switch is not appropriate for actual game titles.

Remarks

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the Xbox Image folder.
  3. Click the General property page.
  4. Set the Don't Modify Hard Disk property to Yes.
