Choose Xbox to Debug Dialog Box

Visual Studio displays its Choose Xbox to Debug Dialog Box dialog box if you attempt to debug an XBE file but have not specified a default Xbox.

In the Xbox Name edit box, type the name of the Xbox on which you want to debug your game. Visual Studio then copies the XBE file onto the Xbox and enters Debug mode.

To prevent the Choose Xbox to Debug dialog box from appearing, set a default Xbox using the Visual Studio Remote Machine property.
