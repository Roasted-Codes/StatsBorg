/TESTVERSION (Version)

/TESTVERSION:version

Where:

version
Version number for the image.

Remarks

This switch should not be used. It has been deprecated in favor of /TESTXBEVERSION.

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Select the Xbox Image folder.
  3. Select the Certificate property page.
  4. Modify the Version property.
