Map DVD

The Map DVD property enables developers to map the XD:\ drive to the DVD during debugging.

Remarks

Set the Map DVD property to Yes if you want to map the XD:\ drive to its DVD during debugging. The default is No. In the default state, the XD:\ drive is mapped to the game's parent directory.

To set this property in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click Debugging.
  3. Set the Map DVD property to Yes or No.
