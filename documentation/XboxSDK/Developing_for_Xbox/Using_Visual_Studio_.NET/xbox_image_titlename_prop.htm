/TITLENAME (Title Name)

/TITLENAME:name

Where:

name
Title name for this image.

Remarks

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the Xbox Image folder.
  3. Click the Certificate property page.
  4. Modify the Title Name property.
