/OUT (File Name)

/OUT:FileName

Where:

FileName
Specifies the name of the output XE image file (*.xbe).

Remarks

This property specifies the name of the file to which the image is written. This is not necessarily the same file name that is used on the XBox. To set the name of the image file on the XBox, use the Remote Path property of the XBox Deployment tool.

This property is also available as the $(ImagePath) macro. It is used in conjunction with the following macros.

MacroDescription
$(ImageDir) Directory to which the primary output file from the build is copied. This is defined as the drive and path and includes the trailing backslash '\'.
$(ImageName) Base name of the primary output file from the build. This does not include the file extension.
$(ImagePath) Absolute path name where the primary output file from the build is copied to. This is defined as drive, path, base name, and file extension.
$(ImageFileName) The file name of the primary output file for the build. This is defined as base name and file extension.
$(ImageExt) The file extension of the primary output file for the build.

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the Xbox Image folder.
  3. Click the General property page.
  4. Modify the File Name property.
