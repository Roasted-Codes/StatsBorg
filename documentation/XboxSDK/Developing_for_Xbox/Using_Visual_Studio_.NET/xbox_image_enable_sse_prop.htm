Enable SSE

The Enable SSE property lets developers compile Xbox games using the streaming SIMD extensions (SSE).

Remarks

The microprocessor used in the Xbox supports the enhanced instruction set for SSE. Selecting Streaming SIMD Extensions (/arch:SSE) as the value for the Enable SSE property causes Visual Studio to compile games with SSE enabled. This has the same effect as using the /arch:sse switch when compiling from the command line.

To set this property in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click Code Generation.
  3. Set the Enable Enhanced Instruction Set property to Streaming SIMD Extensions (/arch:SSE).

Note    The Xbox does not support SSE2.
