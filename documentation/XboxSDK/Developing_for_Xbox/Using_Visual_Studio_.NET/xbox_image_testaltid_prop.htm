/TESTALTID (Alternate ID)

/TESTALTID:number[,key]

Where:

number
Alternate title IDs for access by alternate images.
key
Optional value that specifies a key for the images.

Remarks

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the Xbox Image folder.
  3. Click the Certificate property page.
  4. Modify the Alternate ID property.
