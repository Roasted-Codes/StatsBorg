/NOLIBWARN (Disable Library Warnings)

/NOLIBWARN

Disables the unapproved library warning messages.

Remarks

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the Xbox Image folder.
  3. Click the General property page.
  4. Sets the Disable Library Warnings property to Yes.
