Remote XBE for Debug Session Dialog Box

Visual Studio displays its Remote XBE for Debug Session dialog box if you attempt to debug an XBE file but do not have a default Xbox specified.

In the Xbox Name edit box, type the name of the Xbox on which you want to debug your game. Visual Studio then displays the Choose Xbox to Debug dialog box.

To prevent the Remote XBE for Debug Sessiondialog box from appearing, set a default Xbox using the Visual Studio Remote Machine property.
