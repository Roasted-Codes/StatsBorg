/TESTMEDIATYPES (Media Types)

/TESTMEDIATYPES:number

Where:

number
Specifies the allowed media types for this image.

Remarks

This property can be set to any of the following values.

ConstantValue
XBEIMAGE_MEDIA_TYPE_HARD_DISK 0x00000001
XBEIMAGE_MEDIA_TYPE_DVD_X2 0x00000002
XBEIMAGE_MEDIA_TYPE_DVD_CD 0x00000004
XBEIMAGE_MEDIA_TYPE_CD 0x00000008
XBEIMAGE_MEDIA_TYPE_DVD_5_RO 0x00000010
XBEIMAGE_MEDIA_TYPE_DVD_9_RO 0x00000020
XBEIMAGE_MEDIA_TYPE_DVD_5_RW 0x00000040
XBEIMAGE_MEDIA_TYPE_DVD_9_RW 0x00000080

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the Xbox Image folder.
  3. Click the Certificate property page.
  4. Modify the Media Types property.
