/UDCLUSTER (Utility Drive Cluster Size)

/UDCLUSTER:size

Where:

size
Specifies the utility drive cluster size. By default this is 16384.

Remarks

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the Xbox Image folder.
  3. Click the General property page.
  4. Modify the Utility Drive Cluster Size property.
