/FORMATUD (Format Utility Drive)

/FORMATUD

Always format the utility drive mounted during initialization. This switch is incompatible with /DONTMOUNTUD.

Remarks

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the Xbox Image folder.
  3. Click the General property page.
  4. Set the Format Utility Drive property to Yes.
