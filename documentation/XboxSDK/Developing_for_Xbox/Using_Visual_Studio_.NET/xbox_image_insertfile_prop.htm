/INSERTFILE (Insert File)

/INSERTFILE:file,section,[R][N]

Where:

file
Specifies the name of a file to insert as a section.
section
Specifies the section where the file should be inserted.

R indicates that the file is read-only, and N indicates that there is no preload.

Remarks

To specify multiple files, separate them by semicolons.

To set this Image Builder option in the Visual Studio environment:

  1. Open the Xbox project's Property Pages dialog box.
  2. Click the Xbox Image folder.
  3. Click the General property page.
  4. Modify the Insert File property.
