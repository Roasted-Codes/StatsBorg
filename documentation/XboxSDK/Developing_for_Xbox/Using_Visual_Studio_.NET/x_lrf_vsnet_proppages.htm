Visual Studio .NET Property Pages

The XBox extensions to Visual Studio .NET add two pages to the Visual Studio .NET project property pages: Xbox Image and Xbox Deployment. In addition, the XBox extensions add properties to existing property pages. Help is available for any of the properties on these pages by selecting the appropriate property, then clicking the Help button in the property pages dialog box. The properties are also described in the following pages:

Properties Specific to XBox

Many of the following properties correspond to command-line options for the Xbox Image Builder Tool (ImageBld).

  • /DEBUG (Include Debug Info)
  • /DEFAULTSAVEIMAGE (Save Game Image)
  • /DONTMODIFYHD (Don't Modify Hard Disk)
  • /DONTMOUNTUD (Don't Mount Utility Drive)
  • /FASTCAP (Fast Call-Attribute Profiling)
  • /FORMATUD (Format Utility Drive)
  • /INSERTFILE (Insert File)
  • /LIMITMEM (Limit Memory)
  • /NOLIBWARN (Disable Library Warnings)
  • /NOLOGO (Suppress Startup Banner)
  • /NOPRELOAD (Suppress Preloading)
  • /OUT (File Name)
  • /STACK (Stack Size)
  • /TESTALTID (Alternate ID)
  • /TESTID (Title ID)
  • /TESTLANKEY (LAN Key)
  • /TESTMEDIATYPES (Media Types)
  • /TESTRATINGS (Xbox Game Rating)
  • /TESTREGION (Xbox Game Region)
  • /TESTSERVICE (Service)
  • /TESTSIGNKEY (Sign Key)
  • /TESTVERSION (Version)
  • /TESTXBEVERSION (XBE Version)
  • /TITLEIMAGE (Title Image)
  • /TITLEINFO (Title Info)
  • /TITLENAME (Title Name)
  • /UDCLUSTER (Utility Drive Cluster Size)
  • Skip Deployment
  • Xbox Deployment Tool
