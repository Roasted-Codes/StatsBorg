D3DRECT

Defines a rectangle.

typedef struct _D3DRECT {
    LONG x1;
    LONG y1;
    LONG x2;
    LONG y2;
} D3DRECT;

Members

x1 and y1
Coordinates of the upper-left corner of the rectangle.
x2 and y2
Coordinates of the lower-right corner of the rectangle.
