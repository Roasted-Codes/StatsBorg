XFONT::SetBkColor

Sets the background color of the text.

void __stdcall SetBkColor(
  D3DCOLOR color
);

Parameters

color
Sets the background color of the text. The format of the color must match the format of the surface being drawn on.

Return Values

None.

Remarks

The default background color is black.

Requirements

  Header: Declared in xfont.h
  Import Library: Use xgraphics.lib
