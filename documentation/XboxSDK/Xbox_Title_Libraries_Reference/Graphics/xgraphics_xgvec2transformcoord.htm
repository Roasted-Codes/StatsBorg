XGVec2TransformCoord

Transforms a 2D vector by a given matrix, projecting the result back into w = 1.

XGVECTOR2* XGVec2TransformCoord(
  XGVECTOR2* pOut,
  CONST XGVECTOR2* pV,
  CONST XGMATRIX* pM
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR2 structure that is the result of the operation.
pV
[in] Pointer to the source XGVECTOR2 structure.
pM
[in] Pointer to the source XGMATRIX structure.

Return Values

Pointer to a XGVECTOR2 structure that is the transformed vector.

Remarks

This function transforms the vector, pV (x, y, 0, 1), by the matrix, pM, projecting the result back into w=1.

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec2TransformCoord function can be used as a parameter for another function.
