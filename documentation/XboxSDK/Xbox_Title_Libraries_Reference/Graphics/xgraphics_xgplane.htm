XGPLANE

Describes a plane.

typedef struct XGPLANE {
  FLOAT a, b, c, d;
} XGPLANE;

Members

a, b, c, and d
The a, b, c, and d coefficients, respectively, of the clipping plane in the general plane equation. See Remarks.

Remarks

The members of the XGPLANE structure take the form of the general plane equation. They fit into the general plane equation so that ax + by + cz + dw = 0.

C++ programmers can take advantage of operator overloading and type casting. The C++ implementation of the XGPLANE structure implements overloaded constructors and overloaded unary and binary operators. For more information, see C++ Specific Features.
