IDirect3DPalette8::Lock

Retrieves a pointer to the address of the palette data.

HRESULT Lock(
  D3DCOLOR** ppColors,
  DWORD Flags
);

Parameters

ppColors
[out, retval] a pointer the address of the palette data.
Flags
[in] A zero or one locking flags. If no flags are specified then this call will block until the palette is no longer being used by the hardware.
D3DLOCK_READONLY
The title promises to not write to the palette.
D3DLOCK_NOOVERWRITE
The title promises to not overwrite any existing palette entry.

Return Values

The method returns D3D_OK.

Requirements

  Import Library: Use D3d8.lib.
