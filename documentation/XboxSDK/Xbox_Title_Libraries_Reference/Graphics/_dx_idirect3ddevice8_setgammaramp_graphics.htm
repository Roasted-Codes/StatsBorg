IDirect3DDevice8::SetGammaRamp

Sets the device gamma correction ramp.

void SetGammaRamp(
  DWORD  Flags,
  CONST D3DGAMMARAMP  *pRamp
);

Parameters

Flags
[in] One of the following values can be set.
D3DSGR_CALIBRATE
This flag has no effect on Xbox and is ignored. The ramp will be passed directly to the device.
D3DSGR_NO_CALIBRATION
This flag has no effect on Xbox and is ignored. The ramp will be passed directly to the device.
D3DSGR_IMMEDIATE
Gamma is applied immediately. If the D3DSGR_IMMEDIATE is not specified, the requested gamma ramp is applied right before the next flip.
pRamp
[in] Pointer to a D3DGAMMARAMP structure, representing the gamma correction ramp to be set for the implicit swap chain.

Return Values

None.
