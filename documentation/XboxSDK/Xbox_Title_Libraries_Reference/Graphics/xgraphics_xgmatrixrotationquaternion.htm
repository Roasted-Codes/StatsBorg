XGMatrixRotationQuaternion

Builds a matrix from a quaternion.

XGMATRIX* XGMatrixRotationQuaternion(
  XGMATRIX* pOut,
  CONST XGQUATERNION* pQ
);

Parameters

pOut
[in, out] Pointer to the XGMATRIX structure that is the result of the operation.
pQ
[in] Pointer to the source XGQUATERNION structure.

Return Values

Pointer to a XGMATRIX structure built from the source quaternion.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGMatrixRotationQuaternion function can be used as a parameter for another function.

Requirements
