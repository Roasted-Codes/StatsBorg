XGQuaternionIdentity

Returns the identity quaternion.

XGQUATERNION* XGQuaternionIdentity(
  XGQUATERNION* pOut
);

Parameters

pOut
[in, out] Pointer to the XGQUATERNION structure that is the result of the operation.

Return Values

Pointer to the XGQUATERNION structure that is the identity quaternion.

Remarks

Given a quaternion (x, y, z, w), the XGQuaternionIdentity function will return the quaternion (0, 0, 0, 1).

The return value for this function is the same value returned in the pOut parameter. In this way, the XGQuaternionIdentity function can be used as a parameter for another function.
