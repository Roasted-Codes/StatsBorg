IDirect3DDevice8::BlockUntilVerticalBlank

Blocks the current CPU thread until the next vertical blank.

void BlockUntilVerticalBlank();

Remarks

Event-based blocks are generally done so that other non-rendering threads can run while the current CPU thread is waiting.
