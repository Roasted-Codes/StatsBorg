IDirect3DDevice8::SelectVertexShaderDirect

Selects the vertex shader to execute on the GPU. Unlike IDirect3DDevice8::SelectVertexShader, this function does not require a vertex shader handle.

HRESULT SelectVertexShaderDirect(
    D3DVERTEXATTRIBUTEFORMAT *pVAF,
    DWORD Address
);

Parameters

pVAF
[in] Pointer to a D3DVERTEXATTRIBUTEFORMAT structure that is used to program the GPU's stream registers. This value may be NULL except when switching from fixed-function pipeline to a vertex shader program.
Address
[in] The start location on the GPU of the vertex shader, in the range 0 through 135.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The GPU used by the Xbox supports the loading of multiple vertex shaders into a set of 136 available addresses on the chip. SelectVertexShaderDirect is used to select which of the currently loaded vertex shader programs to use as the current vertex shader program.

The D3DVERTEXATTRIBUTEFORMAT structure is used to program the hardware stream registers to the correct input data stream as required by the vertex shader program. If the newly selected vertex shader program uses the same input data stream format as the previously selected vertex shader program, pVAF can be passed as NULL in order to avoid the redundant reprogramming of the hardware stream registers.

The vertex format of the selected shader may be overridden if SetVertexShaderInputDirect was previously called.

Requirements

Header: Declared in D3d8.h.

Import Library: Use D3d8.lib.
