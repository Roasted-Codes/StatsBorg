IDirect3DPushBuffer8::SetRenderState

Used only within a IDirect3DPushBuffer8::BeginFixup/IDirect3DPushBuffer8::EndFixup bracket to modify the corresponding call to IDirect3DDevice8::SetRenderState as it was recorded inthe push-buffer. Only simple render states (those whose values are less than D3DRS_SIMPLE_MAX) can be fixed-up in push-buffers.

HRESULT SetRenderState(
    DWORD Offset,
    D3DRENDERSTATETYPE State,
    DWORD Value
);

Parameters

Offset
[in] The offset determined during the recording in the push-buffer as returned from IDirect3DDevice8::GetPushBufferOffset just before IDirect3DDevice8::SetRenderState was recorded.
State
[in] Device state variable that is being modified. This parameter must be a member of the D3DRENDERSTATETYPE enumerated type whose value is less than D3DRS_SIMPLE_MAX.
Value
[in] New value for the device render state to be set. The meaning of this parameter is dependent on the value specified for State. For example, if State were D3DRS_SHADEMODE, the second parameter would be one member of the D3DSHADEMODE enumerated type.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in D3d8.h.

Import Library: Use D3d8.lib.
