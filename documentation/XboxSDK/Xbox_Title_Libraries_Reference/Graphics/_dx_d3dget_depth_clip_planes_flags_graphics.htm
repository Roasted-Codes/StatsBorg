D3DGET_DEPTH_CLIP_PLANES_FLAGS

Direct3D keeps track of one set of depth-clipping planes for use when the fixed-function pipeline is active, and keeps track of another set of depth-cliping planes for use when a vertex shader program is active. The D3DGET_DEPTH_CLIP_PLANES_FLAGS enumeration is used to determine which set of clipping planes will be retrieved by IDirect3DDevice8::GetDepthClipPlanes.

typedef enum _D3DGET_DEPTH_CLIP_PLANES_FLAGS{
    D3DGDCP_GET_VERTEXPROGRAM_PLANES = 10,
    D3DGDCP_GET_FIXEDFUNCTION_PLANES = 11,
} D3DGET_DEPTH_CLIP_PLANES_FLAGS;

Constants

D3DGDCP_GET_VERTEXPROGRAM_PLANES
Retrieve the depth-clipping planes used when a vertex shader program is active.
D3DGDCP_GET_FIXEDFUNCTION_PLANES
Retrieve the depth-clipping planes used when the fixed-function pipeline is active.
