XFONT::SetTextColor

Sets the color of the text.

void __stdcall SetTextColor(
  D3DCOLOR color
);

Parameters

color
[in] Sets the color of the text.

Return Values

None.

Remarks

The default color is white.

XFONT does not perform any alpha blending. The alpha value specified in color is copied to the surface as pixels are rendered. As a work-around, alpha-blending can be achieved by rendering the text onto a temporary surface and then rendering that surface onto the scene with alpha-blending enabled.
