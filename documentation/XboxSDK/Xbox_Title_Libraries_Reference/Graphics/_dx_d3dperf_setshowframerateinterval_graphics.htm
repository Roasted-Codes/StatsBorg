D3DPERF_SetShowFrameRateInterval

Sets the frame rate interval determining how often D3DPERF_DumpFrameRateInfo is called.

__inline void WINAPI D3DPERF_SetShowFrameRateInterval(
  DWORD FrameRateIntervalTime
);

Parameters

FrameRateIntervalTime
[in] Number of milliseconds between calls to D3DPERF_DumpFrameRateInfo. This should be set to zero to disable automatic dumping of frame rate information.

Return Values

None.
