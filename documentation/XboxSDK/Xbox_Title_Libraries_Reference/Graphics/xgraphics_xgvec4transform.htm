XGVec4Transform

Transforms a 4-D vector by a given matrix.

XGVECTOR4* XGVec4Transform(
  XGVECTOR4* pOut,
  CONST XGVECTOR4* pV,
  CONST XGMATRIX* pM
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR4 structure that is the result of the operation.
pV
[in] Pointer to the source XGVECTOR4 structure.
pM
[in] Pointer to the source XGMATRIX structure.

Return Values

Pointer to a XGVECTOR4 structure that is the transformed 4-D vector.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec4Transform function can be used as a parameter for another function.

Requirements

  Header: Declared in xgmath.h.
  Import Library: Use xgraphics.lib.
