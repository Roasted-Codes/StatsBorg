XGVec2Lerp

Performs a linear interpolation between two 2D vectors.

XGVECTOR2* XGVec2Lerp(
  XGVECTOR2* pOut,
  CONST XGVECTOR2* pV1,
  CONST XGVECTOR2* pV2,
  FLOAT s
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR2 structure that is the result of the operation.
pV1
[in] Pointer to a source XGVECTOR2 structure.
pV2
[in] Pointer to a source XGVECTOR2 structure.
s
[in] Parameter that linearly interpolates between the vectors.

Return Values

Pointer to a XGVECTOR2 structure that is the result of the linear interpolation.

Remarks

This function performs the linear interpolation based on the following formula: V1 + s(V2-V1).

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec2Lerp function can be used as a parameter for another function.

Requirements

  Header: Declared in xgmath.h.
