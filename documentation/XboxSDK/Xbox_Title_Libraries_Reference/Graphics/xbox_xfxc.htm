Xbox HLSL Compiler (XFXC)

The Xbox HLSL Compiler (XFXC) compiles an ASCII representation of vertex shader into the microcode format used by the Xbox game system. Unlike the Xbox Shader Assembler Tool (XSAsm), XFXC can compile vertex shaders that use HLSL.

USAGE: XFXC [/T target] [/E name] [/Od] [/Vd] [/Zi] [/Zpr|/Zpc] [/Fc file] 
  [/Fa file] [/Fl file] [/Fh file] [/H] [/nologo] FileName [/?]
Option Description

/T target

Sets the target instruction set. If the target is specified, it must be xvs_1_1, which is the default.

/E name

Sets the entrypoint function to be name. If this switch isn't specified, the default function is main.

/Od

Disable optimizations.

/Vd

Disable validation.

/Zi

Enable debugging information.

/Zpr|/Zpc

Determines how matrices are packed. /Zpr specifies row-major order. /Zpc specifies column-major order.

/Fc file

Output shader constant declaration.

/Fa file

Output generated assembly listing.

/Fl file

Output readable microcode listing.

/Fh file

Output header specifying generated microcode.

/H

Output microcode as a .h file.

/nologo

Suppress the copyright message.

FileName

Source file to compile.

/?

Displays usage.
