IDirect3DDevice8::SetOverscanColor

Sets the overscan color.

void SetOverscanColor(
  D3DCOLOR Color
);

Parameters

Color
Sets the color of the overscan region.

Return Values

None.

Remarks

This function sets the screen color outside of the rendering viewport. It typically won't actually appear on the screen unless using the 10×11 pixel aspect ratio.
