IDirect3DDevice8::SetVertexData4ub

Sets the initial values of a given vertex attribute. This function sets persistent vertex attributes that may be used in vertex shaders and Begin/End blocks.

HRESULT IDirect3DDevice8::SetVertexData4ub(
   INT Register,
   BYTE a,
   BYTE b,
   BYTE c,
   BYTE d
   );

Parameters

Register
[in] An integer representing the slot number of the vertex attribute to be set. The following constants may be used for this parameter.
D3DVSDE_POSITION
Register 0.
D3DVSDE_BLENDWEIGHT
Register 1.
D3DVSDE_NORMAL
Register 2.
D3DVSDE_DIFFUSE
Register 3. Color is specified as (red, green, blue, alpha), where red, green, and blue are between 0.0f and 255.0f, and alpha is between 0.0f and 1.0f.
D3DVSDE_SPECULAR
Register 4. Color is specified as (red, green, blue, alpha), where red, green, and blue are between 0.0f and 255.0f, and alpha is between 0.0f and 1.0f.
D3DVSDE_FOG
Register 5. Xbox extension.
D3DVSDE_BACKDIFFUSE
Register 7. Color is specified as (red, green, blue, alpha), where red, green, and blue are between 0.0f and 255.0f, and alpha is between 0.0f and 1.0f. Xbox extension.
D3DVSDE_BACKSPECULAR
Register 8. Color is specified as (red, green, blue, alpha), where red, green, and blue are between 0.0f and 255.0f, and alpha is between 0.0f and 1.0f. Xbox extension.
D3DVSDE_TEXCOORD0
Register 9.
D3DVSDE_TEXCOORD1
Register 10.
D3DVSDE_TEXCOORD2
Register 11.
D3DVSDE_TEXCOORD3
Register 12.
a, b, c, d
[in] 8-bit values used to set the floating-point values associated with a vertex attribute slot. The 8-bit values are interpreted as unsigned integers in the range 0.0 to +256.0. After this call, the specified vertex attribute slot will contain (a, b, c, d).

Return Values

If the method succeeds, the return value is D3D_OK.

Remarks

Vertex attributes will remain set unless overwritten by the vertex buffer data stream.

Requirements

  Header: Declared in D3d8.h.
  Import Library: Use D3d8.lib.
