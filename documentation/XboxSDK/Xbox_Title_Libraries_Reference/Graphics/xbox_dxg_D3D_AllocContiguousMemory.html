D3D_AllocContiguousMemory

Allocates a block of contiguous memory.

void* D3D_AllocContiguousMemory(
  DWORD Size,
  DWORD Alignment
);

Parameters

Size
The size of the allocation in bytes. This value is rounded up the nearest page size.
Alignment
The alignment of the allocation. This must be a power of two and is also rounded up to the nearest page size.

Return Values

If this method succeeds, it returns a write-combined pointer to the block of contiguous memory.

If the memory cannot be allocated, this method returns a NULL value.

Remarks

The memory is suitable for any Direct3D data that will be read by the GPU (this is why its write-combined). However, there will be bad performance results if this is use to allocate memory for Direct3D data read by the CPU, such as Direct3D resource header structures and index buffers.

D3D_AllocContiguousMemory and XPhysicalAlloc both use the same internal function. The following two function calls are equivalent:

D3D_AllocContiguousMemory(Size, Alignment);
XPhysicalAlloc(Size, MAXULONG_PTR, Alignment, PAGE_READWRITE | PAGE_WRITECOMBINE);
