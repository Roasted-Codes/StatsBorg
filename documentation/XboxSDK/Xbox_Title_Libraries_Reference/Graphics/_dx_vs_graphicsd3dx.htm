vs

Provides a method to specify the type and version of the shader code.

vs.mainVer.subVer

Registers

mainVer
Main version number of the vertex shader.
subVer
Sub version number of the vertex shader.

Remarks

This instruction puts the shader assembler into DirectX compatiability mode. The shader will be limited to those features that are supported by DirectX 8.0. Among these limits are a maximum of 128 instructions, 96 constants, the inability to use instruction pairing, and the inability to use the new instructions dph or rcc. To take advantage of the Xbox extensions to the shader assembler, you must specify a different type and version of the shader using xvs, xvss, or xvsw.

The following example illustrates how the vs instruction might be used.

vs.1.0  ; Vertex shader version 1.0.

This instruction is required to be at the beginning of all vertex shaders that are not using Xbox extensions.
