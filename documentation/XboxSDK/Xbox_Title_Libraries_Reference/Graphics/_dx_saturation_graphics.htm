Saturate

rn_sat

Clamps the register to the range from zero to one.

The following example shows how to use this modifier.

mul r0, v0, r0_sat  ; Clamp r0 to the range from zero to one and then
                    ; modulate v0 by the resulting value.

Saturation is typically done when preparing a value to use as a blend factor. When blending between two colors, you need to ensure that any previous math has not left you with a negative value since blending takes place between zero and one.

This modifier is mutally exclusive with all other argument modifiers except Alpha Replicate.

The saturation modifier has a diffent usage between DirectX for the Xbox and DirectX 8.0 for the PC. On Xbox, saturation is an argument modifier and on the PC, it is an instruction modifier. Shader programs written for both platforms will need to take this into account.
