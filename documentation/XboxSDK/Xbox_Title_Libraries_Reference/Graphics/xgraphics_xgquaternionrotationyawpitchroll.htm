XGQuaternionRotationYawPitchRoll

Builds a quaternion with the given yaw, pitch, and roll.

XGQUATERNION* XGQuaternionRotationYawPitchRoll(
  XGQUATERNION* pOut,
  FLOAT Yaw,
  FLOAT Pitch,
  FLOAT Roll
);

Parameters

pOut
[in, out] Pointer to the XGQUATERNION structure that is the result of the operation.
Yaw
[in] Yaw around the y-axis, in radians.
Pitch
[in] Pitch around the x-axis, in radians.
Roll
[in] Roll around the z-axis, in radians.

Return Values

Pointer to a XGQUATERNION structure with the specified yaw, pitch, and roll.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGQuaternionRotationYawPitchRoll function can be used as a parameter for another function.

Requirements

  Header: Declared in xgmath.h.
  Import Library: Use xgraphics.lib.
