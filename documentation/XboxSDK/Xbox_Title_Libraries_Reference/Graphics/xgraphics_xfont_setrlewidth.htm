XFONT::SetRLEWidth

Sets the size of the RLE packet used to compress the internal bitmaps used by the font.

HRESULT __stdcall SetRLEWidth(
  unsigned uRLEWidth
);

Parameters

uRLEWidth
[in] Width of the RLE packet. This value must be between two and eight, inclusive.

Return Values

On success, the function returns S_OK.

Remarks

The width of the RLE packet can only be specified for TrueType fonts. The width may effect the number of of bitmaps that can fit in the internal cache depending on the size and charateristics of the font. By default, uRLEWidth is two.
