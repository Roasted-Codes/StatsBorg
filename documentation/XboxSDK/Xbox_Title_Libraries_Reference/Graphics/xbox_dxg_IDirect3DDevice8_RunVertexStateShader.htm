IDirect3DDevice8::RunVertexStateShader

Causes the specified vertex state shader to execute on the GPU.

HRESULT RunVertexStateShader(
  DWORD Address,
  CONST float* pData
);

Parameters

Address
[in] The start location on the GPU of the vertex shader, in the range 0 through 135.
pData
An array of four float values that are assigned to the V0 register. If a NULL value is passed, the setting is undefined.

Return Values

If the method succeeds, the return value is D3D_OK.

If the method fails, the return value can be D3DERR_INVALIDCALL.
