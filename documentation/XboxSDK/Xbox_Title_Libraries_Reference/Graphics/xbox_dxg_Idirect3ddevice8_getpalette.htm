IDirect3DDevice8::GetPalette

Retrieves the current palette.

HRESULT GetPalette(
  DWORD Stage,
  D3DPalette** ppPalette
);

Parameters

Stage
[in] Texture stage identifier of the palette to retrieve. Stage identifiers are zero-based. Devices can have up to four set textures, so the maximum value allowed for Stage is 3.
ppPalette
[out, retval] Address of a pointer to a IDirect3DPalette8 interface, representing the returned palette.

Return Values

The method returns D3D_OK.

Requirements

  Import Library: Use D3d8.lib.
