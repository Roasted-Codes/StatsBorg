D3DDISPLAYMODE

Describes the display mode.

typedef struct _D3DDISPLAYMODE
{
    UINT            Width;
    UINT            Height;
    UINT            RefreshRate;
    DWORD           Flags;        // Xbox extension
    D3DFORMAT       Format;
} D3DDISPLAYMODE;

Members

Width
Screen width, in pixels.
Height
Screen height, in pixels.
RefreshRate
Refresh rate. The value of 0 indicates an adapter default.
Flags
One or more of the following flags ORed together:
D3DPRESENTFLAG_WIDESCREEN
Indicates support for wide screen.
D3DPRESENTFLAG_INTERLACED
Indicates support for an interlaced display.
D3DPRESENTFLAG_PROGRESSIVE
Indicates support for a progressive display.
D3DPRESENTFLAG_FIELD
Indicates support for field rendering.
D3DPRESENTFLAG_10X11PIXELASPECTRATIO
Set this flag to center the frame buffer on the display. On a TV that is 704 pixels across, this would leave 32 pixels of black border on the left and 32 pixels of black border on the right.
D3DPRESENTFLAG_EMULATE_REFRESH_RATE
Set this flag to allow emulation of refresh rates other than the native display mode refresh rates. For example, to emulate a 50 Hz refresh rate in an NTSC mode, set this flag and set FullScreen_RefreshRateInHz to 50. This flag does not work with 1080i and does not work with overlays.
Format
Member of the D3DFORMAT enumerated type, describing the surface format of the display mode.
