IDirect3DDevice8::GetVisibilityTestResult

Retrieves the results of a previously submitted visibility test.

HRESULT GetVisibilityTestResult(
  DWORD Index,
  UINT* pResult,
  ULONGLONG* pTimeStamp
);

Parameters

Index
[in] An integer that identifies which visibility test is being queried.
pResult
[out] Pointer to an integer where the number of pixels rendered for the test geometry is stored if the visibility test is complete.
pTimeStamp
[out] Pointer to a variable that receives the GPU nanosecond count at the time the visibility test completed. If the timestamp is not desired, this value can be NULL.

Return Values

If this method succeeds, it returns D3D_OK.

If this method fails, it can return D3DERR_TESTINCOMPLETE, indicating that the visibility test is not yet complete. A return value of D3DERR_TESTINCOMPLETE also indicates that the values of the pResult and pTimeStamp parameters have not been updated.

Remarks

This method is used to retrieve the result of a previously submitted visibility test. Calling IDirect3DDevice8::GetVisibilityTestResult with an ID that has not been previously specified in a call to IDirect3DDevice8::EndVisibilityTest will result in an error. If the visibility test has been completed by the hardware, the count of rendered pixels is written to the integer that is referenced by the pResult pointer.

The pTimeStamp parameter can be used to determine the time spent in the GPU during a sequence of operations, by executing a visibility test before and after the desired operations and comparing the resulting timestamps.
