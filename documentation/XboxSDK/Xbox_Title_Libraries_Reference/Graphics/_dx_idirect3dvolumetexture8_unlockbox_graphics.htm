IDirect3DVolumeTexture8::UnlockBox

This function does nothing. It has been provided to ease in the porting of code from DirectX 8.0 to DirectX for the Xbox.

HRESULT UnlockBox(
  UINT Level
);

Parameters

Level
[in] Specifies the level of the volume texture resource to unlock.

Return Values

The method returns D3D_OK.
