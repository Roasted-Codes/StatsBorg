XGVec4CatmullRom

Performs a Catmull-Rom interpolation, using the specified 4-D vectors.

XGVECTOR4* XGVec4CatmullRom(
  XGVECTOR4* pOut,
  CONST XGVECTOR4* pV1,
  CONST XGVECTOR4* pV2,
  CONST XGVECTOR4* pV3,
  CONST XGVECTOR4* pV4,
  FLOAT s
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR4 structure that is the result of the operation.
pV1
[in] Pointer to a source XGVECTOR4 structure, a position vector.
pV2
[in] Pointer to a source XGVECTOR4 structure, a position vector.
pV3
[in] Pointer to a source XGVECTOR4 structure, a position vector.
pV4
[[in] Pointer to a source XGVECTOR4 structure, a position vector.
s
[in] Weighting factor. See Remarks.

Return Values

Pointer to a XGVECTOR4 structure that is the result of the Catmull-Rom interpolation.

Remarks

The XGVec4CatmullRom function interpolates between the position V1 when s is equal to 0, and between the position V2 when s is equal to 1, using Catmull-Rom interpolation.

Requirements
