exp

Provides exponential 2x with full precision to at least 1/220.

exp   vDest, vSrc0

Registers

vDest
Destination register, holding the result of the operation.
vSrc0
Source register, specifying the input argument.

Operation

    SetDestReg();
    SetSrcReg(0);

    float v = m_Source[0].w;

    m_TmpReg.x = m_TmpReg.y = m_TmpReg.z = m_TmpReg.w = (float)pow(2, v);  

    WriteResult();

Expansion

This macro takes twelve instruction slots.

Remarks

This is a scalar operation and takes its input from the .w channel, which must be specified. It always replicates the result into all four output channels.

The exp macro instruction is only partially implemented on the Microsoft® Xbox™ game console. For the Xbox, exp is implemented using the expp instruction, without any additional precision. If you require additional precision, you must implement your own exp function. One approach is to use some variation of the following macro:

// myExp requires 10 instructions, 1 temp register, and 2 constant locations.

#define CE0 83 // Taylor series constants
#define CL1 84

def CE0, 9.61597636e-03, -1.32823968e-03, 1.47491097e-04, -1.08635004e-05
def CE1, 1.00000000e+00, -6.93147182e-01, 2.40226462e-01, -5.55036440e-02

// Ro - output
// Ri - input
// Rt - temporary register
// requires that Rt != Ro && Rt != Ri

macro myExp Ro, Ri, Rt
   EXP %Rt, %Ri.x
   MAD %Rt.w, c[CE0].w, %Rt.y, c[CE0].z
   MAD %Rt.w, %Rt.w, %Rt.y, c[CE0].y
   MAD %Rt.w, %Rt.w, %Rt.y, c[CE0].x
   MAD %Rt.w, %Rt.w, %Rt.y, c[CE1].w
   MAD %Rt.w, %Rt.w, %Rt.y, c[CE1].z
   MAD %Rt.w, %Rt.w, %Rt.y, c[CE1].y
   MAD %Rt.w, %Rt.w, %Rt.y, c[CE1].x
   RCP %Rt.w, %Rt.w
   MUL %Ro, %Rt.w, %Rt.x
endm
