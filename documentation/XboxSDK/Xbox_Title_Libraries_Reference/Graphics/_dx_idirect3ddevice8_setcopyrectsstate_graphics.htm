IDirect3DDevice8::SetCopyRectsState

Sets the state and raster operation (ROP) to be performed when calling IDirect3DDevice8::CopyRects, allowing the title to do various blending operations.

HRESULT WINAPI SetCopyRectsState(
  CONST D3DCOPYRECTSTATE *pCopyRectState,
  CONST D3DCOPYRECTROPSTATE *pCopyRectRopState
);

Parameters

pCopyRectState
[in] Pointer to a D3DCOPYTRECTSTATE structure that describes what state to use with CopyRects.
pCopyRectRopState
[in] Pointer to a D3DCOPYRECTROPSTATE structure that describes the ROP to use with CopyRects. pCopyRectRopState may be NULL so long as 
pCopyRectState->Operation != D3DCOPYRECT_ROP_AND

Return Values

If the function succeeds, the return value is S_OK.

Remarks

A full flush of the graphics pipeline occurs whenever you shift between 3D and 2D operations and therefore it is recommended that you perform all 2D operations like CopyRects at the end of the frame if it is possible.
