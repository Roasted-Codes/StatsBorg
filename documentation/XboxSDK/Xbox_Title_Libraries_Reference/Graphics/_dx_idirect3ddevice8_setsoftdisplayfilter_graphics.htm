IDirect3DDevice8::SetSoftDisplayFilter

Enables a TV encoder filter that removes some visual artifacts appearing in displays that have sharp color transitions.

void WINAPI SetSoftDisplayFilter(
  BOOL Enable
);

Parameters

Enable
[in] Set this parameter to TRUE to enable the filtering. By default, this filtering is off.

Return Values

None.

Remarks

Setting the soft-display filter has no effect unless its called after the first call to either IDirect3DDevice8::Swap or IDirect3DDevice8::Present following a display mode change.

Using this filter causes loss of some detail.
