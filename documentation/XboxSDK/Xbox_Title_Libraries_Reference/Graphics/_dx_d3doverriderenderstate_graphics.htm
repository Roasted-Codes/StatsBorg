D3DOVERRIDERENDERSTATE

Describes whether to override a given render state on the next call to D3DPERF_Reset. 

typedef struct _D3DOVERRIDERENDERSTATE
{
    BOOL Override;
    DWORD Value;
} D3DOVERRIDERENDERSTATE;

Members

Override
If TRUE, the render state will overwritten with the new value.
Value
Value passed into the render state.

Remarks

This structure is used in conjunction with the m_SetRenderStateOverridesOnReset and m_RenderStateOverrides members of the D3DPERF structure. For example if you wanted to change the fill mode after the next call to D3DPERF_Reset: 

D3DOVERRIDERENDERSTATE OverrideFillMode = { TRUE, D3DFILL_WIREFRAME };
D3DPERF *pPerf = D3DPERF_GetStatistics();

pPerf->m_SetRenderStateOverridesOnReset = TRUE;
pPerf->m_RenderStateOverrides[ D3DRS_FILLMODE ] = OverrideFillMode;
