IDirect3DDevice8::EnableOverlay

Enables overlays.

HRESULT WINAPI EnableOverlay(
  BOOL Enable
);

Parameters

Enable
[in] TRUE to enable overlays.

Return Values

If the function succeeds, the return value is S_OK.

Remarks

Overlays must be enabled before they can be updated using IDirect3DDevice8::UpdateOverlay. To use overlays, titles should make a single call of EnableOverlay( TRUE ) followed by as many calls to UpdateOverlay as are necessary. When titles are done with the overlay, they should call EnableOverlay( FALSE ).
