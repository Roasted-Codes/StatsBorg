D3DVSD_SKIP

Specifies how many DWORDs to skip in the vertex.

D3DVSD_SKIP( _DWORDCount ) \
           (D3DVSD_MAKETOKENTYPE(D3DVSD_TOKEN_STREAMDATA) | 0x10000000 | \
           ((_DWORDCount) << D3DVSD_SKIPCOUNTSHIFT))

Parameters

_DWORDCount
Number of DWORDs to skip in the vertex.
