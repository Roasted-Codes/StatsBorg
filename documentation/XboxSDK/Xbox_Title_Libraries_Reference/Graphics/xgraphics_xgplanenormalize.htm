XGPlaneNormalize

Returns the normal of a plane.

XGPLANE* XGPlaneNormalize(
  XGPLANE* pOut,
  CONST XGPLANE* pP
);

Parameters

pOut
[in, out] Pointer to the XGPLANE structure that is the result of the operation.
pP
[in] Pointer to the source XGPLANE structure.

Return Values

Pointer to a XGPLANE structure, representing the normal of the plane.

Remarks

XGPlaneNormalize normalizes a plane so that |a,b,c| == 1.

The return value for this function is the same value returned in the pOut parameter. In this way, the XGPlaneNormalize function can be used as a parameter for another function.

Requirements
