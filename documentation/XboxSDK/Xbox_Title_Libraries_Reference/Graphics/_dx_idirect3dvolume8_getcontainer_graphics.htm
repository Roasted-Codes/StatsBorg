IDirect3DVolume8::GetContainer

Provides access to the parent volume texture object, if this surface is a child level of a volume texture.

HRESULT GetContainer(
  IDirect3DBaseTexture8** ppContainer
);

Parameters

ppContainer
[out, retval] Address of a pointer to fill with the container pointer.

Return Values

If the method succeeds, the return value is D3D_OK.

If there is no container, the return value is E_FAIL.

Requirements
