IDirect3DDevice8::GetPushBufferOffset

Used only with an IDirect3DDevice8::BeginPushBuffer/IDirect3DDevice8::EndPushBuffer bracket to determine the offset of the next command to be recorded with the push-buffer.

HRESULT GetPushBufferOffset(
  DWORD *pOffset
);

Parameters

pOffset
[out, retval] Pointer to the DWORD value containing the offset within the push-buffer of the next Direct3D command to be recorded within the IDirect3DDevice8::BeginPushBuffer/IDirect3DDevice8::EndPushBuffer bracket.

Return Values

If the method succeeds, the return value is S_OK.

Remarks

This offset returned by this function is used to generate fix-ups using the fix-up functions. For more information see Fix-ups.

Requirements
