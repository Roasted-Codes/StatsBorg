XGColorModulate

Blends two colors.

XGCOLOR* XGColorModulate(
  XGCOLOR* pOut,
  CONST XGCOLOR* pC1,
  CONST XGCOLOR* pC2
);

Parameters

pOut
[in, out] Pointer to a XGCOLOR structure that is the result of the operation.
pC1
[in] Pointer to a source XGCOLOR structure.
pC2
[in] Pointer to a source XGCOLOR structure.

Return Values

This function returns a pointer to a XGCOLOR structure that is the result of the blending operation.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGColorModulate function can be used as a parameter for another function.

This function blends together two colors by multiplying matching color components, as shown in the following example.

    pOut->r = pC1->r * pC2->r;
