IDirect3DPalette8::GetSize

Retrieves the size of a palette resource.

D3DPALETTESIZE GetSize();

Parameters

None.

Return Values

Returns the size of a palette. The return value can be any of the following D3DPALETTESIZE values: D3DPALETTE_32, D3DPALETTE_64, D3DPALETTE_128, or D3DPALETTE_256.

Requirements
