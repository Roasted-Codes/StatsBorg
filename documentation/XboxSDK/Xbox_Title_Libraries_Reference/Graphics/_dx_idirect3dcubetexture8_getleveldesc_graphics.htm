IDirect3DCubeTexture8::GetLevelDesc

Retrieves a description of a cube texture level

HRESULT GetLevelDesc(
  UINT Level,
  D3DSURFACE_DESC* pDesc
);

Parameters

Level
[in] Specifies a cube texture level.
pDesc
[out] Pointer to a D3DSURFACE_DESC structure, describing the specified cube texture level.

Return Values

The method returns D3D_OK.

Requirements

  Header: Declared in D3d8.h.
  Import Library: Use D3d8.lib.
