IDirect3DResource8::IsBusy

Returns whether or not the data for this resource is currently being used by the GPU. 

BOOL IsBusy();

Parameters

None.

Return Values

Returns TRUE if the data is being used by the GPU.

Returns FALSE if the data is not in use.

Remarks

This method will always return TRUE if this resource is set as one of the resources currently being used by IDirect3DDevice8. It sets the Lock field in the resource to zero when it returns FALSE.

It is safe to call this method from any thread. This method may safely be called within a tight loop without affecting the performance of the graphics processor.

This method can be used to check to see if a call to Lock or if the final Release would block.
