IDirect3DDevice8::GetBackBufferScale

GetBackBufferScale gets the scale of the back-buffer rendering as it was set by IDirect3DDevice8::SetBackBufferScale.

HRESULT GetBackBufferScale(
  float *pX,
  float *pY
);

Parameters

pX
[out] Pointer to a float that will receive the horizontal scaling factor applied to the back buffer.
pY
[out] Pointer to a float that will receive the vertical scaling factor applied to the back buffer.

Return Values

If the method succeeds, the return value is S_OK.

Requirements

  Header: Declared in D3d8.h.
  Import Library: Use D3d8.lib.
