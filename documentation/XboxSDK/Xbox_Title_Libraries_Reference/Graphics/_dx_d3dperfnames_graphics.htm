D3DPERFNAMES

Holds the names for the different performance items gathered by the driver.

typedef struct _D3DPERFNAMES
{
    char *m_PerformanceCounterNames[D3DPERF_MAX];
    char *m_APICounterNames[D3DAPI_MAX];
    D3DSTATENAME m_RenderStateNames[D3DRS_MAX];
    D3DSTATENAME m_TextureStateNames[D3DTSS_MAX];
} D3DPERFNAMES;

Members

m_PerformanceCounterNames
An array of human-readable names for the non-Direct3D events that the driver counts. This array is indexed using the D3DPERF_PerformanceCounters enumeration.
m_APICounterNames
An array of human-readable names for the Direct3D functions that the driver counts. This array is indexed using the D3DPERF_APICounters enumeration.
m_RenderStateNames
An array of D3DSTATENAME objects that provide human-readable names for the render states. This array is indexed using the D3DRENDERSTATETYPE enumeration.
m_TextureStateNames
An array of D3DSTATENAME objects that provide human-readable names for the texture states. This array is indexed using the D3DTEXTURESTAGESTATETYPE enumeration.

Remarks

This information is generally obtained using a pointer returned from D3DPERF_GetNames.

D3DPERFNAMES *pNames = D3DPERF_GetNames();

The pointer returned by D3DPERF_GetNames should never be freed since it is statically allocated.
