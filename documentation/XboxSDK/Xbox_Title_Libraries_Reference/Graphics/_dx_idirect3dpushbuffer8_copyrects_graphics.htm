IDirect3DPushBuffer8::CopyRects

Used only within a IDirect3DPushBuffer8::BeginFixup/IDirect3DPushBuffer8::EndFixup bracket to modify the corresponding call to IDirect3DDevice8::CopyRects as it was recorded inthe push-buffer. This method copies rectangular subsets of pixels from one surface to another.

HRESULT CopyRects(
    DWORD Offset,
    IDirect3DSurface8 *pSourceSurface,
    IDirect3DSurface8 *pDestinationSurface
);

Parameters

Offset
[in] The offset determined during the recording in the push-buffer as returned from IDirect3DDevice8::GetPushBufferOffset just before IDirect3DDevice8::CopyRects was recorded.
pSourceSurface
[in] Pointer to an IDirect3DSurface8 interface, representing the source surface. This parameter must point to a different surface than pDestinationSurface.
pDestinationSurface
[out] Pointer to an IDirect3DSurface8 interface, representing the destination surface. This parameter must point to a different surface than pSourceSurface.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This method does not allow the title to modify the arrays describing the rectangles used for the source or destination surfaces.

Requirements

Header: Declared in D3d8.h.

Import Library: Use D3d8.lib.
