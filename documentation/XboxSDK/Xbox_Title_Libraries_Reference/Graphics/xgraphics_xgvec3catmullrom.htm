XGVec3CatmullRom

Performs a Catmull-Rom interpolation, using the specified 3D vectors.

XGVECTOR3* XGVec3CatmullRom(
  XGVECTOR3* pOut,
  CONST XGVECTOR3* pV1,
  CONST XGVECTOR3* pV2,
  CONST XGVECTOR3* pV3,
  CONST XGVECTOR3* pV4,
  FLOAT s
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR3 structure that is the result of the operation.
pV1
[in] Pointer to a source XGVECTOR3 structure, a position vector.
pV2
[in] Pointer to a source XGVECTOR3 structure, a position vector.
pV3
[in] Pointer to a source XGVECTOR3 structure, a position vector.
pV4
[[in] Pointer to a source XGVECTOR3 structure, a position vector.
s
[in] Weighting factor. See Remarks.

Return Values

Pointer to a XGVECTOR3 structure that is the result of the Catmull-Rom interpolation.

Remarks

The XGVec3CatmullRom function interpolates between the position V1 when s is equal to 0, and between the position V2 when s is equal to 1, using Catmull-Rom interpolation.

Requirements
