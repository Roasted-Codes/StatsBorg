D3DPERF_GetStatistics

Gets the performance statistics from the driver.

D3DPERF * WINAPI D3DPERF_GetStatistics();

Parameters

None.

Return Values

Returns a pointer to a D3DPERF structure which is statically allocated. This structure should never be freed.

Requirements

  Header: Declared in d3d8perf.h.
  Import Library: Use d3d8i.lib or d3d8d.lib.
