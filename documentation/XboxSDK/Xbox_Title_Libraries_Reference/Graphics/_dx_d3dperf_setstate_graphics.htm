D3DPERF_SetState

Enables or disables the vertex cache and/or file.

HRESULT WINAPI D3DPERF_SetState(
  D3DPERFSTATETYPE State, 
  DWORD Value
);

Parameters

State
[in] Member of the D3DPERFSTATETYPE enumerated type describing what will be enabled or disabled.
Value
[in] Selects the behavior to apply to the vertex cache and/or file. Its value has the following effects:
Value Effect
0 Turns off and disables the vertex cache described by State.
0x80000000 If the vertex cache described by State was enabled, it becomes disabled. If it was disabled, it becomes enabled.
All other values        Turns on and enables the vertex cache decribed by State.

Return Values

If the function succeeds, the return value is S_OK, otherwise, the return value is E_FAIL.

Remarks

This function can be useful in determining how well the vertex caching is being used, or if it's being used at all.

Requirements

  Header: Declared in d3d8perf.h.
  Import Library: Use d3d8i.lib or d3d8d.lib.
