IDirect3DDevice8::GetDisplayFieldStatus

Retrieves the number of Vblank interrupts since startup. For TV output, this function also returns whether the display field is EVEN or ODD. 

HRESULT GetDisplayFieldStatus(
  D3DFIELD_STATUS* pFieldStatus,

);

Parameters

pFieldStatus
[out] A pointer to a D3DFIELD_STATUS structure that contains the number of Vblank interrupts and the display field type. If the output is to a monitor, the type returned is D3DFIELD_PROGRESSIVE.

Return Values

If this method succeeds, it returns S_OK.

Remarks

The Vblank counter can be used to syncronize your application.
