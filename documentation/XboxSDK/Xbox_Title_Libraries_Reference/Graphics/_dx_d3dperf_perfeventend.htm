D3DPERF_PerfEventEnd

Stop monitoring a user performance event.

void WINAPI D3DPERF_PerfEventEnd(
    D3DPERFEvent *pD3DPerfEvent,
    BOOL RecordGpuTime
);

Parameters

pD3DPerfEvent
[in] Pointer to the D3DPERFEvent structure that describes the user performance event that should stop getting monitored.
RecordGpuTime
[in] TRUE to insert a fence for counting GPU times.

Return Values

None.

Requirements

Header: Declared in D3D8Perf.h.

Import Library: Use D3d8i.lib or d3d8d.lib.

See Also

D3DPERF_PerfEventStart, D3DPERF, D3DPERFEvent
