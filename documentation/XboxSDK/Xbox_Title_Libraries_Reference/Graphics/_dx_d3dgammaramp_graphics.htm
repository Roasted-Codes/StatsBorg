D3DGAMMARAMP

Contains red, green, and blue ramp data.

typedef struct _D3DGAMMARAMP {
    BYTE                red  [256];
    BYTE                green[256];
    BYTE                blue [256];
} D3DGAMMARAMP;

Members

red, green, and blue
Array of 256 BYTE elements that describe the red, green, and blue gamma ramps.

Xbox Optimizations

The DirectX 8.0 implementation of this structure was defined as follows:

typedef struct _D3DGAMMARAMP {
    WORD                red  [256];
    WORD                green[256];
    WORD                blue [256];
} D3DGAMMARAMP;
