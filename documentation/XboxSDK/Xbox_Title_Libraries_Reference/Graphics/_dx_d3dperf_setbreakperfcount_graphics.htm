D3DPERF_SetBreakPerfCount

Sets a breakpoint on a specified performance counter.

__inline void WINAPI D3DPERF_SetBreakPerfCount(
  D3DPERF_PerformanceCounters perfctr,
  DWORD BreakCount
);

Parameters

perfctr
[in] Member of the D3DPERF_PerformanceCounters enumeration which specifies on what performance counter the breakpoint is set.
BreakCount
[in] Number of times the performance counter is hit before the break occurs.

Return Values

None.
