XFONT::SetTextAntialiasLevel

Sets the amount of anti-alias information to include in the bitmap.

HRESULT __stdcall SetTextAntialiasLevel(
  unsigned uAntialiasLevel
);

Parameters

uAntialiasLevel
[in] The level of anti-aliasing. This value must be zero, two, or four, where zero specifies that there is no anti-aliasing and four specifies the maximum amount.

Return Values

On success, the function returns S_OK.

Remarks

The anti-aliasing level can only be specified for the a TrueType font.

Anti-aliasing works best on large or bold fonts whose strokes are more than one pixel wide. By default, there is no anti-aliasing.
