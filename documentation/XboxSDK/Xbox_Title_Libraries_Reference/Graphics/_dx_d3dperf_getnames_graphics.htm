D3DPERF_GetNames

Gets the names of the performance counters.

D3DPERFNAMES * WINAPI D3DPERF_GetNames();

Parameters

None.

Return Values

Returns a pointer to a D3DPERFNAMES structure which is statically allocated. This structure should never be freed.

Requirements

  Header: Declared in d3d8perf.h.
  Import Library: Use d3d8i.lib or d3d8d.lib.
