D3DPERFEvent

Describes a given performance event.

typedef struct _D3DPERFEvent{
    WORD Type;
    union {
        char HeaderText[30];
        struct {
            WORD GpuPct;
            DWORD CpuPutPtr;
            ULONGLONG CycleCpuStart;
            DWORD CycleCpuLength;
            DWORD CycleGpuStartDelta;
            DWORD CycleGpuLength;
            DWORD Data;
        } ;
    } ;
} D3DPERFEvent;

Members

Type
A member of the D3DPERFEventsType enumeration. Describes the type of the Direct3D performance event.
HeaderText
NULL terminated string that describes a user performance event. This string gets set after calling D3DPERF_InsertPerfEventHeader.
GpuPct
Describes the percentage that the GPU was busy since the last sample. If GpuPct is 0xffff, this indicates that insufficient samples were taken.
CpuPutPtr
Pointer to the location in the push-buffer at the time given in CycleCpuStart.
CycleCpuStart
Start time of the performance monitoring for the event. CycleCpuStart is measured in rdtsc units (for the Xbox, 1/733 seconds).
CycleCpuLength
Number of rdtsc units spent in the event.
CycleGpuStartDelta
Difference in time (measured in rdtsc units) between when the performance event started and when the GPU processing began.
CycleGpuLength
Time spent by the GPU during the event, measured in rdtsc units.
Data
Number of bytes written to the push-buffer. However, if Type is D3DPERFEvent_RunPushBuffer, Data is the size of the push-buffer.
