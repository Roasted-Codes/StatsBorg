XGPlaneDotCoord

Computes the dot-product of a plane and a 3D vector. The w parameter of the vector is assumed to be 1.

FLOAT XGPlaneDotCoord(
  CONST XGPLANE* pP,
  CONST XGVECTOR3* pV
);

Parameters

pP
[in] Pointer to a source XGPLANE structure.
pV
[in] Pointer to a source XGVECTOR3 structure.

Return Values

The dot-product of the plane and 3D vector.

Remarks

Given a plane (a, b, c, d) and a 3D vector (x, y, z) the return value of this function is a*x + b*y + c*z + d*1. The XGPlaneDotCoord function is useful for determining the plane's relationship with a coordinate in 3D space.
