XFONT::GetRLEWidth

Returns the size of the RLE packet used to compress the internal bitmaps used by the font.

unsigned __stdcall GetRLEWidth();

Parameters

None.

Return Values

On success, the function returns the size of the RLE packet used to compress the internal bitmaps used by the font.

Requirements

  Header: Declared in xfont.h
  Import Library: Use xgraphics.lib
