XGPlaneIntersectLine

Finds the intersection between a plane and a line.

XGVECTOR3* XGPlaneIntersectLine(
  XGVECTOR3* pOut,
  CONST XGPLANE* pP,
  CONST XGVECTOR3* pV1,
  CONST XGVECTOR3* pV2
);

Parameters

pOut
[in, out] Pointer to a XGVECTOR3 structure, identifying the intersection between the specified plane and line.
pP
[in] Pointer to the source XGPLANE structure.
pV1
[in] Pointer to a source XGVECTOR3 structure, defining a line starting point.
pV2
[in] Pointer to a source XGVECTOR3 structure, defining a line ending point.

Return Values

Pointer to a XGVECTOR3 structure that is the intersection between the specified plane and line.

Remarks

If the line is parallel to the plane, NULL is returned.

The return value for this function is the same value returned in the pOut parameter. In this way, the XGPlaneIntersectLine function can be used as a parameter for another function.

Requirements
