xps

Provides a method to specify the type and version of the shader code. This identifies the shader as being a pixel shader that uses Xbox extensions to the shader assembler.

xps.mainVer.subVer

Registers

mainVer
Main version number of the pixel shader. Must be 1.
subVer
Sub version number of the pixel shader. Must be 1.

Remarks

The following example illustrates how the xps instruction might be used.

xps.1.1  ; Xbox pixel shader version 1.1.

This instruction is required to be at the beginning of all pixel shaders that are using Xbox extensions to the shader assembler.
