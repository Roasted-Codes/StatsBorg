XGMatrixReflect

Builds a matrix that reflects the coordinate system about a plane.

XGMATRIX* XGMatrixReflect(
  XGMATRIX* pOut,
  CONST XGPLANE* pPlane
);

Parameters

pOut
[in, out] Pointer to the XGMATRIX structure that is the result of the operation.
pPlane
[in] Pointer to the source XGPLANE structure.

Return Values

Pointer to a XGMATRIX structure that reflects the coordinate system about the source plane.

Remarks

This function normalizes the plane equation before it creates the reflected matrix.

The return value for this function is the same value returned in the pOut parameter. In this way, the XGMatrixReflect function can be used as a parameter for another function.

Requirements
