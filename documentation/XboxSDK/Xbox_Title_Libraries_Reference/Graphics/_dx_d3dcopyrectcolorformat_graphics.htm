D3DCOPYRECTCOLORFORMAT

Defines the supported color formats of the source and destination surfaces used in IDirect3DDevice8::CopyRects.

typedef enum _D3DCOPYRECTCOLORFORMAT    // Xbox extension
{
    D3DCOPYRECT_COLOR_FORMAT_DEFAULT                 = 0,
    D3DCOPYRECT_COLOR_FORMAT_Y8                      = 1,
    D3DCOPYRECT_COLOR_FORMAT_X1R5G5B5_Z1R5G5B5       = 2,
    D3DCOPYRECT_COLOR_FORMAT_X1R5G5B5_O1R5G5B5       = 3,
    D3DCOPYRECT_COLOR_FORMAT_R5G6B5                  = 4,
    D3DCOPYRECT_COLOR_FORMAT_Y16                     = 5,
    D3DCOPYRECT_COLOR_FORMAT_X8R8G8B8_Z8R8G8B8       = 6,
    D3DCOPYRECT_COLOR_FORMAT_X8R8G8B8_O8R8G8B8       = 7,
    D3DCOPYRECT_COLOR_FORMAT_X1A7R8G8B8_Z1A7R8G8B8   = 8,
    D3DCOPYRECT_COLOR_FORMAT_X1A7R8G8B8_O1A7R8G8B8   = 9,
    D3DCOPYRECT_COLOR_FORMAT_A8R8G8B8                = 10,
    D3DCOPYRECT_COLOR_FORMAT_Y32                     = 11,
    D3DCOPYRECT_COLOR_FORMAT_FORCE_DWORD             = 0x7fffffff, /* force 32-bit size enum */
} D3DCOPYRECTCOLORFORMAT;

Constants

D3DCOPYRECT_COLOR_FORMAT_DEFAULT
This is the default CopyRects behavior. If this is set, then D3DCOPYRECT_SRCCOPY must be set as the D3DCOPYRECTOPERATION.
D3DCOPYRECT_COLOR_FORMAT_Y8
The source and destination surfaces specified in CopyRects can be any 8-bit surface format. The only D3DCOPYRECTOPERATIONs allowed with this format are the following:
  • D3DCOPYRECT_SRCCOPY_AND
  • D3DCOPYRECT_ROP_AND
  • D3DCOPYRECT_SRCCOPY
D3DCOPYRECT_COLOR_FORMAT_X1R5G5B5_Z1R5G5B5
The source surface for CopyRects has surface format D3DFMT_X1R5G5B5. The destination surface format is the same except that the alpha channel is set to zero.
D3DCOPYRECT_COLOR_FORMAT_X1R5G5B5_O1R5G5B5
The source surface for CopyRects has surface format D3DFMT_X1R5G5B5. The destination surface format is the same except that the alpha channel is set to one.
D3DCOPYRECT_COLOR_FORMAT_R5G6B5
The source surface and destination surface specified in CopyRects both have the same surface format, D3DFMT_R5G6B5.
D3DCOPYRECT_COLOR_FORMAT_Y16
The source and destination surfaces specified in CopyRects can be any 16-bit surface format. The only D3DCOPYRECTOPERATIONs allowed with this format are the following:
  • D3DCOPYRECT_SRCCOPY_AND
  • D3DCOPYRECT_ROP_AND
  • D3DCOPYRECT_SRCCOPY
D3DCOPYRECT_COLOR_FORMAT_X8R8G8B8_Z8R8G8B8
The source surface for CopyRects has surface format D3DFMT_X8R8G8B8. The destination surface format is the same except that the alpha channel is set to zero.
D3DCOPYRECT_COLOR_FORMAT_X8R8G8B8_O8R8G8B8
The source surface for CopyRects has surface format D3DFMT_X8R8G8B8. The destination surface format is the same except that the alpha channel is set to one.
D3DCOPYRECT_COLOR_FORMAT_X1A7R8G8B8_Z1A7R8G8B8
This source surface for CopyRects has a surface format in which 7 bits are reserved for alpha and each of the color channels uses 8 bits. The surface format is stored as below: 
(alpha << 24) | (red << 16) | (green << 8) | blue
The destination surface format is the same except that the upper-most bit (which was unused in the source) is set to zero.

The only D3DCOPYRECTOPERATIONs allowed with this format are the following:

  • D3DCOPYRECT_SRCCOPY
  • D3DCOPYRECT_SRCCOPY_PREMULT
D3DCOPYRECT_COLOR_FORMAT_X1A7R8G8B8_O1A7R8G8B8
This source surface for CopyRects has a surface format in which 7 bits are reserved for alpha and each of the color channels uses 8 bits. The surface format is stored as below: 
(alpha << 24) | (red << 16) | (green << 8) | blue
The destination surface format is the same except that the upper-most bit (which was unused in the source) is set to one.

The only D3DCOPYRECTOPERATIONs allowed with this format are the following:

  • D3DCOPYRECT_SRCCOPY
  • D3DCOPYRECT_SRCCOPY_PREMULT
D3DCOPYRECT_COLOR_FORMAT_A8R8G8B8
The source surface and destination surface specified in CopyRects both have the same surface format, D3DFMT_A8R8G8B8.

The only D3DCOPYRECTOPERATIONs allowed with this format are the following:

  • D3DCOPYRECT_SRCCOPY
  • D3DCOPYRECT_SRCCOPY_PREMULT
D3DCOPYRECT_COLOR_FORMAT_Y32
The source and destination surfaces specified in CopyRects can be any 32 bit surface format. The only D3DCOPYRECTOPERATIONs allowed with this format are the following:
  • D3DCOPYRECT_SRCCOPY_AND
  • D3DCOPYRECT_ROP_AND
  • D3DCOPYRECT_SRCCOPY
D3DCOPYRECT_COLOR_FORMAT_FORCE_DWORD
Forces this enumeration to compile to 32 bits in size. This value is not used.

Remarks

The color format of the surfaces is specified in the ColorFormat member of the D3DCOPYRECTSTATE structure used in IDirect3DDevice8::SetCopyRectsState and IDirect3DDevice8::GetCopyRectsState.
