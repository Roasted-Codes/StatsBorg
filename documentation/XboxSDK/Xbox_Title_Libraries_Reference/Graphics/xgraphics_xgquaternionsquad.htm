XGQuaternionSquad

Interpolates between quaternions, using spherical quadrangle interpolation. 

XGQUATERNION* XGQuaternionSquad(
  XGQUATERNION* pOut,
  CONST XGQUATERNION* pQ1,
  CONST XGQUATERNION* pQ2,
  CONST XGQUATERNION* pQ3,
  CONST XGQUATERNION* pQ4,
  FLOAT t
);

Parameters

pOut
[in, out] Pointer to the XGQUATERNION structure that is the result of the operation.
pQ1
[in] Pointer to a source XGQUATERNION structure.
pQ2
[in] Pointer to a source XGQUATERNION structure.
pQ3
[in] Pointer to a source XGQUATERNION structure.
pQ4
[in] Pointer to a source XGQUATERNION structure.
t
[in] Parameter that indicates how far to interpolate between the quaternions.

Return Values

Pointer to a XGQUATERNION structure that is the result of the spherical quadrangle interpolation.

Remarks

This function uses the following sequence of spherical linear interpolation operations: Slerp(Slerp(Q1, Q4, t), Slerp(Q2, Q3, t), 2t(1-t))

The return value for this function is the same value returned in the pOut parameter. In this way, the XGQuaternionSquad function can be used as a parameter for another function.
