D3DPERF_Dump

Dumps all of the performance statistics accumulated by the driver into the debug window.

void WINAPI D3DPERF_Dump();

Parameters

None.

Return Values

None.

Remarks

This function will display all

  • Non-Direct3D events performance counters,
  • Direct3D events performance counters,
  • Render state performance counters, and
  • Texture state performance counters.

In addition, the function will display the number of cycles since the last call of D3DPERF_Reset.

Requirements

  Header: Declared in d3d8perf.h.
  Import Library: Use d3d8i.lib or d3d8d.lib.
