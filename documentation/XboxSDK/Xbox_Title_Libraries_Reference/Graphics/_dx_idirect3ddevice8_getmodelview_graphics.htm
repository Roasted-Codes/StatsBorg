IDirect3DDevice8::GetModelView

Gets the last model-view transform.

HRESULT GetModelView(
  D3DMATRIX *pModelView
);

Parameters

pModelView
[out] Pointer to a D3DMATRIX structure containing the last model-view transform, as specified by IDirect3DDevice8::SetModelView.

Return Values

If IDirect3DDevice8::SetModelView currently has control of the GPU's transform registers, the return value is S_OK; otherwise the return value is S_FALSE.

Requirements
