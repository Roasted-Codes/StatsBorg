PSCombinerCount

The PSCombinerCount field allows the number of combiner stages to be set for this pixel shader along with three global constants that affect all combiner stages. It is most convenient to use the PS_COMBINERCOUNT(count, flags) macro to set this field. The macro is defined as below:

#define PS_COMBINERCOUNT(count, flags) (((flags)<<8)|(count))

The count value is an integer from 1 to 8. Zero is an invalid value. Flags can be any of the values below, logically ORed together.

PS_COMBINERCOUNT_MUX_LSB
Controls the behavior of the mux operation for all stages. If PS_COMBINERCOUNT_MUX_LSB is specified, the bit used to mux the output of each stage is the least significant bit of the r0.a register.
PS_COMBINERCOUNT_MUX_MSB
Controls the behavior of the mux operation for all stages. If PS_COMBINERCOUNT_MUX_MSB is specified, the bit used to mux the output of each stage is the most significant bit of the r0.a register. If this flag is set, the behavior is identical to the usual Direct3D operation.
PS_COMBINERCOUNT_SAME_C0
If set, reading the c0 constant from any stage reads the value of the c0 constant in stage 0.
PS_COMBINERCOUNT_UNIQUE_C0
If set, each stage can have a unique c0 constant that may be set and read independently.
PS_COMBINERCOUNT_SAME_C1
If set, reading the c1 constant from any stage reads the value of the c1 constant in stage 0.
PS_COMBINERCOUNT_UNIQUE_C1
If set, each stage can have a unique c1 constant that may be set and read independently.
