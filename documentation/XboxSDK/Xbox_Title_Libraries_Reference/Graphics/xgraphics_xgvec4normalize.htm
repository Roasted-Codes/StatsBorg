XGVec4Normalize

Returns the normalized version of a 4-D vector.

XGVECTOR4* XGVec4Normalize(
  XGVECTOR4* pOut,
  CONST XGVECTOR4* pV
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR4 structure that is the result of the operation.
pV
[in] Pointer to the source XGVECTOR4 structure.

Return Values

Pointer to a XGVECTOR4 structure that is the normalized version of the vector.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec4Normalize function can be used as a parameter for another function.

Requirements

  Header: Declared in xgmath.h.
  Import Library: Use xgraphics.lib.
