XGSUCode_GetVertexShaderLength

Examines shader microcode and determines the length in instructions.

DWORD WINAPI XGSUCode_GetVertexShaderLength(LPCVOID pMicrocode);

Parameters

pMicrocode
A pointer to the microcode.

Return Values

Returns the length of the shader in microinstructions.

If the microcode is not a valid shader, the result is undefined.
