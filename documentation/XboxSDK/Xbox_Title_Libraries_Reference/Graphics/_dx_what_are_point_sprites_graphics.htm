What Are Point Sprites?

Support for point sprites in Microsoft® Direct3D® for Xbox enables the high-performance rendering of points (particle systems). Point sprites are generalizations of generic points that enable arbitrary shapes to be rendered as defined by textures.

Variable sized point sprites are not supported in the fixed-function geometry pipeline. To use variable sized point sprites, programmable vertex processing must be used.

To enable the use of point sprites, either within the programmable geometry pipeline, or within the fixed-function geometry pipeline, both D3DRS_POINTSPRITEENABLE and D3DRS_POINTSCALEENABLE must be set to TRUE using code similar to the following:

// pDevice is assumed to be a valid pointer
// to an IDirect3DDevice8 interface.
pDevice->SetRenderState( D3DRS_POINTSPRITEENABLE, TRUE );
pDevice->SetRenderState( D3DRS_POINTSCALEENABLE, TRUE );

Point sprites only use texture stage 3 and therefore, some additional code is needed:

// pTexture is assumed to be a valid pointer to a texture.
pDevice->SetTexture( 3, pTexture );
