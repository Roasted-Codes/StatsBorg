D3DPERF_InsertPerfEventHeader

Inserts a user-defined performance event header (D3DPERFEvent_Header) into the GPU.

__inline void D3DPERF_InsertPerfEventHeader(
    char *szHeader
);

Parameters

szHeader
[in] NULL terminated string that will be associated with this event.

Return Values

None.

Remarks

D3DPERF_InsertPerfEventHeader inserts a Direct3D performance event and associates the given string with the event. Header events allow titles to insert descriptive strings into the performance event buffer, to describe what is happening in the scene at that time.

Note    The descriptive string is stored in place of the usual CPU and GPU event timers and counters. To store timing information, you must insert a separate user-defined event using D3DPERF_PerfEventStart and D3DPERF_PerfEventEnd.

The following code shows how this might get used.

D3DPERF_InsertPerfEventHeader( "Start updating AI" );
D3DPERFEvent *pPerfEvent = D3DPERF_PerfEventStart( D3DPERFEvent_User0, TRUE );

// Update the AI.
// ...

D3DPERF_PerfEventEnd( pPerfEvent, TRUE );

Requirements

Header: Declared in D3d8perf.h.
