Blue Replicate

rn.b

Replicates the blue channel of the affected register to the alpha channel before processing the instruction. The following example shows how to use this modifier:

mul r0, r0, r1.b    ; This is equivalent to the following:
                    ; mul r0.rgb, r0.rgb, r1.rgb
                    ; + mul r0.a, r0.a, r1.b

This operator can be used in conjunction with all other input modifiers except alpha replicate.
