IDirect3DDevice8::SetScissors

Sets scissor rectangles that may be used to throw away (scissor) all pixels that fall within a certain range.

HRESULT WINAPI SetScissors(
  DWORD Count, 
  BOOL Exclusive, 
  CONST D3DRECT *pRects
);

Parameters

Count
[in] Number of rectangles to scissor. Count must be less than D3DSCISSORS_MAX. Setting Count to zero will reset the scissors to the last IDirect3DDevice8::SetViewport values.
Exclusive
[in] If Exclusive is TRUE, a pixel is killed if it falls into any of the specified scissor rectangles. If Exclusive is FALSE, a pixel is killed only if it does not fall into any of the specified rectangles.
pRects
[in] Pointer to an array of D3DRECT structures whose size is specified by Count. The scissor rectangles are specified using screen-space coordinates.

Return Values

If the method succeeds, the return value is S_OK.

Remarks

Scissoring will take place in the next rendering operation that follows IDirect3DDevice8::SetScissors. The scissoring operation is respected by all rendering operations except IDirect3DDevice8::Clear.

Calling IDirect3DDevice8::SetViewport will reset the scissors state to be one inclusive rectangle that is the size of the viewport. This may be overridden by calling IDirect3DDevice8::SetScissors after setting the viewport.
