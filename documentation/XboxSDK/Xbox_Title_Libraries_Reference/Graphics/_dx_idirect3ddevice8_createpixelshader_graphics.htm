IDirect3DDevice8::CreatePixelShader

Creates a pixel shader.

HRESULT CreatePixelShader(
  CONST D3DPIXELSHADERDEF* pPSDef,
  DWORD* pHandle
);

Parameters

pPSDef
[in] Pointer to a D3DPIXELSHADERDEF structure that specifies the blending operations. This value cannot be NULL.
pHandle
[out, retval] Pointer to the returned pixel shader handle.

Return Values

If the method succeeds, the return value is D3D_OK.

If the method fails, the return value can be one of the following values.

D3DERR_INVALIDCALL
D3DERR_OUTOFVIDEOMEMORY
E_OUTOFMEMORY

Requirements

  Import Library: Use D3d8.lib.
