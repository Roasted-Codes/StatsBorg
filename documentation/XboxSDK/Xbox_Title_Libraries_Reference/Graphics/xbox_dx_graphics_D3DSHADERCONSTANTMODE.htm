D3DSHADERCONSTANTMODE

The D3DSHADERCONSTANTMODE is used to specify the number of constant registers available to the shader.

typedef DWORD D3DSHADERCONSTANTMODE;            // Xbox extension

#define D3DSCM_96CONSTANTS                  0
#define D3DSCM_192CONSTANTS                 1
#define D3DSCM_192CONSTANTSANDFIXEDPIPELINE 2
#define D3DSCM_NORESERVEDCONSTANTS          0x10    // Flag

Constants

D3DSCM_96CONSTANTS
Makes registers 0 through 95 available.
D3DSCM_192CONSTANTS
Makes registers -96 through 95 available. Specifying this flag prevents the usage of the of the fixed function pipeline.
D3DSCM_192CONSTANTSANDFIXEDPIPELINE
This constant is not supported.
D3DSCM_NORESERVEDCONSTANTS
This constant can be ORed into the other constants to tell DirectX not to reserve registers D3DVS_XBOX_RESERVEDCONSTANT1 and D3DVS_XBOX_RESERVEDCONSTANT2. In other words, all 192 constant registers are available when this bit is set. The restriction is that only #pragma SCREENSPACE vertex shaders can be used in this mode.
