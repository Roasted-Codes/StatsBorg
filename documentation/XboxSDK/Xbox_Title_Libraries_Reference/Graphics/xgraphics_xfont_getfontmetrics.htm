XFONT::GetFontMetrics

Gets the vertical metrics for the current font settings.

void __stdcall GetFontMetrics(
  unsigned *puCellHeight,
  unsigned *puDescent
);

Parameters

puCellHeight
[out] Pointer to the returned height of the text cell, measured in pixels.
puDescent
[out] Pointer to the returned height of the descent (units below the baseline), measured in pixels.

Return Values

None.

Remarks

The descent is defined as the amount that a lower case 'g' hangs below the baseline.

Requirements

  Import Library: Use xgraphics.lib
