XGMatrixPerspectiveFovLH

Builds a left-handed perspective projection matrix based on a field of view (FOV).

XGMATRIX* XGMatrixPerspectiveFovLH(
  XGMATRIX* pOut,
  FLOAT fovy,
  FLOAT Aspect,
  FLOAT zn,
  FLOAT zf
);

Parameters

pOut
[in, out] Pointer to the XGMATRIX structure that is the result of the operation.
fovy
[in] Field of view, in radians.
Aspect
[in] Aspect ratio.
zn
[in] Z-value of the near view-plane.
zf
[in] Z-value of the far view-plane.

Return Values

Pointer to a XGMATRIX structure that is a left-handed perspective projection matrix.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGMatrixPerspectiveFovLH function can be used as a parameter for another function.

Requirements

  Import Library: Use xgraphics.lib.
