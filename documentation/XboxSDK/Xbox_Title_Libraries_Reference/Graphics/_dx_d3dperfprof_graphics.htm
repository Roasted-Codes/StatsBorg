D3DPERFPROF

Helps to define the bit-mask history displayed in the m_ProfileData member of D3DPERF.

typedef enum _D3DPERFPROF
{
    D3DPERF_PROF_GPU,
    D3DPERF_PROF_FE,
    D3DPERF_PROF_BE,
    D3DPERF_PROF_MAX
} D3DPERFPROF;

Constants

D3DPERF_PROF_GPU
Indicates the graphics processing unit (GPU) is busy.
D3DPERF_PROF_FE
Indicates that the front end is busy.
D3DPERF_PROF_BE
Indicates that the back end is busy.
D3DPERF_PROF_MAX
Total number of performance profiler masks.
