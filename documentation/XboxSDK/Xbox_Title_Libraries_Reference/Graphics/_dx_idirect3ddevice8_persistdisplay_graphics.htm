IDirect3DDevice8::PersistDisplay

Saves screen contents across quick reboots and device resets.

HRESULT WINAPI PersistDisplay();

Parameters

None.

Return Values

If the method succeeds, the return value is S_OK.

If the device has already been uninitialized, the return value is E_FAIL.

Remarks

This function will persist the contents of the display which was drawn by the last call to IDirect3DDevice8::Present. The function will wait for the last Present to be processed before returning.

To use this function, a title should

  1. Present a transition frame by calling IDirect3DDevice8::Present.
  2. Call IDirect3DDevice8::PersistDisplay.
  3. Call XLaunchNewImage with the appropriate parameters to quick reboot the system, or call IDirect3DDevice8::Reset with new presentation parameters.

PersistDisplay will allocate enough memory to copy the surface and that memory will be freed on the first subsequent call to IDirect3DDevice8::Present.
