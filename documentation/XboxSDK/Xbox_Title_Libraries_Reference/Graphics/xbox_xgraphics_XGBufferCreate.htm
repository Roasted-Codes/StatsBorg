XGBufferCreate

Creates a buffer of a specified size.

HRESULT XGBufferCreate(
  DWORD numBytes,
  LPXGBUFFER* ppBuffer
);

Parameters

numBytes
[in] The size of the buffer in BYTES.
ppBuffer
[out] A pointer to the created buffer.

Return Values

If this method succeeds, it returns S_OK.

Requirements

  Header: Declared in xgraphics.h.
  Import Library: Use xgraphics.lib.
