XGVec3Normalize

Returns the normalized version of a 3D vector.

XGVECTOR3* XGVec3Normalize(
  XGVECTOR3* pOut,
  CONST XGVECTOR3* pV
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR3 structure that is the result of the operation.
pV
[in] Pointer to the source XGVECTOR3 structure.

Return Values

Pointer to a XGVECTOR3 structure that is the normalized version of the specified vector.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec3Normalize function can be used as a parameter for another function.

Requirements

  Header: Declared in xgmath.h.
  Import Library: Use xgraphics.lib.
