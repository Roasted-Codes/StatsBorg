IDirect3DDevice8::GetScreenSpaceOffset

Gets the screen-space offset which was set by IDirect3DDevice8::SetScreenSpaceOffset

HRESULT GetScreenSpaceOffset(
  float *pX,
  float *pY
);

Parameters

pX
[out] Pointer to a float that will receive the horizontal offset.
pY
[out] Pointer to a float that will receive the vertical offset.

Return Values

If the method succeeds, the return value is S_OK.

Requirements

  Header: Declared in D3d8.h.
  Import Library: Use D3d8.lib.
