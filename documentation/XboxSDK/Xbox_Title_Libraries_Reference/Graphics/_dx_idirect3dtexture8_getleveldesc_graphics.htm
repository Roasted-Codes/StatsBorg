IDirect3DTexture8::GetLevelDesc

Retrieves a level description of a texture resource.

HRESULT GetLevelDesc(
  UINT Level,
  D3DSURFACE_DESC* pDesc
);

Parameters

Level
[in] The texture level for which to get the description.
pDesc
[out] Pointer to a D3DSURFACE_DESC structure.

Return Values

The method returns D3D_OK.
