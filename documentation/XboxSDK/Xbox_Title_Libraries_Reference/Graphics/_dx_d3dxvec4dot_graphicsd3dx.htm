D3DXVec4Dot

Determines the dot-product of two 4-D vectors.

FLOAT D3DXVec4Dot(
  CONST D3DXVECTOR4* pV1,
  CONST D3DXVECTOR4* pV2
);

Parameters

pV1
[in] Pointer to a source D3DXVECTOR4 structure.
pV2
[in] Pointer to a source D3DXVECTOR4 structure.

Return Values

The dot-product.
