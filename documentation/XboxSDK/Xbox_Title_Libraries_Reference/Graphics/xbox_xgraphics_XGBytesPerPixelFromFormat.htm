XGBytesPerPixelFromFormat

Returns the bytes per texel of a format.

DWORD XGBytesPerPixelFromFormat(
  D3DFORMAT Format
);

Parameters

Format
The format to be checked.

Return Values

If this method succeeds, it returns the number of BYTES per texel for the provided format.

Requirements
