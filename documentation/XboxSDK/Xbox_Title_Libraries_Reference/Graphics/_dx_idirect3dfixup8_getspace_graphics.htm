IDirect3DFixup8::GetSpace

Returns the amount of free space remaining in the fix-up object.

HRESULT GetSpace(
  DWORD *pSpace
);

Parameters

pSpace
[out, retval] Pointer to a DWORD value containing the size in bytes of the free space remaining in the fix-up object.

Return Values

If the method succeeds, the return value is S_OK.

Requirements
