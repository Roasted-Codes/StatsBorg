IDirect3DPushBuffer8::EndFixup

Ends a fix-up bracket begun with IDirect3DPushBuffer8::BeginFixup.

HRESULT EndFixup();

Parameters

None.

Return Values

If this method succeeds, it returns S_OK. The method will return D3DERR_BUFFERTOOSMALL if a fix-up object was created and there wasn't enough room to record all of the fix-ups.
