IDirect3DDevice8::GetShaderConstantMode

Retrieves the constant register mode for the current vertex shader.

HRESULT GetShaderConstantMode(
  D3DSHADERCONSTANTMODE* pMode

);

Parameters

pMode
[out, retval] A pointer to a DWORD value consisting of an ORing of one or more values of the D3DSHADERCONSTANTMODE type. This specifies number of constant registers available.

Return Values

If the method succeeds, the return value is S_OK.

If the method fails, the return value can be D3DERR_INVALIDCALL.

Requirements

  Header: Declared in D3d8.h.
  Import Library: Use D3d8.lib.
