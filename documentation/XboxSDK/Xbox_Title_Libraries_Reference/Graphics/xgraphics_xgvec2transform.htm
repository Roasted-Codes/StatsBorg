XGVec2Transform

Transforms a 2D vector by a given matrix.

XGVECTOR4* XGVec2Transform(
  XGVECTOR4** pOut,
  CONST XGVECTOR2* pV,
  CONST XGMATRIX* pM
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR4 structure that is the result of the operation.
pV
[in] Pointer to the source XGVECTOR2 structure.
pM
[in] Pointer to the source XGMATRIX structure.

Return Values

Pointer to a XGVECTOR4 structure that is the transformed vector.

Remarks

This function transforms the vector, pV (x, y, 0, 1), by the matrix, pM.

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec2Transform function can be used as a parameter for another function.

Requirements
