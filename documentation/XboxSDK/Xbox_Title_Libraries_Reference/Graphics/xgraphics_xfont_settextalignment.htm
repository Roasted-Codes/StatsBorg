XFONT::SetTextAlignment

Sets how coordinates passed to XFONT::TextOut are handled.

void __stdcall SetTextAlignment(
  unsigned uMode
);

Parameters

uMode
[in] Sets the part of the text to which the coordinates passed to XFONT::TextOut refer. The value must be one of the vertical alignment flags ORed with a horizontal alignment flag.
Vertical Alignment Flags Meaning
XFONT_BASELINE The y coordinate refers to the baseline of the text.
XFONT_BOTTOM The y coordinate refers to the bottom of the character cell.
XFONT_TOP The y coordinate refers to the top of the character cell.

Horizontal Alignment Flags Meaning
XFONT_CENTER The x coordinate refers to the center of the text.
XFONT_LEFT The x coordinate refers to the leftmost edge of the text.
XFONT_RIGHT The x coordinate refers to the rightmost edge of the text.

Return Values

None.

Remarks

The default alignment is XFONT_TOP | XFONT_LEFT.
