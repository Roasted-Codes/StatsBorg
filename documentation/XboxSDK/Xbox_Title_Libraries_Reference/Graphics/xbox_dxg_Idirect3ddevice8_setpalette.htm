IDirect3DDevice8::SetPalette

Loads a palette into a texture stage.

HRESULT SetPalette(
  DWORD Stage,
  D3DPalette* pPalette
);

Parameters

Stage
The texture stage in which to set this palette. Stage identifiers are zero-based. Devices can have up to four set textures, so the maximum value allowed for Stage is 3.
pPalette
The palette to set.

Return Values

The method returns D3D_OK.

Remarks

The palette will only be used if the D3DFORMAT for the texture set in the same stage is D3DFMT_P8.
