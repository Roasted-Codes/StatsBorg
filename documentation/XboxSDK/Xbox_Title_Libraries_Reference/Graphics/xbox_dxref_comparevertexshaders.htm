XGSUCode_CompareVertexShaders

Compares two vertex shaders to see if they produce equivalent results.

HRESULT WINAPI XGSUCode_CompareVertexShaders(
  LPCVOID pMicrocodeA, 
  LPCVOID pMicrocodeB,
  LPXGBUFFER *ppErrorLog
);

Parameters

pMicrocodeA
[in] Pointer to a vertex shader microcode program.
pMicrocodeB
[in] Pointer to a vertex shader microcode program.
ppErrorLog
[out] The differences between the two shader results are written here. Pass NULL if there is no need to see the log.

Return Values

The function returns S_OK if the two shaders produce equivalent results. If there was an internal error or if the two shaders fail to produce equivalent results, the function returns a failure code. If either microcode program is not a valid vertex shader, the result is undefined.

Requirements
