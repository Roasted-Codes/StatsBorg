D3D__SingleStepPusher

Causes Direct3D to block after every Direct3D function call.

BOOL D3D__SingleStepPusher;

Remarks

Set D3D__SingleStepPusher to TRUE to cause Direct3D to make an implicit call to IDirect3DDevice8::BlockUntilIdle after every Direct3D function call. This is useful for tracking unusual hardware crashes and hangs down to the problem function call.

Requirements

  Header: Declared in d3d8.h.

  Import Library: Use D3d8d.lib.
