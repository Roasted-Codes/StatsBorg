IDirect3DDevice8::GetOverscanColor

Gets the overscan color.

D3DCOLOR GetOverscanColor();

Parameters

None.

Return Values

Returns the overscan color.

Remarks

The overscan color can be set by calling IDirect3DDevice8::SetOverscanColor.
