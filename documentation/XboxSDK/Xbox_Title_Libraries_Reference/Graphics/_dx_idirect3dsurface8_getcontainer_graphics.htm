IDirect3DSurface8::GetContainer

Provides access to the parent cube texture or texture (mipmap) object, if this surface is a child level of a cube texture or a mipmap.

HRESULT GetContainer(
  IDirect3DBaseTexture8** ppContainer
);

Parameters

ppContainer
[out, retval] Address of a pointer to fill with the container pointer.

Return Values

If the method succeeds, the return value is D3D_OK.

If there is no container, the return value is E_FAIL.

Remarks

If the surface is created using IDirect3DDevice8::CreateImageSurface, IDirect3DDevice8::CreateRenderTarget or IDirect3DDevice8::CreateDepthStencilSurface then GetContainer will return NULL.
