XGVec3Cross

Determines the cross-product of two 3D vectors.

XGVECTOR3* XGVec3Cross(
  XGVECTOR3* pOut,
  CONST XGVECTOR3* pV1,
  CONST XGVECTOR3* pV2
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR3 structure that is the result of the operation.
pV1
[in] Pointer to a source XGVECTOR3 structure.
pV2
[in] Pointer to a source XGVECTOR3 structure.

Return Values

Pointer to a XGVECTOR3 structure that is the cross product of two 3D vectors.

Remarks

This function determines the cross-product with the following code.

    XGVECTOR3 v;
 
    v.x = pV1->y * pV2->z - pV1->z * pV2->y;
    v.y = pV1->z * pV2->x - pV1->x * pV2->z;
    v.z = pV1->x * pV2->y - pV1->y * pV2->x;
 
    *pOut = v;

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec3Cross function can be used as a parameter for another function.
