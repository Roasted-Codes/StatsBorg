IDirect3DDevice8::GetProjectionViewportMatrix

Gets a copy of the current projection/viewport.

HRESULT GetProjectionViewportMatrix(
  D3DMATRIX *pProjectionViewport
);

Parameters

pProjectionViewport
[out] Pointer to a D3DMATRIX structure containing the current projection/viewport for use with IDirect3DDevice8::SetModelView.

Return Values

If the method succeeds, the return value is S_OK.

Remarks

This matrix will change whenever IDirect3DDevice8::SetViewport, IDirect3DDevice8::SetTransform (with D3DTS_PROJECTION), or IDirect3DDevice8::SetRenderState (with either D3DRS_ZENABLE or D3DRS_MULTISAMPLETYPE) is called.
