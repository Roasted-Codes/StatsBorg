IDirect3DDevice8::SelectVertexShader

Selects the vertex shader to execute on the GPU.

HRESULT SelectVertexShader(
  DWORD Handle,
  DWORD Address
);

Parameters

Handle
The handle of the vertex shader to execute. This value may be NULL. Handle cannot be NULL when switching from the fixed-function pipeline to a vertex shader program.
Address
The start location on the GPU of the vertex shader, in the range 0 through 135.

Return Values

If the method succeeds, the return value is D3D_OK.

If the method fails, the return value can be D3DERR_INVALIDCALL.

Remarks

The GPU used by the Xbox supports the loading of multiple vertex shaders into a set of 136 available addresses on the chip. SelectVertexShader is used to select which of the currently loaded vertex shader programs, as loaded by IDirect3DDevice8::LoadVertexShader, to use as the current vertex shader program.

The vertex shader handle is used to program the hardware streams registers to the correct input data stream as required by the vertex shader program. If the newly selected vertex shader program uses the same input data stream format as the previously selected vertex shader program, Handle can be passed as NULL in order to avoid the redundant reprogramming of the hardware streams registers.

The vertex format of the selected shader may be overridden if SetVertexShaderInputDirect was previously called.
