D3DFIELDTYPE

Describes the display field type.

typedef enum _D3DFIELDTYPE {
    D3DFIELD_ODD            = 1,
    D3DFIELD_EVEN           = 2,
    D3DFIELD_PROGRESSIVE    = 3,
    D3DFIELD_FORCE_DWORD    = 0x7fffffff,
} D3DFIELDTYPE;

Constants

D3DFIELD_ODD
The TV display field is odd.
D3DFIELD_EVEN
The TV display field is even.
D3DFIELD_PROGRESSIVE
The output is to a monitor.
D3DFIELD_FORCE_DWORD
Forces this enumeration to compile to 32 bits in size. This value is not used.
