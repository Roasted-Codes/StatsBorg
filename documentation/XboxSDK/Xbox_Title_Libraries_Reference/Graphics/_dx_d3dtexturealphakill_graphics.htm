D3DTEXTUREALPHAKILL

typedef enum _D3DTEXTUREALPHAKILL { // Xbox extension
  D3DTALPHAKILL_DISABLE = 0,
  D3DTALPHAKILL_ENABLE = 4,
  D3DTALPHAKILL_FORCE_DWORD = 0x7fffffff
} D3DTEXTUREALPHAKILL;

Constants

D3DTALPHAKILL_DISABLE
Disables Alphakill.
D3DTALPHAKILL_ENABLE
Enables Alphakill.
D3DTALPHAKILL_FORCE_DWORD
Forces this enumeration to compile to 32 bits in size. This value is not used.
