IDirect3DDevice8::GetGammaRamp

Retrieves the gamma correction ramp.

void GetGammaRamp(
  D3DGAMMARAMP  *pRamp
);

Parameters

pRamp
[in, out] Pointer to an application-supplied D3DGAMMARAMP structure to fill with the gamma correction ramp.

Return Values

None.

Remarks

This function will return the gamma ramp set by the last IDirect3DDevice8::SetGammaRamp or the default gamma ramp of the device.
