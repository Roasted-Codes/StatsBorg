Combiner-Related Fields

The following sections provide a description of the combiner-related fields of D3DPIXELSHADERDEF.

The following controls the number of combiner stages as well as how the c0 and c1 constants are handled for each stage.

  • PSCombinerCount

The following fields are present in each of the eight combiner stages.

  • PSRGBInputs[0-7]
  • PSAlphaInputs[0-7]
  • PRSRGBOutputs[0-7]
  • PSAlphaOutputs[0-7]

Each combiner stage is divided into an RGB portion and an alpha portion that operate in parallel. Inputs and outputs are specified separately for each portion of the stage. Each portion has four inputs and up to three outputs.
