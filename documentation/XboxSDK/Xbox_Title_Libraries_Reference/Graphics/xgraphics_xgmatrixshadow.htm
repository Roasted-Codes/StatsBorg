XGMatrixShadow

Builds a matrix that flattens geometry into a plane.

XGMATRIX* XGMatrixShadow(
  XGMATRIX* pOut,
  CONST XGVECTOR4* pLight,
  CONST XGPLANE* pPlane
);

Parameters

pOut
[in, out] Pointer to the XGMATRIX structure that is the result of the operation.
pLight
[in] Pointer to a XGVECTOR4 structure describing the light's position.
pPlane
[in] Pointer to the source XGPLANE structure.

Return Values

Pointer to a XGMATRIX structure that flattens geometry into a plane.

Remarks

The XGMatrixShadow function flattens geometry into a plane, as if casting a shadow from a light.

The return value for this function is the same value returned in the pOut parameter. In this way, the XGMatrixShadow function can be used as a parameter for another function.

Requirements
