IDirect3DDevice8::GetPixelShaderFunction

Retrieves the pixel shader function.

HRESULT GetPixelShaderFunction(
   DWORD Handle,
   D3DPIXELSHADERDEF* pData
);

Parameters

Handle
[in] Handle to the referred to pixel shader.
pData
[out] Pointer to a D3DPIXELSHADERDEF structure that, on the successful return of the function, will contain the pixel shader referred to by the Handle.

Return Values

If the method succeeds, the return value is S_OK.

Requirements

  Header: Declared in D3d8.h.
  Import Library: Use D3d8.lib.
