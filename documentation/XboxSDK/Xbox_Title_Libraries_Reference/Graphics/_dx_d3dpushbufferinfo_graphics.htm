D3DPUSHBUFFERINFO

Holds information on the push-buffer since the last call to D3DPERF_Reset.

typedef struct _D3DPUSHBUFFERINFO
{
    DWORD PushBufferSize;
    DWORD PushSegmentSize;
    DWORD PushSegmentCount;

    DWORD *pPushBase;
    DWORD *pPushLimit;

    ULONGLONG PushBufferBytesWritten;

} D3DPUSHBUFFERINFO;

Members

PushBufferSize
Size of the push-buffer in bytes.
PushSegmentSize
Size of the push-buffer segment in bytes.
PushSegmentCount
The maximum number of active segments. This value is the size of the push-buffer divided by the KickOffSize.
pPushBase
Pointer to the start of the memory for the push-buffer.
pPushLimit
Pointer to the first DWORD value following the end of the push-buffer.
PushBufferBytesWritten
Estimate of number of bytes written to the push-buffer since last call of D3DPERF_Reset.

Remarks

This structure is generally filled using the function D3DPERF_GetPushBufferInfo.
