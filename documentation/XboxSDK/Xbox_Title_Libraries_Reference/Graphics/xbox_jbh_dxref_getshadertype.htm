XGSUCode_GetVertexShaderLength

Examines shader microcode and determines the type of shader.

DWORD WINAPI XGSUCode_GetVertexShaderLength(LPCVOID pMicrocode);

Parameters

pMicrocode
A pointer to the microcode.

Return Values

Returns the type of shader that is pointed to by pMicrocode. Due to implementation restrictions, the SASMT_SCREENSPACE bit will not be set, even if the microcode is for a screen space shader.

If the microcode is invalid, the result is undefined.

Requirements
