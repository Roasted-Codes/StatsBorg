IDirect3DResource8::Register

Initializes the Data field of a manually allocated resource.

VOID Register(
  void* pBase
);

Parameters

pBase
[in] Address to the base of the data allocation.

Return Values

None.

Remarks

This method takes pBase, adds the current value of the resource's Data field to it, maps it to a physical memory address and stores that back into the Data field. This method should be used to initialize any resource that has not been created by one of the IDirect3DDevice8::Create methods.

This method does not modify any state except for the Data field in the resource so it is safe to call this method from any thread regardless of whether another thread is currently using the device.
