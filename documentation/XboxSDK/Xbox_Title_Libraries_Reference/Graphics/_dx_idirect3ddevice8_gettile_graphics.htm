IDirect3DDevice8::GetTile

Queries a tile configuration. Tiles are GPU views that significantly improve rendering speed for render targets and depth buffers by reorganizing memory to improve bandwidth, and by allowing depth buffers to be compressed.

HRESULT WINAPI GetTile(
  DWORD Index, 
  D3DTILE *pTile
);

Parameters

Index
[in] Index is the zero-based index of the tile to be used. There are a total of D3DTILE_MAXTILES (8) tiles available. Tiles 0 and 1 are reserved by Direct3D for the primary render target and primary depth buffer.
pTile
[out] Pointer to a D3DTILE structure, describing the tile at the given Index. On return, pTile->pMemory is NULL if the corresponding tile is currently disabled.

Return Values

If the method succeeds, the return value is S_OK.
