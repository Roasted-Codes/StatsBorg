D3DSWAPDATA

Describes the current swap.

typedef struct _D3DSWAPDATA {  // Xbox extension
    DWORD           Swap;
    DWORD           SwapVBlank;
    DWORD           MissedVBlanks;
    DWORD           TimeUntilSwapVBlank;
    DWORD           TimeBetweenSwapVBlanks;
} D3DSWAPDATA;

Members

Swap
The count of the current swap.
SwapVBlank
The vertical blank count at which the current swap will be made visible.
MissedVBlanks
The number of missed vertical blanks.
TimeUntilSwapVBlank
The approximate time until the next vertical blank at which the swap will be made visible.
TimeBetweenSwapVBlanks
The expected time between successive vertical blanks in which swaps were made visible. If using NTSC, this should correspond to 1/60 seconds with D3DPRESENT_INTERVAL_ONE, and 2/60 seconds with D3DPRESENT_INTERVAL_TWO.

Remarks

This structure is used by the callback function set on the device by IDirect3DDevice8::SetSwapCallback.

The time values reported in TimeUntilSwapVBlank and TimeBetweenSwapVBlanks are in 'rdtsc' units.
