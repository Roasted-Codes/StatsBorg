IDirect3DDevice8::GetPersistedSurface

Retrieves a surface that holds the bits being displayed after a call to IDirect3DDevice8::PersistDisplay.

HRESULT GetPersistedSurface(
   IDirect3DSurface8 **ppSurface
);

Parameters

ppSurface
[out, retval] Address of a pointer to an IDirect3DSurface8 interface, representing the returned persistent surface.

Return Values

If the method succeeds, the return value is D3D_OK.

If PersistDisplay has not been called since the last Present call, there is no persistent surface and the pointer returned is NULL.

The persistent surface exists only until the next call to IDirect3DDevice8::Present. The pointer returned from GetPersistentSurface is no longer valid after IDirect3DDevice8::Present has been called.

The persistent surface should not be modified in any way. The graphics hardware refreshes the display directly from this buffer, and any attempt to modify its contents may result in display tearing or other problems.

Note  Unlike other Direct3D functions such as IDirect3DDevice8::GetBackBuffer, calling this method will not increase the internal reference count on the returned IDirect3DSurface8 interface. The persistent surface is not reference-counted and only exists until the next call to Present.

Requirements

  Header: Declared in D3d8.h.
  Import Library: Use D3d8.lib.
