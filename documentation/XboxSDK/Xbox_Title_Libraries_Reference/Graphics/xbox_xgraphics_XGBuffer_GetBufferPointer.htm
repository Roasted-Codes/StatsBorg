XGBuffer_GetBufferPointer

Retrieves a pointer to a buffer.

LPVOID XGBuffer_GetBufferPointer(
  XGBuffer* pThis
);

Parameters

pThis
[in, out] A pointer to a valid XGBuffer structure.

Return Values

If this method succeeds, it returns the pointer to a buffer.
