IDirect3DDevice8::SetVertexDataColor

Sets the initial values of a given vertex attribute. This function sets persistent vertex attributes that may be used in vertex shaders and Begin/End blocks.

HRESULT IDirect3DDevice8::SetVertexDataColor(
   INT Register,
   D3DCOLOR Color
);

Parameters

Register
[in] An integer representing the slot number of the vertex attribute to be set. The following constants may be used for this parameter.
D3DVSDE_POSITION
Register 0.
D3DVSDE_BLENDWEIGHT
Register 1.
D3DVSDE_NORMAL
Register 2.
D3DVSDE_DIFFUSE
Register 3.
D3DVSDE_SPECULAR
Register 4.
D3DVSDE_FOG
Register 5. Xbox extension.
D3DVSDE_BACKDIFFUSE
Register 7.
D3DVSDE_BACKSPECULAR
Register 8.
D3DVSDE_TEXCOORD0
Register 9.
D3DVSDE_TEXCOORD1
Register 10.
D3DVSDE_TEXCOORD2
Register 11.
D3DVSDE_TEXCOORD3
Register 12.
Color
[in] A D3DCOLOR value used to specify the value assigned to the register.

Return Values

If the method succeeds, the return value is D3D_OK.

Remarks

Assuming you have pDevice, a valid pointer to an IDirect3DDevice8 interface, the following two lines are equivalent: 

pDevice->SetVertexData4ub( register, r, g, b, a );
pDevice->SetVertexDataColor( register, D3DCOLOR_ARGB( a, r, g, b ) );

Vertex attributes will remain set unless overwritten by the vertex buffer data stream.

Requirements

  Import Library: Use D3d8.lib.
