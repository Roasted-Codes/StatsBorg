XGUnswizzleBox

Unswizzles a box from a texture into a buffer.

void WINAPI XGUnswizzleBox(
  LPCVOID pSource,
  DWORD Width,
  DWORD Height,
  DWORD Depth,
  D3DBOX *pBox,
  LPVOID pDest,
  DWORD RowPitch,
  DWORD SlicePitch, 
  XGPOINT3D *pPoint,
  DWORD BytesPerPixel
);

Parameters

pSource
The source texture.
Width
The width of the entire source texture.
Height
The height of the entire source texture.
Depth
The depth of the entire destination texture.
pBox
The rectangle within the texture to copy.
pDest
The destination buffer.
RowPitch
Byte offset from the left edge of one row to the left edge of the next row.
SlicePitch
Byte offset from the top-left of one slice to the top-left of the next deepest slice.
pPoint
Where to copy the rectangle to.
BytesPerPixel
The number of bytes per pixel.

Return Values

None.

Requirements

  Header: Declared in xgraphics.h.
  Import Library: Use xgraphics.lib
