IDirect3DDevice8::GetScissors

Retrieves information about the current scissor rectangles.

HRESULT WINAPI GetScissors(
  DWORD *pCount, 
  BOOL *pExclusive, 
  D3DRECT *pRects
);

Parameters

pCount
[out] Pointer to a DWORD value that contains the number of scissoring rectangles. This is the size of the array pointed to by pRects and must be less than D3DSCISSORS_MAX.
pExclusive
[out] Pointer to a BOOL value that if TRUE, a pixel will be killed if it falls into any of the rectangles in the array pointed to by pRects. If the value pointed to is FALSE, a pixel will be killed only if does not fall into any of the rectangles pointed to by pRects.
pRects
[out] Pointer to an array of D3DRECT structures.

Return Values

If the method succeeds, the return value is S_OK.

Requirements
