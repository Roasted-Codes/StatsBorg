XGVec4Subtract

Subtracts two 4-D vectors.

XGVECTOR4* XGVec4Subtract(
  XGVECTOR4* pOut,
  CONST XGVECTOR4* pV1,
  CONST XGVECTOR4* pV2
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR4 structure that is the result of the operation.
pV1
[in] Pointer to a source XGVECTOR4 structure.
pV2
[in] Pointer to a source XGVECTOR4 structure.

Return Values

Pointer to a XGVECTOR4 structure that is the difference of two vectors.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec4Subtract function can be used as a parameter for another function.

Requirements

  Import Library: Use xgraphics.lib.
