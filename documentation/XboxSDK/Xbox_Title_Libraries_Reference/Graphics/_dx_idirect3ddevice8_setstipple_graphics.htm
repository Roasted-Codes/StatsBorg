IDirect3DDevice8::SetStipple

Sets a stipple pattern used when drawing polygons (when D3DPRIMITIVETYPE is D3DPT_POLYGON.)

void SetStipple(
    CONST DWORD *pPattern
);

Parameters

pPattern
[in] Pointer to 32 DWORDs which determine the 32×32 bit polygon stipple pattern.

Return Values

None.

Remarks

When D3DRS_STIPPLEENABLE is set to TRUE, and the primitive being drawn is a polygon, the polygon stipple is indexed using the low five bits of the screen-space X and Y coordinates. If the indexed bit is zero, the pixel is killed.

Requirements

Header: Declared in D3d8.h.

Import Library: Use D3d8.lib.
