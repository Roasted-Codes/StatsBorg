XGPlaneFromPoints

Constructs a plane from three points.

XGPLANE* XGPlaneFromPoints(
  XGPLANE* pOut,
  CONST XGVECTOR3* pV1,
  CONST XGVECTOR3* pV2,
  CONST XGVECTOR3* pV3
);

Parameters

pOut
[in, out] Pointer to the XGPLANE structure that is the result of the operation.
pV1
[in] Pointer to a XGVECTOR3 structure, defining one of the points used to construct the plane.
pV2
[in] Pointer to a XGVECTOR3 structure, defining one of the points used to construct the plane.
pV3
[in] Pointer to a XGVECTOR3 structure, defining one of the points used to construct the plane.

Return Values

Pointer to the XGPLANE structure constructed from the given points.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGPlaneFromPoints function can be used as a parameter for another function.

Requirements
