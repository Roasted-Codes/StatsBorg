D3DPIXELSHADERDEF_FILE

Describes the file structure of a pre-compiled pixel shader program as generated by xsasm.

typedef struct _D3DPixelShaderDefFile
{
    DWORD               FileID; 
    D3DPIXELSHADERDEF   Psd;
} D3DPIXELSHADERDEF_FILE;

Members

FileID
Uniquely identifies the file as a pixel shader binary. For a valid pixel shader, this should be the value D3DPIXELSHADERDEF_FILE_ID.
Psd
A structure containing a directly defined pixel shader definition. For more information, see either Directly Defined Pixel Shaders or D3DPIXELSHADERDEF.

Remarks

The following program demonstrates how this structure may be used.

// Open the pixel shader file
HANDLE hFile = CreateFile( "MyShader.xpu", GENERIC_READ, FILE_SHARE_READ, 
                           NULL, OPEN_EXISTING, 0, NULL );
if( INVALID_HANDLE_VALUE == hFile )
  return E_FAIL;

// Load the pre-compiled pixel shader microcode
D3DPIXELSHADERDEF_FILE psdf;
DWORD dwBytesRead;
ReadFile( hFile, &psdf, sizeof(D3DPIXELSHADERDEF_FILE), &dwBytesRead, NULL );
CloseHandle( hFile );
