IDirect3DTexture8::UnlockRect

This function does nothing. It has been provided to ease in the porting of code from DirectX 8.0 to DirectX for the Xbox.

HRESULT UnlockRect(
  UINT Level
);

Parameters

Level
[in] Specifies the level of the texture to unlock.

Return Values

The method returns D3D_OK.
