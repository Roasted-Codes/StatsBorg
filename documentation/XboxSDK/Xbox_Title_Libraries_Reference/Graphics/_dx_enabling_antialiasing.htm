Enabling Antialiasing

Antialiasing is enabled by specifying something other than D3DMULTISAMPLE_NONE for the MultiSampleType member of D3DPRESENT_PARAMETERS when creating or resetting the device.

// This sample assumes that pD3D is a valid pointer to an IDirect3D8
// interface and the pDevice is a pointer to an IDirect3DDevice8
// interface.

// Set up the structure used to create the device.
D3DPRESENT_PARAMETERS d3dPP;
ZeroMemory( &d3dPP, sizeof(d3dPP) );
d3dPP.BackBufferWidth       = 640;
d3dPP.BackBufferHeight      = 480;
d3dPP.BackBufferFormat      = D3DFMT_X8R8G8B8;
d3dPP.BackBufferCount       = 1;
d3dPP.MultiSampleType       = D3DMULTISAMPLE_4_SAMPLES_MULTISAMPLE_LINEAR;
d3dPP.SwapEffect            = D3DSWAPEFFECT_DISCARD;

// Create the device.
if ( FAILED( pD3D->CreateDevice( 0, D3DDEVTYPE_HAL, NULL,
                                 D3DCREATE_HARDWARE_VERTEXPROCESSING,
                                 &d3dPP, &pDevice ) ) )
     return E_FAIL;

By setting the MultiSampleType member to be a multisampling type (as opposed to a supersampling type), multisampling is enabled. This is unlike the behavior under DirectX 8.0 for the PC where it is necessary to turn on multisampling using the render state D3DRS_MULTISAMPLEANTIALIAS.
