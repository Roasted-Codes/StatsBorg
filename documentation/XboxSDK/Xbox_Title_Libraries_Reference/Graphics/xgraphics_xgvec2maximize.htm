XGVec2Maximize

Returns a 2D vector that is made up of the largest components of two 2D vectors.

XGVECTOR2* XGVec2Maximize(
  XGVECTOR2* pOut,
  CONST XGVECTOR2* pV1,
  CONST XGVECTOR2* pV2
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR2 structure that is the result of the operation.
pV1
[in] Pointer to a source XGVECTOR2 structure.
pV2
[in] Pointer to a source XGVECTOR2 structure.

Return Values

Pointer to a XGVECTOR2 structure that is made up of the largest components of the two vectors.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec2Maximize function can be used as a parameter for another function.
