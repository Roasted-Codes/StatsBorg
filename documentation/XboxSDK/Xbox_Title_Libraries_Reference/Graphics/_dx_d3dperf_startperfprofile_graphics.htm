D3DPERF_StartPerfProfile

Starts the graphics processing unit (GPU) profiler.

BOOL WINAPI D3DPERF_StartPerfProfile();

Parameters

None.

Return Values

If the function successfully starts the profiling, it will return TRUE; otherwise, it returns FALSE.

Remarks

Enabling the profiler will cause an interupt every 100 ns, which checks the busy/idle status of the GPU.

If the Intel VTune Performance Analyzer for Xbox is enabled, the GPU profiler's timing may be inaccurate.

Requirements

  Header: Declared in d3d8perf.h.
  Import Library: Use d3d8i.lib or d3d8d.lib.
