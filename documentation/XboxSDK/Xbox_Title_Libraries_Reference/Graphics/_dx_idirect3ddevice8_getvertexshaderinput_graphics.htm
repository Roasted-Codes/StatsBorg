IDirect3DDevice8::GetVertexShaderInput

Gets the byte offset from the start of the vertex buffer data. It is used to check the values that were set by IDirect3DDevice8::SetVertexShaderInput.

HRESULT GetVertexShaderInput(
  DWORD *pHandle,
  UINT *pStreamCount,
  D3DSTREAM_INPUT *pStreamInputs
);

Parameters

pHandle
[out] Pointer to a DWORD value, holding the handle of the vertex shader last specified in the call to IDirect3DDevice8::SetVertexShaderInput.
pStreamCount
[out] Pointer to a UINT value, holding the stream count as specified in the last call to IDirect3DDevice::SetVertexShaderInput.
pStreamInputs
[out] Pointer to an array of D3DSTREAM_INPUT structures used to describe the vertex buffers and the offsets from the starts of those vertex buffers as they were specified in the last call to IDirect3DDevice8::SetVertexShaderInput. This call does not check pStreamCount to see if there is enough room.

Return Values

If IDirect3DDevice8::SetVertexShaderInput currently has control of the GPU's stream registers, the method returns S_OK; otherwise it returns S_FALSE.

Remarks

If using this method just for its return value, you can pass NULL for each of its parameters.
