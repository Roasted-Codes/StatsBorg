XGIsSwizzledFormat

Returns whether a texture format is swizzled or not.

BOOL XGIsSwizzledFormat(
  D3DFORMAT Format
);

Parameters

Format
The format to be checked.

Return Values

If the specified format is swizzled, this method returns TRUE. Otherwise, this method returns FALSE.
