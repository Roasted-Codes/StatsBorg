D3DTRANSFORMSTATETYPE

Defines constants that describe transformation state values. 

typedef enum _D3DTRANSFORMSTATETYPE {
    D3DTS_VIEW          = 0,
    D3DTS_PROJECTION    = 1,
    D3DTS_TEXTURE0      = 2,
    D3DTS_TEXTURE1      = 3,
    D3DTS_TEXTURE2      = 4,
    D3DTS_TEXTURE3      = 5,
    D3DTS_WORLD         = 6,
    D3DTS_WORLD1        = 7,
    D3DTS_WORLD2        = 8,
    D3DTS_WORLD3        = 9,

    D3DTS_MAX           = 10,
    D3DTS_FORCE_DWORD     = 0x7fffffff,
} D3DTRANSFORMSTATETYPE;

Constants

D3DTS_VIEW
Identifies the transformation matrix being set as the view transformation matrix. The default value is NULL (the identity matrix).
D3DTS_PROJECTION
Identifies the transformation matrix being set as the projection transformation matrix. The default value is NULL (the identity matrix).
D3DTS_TEXTURE0 through D3DTS_TEXTURE3
Identifies the transformation matrix being set for the specified texture stage.
D3DTS_WORLD
Identifies the transformation matrix being set as the world transformation matrix.
D3DTS_WORLD1
Identifies the transformation matrix that can be used to blend vertices. This matrix is used with the corresponding blending weight value specified in the vertex format.
D3DTS_WORLD2
Identifies the transformation matrix that can be used to blend vertices. This matrix is used with the corresponding blending weight value specified in the vertex format.
D3DTS_WORLD3
Identifies the transformation matrix that can be used to blend vertices. This matrix is used with the corresponding blending weight value specified in the vertex format.
D3DTS_MAX
This constant is not supported in this release of the Xbox video game system.
D3DTS_FORCE_DWORD
Forces this enumeration to compile to 32 bits in size. This value is not used.

Remarks

The transform states in the range 256 through 511 are reserved to store up to 256 matrices that can be indexed using 8-bit indices. The D3DTS_WORLDMATRIX macro can be used to map indices in the range 0 through 255 to the corresponding transform states.

Xbox Optimizations

The constants for this enumeration have been redefined. The DirectX 8.0 implementation was defined as follows:
