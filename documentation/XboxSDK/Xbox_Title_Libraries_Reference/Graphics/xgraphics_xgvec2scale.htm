XGVec2Scale

Scales a 2D vector.

XGVECTOR2* XGVec2Scale(
  XGVECTOR2* pOut,
  CONST XGVECTOR2* pV,
  FLOAT s
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR2 structure that is the result of the operation.
pV
[in] Pointer to the source XGVECTOR2 structure.
s
[in] Scaling value.

Return Values

Pointer to a XGVECTOR2 structure that is the scaled vector.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec2Scale function can be used as a parameter for another function.

Requirements

  Header: Declared in xgmath.h.
  Import Library: Use xgraphics.lib.
