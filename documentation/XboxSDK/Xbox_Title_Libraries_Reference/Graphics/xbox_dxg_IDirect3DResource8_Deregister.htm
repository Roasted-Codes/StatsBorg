IDirect3DResource8::BlockUntilNotBusy

Blocks until the graphics processor is no longer using this resource.

VOID BlockUntilNotBusy();

Parameters

None.

Return Values

None.

Remarks

Use this method to ensure that it is safe to destroy a manually allocated resource. The method will return only once the resource is no longer being used by the graphics processor, can be locked without blocking, and can be safely destroyed.

The title is responsible to ensure that any references to this resource from IDirect3DDevice8 have been released because such a situation will cause this method to block forever.
