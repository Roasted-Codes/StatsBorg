IDirect3DPushBuffer8::SetPalette

Used only within a IDirect3DPushBuffer8::BeginFixup/IDirect3DPushBuffer8::EndFixup bracket to modify the corresponding call to IDirect3DDevice8::SetPalette as it was recorded in the push-buffer. Loads a palette into a texture stage.

HRESULT SetPalette(
  DWORD Offset,
  DWORD Stage,
  D3DPalette* pPalette
);

Parameters

Offset
[in] The offset determined during the recording in the push-buffer as returned from IDirect3DDevice8::GetPushBufferOffset just before IDirect3DDevice8::SetPalette was recorded.
Stage
The texture stage in which to set this palette. Stage identifiers are zero-based. Devices can have up to four set textures, so the maximum value allowed for Stage is 3.
pPalette
The palette to set.

Return Values

The method returns D3D_OK.

Remarks

The palette will only be used if the D3DFORMAT for the texture set in the same stage is D3DFMT_P8.
