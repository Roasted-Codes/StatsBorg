IDirect3DDevice8::GetStipple

Returns the currently set polygon stipple pattern.

void GetStipple(
    DWORD *pPattern
);

Parameters

pPattern

[out] Address of 32 DWORDs that will receive the 32×32 bit polygon stipple pattern. The output pixels are determined by the following equation:
Pixel[x][y] = StipplePattern[x][31 - y]

where x is the bit position and 31 − y is the offset. The bits are stored in little-endian format.

Return Values

None.

Requirements

Header: Declared in D3d8.h.

Import Library: Use D3d8.lib.
