IDirect3DDevice8::GetVertexShaderType

Retrieves the type of the specified vertex shader.

HRESULT GetVertexShaderType(
  DWORD  Handle,
  DWORD  *pType
);

Parameters

Handle
[in] The handle of the vertex shader.
pType
[out] pointer to a DWORD value that contains the microcode type for the specified vertex shader. The following are valid values.
D3DSMT_VERTEXSHADER
An ordinary vertex shader.
D3DSMT_READWRITE_VERTEXSHADER
A vertex shader that can write to the constant registers.
D3DSMT_VERTEXSTATESHADER
A vertex state shader.

Return Values

If the method succeeds, the return value is D3D_OK.

If the method fails, the return value can be D3DERR_INVALIDCALL.

Requirements
