IDirect3DDevice8::EndPush

EndPush must be called after IDirect3DDevice8::BeginPush, once the data is completely written to the push-buffer.

HRESULT EndPush(
  DWORD *pPush
);

Parameters

pPush
[in] pPush should point to the first DWORD following the data written to the push-buffer.

Return Values

If the method succeeds, the return value is S_OK.

Remarks

For an example of how to use EndPush, see the remarks section of BeginPush.

After calling EndPush, the data written to the push-buffer may not be modified.

Requirements

  Import Library: Use D3d8.lib.
