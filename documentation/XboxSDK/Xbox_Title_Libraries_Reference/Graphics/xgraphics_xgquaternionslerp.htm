XGQuaternionSlerp

Interpolates between two quaternions, using spherical linear interpolation.

XGQUATERNION* XGQuaternionSlerp(
  XGQUATERNION* pOut,
  CONST XGQUATERNION* pQ1,
  CONST XGQUATERNION* pQ2,
  FLOAT t
);

Parameters

pOut
[in, out] Pointer to the XGQUATERNION structure that is the result of the operation.
pQ1
[in] Pointer to a source XGQUATERNION structure.
pQ2
[in] Pointer to a source XGQUATERNION structure.
t
[in] Parameter that indicates how far to interpolate between the quaternions.

Return Values

Pointer to a XGQUATERNION structure that is the result of the interpolation.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGQuaternionSlerp function can be used as a parameter for another function.

Requirements

  Header: Declared in xgmath.h.
  Import Library: Use xgraphics.lib.
