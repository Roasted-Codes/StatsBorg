D3DPERF_Reset

Resets all performance counters to zero.

void WINAPI D3DPERF_Reset();

Parameters

None.

Return Values

None.

Remarks

If the m_SetRenderStateOverridesOnReset member of D3DPERF is TRUE, those render states set to be overwritten by the m_RenderStateOverrides member of D3DPERF will be given their new render states and m_SetRenderStateOverridesOnReset will be set to FALSE.

Requirements

  Header: Declared in d3d8perf.h.
  Import Library: Use d3d8i.lib or d3d8d.lib.
