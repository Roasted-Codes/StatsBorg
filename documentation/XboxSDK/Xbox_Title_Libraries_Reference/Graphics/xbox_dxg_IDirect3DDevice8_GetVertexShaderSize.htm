IDirect3DDevice8::GetVertexShaderSize

Returns the size of the specified vertex shader.

HRESULT GetVertexShaderSize(
  DWORD Handle,
  UINT* pSize
);

Parameters

Handle
[in] Handle to the vertex shader. The value for this parameter can be the handle returned by IDirect3DDevice8::CreateVertexShader.

If the handle is NULL, the legacy pixel pipeline is used.

pSize
[out, retval] Pointer to a UINT value, holding the returned size of the vertex shader.

Return Values

If the method succeeds, the return value is D3D_OK.

If the method fails, the return value can be D3DERR_INVALIDCALL.

Remarks

GetVertexShaderSize can be used in the process of mapping multiple vertex shaders to the available slots (addresses) on the GPU.

The GPU used by the Xbox supports the loading of multiple vertex shaders into a set of 136 available addresses on the chip. Note however that the calling application must keep track of the allocation of addresses per vertex shader. No error checking for overwriting a previously allocated set of addresses is provided.
