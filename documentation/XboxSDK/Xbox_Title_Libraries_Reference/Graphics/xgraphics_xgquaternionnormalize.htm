XGQuaternionNormalize

Returns the normal of a quaternion.

XGQUATERNION* XGQuaternionNormalize(
  XGQUATERNION* pOut,
  CONST XGQUATERNION* pQ
);

Parameters

pOut
[in, out] Pointer to the XGQUATERNION structure that is the result of the operation.
pQ
[in] Pointer to the source XGQUATERNION structure.

Return Values

Pointer to a XGQUATERNION structure that is the normal of the quaternion.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGQuaternionNormalize function can be used as a parameter for another function.

Requirements

  Import Library: Use xgraphics.lib.
