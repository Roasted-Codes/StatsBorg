ID3DXTechnique::Validate

Validates the technique.

HRESULT Validate();

Return Values

If the method succeeds, the return value is S_OK.

If the method fails, the return value can be one of the following values.

D3DERR_INVALIDCALL
D3DXERR_INVALIDDATA
E_OUTOFMEMORY
