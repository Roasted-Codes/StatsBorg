XFONT::TextOut

Draws the text on a surface.

HRESULT __stdcall TextOut(
  IDirect3DSurface8 *pSurface,
  LPCWSTR wstr,
  unsigned cch,
  long x,
  long y
);

Parameters

pSurface
[in, out] Pointer to the IDirect3DSurface8 interface where the text should be rendered.
wstr
[in] Pointer to the string that will be rendered.
cch
[in] The length of the string. cch can be -1 if wstr points to a NULL-terminated string.
x
[in] The x-coordinate of the reference point used to align the string. See the Remarks section for more details.
y
[in] The y-coordinate of the reference point used to align the string. See the Remarks section for more details.

Return Values

On success, the function returns S_OK.

Remarks

XFONT::TextOut does not work on swizzled surfaces.

The meaning of the x and y coordinates specified in the parameters is determined by the alignment set using XFONT::SetTextAlignment. By default, the coordinates refer to the upper-left corner of the text cell.
