D3DPERF_SetWaitCycleTimeThreshold

Sets the threshold cycle count and handler function for hitting wait time thresholds.

__inline void WINAPI D3DPERF_SetWaitCycleTimeThreshold(
  PFNCycleThresholdHandler pfnCycleThresholdHandler,
  DWORD DumpWaitCycleTimesThreshold
);

Parameters

pfnCycleThresholdHandler
[in] Pointer to a function of type PFNCycleThresholdHandler that is called when hitting wait time thresholds. PFNCycleThresholdHandler is defined as follows: 
typedef void (WINAPI *PFNCycleThresholdHandler)(
  D3DPERF_PerformanceCounters perfctr,
  ULONGLONG Cycles
);
DumpWaitCycleTimesThreshold
[in] The number of waiting cycles that must be hit before calling the function pointed to by pfnCycleThresholdHandler.

Return Values

None.
