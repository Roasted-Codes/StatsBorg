D3DCOPYRECTOPERATION

Defines the operation that will be performed between the source and destination surfaces when using IDirect3DDevice8::CopyRects.

typedef enum _D3DCOPYRECTOPERATION    // Xbox extension
{
    D3DCOPYRECT_SRCCOPY_AND         = 0,
    D3DCOPYRECT_ROP_AND             = 1,
    D3DCOPYRECT_BLEND_AND           = 2,
    D3DCOPYRECT_SRCCOPY             = 3,
    D3DCOPYRECT_SRCCOPY_PREMULT     = 4,
    D3DCOPYRECT_BLEND_PREMULT       = 5,
    D3DCOPYRECT_FORCE_DWORD         = 0x7fffffff, /* force 32-bit size enum */
} D3DCOPYRECTOPERATION;

Constants

The equations in the constants below use the following:

Match = ColorKeyEnable &&
		ColorKeyValue.a != 0 &&
		ColorKeyValue.rgb == Source.rgb
Temp.argb = Match ? 0 : Source.argb

ColorKeyEnable and ColorKeyValue are refer to the members of the D3DCOPYRECTSTATE structure used in IDirect3DDevice8::SetCopyRectsState and IDirect3DDevice8::GetCopyRectsState.

D3DCOPYRECT_SRCCOPY_AND
Dest.argb = Temp.argb
Dest.writeenable = (Temp.a != 0 )
D3DCOPYRECT_ROP_AND
Dest.argb = ROP3(Rop, Pattern.argb, Temp.argb, Dest.argb)
Dest.writeenable = (Temp.a != 0) && (Pattern.a != 0)
D3DCOPYRECT_BLEND_AND
omega0 = ~(BlendAlpha * Temp.a)
omega1 = (BlendAlpha * Temp.a)
Dest.argb = omega0 * Dest.argb + omega1 * Temp.argb
BlendAlpha refers to the member of the D3DCOPYRECTSTATE structure used in SetCopyRectsState and GetCopyRectsState.
D3DCOPYRECT_SRCCOPY
Dest.argb = Temp.argb
D3DCOPYRECT_SRCCOPY_PREMULT
Dest.argb = (Temp.argb * PremultAlpha.argb)
PremultAlpha refers to the member of the D3DCOPYRECTSTATE structure used in SetCopyRectsState and GetCopyRectsState.
D3DCOPYRECT_BLEND_PREMULT
omega = ~(PremultAlpha.a * Temp.a)
Dest.argb = min(~0, omega * Dest.argb + PremultAlpha.argb * Temp.argb)
PremultAlpha refers to the member of the D3DCOPYRECTSTATE structure used in SetCopyRectsState and GetCopyRectsState.
D3DCOPYRECT_FORCE_DWORD
Forces this enumeration to 32 bits in size. This value is not used.

Remarks

The operation is specified in the Operation member of the D3DCOPYRECTSTATE structure used in IDirect3DDevice8::SetCopyRectsState and IDirect3DDevice8::GetCopyRectsState.
