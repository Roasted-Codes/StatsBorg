IDirect3DDevice8::GetOverlayUpdateStatus

Retrieves whether or not an overlay is pending display at the next vertical blank. 

BOOL GetOverlayUpdateStatus(
);

Parameters

None.

Return Values

This method returns FALSE if the overlay is still pending display at the next vertical blank. Otherwise, this method returns TRUE.
