IDirect3DDevice8::SetFlickerFilter

Adjusts the aggressiveness of the flicker filter.

void WINAPI SetFlickerFilter(
  DWORD Filter
);

Parameters

Filter
[in] Sets the filtering level, which must be between zero and five. Setting Filter to zero turns the flicker filter off, and setting it to five turns on the maximum level of filtering. By default, the flicker filter is on and is set at level five.

Return Values

None.

Remarks

Setting the flicker filter has no effect unless its called after the first call to either IDirect3DDevice8::Swap or IDirect3DDevice8::Present following a display mode change.
