D3DSTATENAME

Provides a human-readable name for an individual render state or texture state.

typedef struct _D3DSTATENAME
{
    DWORD m_State;
    char *m_Name;
} D3DSTATENAME;

Members

m_State
Member of either D3DTEXTURESTAGESTATETYPE or D3DRENDERSTATETYPE.
m_Name
Pointer to a NULL-terminated string containing the name of the texture or render state enumerated in m_State.
