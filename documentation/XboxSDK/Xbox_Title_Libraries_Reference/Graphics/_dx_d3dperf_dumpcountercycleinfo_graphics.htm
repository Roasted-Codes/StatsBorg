D3DPERF_DumpCounterCycleInfo

This is the default handler for the DumpWaitCycleTimesThreshold. It dumps information about PERF_OBJECTLOCK_WAITS.

void WINAPI D3DPERF_DumpCounterCycleInfo(
  D3DPERF_PerformanceCounters perfctr,
  ULONGLONG Cycles
);

Parameters

perfctr
[in] Member of the D3DPERF_PerformanceCounters enumeration. perfctr describes the performance counter about which cycle information is to be dumped.
Cycles
[in] The number of cycles since the last call to D3DPERF_Reset.

Return Values

None.

Remarks

This is the default function used to display cycle information. It is called automatically when the threshold cycle count is hit as was set in the m_DumpWaitCycleTimesThreshold member of D3DPERF.

This function displays the number of OBJECTLOCK_WAITS and the number of cycles and time (in milliseconds) since the last call to D3DPERF_RESET.

This function can be used as the basis for developing a customized handler. It is defined as below:

extern "C"
void WINAPI D3DPERF_DumpCounterCycleInfo(D3DPERF_PerformanceCounters perfctr,
  ULONGLONG Cycles)
{
  if(perfctr == PERF_OBJECTLOCK_WAITS)
  {
    DbgPrint("#%Lu) %s %I64u cycles (%Lums)\n",
      g_PerfCounters.m_PerformanceCounters[perfctr].Count,
      g_PerfNames.m_PerformanceCounterNames[perfctr],
      Cycles,
      Cycles / 733000);
    }
}
