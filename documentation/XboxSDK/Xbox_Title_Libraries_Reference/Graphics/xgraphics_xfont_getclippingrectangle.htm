XFONT::GetClippingRectangle

Gets the clipping rectangle used for the text cell.

D3DRECT *__stdcall GetClippingRectangle();

Parameters

None.

Return Values

If no clipping rectangle is set, the function returns NULL. Otherwise, it returns a pointer to a D3DRECT structure that describes the clipping rectangle.

Requirements

  Header: Declared in xfont.h
  Import Library: Use xgraphics.lib
