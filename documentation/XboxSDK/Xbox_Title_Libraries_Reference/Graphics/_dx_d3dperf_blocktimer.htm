D3DPERF_BlockTimer

Used for inserting user-defined performance events. When an instance of the structure is created, performance information starts getting spewed. The spew stops when the instance of the structure goes out of scope.

struct D3DPERF_BlockTimer 
{
    D3DPERFEvent *m_pD3DPerfEvent;
    BOOL          m_bRecordGpuTime;

    __forceinline D3DPERF_BlockTimer(char *pszHeaderString,
                                     BOOL  bRecordGpuTime,
                                     WORD  EventType = D3DPERFEvent_User0);
    __forceinline ~D3DPERF_BlockTimer();
};

Members

m_pD3DPerfEvent
Pointer to the D3DPERFEvent structure that will store information about the user-defined performance event.
m_bRecordGpuTime
TRUE to insert a fence for counting GPU times.
D3DPERF_BlockTimer
pszHeaderString is a NULL-terminated string used as the event header. bRecordTime should be TRUE if the title wants to insert fences that will be used to keep track of GPU times. EventType is an optional argument that is a member of the D3DPERFEventsType enumerated type. EventType describes the type of performance event to be recorded.

Remarks

The following code snippet shows how this structure can be used monitor performance in a function that updates a title's AI.

void UpdateAI()
{
   D3DPERF_BlockTimer MyTimer( "Updating AI", TRUE );
   
   // Handle updating the AI.
   // ...
   
   return;
   
   // When MyTimer goes out of scope, performance monitoring for this event
   // will automatically stop
}
