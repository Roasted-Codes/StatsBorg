Determining Video Settings

The video settings set in the dashboard determine what kind of device can be created using IDirect3D8::CreateDevice. There are two basic methods of determining these settings. One can use the XGetVideoFlags function which returns a DWORD consisting of the various video settings ORed together. Alternatively, since the valid video modes are influenced by the settings in the dashboard, one can directly determine the valid video modes by calling IDirect3D8::EnumAdapterModes. The number of valid video modes can be retrieved using IDirect3D::GetAdapterModeCount.
