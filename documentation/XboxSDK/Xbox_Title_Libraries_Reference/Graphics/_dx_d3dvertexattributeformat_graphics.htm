D3DVERTEXATTRIBUTEFORMAT

Used with IDirect3DDevice8::SelectVertexShaderDirect, IDirect3DDevice8::SetVertexShaderInputDirect, and IDirect3DPushBuffer8::SetVertexShaderInputDirect to describe each of the 16 possible vertex shader inputs.

typedef struct{
    D3DVERTEXSHADERINPUT Input[16];
} D3DVERTEXATTRIBUTEFORMAT;

Members

Input
Array of D3DVERTEXSHADERINPUT structures describing each possible input. Unused inputs need to have the Format member of D3DVERTEXSHADERINPUT set to D3DVSDT_NONE.

Requirements

Header: Declared in D3dtyps.h.
