XFONT::GetTextExtent

Gets the width of a string in pixels.

HRESULT __stdcall GetTextExtent(
  LPCWSTR wstr,
  unsigned cch,
  unsigned *puWidth
);

Parameters

wstr
[in] Pointer to the string whose width will be computed.
cch
[in] The length of the string. cch can be -1 if wstr points to a NULL-terminated string.
puWidth
[out] Pointer to the returned width of the string, in pixels.

Return Values

On success, the function returns S_OK.

Remarks

The width of the string includes any intercharacter spacing as set by XFONT::SetIntercharacterSpacing.

Requirements
