D3DSWATHWIDTH

Defines performance parameters for texture fill rates.

typedef enum _D3DSWATHWIDTH {
    D3DSWATH_8                  = 0,
    D3DSWATH_16                 = 1,
    D3DSWATH_32                 = 2,
    D3DSWATH_64                 = 3,
    D3DSWATH_128                = 4,
    D3DSWATH_OFF                = 0xf,
    D3DSWATH_FORCE_DWORD        = 0x7fffffff, /* force 32-bit size enum */
} D3DSWATHWIDTH;

Constants

D3DSWATH_8
Sets the swath width to 8 pixels.
D3DSWATH_16
Sets the swath width to 16 pixels.
D3DSWATH_32
Sets the swath width to 32 pixels.
D3DSWATH_64
Sets the swath width to 64 pixels.
D3DSWATH_128
Sets the swath width to 128 pixels.
D3DSWATH_OFF
Disables swathing.
D3DSWATH_FORCE_DWORD
Forces this enumeration to compile to 32 bits in size. This value is not used.

Remarks

The values in this enumerated type are used by the D3DRS_SWATHWIDTH render state. When a triangle is rendered, it is rendered in vertical columns of pixels called "swaths". The optimal swath width for a scene can improve the performance of the texture cache. The optimal value can best be found by experimenting with different values.
