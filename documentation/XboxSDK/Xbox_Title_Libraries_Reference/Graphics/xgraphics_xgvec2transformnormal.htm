XGVec2TransformNormal

Transforms the 2D vector normal by the given matrix.

XGVECTOR2* XGVec2TransformNormal(
  XGVECTOR2* pOut,
  CONST XGVECTOR2* pV,
  CONST XGMATRIX* pM
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR2 structure that is the result of the operation.
pV
[in] Pointer to the source XGVECTOR2 structure.
pM
[in] Pointer to the source XGMATRIX structure.

Return Values

Pointer to a XGVECTOR2 structure that is the transformed vector.

Remarks

This function transforms the vector normal (x, y, 0, 0) of the vector, pV, by the matrix, pM.

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec2TransformNormal function can be used as a parameter for another function.
