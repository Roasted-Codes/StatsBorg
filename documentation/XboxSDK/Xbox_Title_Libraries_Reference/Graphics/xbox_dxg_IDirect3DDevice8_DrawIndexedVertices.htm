IDirect3DDevice8::DrawIndexedVertices

Renders the specified geometric primitive, based on indexing into an array of vertices.

HRESULT DrawIndexedVertices(
  D3DPRIMITIVETYPE PrimitiveType,
  UINT VertexCount,
  CONST WORD *pIndexData,
);

Parameters

PrimitiveType
[in] Member of the D3DPRIMITIVETYPE enumerated type, describing the type of primitive to render.
VertexCount
[in] Number of vertices to render. The maximum number of primitives allowed is determined by checking the MaxPrimitiveCount member of the D3DCAPS8 structure.
pIndexData
[in] Pointer to an array of 16-bit indices.

Return Values

If the method succeeds, the return value is D3D_OK.

If the method fails, the return value can be D3DERR_INVALIDCALL.

Remarks

The pointer pIndexData is preferred to have 16-byte alignment, but can be word-aligned if necessary.

The function DrawIndexedVertices is slightly faster than DrawIndexedPrimitive.

In order to use the D3DPT_POINTLIST for the PrimitiveType parameter of DrawIndexedVertices, D3DRS_POINTSPRITEENABLE must be set to TRUE using SetRenderState.

Requirements
