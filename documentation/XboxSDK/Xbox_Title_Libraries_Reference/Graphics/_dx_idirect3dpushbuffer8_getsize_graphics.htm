IDirect3DPushBuffer8::GetSize

Gets the size of the push-buffer program.

HRESULT GetSize(
  DWORD *pSize
);

Parameters

pSize
[out] Pointer to a DWORD value containing the size in bytes of the push-buffer program.

Return Values

If this method succeeds, it returns S_OK.
