XGMatrixTranspose

Returns the matrix transpose of a matrix.

XGMATRIX* XGMatrixTranspose(
  XGMATRIX* pOut,
  CONST XGMATRIX* pM
);

Parameters

pOut
[in, out] Pointer to the XGMATRIX structure that is the result of the operation.
pM
[in] Pointer to the source XGMATRIX structure.

Return Values

Pointer to the XGMATRIX structure that is the matrix transpose of the matrix.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGMatrixTranspose function can be used as a parameter for another function.

Requirements

  Header: Declared in xgmath.h.
  Import Library: Use xgraphics.lib.
