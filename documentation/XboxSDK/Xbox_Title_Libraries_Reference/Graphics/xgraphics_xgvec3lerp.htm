XGVec3Lerp

Performs a linear interpolation between two 3D vectors.

XGVECTOR3* XGVec3Lerp(
  XGVECTOR3* pOut,
  CONST XGVECTOR3* pV1,
  CONST XGVECTOR3* pV2,
  FLOAT s
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR3 structure that is the result of the operation.
pV1
[in] Pointer to a source XGVECTOR3 structure.
pV2
[in] Pointer to a source XGVECTOR3 structure.
s
[in] Parameter that linearly interpolates between the vectors.

Return Values

Pointer to a XGVECTOR3 structure that is the result of the linear interpolation.

Remarks

This function performs the linear interpolation based on the following formula: V1 + s(V2-V1).

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec3Lerp function can be used as a parameter for another function.

Requirements

  Header: Declared in xgmath.h.
  Import Library: Use xgraphics.lib.
