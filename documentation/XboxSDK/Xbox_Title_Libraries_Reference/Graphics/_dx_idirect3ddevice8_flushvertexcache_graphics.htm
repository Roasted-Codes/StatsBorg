IDirect3DDevice8::FlushVertexCache

Flushes the vertex cache

void FlushVertexCache();

Parameters

None.

Return Values

None.

Remarks

This function has the same behavior as calling IDirect3DVertexBuffer8::Lock without the D3DLOCK_NOFLUSH flag. It has been provided so that titles can avoid calling Lock if they want to and to facilitate DMA loads of vertex data.

Requirements

Header: Declared in D3d8.h.

Import Library: Use D3d8.lib.
