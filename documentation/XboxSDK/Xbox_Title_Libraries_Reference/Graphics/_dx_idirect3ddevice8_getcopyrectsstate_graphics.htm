IDirect3DDevice8::GetCopyRectsState

Retrieves the state and raster operation (ROP) to be performed when calling IDirect3DDevice8::CopyRects, allowing the title to do various blending operations.

HRESULT WINAPI GetCopyRectsState(
  D3DCOPYRECTSTATE *pCopyRectState,
  D3DCOPYRECTROPSTATE *pCopyRectRopState
);

Parameters

pCopyRectState
[in, out] Address of a D3DCOPYTRECTSTATE structure that receives the state that will be used with CopyRects.
pCopyRectRopState
[in, out] Address of a D3DCOPYRECTROPSTATE structure that receives the ROP that will be used with CopyRects.

Return Values

If the function succeeds, the return value is S_OK.

Requirements

  Header: Declared in D3d8.h.
  Import Library: Use D3d8.lib.

IDirect3DDevice8::CopyRects, IDirect3DDevice8::SetCopyRectsState
