XGColorAdd

Adds two color values together to create a new color value.

XGCOLOR* XGColorAdd(
  XGCOLOR* pOut,
  CONST XGCOLOR* pC1,
  CONST XGCOLOR* pC2
);

Parameters

pOut
[in, out] Pointer to a XGCOLOR structure that is the result of the operation.
pC1
[in] Pointer to a source XGCOLOR structure.
pC2
[in] Pointer to a source XGCOLOR structure.

Return Values

This function returns a pointer to a XGCOLOR structure that is the sum of two color values.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGColorAdd function can be used as a parameter for another function.

Requirements

  Header: Declared in xgmath.h.
  Import Library: Use xgraphics.lib.
