XGVec3TransformCoord

Transforms a 3D vector by a given matrix, projecting the result back into w = 1.

XGVECTOR3* XGVec3TransformCoord(
  XGVECTOR3* pOut,
  CONST XGVECTOR3* pV,
  CONST XGMATRIX* pM
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR3 structure that is the result of the operation.
pV
[in] Pointer to the source XGVECTOR3 structure.
pM
[in] Pointer to the source XGMATRIX structure.

Return Values

Pointer to a XGVECTOR3 structure that is the transformed vector.

Remarks

This function transforms the vector, pV (x, y, z, 1), by the matrix, pM, projecting the result back into w=1.

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec3TransformCoord function can be used as a parameter for another function.

Requirements
