XGToRadian

Converts degrees into radians.

XGToRadian( degree ) ((degree) * (XG_PI / 180.0f))

Parameters

degree
The value, in degrees, to convert to radians.

Return Values

The macro returns the degree value in radians.

Remarks

The CONSTant XG_PI is declared with the following define.

#define XG_PI    ((FLOAT)  3.141592654f)
