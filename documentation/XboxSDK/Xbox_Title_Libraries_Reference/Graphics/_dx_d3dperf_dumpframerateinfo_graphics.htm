D3DPERF_DumpFrameRateInfo

Dumps frame rate information into the debug window.

void WINAPI D3DPERF_DumpFrameRateInfo();

Parameters

None.

Return Values

None.

Remarks

This is the default function used to display frame rate information. The rate at which at this infomation is dumped is set by the m_FrameRateIntervalTime member of D3DPERF. To completely disable the output, set m_FrameRateIntervalTime to zero.

The output of this function is controled using the m_dwDumpFPSInfoMask member of D3DPERF.

Requirements

  Header: Declared in d3d8perf.h.
  Import Library: Use d3d8i.lib or d3d8d.lib.
