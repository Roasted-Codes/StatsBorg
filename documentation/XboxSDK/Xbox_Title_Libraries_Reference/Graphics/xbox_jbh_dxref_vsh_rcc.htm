rcc

Computes the clamped reciprocal of the source scalar.

rcc   vDest, vSrc0

Registers

vDest
Destination register, holding the result of the operation.
vSrc0
Source register, specifying the input argument.

Operation

The following code fragment shows the operations performed by the rcc instruction to write a result to the destination.

    SetDestReg();
    SetSrcReg(0);

    float input = m_Source[0].w;
    if(input < -1.84467e+19f)
    {
        input = -1.84467e+19f;
    }
    else if(input > -5.42101e-20 && input < 0)
    {
        input = -5.42101e-020;
    }
    else if(input >= 0 && input < 5.42101e-20)
    {
        input = 5.42101e-20;
    }
    else if(input > 1.84467e+19f)
    {
        input = 1.84467e+19f;
    }
	
    if( input == 1.0f )
    {
        // Must be exactly 1.0
        m_TmpReg.x = m_TmpReg.y = m_TmpReg.z = m_TmpReg.w = 1.0f;
    }
    else
    {
        m_TmpReg.x = m_TmpReg.y = m_TmpReg.z = m_TmpReg.w = 1.0f/input;
    }

    WriteResult();

Remarks

The following example illustrates how the rcc instruction might be used.

rcc r1.x, r0
