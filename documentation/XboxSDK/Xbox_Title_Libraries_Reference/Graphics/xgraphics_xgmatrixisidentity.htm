XGMatrixIsIdentity

Determines if a matrix is an identity matrix.

BOOL XGMatrixIsIdentity(
  CONST XGMATRIX* pM
);

Parameters

pM
[in] Pointer to the XGMATRIX structure that will be tested for identity.

Return Values

If the matrix is an identity matrix, this function returns TRUE. Otherwise, this function returns FALSE.

Requirements
