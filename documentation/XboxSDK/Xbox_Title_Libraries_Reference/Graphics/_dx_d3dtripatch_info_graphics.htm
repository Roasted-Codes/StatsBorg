D3DTRIPATCH_INFO

Describes a triangular high-order patch.

typedef struct _D3DTRIPATCH_INFO {
    UINT                StartVertexOffset;
    UINT                NumVertices;
    D3DBASISTYPE        Basis;
    D3DORDERTYPE        Order;
} D3DTRIPATCH_INFO;

Members

StartVertexOffset
Starting vertex offset, in number of vertices.
NumVertices
Number of vertices.
Basis
Member of the D3DBASISTYPE enumerated type, defining the basis type for the triangular high-order patch. This must be set to D3DBASIS_BEZIER.
Order
Member of the D3DORDERTYPE enumerated type, defining the order type for the triangular high-order patch.

Remarks

The table below shows the order of the vertices for each of the tri-patches supported on Xbox:

Linear Bezier
Cubic Bezier
Quintic Bezier

Xbox Optimizations

Xbox does not support either D3BASIS_BSPLINE or D3DBASIS_INTERPOLATE as a basis type for triangular patches.
