IDirect3DFixup8::Reset

Resets a fix-up object after it gets full so that the fix-up object may be used again in an IDirect3DPushBuffer8::BeginFixup/IDirect3DPushBuffer8::EndFixup bracket.

HRESULT Reset();

Parameters

None.

Return Values

If the method succeeds, the return value is S_OK.

Remarks

Similarly to how IDirect3DVertexBuffer8::Lock may be used, this function blocks the CPU until the GPU is done with any of the old contents of the fix-up object, including the most recently specified fix-ups. For this reason, it's recommended that callers employ two "ping-pong" fix-up objects, and call IDirect3DFixup8::Reset just before a fix-up object is reused (not immediately after a fix-up object gets full). It's also recommended that the fix-up objects be large enough so that they don't have to be frequently IDirect3DFixup8::Reset on every frame.
