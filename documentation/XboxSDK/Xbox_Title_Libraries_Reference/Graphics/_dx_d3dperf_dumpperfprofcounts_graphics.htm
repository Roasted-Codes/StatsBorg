D3DPERF_DumpPerfProfCounts

Dumps the GPU's profile data to the debug output window.

void WINAPI D3DPERF_DumpPerfProfCounts();

Parameters

None.

Return Values

None.

Remarks

This function will display information about the total number of busy counts for each unit read while the profiler was enabled. It will also display the total number of profile samples read.

Requirements

  Header: Declared in d3d8perf.h.
  Import Library: Use d3d8i.lib or d3d8d.lib.
