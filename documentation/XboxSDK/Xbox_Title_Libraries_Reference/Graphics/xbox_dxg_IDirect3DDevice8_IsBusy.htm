IDirect3DDevice8::IsBusy

Returns whether the graphics hardware is in use.

BOOL IsBusy();

Parameters

None.

Return Values

If the graphics hardware is in use, this method returns TRUE.

If the graphics hardware is not in use, this method returns FALSE.
