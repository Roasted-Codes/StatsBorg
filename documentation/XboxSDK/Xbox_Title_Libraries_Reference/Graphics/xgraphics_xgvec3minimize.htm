XGVec3Minimize

Returns a 3D vector that is made up of the smallest components of two 3D vectors.

XGVECTOR3* XGVec3Minimize(
  XGVECTOR3* pOut,
  CONST XGVECTOR3* pV1,
  CONST XGVECTOR3* pV2
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR3 structure that is the result of the operation.
pV1
[in] Pointer to a source XGVECTOR3 structure.
pV2
[in] Pointer to a source XGVECTOR3 structure.

Return Values

Pointer to a XGVECTOR3 structure that is made up of the smallest components of the two vectors.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec3Minimize function can be used as a parameter for another function.
