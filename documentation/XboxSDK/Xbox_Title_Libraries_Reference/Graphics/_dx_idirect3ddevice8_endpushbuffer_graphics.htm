IDirect3DDevice8::EndPushBuffer

Ends a push-buffer recording bracket.

HRESULT EndPushBuffer();

Parameters

None.

Return Values

If the method succeeds, the return value is S_OK. The function will return D3DERR_BUFFERTOOSMALL if the push-buffer was too small to hold all of the Direct3D function calls that were recorded beween IDirect3DDevice8::BeginPushBuffer and IDirect3DDevice8::EndPushBuffer.

Requirements

  Header: Declared in D3d8.h.
  Import Library: Use D3d8.lib.
