XFONT::GetTextAntialiasLevel

Returns the anti-aliasing level as set by XFONT::SetTextAntialiasLevel.

unsigned __stdcall GetTextAntialiasLevel();

Parameters

None.

Return Values

The function returns the level of anti-aliasing. This value is either zero, two, or four, where zero specifies that there is no anti-aliasing and four specifies the maximum amount.

Requirements

  Header: Declared in xfont.h
  Import Library: Use xgraphics.lib
