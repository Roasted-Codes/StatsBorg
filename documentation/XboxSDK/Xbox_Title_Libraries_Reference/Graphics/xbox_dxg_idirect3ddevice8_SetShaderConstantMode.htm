IDirect3DDevice8::SetShaderConstantMode

Sets the number of constant registers available to vertex shader programs.

HRESULT SetShaderConstantMode(
  D3DSHADERCONSTANTMODE pMode

);

Parameters

pMode
[in] One or more members of the D3DSHADERCONSTANTMODE type, ORed together. This specifies number of constant registers available.

Return Values

If the method succeeds, the return value is S_OK.

If the method fails, the return value can be D3DERR_INVALIDCALL.

Remarks

The fixed function pipeline cannot be used when in the D3DSCM_192CONSTANTS mode.

When switching from a 192-constant mode to D3DSCM_96CONSTANTS, the contents of all the constant registers in the range –96 through –1 will be lost.

Constant registers D3DVS_XBOX_RESERVEDCONSTANT1 and D3DVS_XBOX_RESERVEDCONSTANT2 (–38 and –37) are reserved except when in D3DSCM_NORESERVEDCONSTANTS mode.
