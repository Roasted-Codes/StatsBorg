D3DXSetDXT3DXT5

Sets premultiplied alpha format for DXT surface and texture processing.

void WINAPI D3DXSetDXT3DXT5(
  BOOL Set
);

Parameters

Set
[in] When TRUE, Direct3DX surfaces will be processed with non-premultiplied alpha (that is, DXT3/DXT5 format). When FALSE, surfaces will be processed with premultiplied alpha (that is, DXT2/DXT4 format).

Return Values

None.

Remarks

By default, surfaces are processed with premultiplied alpha (DXT2/DXT4 format).

In Direct3D on the PC, D3DFMT_DXT2 and D3DFMT_DXT3 are two distinct texture formats. On Xbox, the hardware treats these two formats the same so the value of their enumerants is the same. If you call D3DXCreateTextureFromFileEx and request that the resulting texture be in D3DFMT_DXT3, Direct3DX on Xbox has no way to determine whether you really mean DXT3 or DXT2. The same situation applies for DXT5 and DXT4. The only difference between these two texture types is that one type (DXT2 and DXT4) requires that each of the RGB channels be multiplied by the Alpha channel before the compression takes place. With DXT3 and DXT5, no premultiplication of alpha occurs.

When an application is using Direct3DX to load Direct3DX textures on Xbox, the application should call D3DXSetDXT3DXT5 to indicate whether a DXT3/DXT5 texture is or a DXT2/DXT4 texture.
