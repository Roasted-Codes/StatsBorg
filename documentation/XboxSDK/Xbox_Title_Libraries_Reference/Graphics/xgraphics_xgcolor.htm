XGCOLOR

Describes color values.

typedef struct XGCOLOR {
  FLOAT r, g, b, a;
} XGCOLOR;

Members

r
Red component of the color.
g
Green component of the color.
b
Blue component of the color.
a
Alpha component of the color.
