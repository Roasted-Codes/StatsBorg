XGVec3Project

Projects a vector from object space into screen space.

XGVECTOR3* XGVec3Project(
  XGVECTOR3* pOut,
  CONST XGVECTOR3* pV
  CONST D3DVIEWPORT8* pViewport,
  CONST XGMATRIX* pProjection
  CONST XGMATRIX* pView,
  CONST XGMATRIX* pWorld
);

Parameters

pOut
[in, out] Pointer to the XGVECTOR3 structure that is the result of the operation.
pV
[in] Pointer to the source XGVECTOR3 structure.
pViewport
[in] Pointer to a D3DVIEWPORT8 structure, representing the viewport.
pProjection
[in] Pointer to a XGMATRIX structure, representing the projection matrix.
pView
[in] Pointer to a XGMATRIX structure, representing the view matrix.
pWorld
[in] Pointer to a XGMATRIX structure, representing the world matrix.

Return Values

Pointer to a XGVECTOR3 structure that is the vector projected from object space to screen space.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGVec3Project function can be used as a parameter for another function.

Requirements

  Import Library: Use xgraphics.lib.
