XGWriteSurfaceOrTextureToXPR

This utility functions allows a single surface or texture to be saved to a packed resource file (.xpr). All surface and texture formats supported by Direct3D are supported by this function.

HRESULT WINAPI XGWriteSurfaceOrTextureToXPR( 
  IDirect3DResource8 *pResource,
  const char *cPath,
  BOOL bWriteSurfaceAsTexture
);

Parameters

pResource
[in] A pointer to the surface or texture.
cPath
[in] Null-terminated character string that contains the drive, path, and filename for the output file. If the file exists, it will be overwritten. The filename should have an .xpr or .xbx extension.
bWriteSurfaceAsTexture
[in] If pResource is a surface and bWriteSurfaceAsTexture is TRUE, the resource is converted to a texture before it is written to the .xpr file.

Return Values

If this method succeeds it returns S_OK.

Requirements

  Header: Declared in xgraphics.h.
  Import Library: Use xgraphics.lib.
