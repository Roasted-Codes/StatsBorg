XGCompileDrawIndexedVertices

Creates an IDirect3DDevice8::DrawIndexedVertices push-buffer. This can be run on either the Xbox or on the PC at author time.

HRESULT XGCompileDrawIndexedVertices(
  void *pBuffer,
  DWORD *pSize,
  D3DPRIMITIVETYPE PrimitiveType,
  UINT VertexCount,
  CONST WORD *pIndexData
);

Parameters

pBuffer
[out] Pointer to a previously allocated buffer that will hold the data for the push-buffer.
pSize
[in, out] At call time, pSize is a pointer to a DWORD value containing the size of the buffer pointed at by pBuffer. On the return of the function, pSize will point to a DWORD value containing the size of the resulting push-buffer. This value is set even if the function fails.
PrimitiveType
[in] Member of the D3DPRIMITIVETYPE enumerated type, describing the type of primitive that will be rendered when the push-buffer is run.
VertexCount
[in] Number of vertices to render. The maximum number of primitives is 32 million.
pIndexData
[in] Pointer to an array of 16-bit indices.

Return Values

If this method succeeds, it returns S_OK. If the buffer pointed to by pBuffer is too small, the function returns D3DERR_BUFFERTOOSMALL.

Remarks

This function may be used to save a pre-compiled push-buffer data to a file. For example,

// Set up the vertex buffer and array of indices.
// Set the stream source and vertex shader.
// ...

DWORD Size = INIT_SIZE;		// Needs to be >= push-buffer size

// Get the compiled push-buffer data.
void *pData = D3D_AllocContiguousMemory( Size, D3DPUSHBUFFER_ALIGNMENT );
XGCompileDrawIndexedVertices( pData,
                              &Size,
                              D3DPT_TRIANGLELIST,
                              VertexCount,
                              pIndexData );

// Now copy the buffer pointed to by pData to a file

To use the push-buffer data you will need to manually fill an IDirect3DPushBuffer8 object.

void *pData = D3D_AllocContiguousMemory( Size, D3DPUSHBUFFER_ALIGNMENT );

// Copy the push-buffer data from the file to the buffer pointed to by pData
// ...

// Now set up the push-buffer resource
IDirect3DPushBuffer8 PushBuffer;
ZeroMemory( &PushBuffer, sizeof ( PushBuffer ) );
PushBuffer.Common = 1 | D3DCOMMON_TYPE_PUSHBUFFER;
PushBuffer.Size = Size;
PushBuffer.AllocationSize = Size;
PushBuffer.Register( pData );

// Finally, you can run the push-buffer.
pDevice->RunPushBuffer( &PushBuffer, NULL );
