XGMatrixInverse

Calculates the inverse of a matrix.

XGMATRIX* XGMatrixInverse(
  XGMATRIX* pOut,
  FLOAT* pDeterminant,
  CONST XGMATRIX* pM
);

Parameters

pOut
[in, out] Pointer to the XGMATRIX structure that is the result of the operation.
pDeterminant
[in, out] After this function returns, pointer to a FLOAT value that is the determinant of the matrix, pM.
pM
[in] Pointer to the source XGMATRIX structure.

Return Values

Pointer to a XGMATRIX structure that is the inverse of the matrix.

Remarks

This function accepts any arbitrary matrix.

Matrix inversion may fail, in which case NULL is returned by the XGMatrixInverse function. The determinant of pM is returned if the pDeterminant parameter is non-NULL.

The return value for this function is the same value returned in the pOut parameter. In this way, the XGMatrixInverse function can be used as a parameter for another function.

Requirements
