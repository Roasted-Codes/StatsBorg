XFONT::SetBkMode

Sets whether the background of the text cell is filled with the background color.

void __stdcall SetBkMode(
  int iBkMode
);

Parameters

iBkMode
[in] Sets the fill style. This parameter must be one of the following values:
Value Meaning
XFONT_TRANSPARENT The background of the text cell is filled with the background color of the surface on which the text is rendered.
XFONT_OPAQUE The background of the text cell is filled with the color set using XFONT::SetBkColor.

Return Values

None.

Remarks

The default fill style is XFONT_TRANSPARENT.
