D3DPERF_EndEvent

D3DPERF_EndEvent is used along with D3DPERF_BeginEvent to define the end of a user-defined event that happens over a period of time. This event will then appear in the PIX Timeline Window.

INT D3DPERF_EndEvent();

Parameters

None.

Return Values

Returns the level of the hierarchy this event is ending in. If an error occurs, this value will be negative.

Remarks

Each call to D3DPERF_EndEvent must be paired with a corresponding call to D3DPERF_BeginEvent. These functions must bracket the event that is to be captured, and must not be separated by a call to Swap or Present.

In release builds of Direct3D, this function does nothing.

Requirements

Header: Declared in D3d8perf.h.

Import Library: Use Use d3d8i.lib or D3d8d.lib.
