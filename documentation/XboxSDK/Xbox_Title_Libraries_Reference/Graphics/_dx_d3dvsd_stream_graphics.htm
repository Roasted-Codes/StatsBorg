D3DVSD_STREAM

Sets the current stream.

D3DVSD_STREAM( _StreamNumber ) \
             (D3DVSD_MAKETOKENTYPE(D3DVSD_TOKEN_STREAM) | (_StreamNumber))

Parameters

_StreamNumber
Specifies the stream from which to get data. Possible values range from 0 through dwMaxStreams - 1.
