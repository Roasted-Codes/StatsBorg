D3DVBLANKDATA

Describes the current vertical blank.

typedef struct _D3DVBLANKDATA {  // Xbox extension
    DWORD           VBlank;
    DWORD           Swap;
    DWORD           Flags;
} D3DVBLANKDATA;

Members

VBlank
The vertical blank count. If the current video mode is interlaced, VBlank will be even if the current field is even and it will be odd if the current field is odd.
Swap
The count of the most recent swap that was shown. If the swap was shown, starting on the vertical blank that initialized D3DVBLANKDATA, the Swap member will reflect the count of that swap.
Flags
The Flags member will be one of the following values:
Value Description
0 This is the value if Direct3D neither handled a swap on the vertical blank nor expected to (such as when the FullScreen_PresentationInterval member of D3DPRESENT_PARAMETERS is D3DPRESENT_INTERVAL_TWO and it's on the first vertical blank).
D3DVBLANK_SWAPMISSED This flag is set if, according to the FullScreen_PresentationInterval member of D3DPRESENT_PARAMETERS, Direct3D expected to show a swap on this vertical blank, but the title didn't make the swap in time (the title missed a frame).
D3DVBLANK_SWAPDONE This flag is set if a swap was shown durnig the vertical blank.

Remarks

This structure is used by the callback function set on the device by IDirect3DDevice8::SetVerticalBlankCallback.
