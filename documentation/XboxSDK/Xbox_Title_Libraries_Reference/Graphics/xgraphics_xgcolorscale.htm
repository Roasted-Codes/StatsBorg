XGColorScale

Scales a color value.

XGCOLOR* XGColorScale( 
  XGCOLOR* pOut,
  CONST XGCOLOR* pC,
  FLOAT s 
);

Parameters

pOut
[in, out] Pointer to a XGCOLOR structure that is the result of the operation.
pC
[in] Pointer to a source XGCOLOR structure.
s
[in] Scale factor. It scales the color, treating it like a 4-D vector. There are no limits on the value of s. If s is 1, the resulting color is the original color.

Return Values

This function returns a pointer to a XGCOLOR structure that is the scaled color value.

Remarks

The return value for this function is the same value returned in the pOut parameter. In this way, the XGColorScale function can be used as a parameter for another function.

This function computes the scaled color value by multiplying the color components of the XGCOLOR structure by the specified scale factor, as shown in the following example.

    pOut->r = pC->r * s;
