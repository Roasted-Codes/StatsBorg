D3DTEXTURECOLORKEYOP

Defines the color-key operation to perform.

typedef enum _D3DTEXTURECOLORKEYOP {  // Xbox extension
    D3DTCOLORKEYOP_DISABLE      = 0,
    D3DTCOLORKEYOP_ALPHA        = 1,
    D3DTCOLORKEYOP_RGBA         = 2,
    D3DTCOLORKEYOP_KILL         = 3,

    D3DTCOLORKEYOP_MAX          = 4,
    D3DTCOLORKEYOP_FORCE_DWORD  = 0x7fffffff, /* force 32-bit size enum */
} D3DTEXTURECOLORKEYOP;

Constants

D3DTCOLORKEYOP_DISABLE
Disables color-key operations.
D3DTCOLORKEYOP_ALPHA
Alpha is set to set on a color-key match.
D3DTCOLORKEYOP_RGBA
All channels (red, green, blue, and alpha) are set to zero on a color-key match.
D3DTCOLORKEYOP_KILL
Kills the pixel if any of its texels that were filtered with a non-zero weight were color-keyed.
D3DTCOLORKEYOP_MAX
This value is not used.
D3DTCOLORKEYOP_FORCE_DWORD
Forces this enumeration to compile to 32 bits in size. This value is not used.
