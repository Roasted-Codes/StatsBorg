IDirect3DDevice8::InsertFence

Inserts a fence into the push buffer.

DWORD InsertFence();

Return Values

The function returns the handle to the fence.

Remarks

This fence can be used to determine when all of the commands in the push buffer before the fence have been processed. It will also allow resources used for drawing to be locked without blocking if those resources are no longer set on the device and the fence has been executed.
