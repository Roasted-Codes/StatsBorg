D3DFRONT

Specifies whether the front face is defined as clockwise or counterclockwise.

typedef enum _D3DFRONT {        // Xbox extension
    D3DFRONT_CW                 = 0x900,
    D3DFRONT_CCW                = 0x901,
    D3DFRONT_FORCE_DWORD        = 0x7fffffff, /* force 32-bit size enum */
} D3DFRONT;

Constants

D3DFRONT_CW
The front face is defined clockwise.
D3DFRONT_CCW
The front face is defined counterclockwise.
D3DFRONT_FORCE_DWORD
Forces this enumeration to compile to 32 bits in size. This value is not used.

Remarks

The values in this enumerated type are used by the D3DRS_FRONTFACE render state.
