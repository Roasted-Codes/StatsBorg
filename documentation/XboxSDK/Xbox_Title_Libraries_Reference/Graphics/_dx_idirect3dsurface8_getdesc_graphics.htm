IDirect3DSurface8::GetDesc

Retrieves a description of the surface.

HRESULT GetDesc(
  D3DSURFACE_DESC* pDesc
);

Parameters

pDesc
[out] Pointer to a D3DSURFACE_DESC structure, describing the surface.

Return Values

The method returns D3D_OK.
