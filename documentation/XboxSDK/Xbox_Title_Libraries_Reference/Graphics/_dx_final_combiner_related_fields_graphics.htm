Final Combiner-Related Fields

The following sections provide a description of the final combiner-related fields of D3DPIXELSHADERDEF.

The next two fields specify the inputs to the final combiner. The final combiner is typically used to add the specular color in the v1 register to the fragment color and also to blend in the fog color. If both final combiner fields are set to 0, the usual function of the final combiner is retained. However, if either of the final combiner fields is set to a non-zero value, the final combiner operation is completely specified by the pixel shader. No outputs need be set for the final combiner. The RGB and alpha results are set as follows:

rgbout = a*b + (1-a)*c + d
alphaout = alpha channel of g
prod = e*f
