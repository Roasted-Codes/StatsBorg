XGWriteSurfaceToFile

Allows the contents of a surface to be written to a 24-bit .bmp file.

HRESULT XGWriteSurfaceToFile(
  IDirect3DSurface8* pSurf,
  const char* cPath
);

Parameters

pSurf
[in] A pointer to an IDirect3DSurface8 surface.
cPath
[in] A pointer to a character string that contains the drive, path, and filename for the output file. If the file exists, it will be overwritten. The filename should have a .bmp extension.

Return Values

If this method succeeds it returns S_OK.

Remarks

The following surface formats are supported:

  • D3DFMT_LIN_A8R8G8B8
  • D3DFMT_LIN_X8R8G8B8
  • D3DFMT_LIN_R5G6B5
  • D3DFMT_LIN_X1R5G5B5

These are the formats that are possible for a front buffer. Swizzled formats are not currently supported.

To use this function, create a temporary buffer 3 * height * width (in bytes) big. If this allocation fails, the file is written more slowly using a much smaller buffer.

Requirements

  Header: Declared in xgraphics.h.
  Import Library: Use xgraphics.lib.
