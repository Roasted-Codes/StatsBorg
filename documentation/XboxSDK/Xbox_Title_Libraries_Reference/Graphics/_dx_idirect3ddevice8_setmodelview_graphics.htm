IDirect3DDevice8::SetModelView

This method is a faster alternative to IDirect3DDevice8::SetTransform, and it works with push-buffer fix-ups.

HRESULT SetModelView(
  CONST D3DMATRIX *pModelView, 
  CONST D3DMATRIX *pInverseModelView, 
  CONST D3DMATRIX *pComposite
);

Parameters

pModelView
[in] Pointer to the model-view transform.
pInverseModelView
[in] Pointer to the inverse of the model-view transform. This may be NULL if lighting and texture generation are disabled.
pComposite
[in] Pointer to a D3DMATRIX that is the product of the model-view transform and the projection/viewport (as returned by IDirect3DDevice8::GetProjectionViewportMatrix matrices).

Return Values

If the method succeeds, the return value is S_OK.

Remarks

This method can only be called while D3DRS_VERTEXBLEND is FALSE. It also cannot be called while in D3DSCM_192CONSTANTS mode because it would overwrite some of the negative shader constants.

When IDirect3DDevice8::SetModelView is called with a non-NULL pModelView pointer, the method takes control of the GPU's transform registers until IDirect3DDevice8::SetModelView is called with a NULL pModelView pointer. While the function has control, any calls to IDirect3DDevice8::SetTransform will be effectively ignored.
