Xbox Direct3D Resources

A resource is an instance of any of the structures defined in d3d8.h that inherit from IDirect3DResource8, including all textures, surfaces, volumes, vertex buffers, index buffers, palettes, and push-buffers. These resources include a header that contains all of the information needed by the driver to process the resource and a data allocation that contains the information used by the graphics processor.

The resource headers are used only by the driver and so should be allocated from normal cached memory. The header contains all of the information needed by the driver to set up the hardware so it can use the information stored in the data allocation.

The information in this overview applies to the following interfaces:

  • IDirect3DCubeTexture8
  • IDirect3DIndexBuffer8
  • IDirect3DPalette8
  • IDirect3DPushBuffer8
  • IDirect3DSurface8
  • IDirect3DTexture8
  • IDirect3DVertexBuffer8
  • IDirect3DVolume8
  • IDirect3DVolumeTexture8
