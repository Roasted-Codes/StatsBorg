xvsw

Provides a method to specify the type and version of the Xbox read/write vertex shader code.

xvsw.mainVer.subVer

Registers

mainVer
Main version number of the Xbox read/write vertex shader.
subVer
Sub version number of the Xbox read/write vertex shader.

Remarks

An Xbox read/write vertex shader is like a normal Xbox vertex shader, except that:

  • It can write to constant registers. The values of the constant registers are preserved across invocations of the shader.
  • It runs roughly six times slower than a normal Xbox vertex shader.

The following example illustrates how the xvsw instruction might be used.

xvsw.1.1  ; Xbox read/write vertex shader version 1.1.
mov r0,c[0]    ; c[0] = c[0] + c[1]
add r0,r0,c[1]
mov c[0],r0
mov oD0,r0
mov oPos,v0

This instruction is required to be at the beginning of all Xbox read/write vertex shaders.
