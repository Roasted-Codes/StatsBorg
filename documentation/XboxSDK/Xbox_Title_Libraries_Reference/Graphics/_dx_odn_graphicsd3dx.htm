oDn

The output data registers, which are used to output vertex color data. Specifically, these are an array of output data registers that are iterated and directly routed to the pixel shader. Typically, these registers are used for colors.

Note  Direct3D 8.0 on the PC defines two color outputs for vertex shaders: oD0 and oD1. Direct3D on Xbox defines two additional registers, oB0 and oB1. These two registers can be used to store the color value for the back side of a polygon. The Xbox GPU A-2 sends oD0 and oD1 on to the pixel shader in the v0 and v1 registers if the triangle is front-faced. Conversely, it passes oB0 and oB1 to pixel shader registers v0 and v1 if the triangle is back-faced.
