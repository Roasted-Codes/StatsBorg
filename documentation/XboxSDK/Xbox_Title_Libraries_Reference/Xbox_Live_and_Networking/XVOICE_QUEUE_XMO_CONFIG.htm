XVOICE_QUEUE_XMO_CONFIG

The XVOICE_QUEUE_XMO_CONFIG structure stores the configuration information for an voice-queue Xbox media object.

typedef struct{
    DWORD dwMsOfDataPerPacket;
    DWORD dwCodecBufferSize;
    DWORD dwMaxDelay;
    DWORD dwMinDelay;
    DWORD dwInitialHighWaterMark;
    DOUBLE dIdealQuality;
} XVOICE_QUEUE_XMO_CONFIG, *LPXVOICE_QUEUE_XMO_CONFIG;

Members

dwMsOfDataPerPacket
Specifies the number of milliseconds of data in each packet. The default value for this member is 20.
dwCodecBufferSize
Specifies the buffer size for the Codec. Obtain this value by calling the IXVoiceEncoder::GetCodecBufferSize method. This member has no default value; the title must supply a non-zero value for this member.
dwMaxDelay
Specifies the maximum delay for the queue in milliseconds. Adjust the value of this member in order to eliminate jitter. The default value for this member is 2000.
dwMinDelay
Specifies the minimum delay for the queue in milliseconds. Adjust the value of this member in order to eliminate jitter. The default value for this member is 80.
dwInitialHighWaterMark
Specifies the high water mark in milliseconds for the adaptive queuing algorithm. The default value is 400.
dIdealQuality
Specifies the ideal quality for the high-water mark. The range of values for this member is 0.0 (no errors) to 1.0 (all errors). The default value is 0.005.

Remarks

The XVoiceQueueCreateMediaObject functions takes a pointer to this structure as one of its parameters.

To obtain the default value for a member of the structure, set that member to zero. Since a value of zero specifies the default value, it is not possible to specify zero as an actual value. If a title wishes to set a member's value to zero, it should instead set the value to a very small number. An example would be setting the dIdealQuality member to 0.00001.
