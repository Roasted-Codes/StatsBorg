XOnlineStringLookupGetResults

Retrieves the results from a call to XOnlineStringLookup.

HRESULT XOnlineStringLookupGetResults(
    XONLINETASK_HANDLE hTask,
    BYTE *pbBuffer,
    DWORD *pdwBufferSize,
    WCHAR **ppwszStrings,
    WORD *pwNumStrings
);

Parameters

hTask
[in] An XONLINETASK_HANDLE returned from a call to XOnlineStringLookup.
pbBuffer
[in, out] Byte array to hold the Unicode strings. This buffer must be large enough to hold the the strings and their terminating NULL characters.
pdwBufferSize
[in, out] Size of the buffer pointed to by the pbBuffer parameter.

Note    The buffer size is specified in bytes, not characters.

ppwszStrings
[in, out] Array of Unicode strings. This array will be filled in with pointers to Unicode strings stored in the buffer specified by pbBuffer. The order of the strings is the same as the IDs specified to XOnlineStringLookup.
pwNumStrings
[in, out] Number of Unicode strings returned. This should be the same as the number of IDs passed to XOnlineStringLookup.

Return Values

Returns S_OK if successful. If a string ID is not found, this function returns an empty (non-NULL) Unicode string. Other errors cause an error code to be returned. Among these are the value Xonline_E_Overflow_HR, which is returned when the buffer pointed to by the pbBuffer parameter is not large enough.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
