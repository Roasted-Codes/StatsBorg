XONLINE_FRIEND

Stores information about a user on the Friends List. 

typedef struct{
    XUID xuid;
    CHAR szGamertag[XONLINE_GAMERTAG_SIZE];
    DWORD dwFriendState;
    FILETIME gameinviteTime;
    XNKID sessionID;
    DWORD dwTitleID;
    BYTE StateDataSize;
    BYTE StateData[MAX_STATEDATA_SIZE];
    BYTE bReserved;
} XONLINE_FRIEND, *PXONLINE_FRIEND;

Members

xuid
Unique identifier of the player.
szGamertag

Null-terminated string that contains the user's Gamertag. The format of the Gamertag must adhere to the following rules:
  • Minimum of one character in length.
  • Maximum of 15 characters in length.
  • Can include only the following characters: A–Z, a–z, 0–9, and [space].
  • Cannot begin with a [space] character.
  • Cannot have multiple contiguous [space] characters.
  • Must begin with an alphabetic character (A–Z, a–z).
  • Cannot end with space.

The Gamertag is case insensitive for comparisons, but case should be preserved when stored. For example, Ubba and UBBA are the same and, hence, cannot both exist as accounts in the system, but whichever one was entered into the system, the case is preserved.

dwFriendState
State of the player. This member is a combination of one or more of the following values:
Value Description
XONLINE_FRIENDSTATE_FLAG_INVITEACCEPTED The user represented by this XONLINE_FRIEND structure has accepted an invitation from the user who enumerated the Friends List.
XONLINE_FRIENDSTATE_FLAG_INVITEREJECTED The user represented by this XONLINE_FRIEND structure has rejected an invitation from the user who enumerated the Friends List.
XONLINE_FRIENDSTATE_FLAG_JOINABLE The user is in a session that other players can join.
XONLINE_FRIENDSTATE_FLAG_ONLINE The user is online.
XONLINE_FRIENDSTATE_FLAG_RECEIVEDREQUEST The user who enumerated the Friends List has received a Friend request from the user represented by this XONLINE_FRIEND structure. This flag is cleared when the request is answered
XONLINE_FRIENDSTATE_FLAG_SENTREQUEST The user who enumerated the Friends List has sent a Friend request to the user represented by this XONLINE_FRIEND structure. This flag is cleared when the request is answered
XONLINE_FRIENDSTATE_FLAG_PLAYING The user is playing a game.
XONLINE_FRIENDSTATE_FLAG_RECEIVEDINVITE The user who enumerated the Friends List has received an invitation from the user represented by this XONLINE_FRIEND structure. This flag remains set until the invitation is revoked using XOnlineFriendsRevokeGameInvite or the inviting user's session ID changes. The user's session ID can be changed by calling XOnlineNotificationSetState
XONLINE_FRIENDSTATE_FLAG_SENTINVITE The user who enumerated the Friends List has sent an invitation to the user represented by this XONLINE_FRIEND structure. This flag remains set until the invitation is revoked using XOnlineFriendsRevokeGameInvite or the inviting user's session ID changes. The user's session ID can be changed by calling XOnlineNotificationSetState
XONLINE_FRIENDSTATE_FLAG_VOICE The user has voice capability.
gameinviteTime
FILETIME structure that contains the time that this friend sent an invitation to the current user.
sessionID
An XNKID structure that contains the session identifier. If this structure is all zeros, the user is not in a session.
dwTitleID
The title identifier. If this value is zero, the title identifier is unknown. Use the XOnlineFriendsGetTitleName function to retrieve the corresponding title name.
StateDataSize
State data size, in bytes.
StateData
Array of BYTEs containing the state data.
bReserved
This member is reserved for future use. Do not use.
