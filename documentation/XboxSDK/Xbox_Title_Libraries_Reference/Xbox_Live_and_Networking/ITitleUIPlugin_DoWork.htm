ITitleUIPlugin::DoWork

Perform periodic work required by the UI plug-in. DoWork is called by the Xbox Drop-in UI engine every time ILiveEngine::DoWork is called (typically once per frame).

HRESULT DoWork();

Parameters

None.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

DoWork is called after other calls are made to the plug-in for changes that need to occur in that DoWork session.

The Xbox Drop-in UI engine currently ignores the return value of DoWork.

The default ITitleUIPlugin implementation restores changed IDirect3DDevice8 states before returning from DoWork.

Requirements
