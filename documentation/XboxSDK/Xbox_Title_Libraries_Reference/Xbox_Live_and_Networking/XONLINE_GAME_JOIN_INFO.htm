XONLINE_GAME_JOIN_INFO

Contains the information required for a player to join a game.

typedef struct _XONLINE_GAME_JOIN_INFO{
    XUID xuidJoinedUser;
    CHAR szJoinedUserGamertag[XONLINE_GAMERTAG_SIZE];
    DWORD dwTitleID;
    XNKID SessionID;
} XONLINE_GAME_JOIN_INFO, *PXONLINE_GAME_JOIN_INFO;

Members

xuidJoinedUser
Player's ID.
szJoinedUserGamertag
Player's Gamertag.
dwTitleID
ID of the title.
SessionID
ID of the session in the competition that the player wants to join.
