XOnlineStartup

Performs initialization and startup for Xbox Live functions.

HRESULT XOnlineStartup(
    const XONLINE_STARTUP_PARAMS *pxosp
);

Parameters

pxosp
[in] Pointer to an XONLINE_STARTUP_PARAMS structure that specifies initialization parameters. This parameter can be NULL if default startup parameters are desired.

Return Values

If the function succeeds, the return value is S_OK.

Remarks

This function must be successfully called before any other Xbox Live functions can be called. When finished using all Xbox Live functions, call XOnlineCleanup.

XOnlineStartup internally automatically calls XNetStartup and WSAStartup with default parameters in order to initialize the Xbox Secure Network Libary and the Winsock layer. To specify non-default startup parameters for XNetStartup or WSAStartup, call those functions prior to calling XOnlineStartup.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
