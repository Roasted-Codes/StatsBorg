ITitlePlayersListItem

ITitlePlayersListItem implements the player information interface to the Xbox Drop-in UI Players feature. The Players feature calls methods on the interface to obtain information from the title about players displayed in the Players List. Titles must provide their own implementation of ITitlePlayersListItem.

ITitlePlayersListItem instances are added to and removed from the Players List with ILivePlayersList::RegisterPlayer and ILivePlayersList::UnregisterPlayer.

ITitlePlayersListItem methods Compare
GetBitflags
GetName
GetVoiceStatus
GetXUID
IsTalking

The ITitlePlayersListItem interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.
