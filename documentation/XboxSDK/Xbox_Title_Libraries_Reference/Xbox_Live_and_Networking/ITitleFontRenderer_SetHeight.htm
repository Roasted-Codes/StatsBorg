ITitleFontRenderer::SetHeight

Sets the height of the text cell.

HRESULT SetHeight(
    DWORD Height
);

Parameters

Height
[in] Total height of the text cell, in pixels.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

All further text rendered by DrawText should be rendered with a height of Height (until SetHeight is called again to set a new height).

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ITitleFontRenderer::DrawText, ITitleFontRenderer::SetColor, D3DCOLOR
