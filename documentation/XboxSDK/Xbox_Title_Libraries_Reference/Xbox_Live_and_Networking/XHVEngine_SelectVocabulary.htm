IXHVEngine::SelectVocabulary

Selects which speech recognition vocabularies are active for the specified local talker.

HRESULT SelectVocabulary(
    DWORD dwLocalPort,
    DWORD dwVocabsCount,
    const XHV_SR_VOCAB_SELECTION *pVocabSelections
);

Parameters

dwLocalPort
[in] Controller port of the local talker.
dwVocabsCount
[in] The number of vocabularies in pVocabSelections
pVocabSelections
[in] An array of XHV_SR_VOCAB_SELECTION values specifying the vocabularies to activate.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This method selects as active one or more vocabularies in the speech recognition bank currently registered with the speech recognition engine. Vocabularies are specified in pVocabSelections by their enumerated values as determined by the BuildSpeechRecBank tool, which creates a header file containing the vocabulary section values.

Call SelectVocabulary only after successfully calling RegisterSpeechRecognitionBank.

Active vocabularies cannot be changed while speech recognition is in progress. SelectVocabulary will return a failure code if such an attempt is made.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
