XOnlineFriendsJoinGame

Enables a user to join an online-game session being played by a member of the user's Friends List.

HRESULT XOnlineFriendsJoinGame(
    DWORD dwUserIndex,
    const XONLINE_FRIEND *pToFriend
);

Parameters

dwUserIndex
[in] Controller index of the user who is choosing to join the online-game session.
pToFriend
[in] Pointer to an XONLINE_FRIEND structure for the friend who is playing the game to be joined.

Return Values

If XOnlineFriendsJoinGame succeeds, the return value is S_OK.

If XOnlineFriendsJoinGame fails, the return value is one of the following error codes.

Value Description
XONLINE_E_NOTIFICATION_SAME_TITLE The invitation is to join the same title as the one currently running. The game should allow the user to join another instance of the same title. The XOnlineFriendsJoinGame function is not necessary in this case. To avoid this situation, use the XOnlineTitleIdIsSameTitle function to check the dwTitleID of the XONLINE_FRIEND structure.
XONLINE_E_NOTIFICATION_NO_TASK The Friends startup task is not active. You must first call XOnlineFriendsStartup to create the startup task.
Other See Online Errors for information about other possible errors.

Remarks

If the user chooses to join a game session for a title that is different than the title currently running on the Xbox console, XOnlineFriendsJoinGame writes information to the Xbox hard disk that enables the new title to join the game session. The new title retrieves this information by calling XOnlineFriendsGetAcceptedGameInvite. The new title can inspect the flags of the InvitingFriend member of the returned XONLINE_ACCEPTED_GAMEINVITE structure to determine if the information on the hard disk is for a cross-title join or cross-title invitation.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
