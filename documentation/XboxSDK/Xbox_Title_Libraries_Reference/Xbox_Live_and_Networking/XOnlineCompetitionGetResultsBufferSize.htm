XOnlineCompetitionGetResultsBufferSize

Retrieves the buffer size required for the results of a search initiated by functions such as XOnlineCompetitionSearch and XOnlineCompetitionTopology.

DWORD XOnlineCompetitionGetResultsBufferSize(
    DWORD dwResultsPerPage,
    DWORD dwNumSpecs,
    const XONLINE_ATTRIBUTE_SPEC *pSpecs
);

Parameters

dwResultsPerPage
[in] Number of results per page. Maximum is 255.
dwNumSpecs
[in] Number of attribute specifiers.
pSpecs
[in] Array of attribute specifiers.

Return Values

Returns the size, in bytes, required for the results buffer.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.

See Also

XOnlineCompetitionSearch, XOnlineCompetitionSearchGetResults, XOnlineCompetitionTopology, XOnlineCompetitionTopologyGetResults, XOnlineCompetitionTopologySingleElimination, XOnlineCompetitionTopologySingleEliminationGetResults
