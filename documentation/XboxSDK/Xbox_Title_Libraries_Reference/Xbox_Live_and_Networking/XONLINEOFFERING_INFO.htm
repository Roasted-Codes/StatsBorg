XONLINEOFFERING_INFO

Contains details about an online offering.

typedef struct{
    XOFFERING_ID OfferingId;  // Offering ID
    DWORD dwOfferingType;  // Offering type
    DWORD dwBitFlags;  // Package-specific flags
    DWORD dwPackageSize;  // Package wire size (bytes)
    DWORD dwInstallSize;  // Installed size (blocks)
    FILETIME ftActivationDate;  // Activation date of package
    DWORD dwRating;  // Package rating
    WORD fOfferingFlags;  // Per-offering flags
    DWORD dwTitleSpecificData;  // Size of data blob (bytes)
    PBYTE pbTitleSpecificData;  // Pointer to data blob
} XONLINEOFFERING_INFO, *PXONLINEOFFERING_INFO;

Members

OfferingId
Identifier of the online offering. The high 32 bits correspond to the TitleID. The low 32 bits are used by the title to uniquely identify offerings belonging to that title.
dwOfferingType
Value specifying the type of online offering. This value can be one of the following:
Value Description
XONLINE_OFFERING_CONTENT The offering is some kind of downloadable content.
XONLINE_OFFERING_SUBSCRIPTION The offering is some kind of subscription.
dwBitFlags
Any package-specific flags.
dwPackageSize
Size, in bytes, of the downloadable content.
dwInstallSize
Size, in blocks, of the content after installation.
ftActivationDate
FILETIME value specifying the activation date of the offering.
dwRating
Rating of the offering.
fOfferingFlags

Per-offering flags. Can be a combination of the following values:
Value Description
XONLINE_OFFERING_IS_NOT_FREE The offering requires a purchase (it is not free)

The XOnlineOfferingIsFree macro, when passed the entire fOfferingFlags value, will indicate whether an offering is free or not, and is the preferred method of testing the flags in this member (rather than testing them directly).

dwTitleSpecificData
Size, in bytes, of the title specific data related to the offering. The maximum size of the title specific data is XONLINEOFFERING_ENUM_MAX_TITLE_DATA_SIZE (8000) bytes.
pbTitleSpecificData
Address of a buffer containing the title specific data related to the offering.

Requirements
