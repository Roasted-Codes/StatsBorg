IPluginSupport::GetLayout

Retrieves, from the skin file, the skin layout information for the layout with the specified screen and object resource identifiers.

HRESULT GetLayout(
    DWORD ScreenResID,
    DWORD ObjectResID,
    UIX_SKIN_LAYOUT_INFO **ppLayout
);

Parameters

ScreenResID
[in] Resource identifier of the screen.
ObjectResID
[in] Resource identifier of the object.
ppLayout
[out] Pointer to a UIX_SKIN_LAYOUT_INFO structure that receives the layout information.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
