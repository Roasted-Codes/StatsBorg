XOnlineQueryFindFromIds

Queries a dataset using a list of entity IDs and a query procedure.

HRESULT XOnlineQueryFindFromIds(
    DWORD dwDatasetId,
    DWORD dwProcIndex,
    DWORD dwNumResultSpecs,
    const XONLINE_ATTRIBUTE_SPEC *pAttributeSpecs,
    DWORD dwNumEntityIds,
    const XENTITY_ID *pEntityIds,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE *phTask
);

Parameters

dwDatasetId
[in] ID of the dataset to query.
dwProcIndex
[in] Index number of the query procedure.
dwNumResultSpecs
[in] Number of attributes in the array specified by the pAttributeSpecs parameter.
pAttributeSpecs
[in] Array of XONLINE_ATTRIBUTE_SPEC structures that specify the type and size, in bytes, of each result attribute. The first item in this array must have the type XONLINE_QUERY_ENTITY_ID. Each subsequent structure must match the order and type of return value from the query.
dwNumEntityIds
[in] Number of entity IDs in the array specified by the pEntityIds parameter. Must be at least 1 and no greater than 10.
pEntityIds
[in] Array of entity IDs.
hWorkEvent
[in, optional] Handle to an event you want to associate with this task. If you do not want to assign an event, set this parameter to NULL.
phTask
[out] Pointer to an XONLINETASK_HANDLE type. The handle is associated with the task started by XOnlineQueryFindFromIds.

Return Values

Returns S_OK if successful, or an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
