XOnlineStatUnitEnumerateGetResults

Retrieves unit stats information requested by the XOnlineStatUnitEnumerate function.

HRESULT XOnlineStatUnitEnumerateGetResults(
    XONLINETASK_HANDLE hTask,
    PXONLINE_STAT_UNIT pUnits,
    DWORD dwStatCount,
    XONLINE_STAT *pStats,
    DWORD *pdwReturnedResults
);

Parameters

hTask
[in] Handle to the task returned by the XOnlineStatUnitEnumerate function.
pUnits
[out] Pointer to an array of XONLINE_STAT_UNIT structures. The number of structures should be the same as the value specified by the dwMaxUnitsToReturn parameter in the call to XOnlineStatUnitEnumerate.
dwStatCount
[in] Number of XONLINE_STAT structures pointed to by the pStats parameter.
pStats
[out] Pointer to an array of XONLINE_STAT structures that receives the retrieved stats. The number of structures in this array should be equal to the number of entries requested times the number of stats requested per entry—that is, the product of the values specified by dwStatCount and the dwMaxUnitsToReturn parameter in the call to XOnlineStatUnitEnumerate.
pdwReturnedResults
[out] Pointer to a DWORD variable that receives the number of units actually returned.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
