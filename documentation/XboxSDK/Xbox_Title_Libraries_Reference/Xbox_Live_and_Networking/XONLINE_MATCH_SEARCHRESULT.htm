XONLINE_MATCH_SEARCHRESULT

Contains the base session information returned to the client after a successful search. 

typedef struct{
    DWORD dwReserved;
    XNKID SessionID;
    XNADDR HostAddress;
    XNKEY KeyExchangeKey;
    DWORD dwPublicOpen;
    DWORD dwPrivateOpen;
    DWORD dwPublicFilled;
    DWORD dwPrivateFilled;
    DWORD dwNumAttributes;
} XONLINE_MATCH_SEARCHRESULT, *PXONLINE_MATCH_SEARCHRESULT;

Members

dwReserved
This member is reserved.
SessionID
XNKID structure containing the session identifier.
HostAddress
XNADDR structure that contains the Xbox network address of the Xbox that is hosting the game session. This address is not the same as an IP address.
KeyExchangeKey
XNKEY containing the shared encryption key for the session.
dwPublicOpen
Value containing the number of unoccupied public player slots. There is no guarantee that this number of slots will be available when the host is contacted.
dwPrivateOpen
Value containing the number of unoccupied private player slots. There is no guarantee that this number of slots will be available when the host is contacted.
dwPublicFilled
Value containing the number of public player slots that are currently filled.
dwPrivateFilled
Value containing the number of available private player slots that are currently filled.
dwNumAttributes
Value containing the number of attributes that follow this structure. An array that contains the attribute offsets immediately follows the dwNumAttributes member in the buffer. The number of elements in this array equals the value of dwNumAttributes.

Remarks

Use the SessionID and the KeyExchangeKey with the XNetRegisterKey function to register a secure networking connection with the Xbox that is hosting the game session.

Use the SessionID and the HostAddress with the XNetXnAddrToInAddr function to obtain the IN_ADDR of the Xbox that is hosting the online-game session.

Requirements

Header: Declared in Xonline.h.
