UIX_UITEST_PARAMS

Initialization parameters for the UI test feature. This structure is passed in pFeatureParams when calling ILiveEngine::StartFeature with UIX_UITEST_FEATURE. Available only in debug builds.

typedef struct _UIX_FRIENDS_PARAMS{
    SIZE_T StructSize;
} UIX_UITEST_PARAMS, *PUIX_UITEST_PARAMS;

Members

StructSize
Size of the structure. Must be set to the structure size before calling ILiveEngine::StartFeature.

Remarks

This structure must be initialized to zero and StructSize set to the size of the structure before calling ILiveEngine::StartFeature.

Requirements

Header: Declared in Uix.h.
