XONLINE_TEAM

typedef struct _XONLINE_TEAM{
    XUID xuidTeam;
    XONLINE_TEAM_PROPERTIES TeamProperties;
    DWORD dwFlags;
    FILETIME CreationTime;
    DWORD dwMemberCount;
} XONLINE_TEAM, *PXONLINE_TEAM;

Members

xuidTeam
XUID that identifies the team.
TeamProperties
An XONLINE_TEAM_PROPERTIES structure that contains additional information about the team.
dwFlags
Flag values that can be the following:
Value Description
XONLINE_TEAM_MSG_RECRUIT Indicates recruiting a team member.
XONLINE_TEAM_MSG_GAME_INVITE Indicates a game invitation.
CreationTime
A FILETIME structure that indicates when the file was created.
dwMemberCount
Number of members on the team.

Requirements

Header: Declared in Xonline.h.
