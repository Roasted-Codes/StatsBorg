XNQOS

Stores the results from asynchronous QoS probes to multiple remote hosts.

typedef struct{
    UINT cxnqos;
    UINT cxnqosPending;
    XNQOSINFO axnqosinfo[1];
} XNQOS;

Members

cxnqos
Total number of remote hosts being probed.
cxnqosPending
Number of remote hosts for which data has not yet been received. When this member is zero, all probes are complete. This member is always less than or equal to the cxnqos member.
axnqosinfo
Pointer to a variable length array of XNQOSINFO structures. The size of this array is specified by the cxnqos member. Each of the structures in this array contains the results for a single remote host.

Requirements
