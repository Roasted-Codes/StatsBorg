XOnlineStorageGetProgress

Retrieves the amount of work performed so far on an Xbox Live storage upload or download operation. The function returns the amount of work performed as both a percentage and a fraction. 

HRESULT XOnlineStorageGetProgress(
    XONLINETASK_HANDLE hTask,
    DWORD *pdwPercentDone,
    ULONGLONG *pqwNumerator,
    ULONGLONG *pqwDenominator
);

Parameters

hTask
[in] Handle to an asynchronous task for a storage operation. For this parameter, use the task handle returned by XOnlineStorageUpload, XOnlineStorageUploadByServerPath, XOnlineStorageUploadFromMemory, XOnlineStorageDownload, XOnlineStorageDownloadToMemory, XOnlineStorageDeleteFile, or XOnlineStorageEnumerate.
pdwPercentDone
[out, optional] Pointer to a DWORD variable that receives the amount of work performed as a percentage. The percentage is represented as a whole-number value between 0 and 100, inclusive.
pqwNumerator
[out, optional] Pointer to a ULONGULONG (quadword) variable that receives the numerator of the fractional amount of work performed.
pqwDenominator
[out, optional] Pointer to a ULONGULONG (quadword) variable that receives the denominator of the fractional amount of work performed.

Return Values

Returns S_OK if successful.

If the function fails, it returns one of the following error codes.

Value Description
E_INVALIDARG The specified task handle is invalid.
Other See Online Errors for information about other possible errors.

Remarks

Use pqwNumerator and qpwDenominator to aggregate progress from multiple tasks into a single progress bar. To get the aggregate progress, add up all the numerators and divide the result by the sum of the denominators.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
