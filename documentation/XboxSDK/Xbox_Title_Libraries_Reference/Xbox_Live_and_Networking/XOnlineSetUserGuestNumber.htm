XOnlineSetUserGuestNumber

Macro that sets the guest number in the dwUserFlags member of an XUID structure.

#define XOnlineSetUserGuestNumber(dwUserFlags,guestNumber) \
   ((dwUserFlags) = ((dwUserFlags) & ~XONLINE_USER_GUEST_MASK) | \
                    (guestNumber & XONLINE_USER_GUEST_MASK))

Parameters

dwUserFlags
[in, out] Type is DWORD. The dwUserFlags member of an XUID structure.
guestNumber
[in] Type is DWORD. The guest number to set.

Return Values

The return value is the new value of dwUserFlags.

The return value is of type DWORD.

Remarks

XOnlineSetUserGuestNumber sets the guest number of a user in preparation for a call to XOnlineLogon. A guest must log on at the same time as the sponsoring user and use the credentials of that sponsor.

Requirements

Header: Declared in Xonline.h.
