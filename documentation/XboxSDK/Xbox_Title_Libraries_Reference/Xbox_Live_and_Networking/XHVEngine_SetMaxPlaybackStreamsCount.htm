IXHVEngine::SetMaxPlaybackStreamsCount

Sets the maximum number of playback streams available for playing decoded voice data.

HRESULT SetMaxPlaybackStreamsCount(
    DWORD dwStreamsCount
);

Parameters

dwStreamsCount
[in] The maximum number of playback streams.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The default number of available playback streams is the maximum number of remote talkers set when the XHV engine was created (the value of the dwMaxRemoteTalkers member of the XHV_RUNTIME_PARAMS structure passed to XHVEngineCreate). If the maximum number of remote talkers was set to greater than XHV_MAX_PLAYBACK_STREAMS, the title should call SetMaxPlaybackStreamsCount with a value less than or equal to XHV_MAX_PLAYBACK_STREAMS in order to properly limit the number of playback streams. XHV_MAX_PLAYBACK_STREAMS is 16, allowing for 4 players per console each listening to 4 different players all talking at the same time.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
