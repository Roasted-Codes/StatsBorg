XONLINE_STARTUP_PARAMS

Contains the startup parameters for Xbox Live.

typedef struct{
    DWORD dwMaxPrivatePool;
} XONLINE_STARTUP_PARAMS, *PXONLINE_STARTUP_PARAMS;

Members

dwMaxPrivatePool
Maximum size, in bytes, of the private heap that the online functions will collectively use. This heap is used for allocations that must occur during asynchronous task pumping, so that the amount of available memory will not change while tasks are being pumped. Available memory can (and will) still change during creation of the asynchronous tasks. If this value is zero, or NULL is specified for the pxosp parameter of XOnlineStartup, then a reasonable default value will be used.

Remarks

If asynchronous tasks are returning XONLINE_E_OUT_OF_MEMORY when being pumped by XOnlineTaskContinue, then likely the size of the private pool needs to be increased when the network is initialized.
