IXHVEngine::RegisterSpeechRecognitionBank

HRESULT RegisterSpeechRecognitionBank(
    VOID *pvSRBank
);

Parameters

pvSRBank
[in] Pointer to an Xbox speech recognition bank (.xsr) that has been loaded into memory by the application. This parameter must be DWORD aligned.

Note    The .xsr file is created by the Win32 speech recognition tool, BuildSpeechRecBank.

Passing NULL effectively unregisters the current speech recognition bank and performs all the operations associated with freeing the memory.

When passing in a pointer to a new bank while a bank is currently loaded, this method performs the equivalent of RegisterSpeechRecognitionBank(NULL) and then normal registration. The local talkers that were in Speech Recognition (SR) mode at that point are switched to inactive mode and no vocabularies are selected. For a local talker to start speech recognition again, SelectVocabulary must be called.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function initializes the speech recognition engine with a specific recognition file. When the engine is in speech recognition mode, this method should be called after calling EnableProcessingMode.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
