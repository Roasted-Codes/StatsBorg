WSACancelOverlappedIO

Cancels all pending overlapped I/O requests on the specified socket.

int WSACancelOverlappedIO(
  SOCKET s 
);

Parameters

s
[in] Descriptor identifying a socket.

Return Values

If no error occurs, the function returns zero. Otherwise, a value of SOCKET_ERROR is returned, and a specific error code can be retrieved by calling WSAGetLastError.

Error code Meaning
WSANOTINITIALISED A successful WSAStartup call must occur before using this function.
WSAENOTSOCK The descriptor is not a socket.

Remarks

WSACancelOverlappedIO cancels all pending overlapped I/O requests on the specified socket. The hEvent member of the WSAOVERLAPPED structure associated with any pending requests will be set to the signaled state. WSAGetOverlappedResult will return FALSE for such structures, and a subsequent call to WSAGetLastError will return WSAECANCELLED.

Requirements

Header: Declared in winsockx.h, for include use Xtl.h.
Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. Use the secure versions of the library file (Xnets.lib or XOnlines.lib) for final testing.
