ILiveEngine::LogOff

Logs off from the Xbox Live service, including logging off all currently logged on users.

HRESULT LogOff();

Parameters

None.

Return Values

Returns S_OK if successful, S_FALSE if not logged on, or an error code otherwise.

Remarks

Active feature screens will exit when LogOff is called.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ILiveEngine::StartFeature, ILiveEngine::GetExitInfo
