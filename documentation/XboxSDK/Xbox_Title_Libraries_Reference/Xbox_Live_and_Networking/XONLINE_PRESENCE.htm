XONLINE_PRESENCE

typedef struct _XONLINE_PRESENCE{
    XUID xuid;
    DWORD dwUserState;
    XNKID SessionID;
    DWORD dwTitleID;
    BYTE StateDataSize;
    BYTE StateData[MAX_STATEDATA_SIZE];
} XONLINE_PRESENCE, *PXONLINE_PRESENCE;

Members

xuid
Unique identifier of the user.
dwUserState

State of the player. This member is a combination of one or more of the following values:
Value Description
XONLINE_PRESENCE_FLAG_JOINABLE The user is in a session that other players can join.
XONLINE_PRESENCE_FLAG_ONLINE The user is online.
XONLINE_PRESENCE_FLAG_RECEIVEDREQUEST The user who enumerated the Presence list has received an XONLINE_MSG_TYPE_FRIEND_REQUEST message from the user represented by this XONLINE_PRESENCE structure.
XONLINE_PRESENCE_FLAG_PLAYING The user is playing a game.
XONLINE_PRESENCE_FLAG_RECEIVEDINVITE The user who enumerated the Presence List has received an XONLINE_MSG_TYPE_GAME_INVITE message from the user represented by this XONLINE_PRESENCE structure.
XONLINE_PRESENCE_FLAG_VOICE The user has voice capability.
XONLINE_PRESENCE_FLAG_RECEIVEDTEAMRECRUIT The user who enumerated the Presence List has received an XONLINE_MSG_TYPE_TEAM_RECRUIT message from the user represented by this XONLINE_PRESENCE structure.
XONLINE_PRESENCE_FLAG_RECEIVEDCOMPREMINDER The user who enumerated the Presence List has received an XONLINE_MSG_TYPE_COMP_REMINDER message from the user represented by this XONLINE_PRESENCE structure.
XONLINE_PRESENCE_FLAG_RECEIVEDCOMPREQUEST The user who enumerated the Presence List has received an XONLINE_MSG_TYPE_COMP_REQUEST message from the user represented by this XONLINE_PRESENCE structure.
XONLINE_PRESENCE_FLAG_RECEIVEDTITLECUSTOM The user who enumerated the Presence List has received an XONLINE_MSG_TYPE_TITLE_CUSTOM from the user represented by this XONLINE_PRESENCE structure.

SessionID
An XNKID structure that contains the session identifier. If this structure is all zeros, the user is not in a session.
dwTitleID
The title identifier. If this value is zero, the title identifier is unknown.
StateDataSize
State data size, in bytes.
StateData
Array of BYTEs containing the state data.
