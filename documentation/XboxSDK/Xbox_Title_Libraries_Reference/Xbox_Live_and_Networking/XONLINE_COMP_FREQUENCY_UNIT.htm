XONLINE_COMP_FREQUENCY_UNIT

Specifies when a reminder should be sent to participants in a competition.

typedef enum {
    XONLINE_COMP_FREQUENCY_MINUTE,
    XONLINE_COMP_FREQUENCY_HOUR,
    XONLINE_COMP_FREQUENCY_DAILY,
    XONLINE_COMP_FREQUENCY_WEEKLY,
} XONLINE_COMP_FREQUENCY_UNIT;

Constants

XONLINE_COMP_FREQUENCY_MINUTE
A reminder is to be sent after a specified number of minutes.
XONLINE_COMP_FREQUENCY_HOUR
A reminder is to be sent after a specified number of hours.
XONLINE_COMP_FREQUENCY_DAILY
A reminder is to be sent daily.
XONLINE_COMP_FREQUENCY_WEEKLY
A reminder is to be sent weekly on a specified day.
