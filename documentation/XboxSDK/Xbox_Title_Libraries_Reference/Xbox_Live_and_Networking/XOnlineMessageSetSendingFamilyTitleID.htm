XOnlineMessageSetSendingFamilyTitleID

Sets the family title ID for messaging.

HRESULT XOnlineMessageSetSendingFamilyTitleID(
    DWORD dwTitleID
);

Parameters

dwTitleID
[in] Family title ID to use for messages. Passing zero as the family title ID will set the family title ID back to the original value.

Return Values

Returns S_OK if successful, an error code otherwise.

XONLINE_E_INVALID_TITLE_ID is returned when the specified title identifier is neither a valid title identifier nor one of the family title identifiers.

Remarks

Setting and resetting the family title ID generates debug output for informational purposes.

Attempting to use this function to enable a family title ID without first including that ID in the title's certificate results in an error message displayed in the debug output as well as the function not working correctly. A family title ID can be included in a title's certificate by opening the project properties dialog box in Visual Studio and adding the family title ID to the alternate title IDs.

Titles should also pass the sending family title ID to XOnlineMessageEnableReceivingFamilyTitleIDs in order to receive notifications of the incoming messages via XOnlineGetNotificationEx and to have those messages returned by XOnlineMessageEnumerate.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
