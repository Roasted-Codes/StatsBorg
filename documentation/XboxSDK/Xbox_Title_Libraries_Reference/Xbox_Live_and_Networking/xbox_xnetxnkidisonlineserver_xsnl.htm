XNetXnKidIsOnlineServer

Determines if the key identifier is from a client-to-server online session.

BOOL XNetXnKidIsOnlineServer(
    XNKID *pxnkid
);

Parameters

pxnkid
Address of the XNKID structure to test.

Return Values

Returns TRUE if the XNKID represents a game server communicating via a security gateway; otherwise returns FALSE.

Requirements

Header: Declared in Winsockx.h.
