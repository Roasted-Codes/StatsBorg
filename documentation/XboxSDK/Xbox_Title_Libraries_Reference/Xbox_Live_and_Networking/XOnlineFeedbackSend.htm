XOnlineFeedbackSend

Sends feedback about a user of the Xbox Live service. 

HRESULT XOnlineFeedbackSend(
    DWORD dwUserIndex,
    XUID xTargetUser,
    XONLINE_FEEDBACK_TYPE FeedbackType,
    const XONLINE_FEEDBACK_PARAMS *pParams,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] The index of the user on the local Xbox console who is submitting the feedback. The value of this parameter should be between 0 and 3.
xTargetUser
[in] A structure of type XUID that contains the Xbox user ID for the user about whom feedback is being sent.
FeedbackType
[in] Value from the XONLINE_FEEDBACK_TYPE enumeration type that describes the feedback itself.
pParams
[in] Pointer to an XONLINE_FEEDBACK_PARAMS structure that contains additional information about the feedback.
hWorkEvent
[in, optional] Handle to an event that will be signaled when there is work to be done on the send-feedback task. The event will be left unsignaled if the task is currently blocked by an asynchronous operation. This parameter can be NULL if no such event is required.
phTask
[out] Pointer to a variable of type XONLINETASK_HANDLE that receives a task handle representing the send-feedback task. This task sends the feedback data to the online service. Pump this task to completion using XOnlineTaskContinue. When the task is complete, call XOnlineTaskClose to close the task.

Return Values

If XOnlineFeedbackSend succeeds, it returns S_OK.

If XOnlineFeedbackSend fails, it returns one of the following error codes.

Value Description
XONLINE_E_FEEDBACK_NULL_TARGET The specified target PUID is NULL. Provide a valid PUID for the user about whom to register feedback
XONLINE_E_FEEDBACK_BAD_TYPE The specified XONLINE_FEEDBACK_TYPE is not valid. Check the feedback type.
XONLINE_E_FEEDBACK_MISMATCH_NICKNAME_SIZE The specified nickname size disagrees with the actual nickname size. Verify that these agree.
E_OUTOFMEMORY Insufficient memory. Free some memory.
XONLINE_E_FEEDBACK_CANNOT_LOG An unspecified problem is preventing the feedback from being logged. Try again later, or give up and don't log the feedback.
Other See Online Errors for information about other possible errors.

The following table lists the possible values that XOnlineTaskContinue may return when called with the handle for the task created by XOnlineFeedbackSend.

Value Description
E_FAIL The feedback data was not uploaded.

Requirements
