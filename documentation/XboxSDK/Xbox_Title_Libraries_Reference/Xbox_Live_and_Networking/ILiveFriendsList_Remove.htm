ILiveFriendsList::Remove

Removes a friend from the user's friends list.

HRESULT Remove(
    const XONLINE_FRIEND *pFriend
);

Parameters

pFriend
[in] Pointer to an XONLINE_FRIEND structure that specifies the friend to remove.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The specified friend will be permanently removed from the friends list (but can be added again later).

Remove can also be used to cancel a pending friend request.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
