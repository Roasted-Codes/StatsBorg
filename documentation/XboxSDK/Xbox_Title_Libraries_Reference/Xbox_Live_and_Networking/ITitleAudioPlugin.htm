ITitleAudioPlugin

ITitleAudioPlugin implements the audio plug-in interface to the Xbox Drop-in UI engine. The engine calls methods on the interface to provide sound for the user interface. Titles can provide their own implementation or use the default ITitleAudioPlugin obtained with UIXCreateAudioPlugin.

ITitleAudioPlugin methods DoWork
PlaySound

The ITitleAudioPlugin interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.
