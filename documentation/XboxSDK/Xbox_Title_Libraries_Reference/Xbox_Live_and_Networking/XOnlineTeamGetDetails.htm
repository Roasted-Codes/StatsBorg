XOnlineTeamGetDetails

Retrieves information about a particular team.

HRESULT XOnlineTeamGetDetails(
    XONLINETASK_HANDLE hTask,
    XUID xuidTeam,
    XONLINE_TEAM *pTeamInfo
);

Parameters

hTask
[in] An XONLINETASK_HANDLE.
xuidTeam
[in] XUID that uniquely identifies the team.
pTeamInfo
[out] Address of an XONLINE_TEAM structure to contain the team information.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
