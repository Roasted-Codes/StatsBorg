XOnlineFriendsGetLatestByFocus

Retrieves a subset of the Friends List. The title describes the subset by specifying a user that should be in the middle of the list returned, and a number of entries to return prior to that user.

HRESULT XOnlineFriendsGetLatestByFocus(
    DWORD dwUserIndex,
    XUID xuidFriendFocus,
    DWORD dwBeforeFocus,
    DWORD *pdwFriendBuffer,
    PXONLINE_FRIEND pFriendBuffer,
    DWORD *pdwFriendsBefore,
    DWORD *pdwFriendsAfter
);

Parameters

dwUserIndex
[in] Controller index of the current user. The user must have been successfully logged on with a previous call to XOnlineLogon.
xuidFriendFocus
[in] User that should be in the middle of the set of entries returned.
dwBeforeFocus
[in] Number of entries to retrieve before the entry specified by the xuidFriendFocus parameter.
pdwFriendBuffer
[in, out] On input, specifies the number of XONLINE_FRIEND structures in the buffer pointed to by the pFriendBuffer parameter. On output, receives the number of XONLINE_FRIEND structures filled with data that were copied to this buffer.
pFriendBuffer
[out] Pointer to a buffer that receives an array of XONLINE_FRIEND structures for members of the Friends List.
pdwFriendsBefore
[out] Receives the number of entries that reside in the Friends List before the entries returned by this function are entered.
pdwFriendsAfter
[out] Receives the number of entries that reside in the Friends List after the entries returned by this function are entered.

Return Values

XOnlineFriendsGetLatestByFocus always succeeds with a return value of S_OK.

Remarks

You must call the XOnlineFriendsStartup function before calling XOnlineFriendsGetLatestByFocus.

The XOnlineFriendsGetLatestByFocus function attempts to fill in the entire buffer provided by the title. If the number of entries from:

(index of focus user) - dwBeforeFocus

to the end of the Friends List isn't sufficient to fill the buffer, the function automatically moves the starting offset upward so as to fill the buffer as much as possible. For this reason, the title should always check the values returned in the pdwFriendsBefore and pdwFriendsAfter parameters.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
