XOnlineMessageSendGetProgress

Retrieves the progress of a message transmission.

HRESULT XOnlineMessageSendGetProgress(
    XONLINETASK_HANDLE hTask,
    DWORD *pdwPercentDone,
    ULONGLONG *pqwNumerator,
    ULONGLONG *pqwDenominator
);

Parameters

hTask

[in] Task handle obtained from XOnlineMessageSend.

Note    Your title should not free the XONLINE_MSG_HANDLE until the send operation completes.

pdwPercentDone
[out, optional] Address of a DWORD. Upon return, the DWORD contains a two-digit integer indicating the percentage of the transmission that has been completed.
pqwNumerator
[out, optional] Address of a ULONGLONG variable. XOnlineMessageSendGetProgress can optionally report the percentage of the transmission that has been completed as a fraction. If the address in pqwNumerator is not NULL, XOnlineMessageSendGetProgress stores the numerator of the fraction into the ULONGLONG variable pointed to by pqwNumerator.
pqwDenominator
[out, optional] Address of a ULONGLONG variable. XOnlineMessageSendGetProgress can optionally report the percentage of the transmission that has been completed as a fraction. If the address in pqwDenominator is not NULL, XOnlineMessageSendGetProgress stores the denominator of the fraction in the ULONGLONG variable pointed to by pqwDenominator.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

In addition to retrieving the progress of messages sent by XOnlineMessageSend, titles can use this function to obtain the progress of any other message sending operation, such as those initiated by XOnlineGameInviteSend and XOnlineFriendsRequestEx.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
