XOnlineArbitrationExtendRound

Extends the time length of an arbitrated round by a specified number of seconds.

HRESULT XOnlineArbitrationExtendRound(
    const XONLINE_ARB_ID *pArbID,
    WORD wMaxSecondsFromNow,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

pArbID
[in] Arbitration session ID.
wMaxSecondsFromNow
[in] Number of seconds that the round is to be extended. Note that this is relative to the current time.
hWorkEvent
[in, optional] Handle to an event that will be signaled when there is work to be done on this task. This parameter can be NULL if no such event is required.
phTask
[out] Address of an XONLINETASK_HANDLE variable. Upon return, this variable contains the handle of the task initiated by this function.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This functions transmits a request to the Xbox Live service to extend the time allowed for the specified round to complete and for reports to be sent. The session must have originally been registered with the XONLINE_ARB_REGISTER_FLAG_TIME_EXTENDABLE flag.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
