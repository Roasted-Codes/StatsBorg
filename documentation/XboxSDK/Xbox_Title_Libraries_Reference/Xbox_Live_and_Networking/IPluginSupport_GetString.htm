IPluginSupport::GetString

Retrieves the Unicode string with the specified string resource identifier from the skin file.

HRESULT GetString(
    DWORD StringResID,
    LPCWSTR *ppString
);

Parameters

StringResID
[in] Resource identifier of the string.
ppString
[out] Pointer to an LPCWSTR variable that receives a pointer to a null-terminated Unicode string.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The returned string pointer should not be written to or freed.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
