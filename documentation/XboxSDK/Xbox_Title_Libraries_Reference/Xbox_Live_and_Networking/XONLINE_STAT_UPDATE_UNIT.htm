XONLINE_STAT_UPDATE_UNIT

Contains statistical information about a unit in a competition.

typedef struct{
    XUID xuidUnitMembers[XONLINE_STAT_MAX_MEMBERS_IN_UNIT];
    DWORD dwLeaderBoardID;
    DWORD dwConditionalUnitIndex;
    DWORD dwNumStats;
    PXONLINE_STAT pStats;
} XONLINE_STAT_UPDATE_UNIT, *PXONLINE_STAT_UPDATE_UNIT;

Members

xuidUnitMembers
Array containing the IDs of all players in the unit.
dwLeaderBoardID

Leaderboard to be updated.

Note    Leaderboards are also referred to as scoreboards.

dwConditionalUnitIndex
One-based index of XONLINE_STAT_CONDITIONAL procedure that determines whether this update occurs, or zero if always updated.
dwNumStats
Number of items in the array specified by the pStats member.
pStats
Array of XONLINE_STAT structures that contain the updated statistical information.

Requirements

Header: Declared in Xonline.h.
