XOnlineOfferingPriceFormat

Converts a price, represented by an XONLINE_PRICE structure, into a display string.

HRESULT XOnlineOfferingPriceFormat(
    XONLINE_PRICE *pPrice,
    LPWSTR lpFormattedPrice,
    DWORD *pdwLength,
    DWORD dwExtendedCharsFilter
);

Parameters

pPrice
[in, out] Pointer to an XONLINE_PRICE structure that specifies the price to be converted to a display string.
lpFormattedPrice
[in, out] Pointer to a buffer that receives the null-terminated Unicode display string for the price.
pdwLength
[in, out] Pointer to a variable that specifies the size, in bytes, of the buffer pointed to by lpFormattedPrice. Upon return, this value will be set to the size, in bytes, of the string (including the terminating null character) written to the buffer.
dwExtendedCharsFilter
[in] Extended (Unicode) currency characters that the title's font is capable of displaying. Any combination of the following flags may be specified.
Value Description Unicode Character
XO_CURRENCY_EUR Euro (Europe) 0x20AC
XO_CURRENCY_GBP Pound (United Kingdom) 0x00A3
XO_CURRENCY_JPY Yen (Japan) 0x00A5
XO_CURRENCY_KRW Won (Korea) 0x20A9

Return Values

If the functions succeeds, the return value is S_OK.

Remarks

If the buffer passed in is not large enough to contain the display string, the function sets the variable pointed to by pdwLength to the size of buffer needed and returns an error code.

The function will use extended (Unicode) currency symbols when needed and if the title indicates support. Otherwise, equivalent currency marker text in ASCII will be used.

The XONLINE_PRICE structure passed in pPrice must be writeable, as the function may alter the bCurrencyFormat member of that structure.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
