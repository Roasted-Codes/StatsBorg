XOnlineMessageDestroy

Releases the memory associated with a message.

HRESULT XOnlineMessageDestroy(
    XONLINE_MSG_HANDLE hMsg
);

Parameters

hMsg
[in] Handle to the message to be destroyed.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function must not be passed a message that is being used in an active send operation.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.

See Also

XOnlineMessageCreate, XOnlineMessageDelete
