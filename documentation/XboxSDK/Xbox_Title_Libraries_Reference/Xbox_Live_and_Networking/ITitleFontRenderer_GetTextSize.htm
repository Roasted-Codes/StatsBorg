ITitleFontRenderer::GetTextSize

Returns the width and height, in pixels, of a specified text string.

HRESULT GetTextSize(
    LPCWSTR pText,
    DWORD *pWidth,
    DWORD *pHeight
);

Parameters

pText
[in] Pointer to a null-terminated Unicode string.
pWidth
[out] Pointer to a DWORD variable that receives the width, in pixels, of the text.
pHeight
[out] Pointer to a DWORD variable that receives the height, in pixels, of the text.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

GetTextSize returns the width and height of the specified string as it would currently be rendered. The width includes intercharacter spacing.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ITitleFontRenderer::SetHeight, ITitleFontRenderer::SetColor
