XOnlineCompetitionSessionRegister

Registers a player for a session in a competition.

HRESULT XOnlineCompetitionSessionRegister(
    const XONLINE_ARB_ID *pArbitrationID,
    WORD wMaxRoundSeconds,
    DWORD dwArbitrationRegisterFlags,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

pArbitrationID
[in] Arbitration session ID. Obtain this value by calling XOnlineArbitrationCreateRoundID.
wMaxRoundSeconds
[in] Maximum length of the round, in seconds.
dwArbitrationRegisterFlags
[in] Composed of one or more arbitration registration flags combined together using a bitwise OR operation. The flags must include XONLINE_ARB_REGISTER_FLAG_USER_COMPETITION or XONLINE_ARB_REGISTER_FLAG_PUBLISHER_COMPETITION, depending on whether the competition is created by a user or a publisher. For more information about arbitration registration flags, see the dwFlags parameter of the XOnlineArbitrationRegister function.
hWorkEvent
[in, optional] Handle of an event that you want to associate with this task. The event is triggered when the task completes. If you do not want to associate a handle with this event, set this parameter to NULL.
phTask
[out] Address of a XONLINETASK_HANDLE variable. Upon return, the XONLINETASK_HANDLE variable contains the handle of the task initiated by this function.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Competitions are composed of one or more sessions. This function registers a player for a particular session in a competition.

Requirements
