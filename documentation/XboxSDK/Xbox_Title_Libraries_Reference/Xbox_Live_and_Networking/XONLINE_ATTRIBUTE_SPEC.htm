XONLINE_ATTRIBUTE_SPEC

Contains the layout of an attribute.

typedef struct _XONLINE_ATTRIBUTE_SPEC{
    DWORD dwType;
    DWORD dwLength;
} XONLINE_ATTRIBUTE_SPEC, *PXONLINE_ATTRIBUTE_SPEC;

Members

dwType
Value that specifies the attribute type. Can be one of the following values:
Type Description
X_ATTRIBUTE_DATATYPE_BLOB Specifies that this attribute contains binary data.
X_ATTRIBUTE_DATATYPE_INTEGER Specifies that this attribute is an integer.
X_ATTRIBUTE_DATATYPE_STRING Specifies that this attribute is a string.
dwLength
Value that specifies the size of the attribute. For string attributes, the length is in wide characters. For binary data, the size is expressed in number of bytes. For integer attributes, this value is set to zero.

Requirements

Header: Declared in Xonline.h.
