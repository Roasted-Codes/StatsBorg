XONLINE_MSG_SEND_RESULT

Contains the results of a message transmission. These results are passed to the sender when the message is received or when it expires.

typedef struct{
    XUID xuidRecipient;
    HRESULT hr;
    DWORD dwMessageID;
} XONLINE_MSG_SEND_RESULT, *PXONLINE_MSG_SEND_RESULT;

Members

xuidRecipient
ID of the sender.
hr
Result of the send operation.
dwMessageID
Message ID, if the send operation was successful.

Requirements

Header: Declared in Xonline.h.

XOnlineMessageSendGetResults, XOnlineMessageRevoke
