ILiveEngine::GetProperty

Retrieves the current value of an ILiveEngine property.

HRESULT GetProperty(
    UIX_PROPERTY_TYPE Property,
    DWORD *pValue
);

Parameters

Property
[in] UIX_PROPERTY_TYPE value specifying the property to retrieve.
pValue
[out] Pointer to a DWORD variable that receives the current value of the property specified in Property.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

GetProperty returns the current value of generic ILiveEngine properties. Property values can be set with SetProperty.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
