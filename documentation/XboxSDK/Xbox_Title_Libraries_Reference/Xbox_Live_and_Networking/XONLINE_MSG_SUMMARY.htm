XONLINE_MSG_SUMMARY

Contains message summary information.

typedef struct{
    XUID xuidSender;
    BYTE bMsgType;
    ULONGLONG qwMessageContext;
    FILETIME ftSentTime;
    DWORD dwMessageID;
    DWORD dwMessageFlags;
    DWORD dwSenderTitleID;
    WORD wExpireMinutes;
    WORD cbDetails;
    char szSenderName[XONLINE_GAMERTAG_SIZE];
} XONLINE_MSG_SUMMARY, *PXONLINE_MSG_SUMMARY;

Members

xuidSender
ID of the sender.
bMsgType
Message type.
qwMessageContext
Message-specific context value.
ftSentTime
Time at which message was sent.

Note    This time is in UTC.

dwMessageID
ID of the message.
dwMessageFlags
Set of one or more message flags combined with a bitwise OR operation. For more information on message flags, see XOnlineMessageCreate.
dwSenderTitleID
ID of the title in which the message was sent.
wExpireMinutes
Number of minutes from the time the message was sent until the message expires.
cbDetails
Size of details blob, excluding downloadable content.
szSenderName
String that holds the Gamertag of the sender.

Requirements

Header: Declared in Xonline.h.

XOnlineMessageDetailsGetResultsSummary, XOnlineMessageSummary, XOnlineMessageEnumerate
