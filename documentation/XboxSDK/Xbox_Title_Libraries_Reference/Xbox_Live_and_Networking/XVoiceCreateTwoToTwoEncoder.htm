XVoiceCreateTwoToTwoEncoder

Returns an IXVoiceEncoder interface for a two-to-two encoder. 

HRESULT XVoiceCreateTwoToTwoEncoder(
    LPXVOICEENCODER *ppMediaObject
);

Parameters

ppMediaObject
[out] Address of a pointer of type LPXVOICEENCODER. This pointer receives the address of the IXVoiceEncoder interface.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The interface is returned with a reference count of one. If only a single thread is accessing the encoder, the title need not call the AddRef method on the interface.

When the title no longer needs the encoder it should call the Release method on the interface. Calling the Release method deallocates the memory associated with the encoder.

The IXVoiceEncoder::ProcessMultiple method for the two-to-two encoder takes two input channels. The encoder compresses the data from each of these input channels and sends the compressed results to the corresponding output channel.

Requirements

Header: Declared in Xvoice.h.

Import Library: Use Xvoice.lib.
