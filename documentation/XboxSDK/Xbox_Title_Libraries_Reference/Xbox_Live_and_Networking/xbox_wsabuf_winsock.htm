WSABUF

The WSABUF structure enables the creation or manipulation of a data buffer.

typedef struct __WSABUF {
  u_long      len;
  char FAR    *buf;
} WSABUF, FAR * LPWSABUF;

Members

len
Length of the buffer.
buf
Pointer to the buffer.

Requirements

Header: Declared in winsockx.h, for include use Xtl.h.
