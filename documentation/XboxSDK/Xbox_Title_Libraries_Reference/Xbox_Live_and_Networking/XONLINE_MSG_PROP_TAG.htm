XONLINE_MSG_PROP_TAG

Builds a property tag from a type and an ID.

#define XONLINE_MSG_PROP_TAG(type,id) ((BYTE)(type) << 8 | (BYTE)(id))

Parameters

type
[in] Type is BYTE. Property type.
id
[in] Type is BYTE. Property ID.

Return Values

This macro evaluates to a property tag.

Requirements

Header: Declared in Xonline.h.
