ILiveFriendsList::GetFriendByXUID

Retrieves detailed information for the friend with the specified user identifier.

HRESULT GetFriendByXUID(
    XUID FriendXUID,
    XONLINE_FRIEND *pFriend
);

Parameters

FriendXUID
[in] An XUID identifying the friend to retrieve from the friends list. Only the qwUserID member of the structure need be set.
pFriend
[out] Pointer to an XONLINE_FRIEND structure that receives the friend information.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
