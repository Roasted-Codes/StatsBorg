XOnlineStatReset

Resets all statistical information for the specified user on the specified leaderboard. 

HRESULT XOnlineStatReset(
    XUID xuid,
    DWORD dwLeaderBoardId,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

xuid
Unique online ID for the user. This user must be logged on from the local Xbox console.
dwLeaderBoardId
Leaderboard on which the user's statistics should be reset. Leaderboards are defined by title developers. If this parameter is zero, the user's statistics are reset on all leaderboards.
hWorkEvent
[in, optional] Optional. Handle to a caller-created event object. This object will be signalled when there is work to do on the task returned in the phTask parameter. The hWorkEvent parameter is optional; the caller may pass NULL for this parameter. A value of NULL indicates that the title will poll XOnlineTaskContinue.
phTask
[out] Address of an XONLINETASK_HANDLE that receives the statistics reset task. Pump this handle to completion by calling XOnlineTaskContinue. Call XOnlineTaskClose to close the task.

Note    To throttle requests sent to the Statistics Service, XOnlineTaskContinue will return XONLINE_E_TASK_ABORTED_BY_DUPLICATE if the task is still being pumped when a second operation is started.

Return Values

If XOnlineStatReset succeeds, the return value is S_OK.

If XOnlineStatReset fails, the return value is one of the following error codes.

Value Description
E_OUTOFMEMORY Insufficient memory. Free some memory.
Other See Online Errors for information about other possible errors.

The following table lists the possible values that XOnlineTaskContinue may return when called with the handle for the task created by XOnlineStatReset.

Value Description
XONLINE_E_STAT_PERMISSION_DENIED The user does not have permission to update the information.
XONLINE_E_STAT_BAD_REQUEST The message is empty, the attribute count equals zero, and so on. Check the values you are passing in. If they are all correct, this could indicate a problem on the server.

Remarks

You can reset statistics only for users that are logged on from the local Xbox console.

For testing on PartnerNet only, if the title specifies a value of zero for the qwUserID member of the xuid parameter, the Statistics service resets all user data on the leaderboard specified by the dwLeaderBoardId parameter. If both the qwUserID member and the dwLeaderBoardId parameter are zero, the Statistics service resets all user data on all leaderboards for that title. These special testing features do not work with the Statistics Sample, however.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
