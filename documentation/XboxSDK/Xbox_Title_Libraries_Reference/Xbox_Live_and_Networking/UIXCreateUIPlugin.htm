UIXCreateUIPlugin

Creates the default Xbox Drop-in UI plug-in interface.

HRESULT UIXCreateUIPlugin(
    ITitleFontRenderer *pFont,
    ITitleUIPlugin **ppUIPlugin
);

Parameters

pFont
[in] Pointer to an ITitleFontRenderer interface that the UI plug-in will use for text rendering.
ppUIPlugin
[out] Address of an ITitleUIPlugin pointer that receives the UI plug-in interface.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

ITitleUIPlugin implements the UI plug-in interface to the Xbox Drop-in UI engine. The engine calls ITitleUIPlugin methods to provide user interface elements and interaction. Titles can provide their own implementation or use the default ITitleUIPlugin obtained with UIXCreateUIPlugin.

To free the UI plug-in interface when it is no longer needed, call Release on the ITitleUIPlugin pointer.
