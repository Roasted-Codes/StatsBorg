XOnlineIsUserVoiceAllowed

Macro that, given the dwUserFlags value from an XUID structure, determines whether a user is permitted to use voice chat.

#define XOnlineIsUserVoiceAllowed(dwUserFlags) \
   (((dwUserFlags) & XONLINE_USER_VOICE_NOT_ALLOWED) == 0)

Parameters

dwUserFlags
[in] Type is DWORD. The dwUserFlags value from an XUID structure contained in an XONLINE_USER structure.

Return Values

The return value is TRUE if the user is permitted to use voice chat; FALSE otherwise.

The return value is of type BOOL.

Remarks

XOnlineIsUserVoiceAllowed accepts the entire dwUserFlags from an XUID structure and determines whether that user is permitted to use voice communication. Titles must enforce this restriction by not allowing restricted users access to voice chat functionality. This restriction reflects the parental control setting in the Xbox Dashboard.

Requirements

Header: Declared in Xonline.h.
