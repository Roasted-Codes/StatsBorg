XNetServerToInAddr

Retrieves the secure IP address from the IP address of the server and the service identifier.

INT XNetServerToInAddr(
    const IN_ADDR ina,
    DWORD dwServiceId,
    IN_ADDR *pina
);

Parameters

ina
An IN_ADDR structure that specifies the IP address of the server.
dwServiceId
Service identifier. The service specified should match one of the services that was specified and granted in the call to XOnlineLogon.
pina
Address of an in_addr structure that receives the secure IP address.

Return Values

Returns zero if successful, an error code otherwise.

Requirements

Header: Declared in Winsockx.h. for include use Xtl.h.

Import Library: Use XOnline.lib. Use the secure versions of the library file (XOnlines.lib) for final testing.
