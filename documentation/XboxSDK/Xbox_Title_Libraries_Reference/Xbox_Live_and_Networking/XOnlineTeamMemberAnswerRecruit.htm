XOnlineTeamMemberAnswerRecruit

Answers a recruiting request made by XOnlineTeamMemberRecruit or XOnlineTeamMemberRecruitByName.

HRESULT XOnlineTeamMemberAnswerRecruit(
    DWORD dwUserIndex,
    XUID xuidTeam,
    XONLINE_PEER_ANSWER_TYPE RecruitAnswer,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the user making the reply.
xuidTeam
[in] XUID that uniquely identifies the team that the reply is being made to.
RecruitAnswer
[in] Accepts or rejects the recruit request. Can be one of the following XONLINE_PEER_ANSWER_TYPE values:
Value Description
XONLINE_PEER_ANSWER_NO Rejects the request.
XONLINE_PEER_ANSWER_YES Accepts the request.
XONLINE_PEER_ANSWER_NEVER Rejects the request indefinitely and prevents the recruiting player from sending to the user again.
hWorkEvent
[in, optional] Handle to an event.
phTask
[out] Pointer to an XONLINETASK_HANDLE returned.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
