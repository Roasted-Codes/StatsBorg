ILivePlayersList::RegisterPlayer

Adds a player to the list displayed by the Players feature.

HRESULT RegisterPlayer(
    CONST ITitlePlayersListItem *pPlayer
);

Parameters

pPlayer
[in] Pointer to an ITitlePlayersListItem that represents the player to add.

Return Values

Returns S_OK if successful, an error code otherwise.
