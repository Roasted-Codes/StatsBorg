LD_DOWNLOADER

The game title passes the LD_DOWNLOADER structure to Downloader.xbe as the pLaunchData parameter of XLaunchNewImage. When Downloader.xbe reboots to the title, it passes this structure back to the title as the pLaunchData parameter of XLaunchNewImage. 

typedef struct _LD_DOWNLOADER{
    DWORD dwID;
    DWORD dwBitFilter;
    BYTE bPremiumLogonPort;
    BYTE Reserved1[3];
    DWORD Reserved2;
    XONLINE_LOGON_STATE LogonState;
    BYTE UserDefined[MAX_LAUNCH_DATA_SIZE - (20 + XONLINE_LOGON_STATE_SIZE)];
} LD_DOWNLOADER, *PLD_DOWNLOADER;

Members

dwID
Value that identifies the structure as downloader launch data. The title sets this member to LAUNCH_DATA_DOWNLOADER_ID before launching Downloader.xbe.
dwBitFilter
Value that identifies a particular category of content. Only content that has a matching bit value is presented to the user. Use the dwBitFilter value to identify content for a particular language, country, or region.
bPremiumLogonPort
Zero-based controller index of the user who will select the downloadable content. The title may specify only a single user; other users will not be able to interact with Downloader.xbe. The specified user may not be a guest.
Reserved1
This member is reserved. Specify zero values for the elements of this array.
Reserved2
This member is reserved. Specify zero for the value of this member.
LogonState
Logon state at the time that the title launches Downloader.xbe.
UserDefined
Opaque data provided by the title. Downloader.xbe passes this data back to the title when it reboots.

Requirements

Header: Declared in Xonline.h.
