XOnlineFriendsGetTitleName

Retrieves the name of the game title that has the specified title ID.

HRESULT XOnlineFriendsGetTitleName(
    DWORD dwTitleId,
    DWORD dwLanguage,
    DWORD dwMaxTitleNameChars,
    LPWSTR lpTitleName
);

Parameters

dwTitleId
[in] Title ID for which to retrieve the name.
dwLanguage

[in] Language in which the title name should be returned. This parameter must be one of the following values:
Value Description
XC_LANGUAGE_ENGLISH English
XC_LANGUAGE_JAPANESE Japanese
XC_LANGUAGE_GERMAN German
XC_LANGUAGE_FRENCH French
XC_LANGUAGE_SPANISH Spanish
XC_LANGUAGE_ITALIAN Italian

To specify the language currently in use by the Xbox console (in other words, the language set by the user through the Xbox Dashboard), pass the return value of XGetLanguage as dwLanguage.

dwMaxTitleNameChars
[in] Number of characters in the buffer pointed to by the wszTitleName parameter, including the trailing null. Must be at least MAX_TITLENAME_LEN.
lpTitleName
[out] Pointer to a Unicode string that receives the name of the title specified by the dwTitleId and dwLanguage parameters. The title allocates the memory for this buffer.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The XONLINE_FRIEND.titleID member contains the ID of the title being played by a user on the friends list.

The title should call this function for each user on the friends list whose XONLINE.friendState includes the XONLINE_FRIENDSTATE_FLAG_ONLINE flag. The title should retrieve and display this title name information whenever it displays the friends list to the user.

XOnlineFriendsGetTitleName returns an empty string if the title name has not yet been cached locally. Titles should continue to poll the function with the desired title ID until the string is non-empty.

The asynchronous task created by the XOnlineFriendsEnumerate function must be active in order to successfully call XOnlineFriendsGetTitleName.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
