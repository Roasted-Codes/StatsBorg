XOnlineMessageSetProperty

Associates a property with a message.

HRESULT XOnlineMessageSetProperty(
    XONLINE_MSG_HANDLE hMsg,
    WORD wPropTag,
    DWORD dwPropValueSize,
    const VOID *pPropValue,
    DWORD dwAttachmentFlags
);

Parameters

hMsg
[in] Message handle obtained from XOnlineMessageCreate.
wPropTag
[in] Property's unique identifier.
dwPropValueSize
[in] Size, in bytes, of the property value.
pPropValue
[in] Pointer to the property's actual value.
dwAttachmentFlags
[in] Flags indicating the attachment type. These are ignored when the property type is not XONLINE_MSG_PROP_TYPE_ATTACHMENT.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Use the XOnlineMessageGetPropType macro to retrieve the property type from the property tag. If the property type is XONLINE_MSG_PROP_TYPE_ATTACHMENT and your title does not pass the XONLINE_MSG_ATTACHMENT_FLAG_DIRECTORY through the dwAttachmentFlags parameter, the property value (in the pPropValue parameter) must point to a memory buffer that remains valid until the send operation completes. If the property type is XONLINE_MSG_PROP_TYPE_ATTACHMENT and your title sets the XONLINE_MSG_ATTACHMENT_FLAG_DIRECTORY flag, the property value must point to an ANSI path string to a local directory whose contents must not be changed until the send operation completes.

Note    Attachments are not available in the November XDK.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
