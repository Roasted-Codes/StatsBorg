ITitleFontRenderer::SetColor

Sets the text color.

HRESULT SetColor(
    D3DCOLOR Color
);

Parameters

Color
[in] A D3DCOLOR value specifying the text color.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

All further text rendered by DrawText should be rendered in Color (until SetColor is called again to set a new color).

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ITitleFontRenderer::DrawText, ITitleFontRenderer::SetHeight, D3DCOLOR
