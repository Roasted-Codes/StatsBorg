XNetXnKidIsOnlinePeer

Determines if the key identifier is from a peer-to-peer online session.

BOOL XNetXnKidIsOnlinePeer(
    XNKID *pxnkid
);

Parameters

pxnkid
Address of the XNKID structure to test.

Return Values

Returns TRUE if the XNKID represents an Xbox on the Internet who is signed into the Xbox Live service; otherwise returns FALSE.

Requirements

Header: Declared in Winsockx.h.
