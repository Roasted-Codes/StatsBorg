XOnlineMessageDownloadAttachmentToMemoryGetResults

Returns the results from calling XOnlineMessageDownloadAttachmentToMemory.

HRESULT XOnlineMessageDownloadAttachmentToMemoryGetResults(
    XONLINETASK_HANDLE hDownloadTask,
    PBYTE *ppbReceivedData,
    DWORD *pdwReceivedDataSize,
    DWORD *pdwTotalDataSize
);

Parameters

hDownloadTask
[in] A task handle obtained from the XOnlineMessageDownloadAttachmentToMemory function.
ppbReceivedData
[out, optional] Pointer into which the data was received. This is the same pointer that was passed to XOnlineMessageDownloadAttachmentToMemory.
pdwReceivedDataSize
[out, optional] Number of bytes actually received.
pdwTotalDataSize
[out, optional] Number of bytes available on the server (pdwReceivedDataSize is equal to pdwTotalDataSize if everything is received).

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function should not be called until pumping of the download task has returned a value other than XONLINETASK_S_RUNNING.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
