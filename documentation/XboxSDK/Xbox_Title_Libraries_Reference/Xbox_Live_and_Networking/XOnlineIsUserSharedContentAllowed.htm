XOnlineIsUserSharedContentAllowed

Macro that, given the dwUserFlags value from an XUID structure, determines whether a user is permitted to use shared content. 

#define XOnlineIsUserSharedContentAllowed(dwUserFlags) \
	(((dwUserFlags) & XONLINE_USER_SHARED_CONTENT_NOT_ALLOWED) == 0)

Parameters

dwUserFlags
[in] Type is DWORD. The dwUserFlags value from an XUID structure contained in an XONLINE_USER structure.

Return Values

The return value is TRUE if the user is permitted to use shared content; FALSE otherwise.

The return value is of type BOOL.

Remarks

XOnlineIsUserSharedContentAllowed accepts the entire dwUserFlags from an XUID structure and determines whether that user is permitted to use shared content. Titles must enforce this restriction by not allowing restricted users access to shared content. This restriction reflects the shared content setting in the Xbox Dashboard.
