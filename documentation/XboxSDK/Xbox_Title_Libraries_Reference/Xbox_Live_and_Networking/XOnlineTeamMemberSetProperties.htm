XOnlineTeamMemberSetProperties

Sets the properties for a team member.

HRESULT XOnlineTeamMemberSetProperties(
    DWORD dwUserIndex,
    XUID xuidTeam,
    XUID xuidTeamMember,
    const XONLINE_TEAM_MEMBER_PROPERTIES *pTeamMemberProperties,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the user setting the team member properties.
xuidTeam
[in] XUID that uniquely identifies the team.
xuidTeamMember
[in] XUID that uniquely identifies the team member.
pTeamMemberProperties
[in] Pointer to an XONLINE_TEAM_MEMBER_PROPERTIES structure containing the properties of the team member.
hWorkEvent
[in, optional] Handle to an event.
phTask
[out] Pointer to an XONLINETASK_HANDLE returned.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
