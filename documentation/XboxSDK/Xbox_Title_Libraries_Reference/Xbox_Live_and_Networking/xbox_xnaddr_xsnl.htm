XNADDR

The XNADDR structure contains an Xbox network address. XNADDR is used by XSNL functions such as XNetXnAddrToInAddr, XNetInAddrToXnAddr, and XNetUnregisterInAddr, and is returned in the Xbox Live XONLINE_MATCH_SEARCHRESULT structure.

typedef struct {
    IN_ADDR ina;
    IN_ADDR inaOnline;
    WORD wPortOnline;
    BYTE abEnet[6];
    BYTE abOnline[20];
} XNADDR;

Members

ina
Value containing the IP address of the host.
inaOnline
Value containing the online IP address. (The value will be zero if the user is not online.)
wPortOnline
Value specifying the online port number. (The value will be zero if the user is not online.)
abEnet
Array of bytes containing the Ethernet MAC address.
abOnline
Array of bytes containing the online identification.

Remarks

XNADDR contains the information required to completely and uniquely identify an Xbox on either a system-link network or on the Internet, even if that Xbox is communicating through network address translation (NAT).

Call XNetXnAddrToInAddr with an XNADDR and a session ID to obtain an IP address to use in Winsock calls.

Titles should treat XNADDR as an opaque structure. The structure members are used internally by XSNL functions and need not be accessed by the title.

Requirements

  Header: Declared in Winsockx.h, for include use Xtl.h.
