XOnlineQueryFindFromIdsGetResults

Obtains the results of the task started by XOnlineQueryFindFromIds.

HRESULT XOnlineQueryFindFromIdsGetResults(
    XONLINETASK_HANDLE hTask,
    DWORD *pdwReturnedResults,
    PVOID pResults
);

Parameters

hTask
[in] Task handle obtained from XOnlineQueryFindFromIds.
pdwReturnedResults
[out] Address of a DWORD. Upon return, the DWORD contains the number of items in the search results.
pResults
[out] Array containing the search results.

Return Values

Returns S_OK if successful, or an error code otherwise.

Remarks

XOnlineQueryFindFromIdsGetRsults should only be called after successful competition of the task started by XOnlineQueryFindFromIds.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
