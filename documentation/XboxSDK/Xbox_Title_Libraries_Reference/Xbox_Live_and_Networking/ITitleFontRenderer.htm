ITitleFontRenderer

ITitleFontRenderer implements the font rendering plug-in interface to the Xbox Drop-in UI engine. The engine calls methods on the interface to render text for the user interface. Titles must provide an ITitleFontRenderer implementation.

ITitleFontRenderer methods DrawText
GetTextSize
SetColor
SetHeight

The ITitleFontRenderer interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

Remarks

A simple, example ITitleFontRenderer implementation using the XFONT interface is provided in Using Xbox Drop-in UI. ITitleFontRenderer methods map nearly directly to XFONT methods.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
