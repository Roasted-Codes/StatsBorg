XOnlineMessageRevoke

Revokes a message that has already been sent.

HRESULT XOnlineMessageRevoke(
    DWORD dwUserIndex,
    DWORD dwNumMsgSendResults,
    const XONLINE_MSG_SEND_RESULT *pMsgSendResults,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player. The player must have been successfully logged on with a previous call to XOnlineLogon.
dwNumMsgSendResults
[in] Number of recipients whose copy of the sent message should be revoked.
pMsgSendResults
[in] Pointer to the XONLINE_MSG_SEND_RESULT array obtained from XOnlineMessageSendGetResults.
hWorkEvent
[in, optional] Handle of the event you want triggered every time there is work to be performed on this task. If you do not want to assign an event, set this parameter to NULL.
phTask
[out] Pointer to an XONLINETASK_HANDLE type. The handle is associated with the task started by this function.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
