XONLINE_SERVICE_INFO

Contains information about a requested online service.

typedef struct{
    DWORD dwServiceID;
    IN_ADDR serviceIP;
    WORD wServicePort;
    WORD wReserved;
} XONLINE_SERVICE_INFO, *PXONLINE_SERVICE_INFO;

Members

dwServiceID
Identifier of the service requested. The value can be one of the following:
Value
XONLINE_BILLING_OFFERING_SERVICE
XONLINE_FEEDBACK_SERVICE
XONLINE_MATCHMAKING_SERVICE
XONLINE_NICKNAME_VERIFICATION_SERVICE
XONLINE_STATISTICS_SERVICE
serviceIP
IN_ADDR structure containing the service IP address.
wServicePort
Port used to communicate with the service.
wReserved
Reserved. Do not use.

Requirements

Header: Declared in Xonline.h.
