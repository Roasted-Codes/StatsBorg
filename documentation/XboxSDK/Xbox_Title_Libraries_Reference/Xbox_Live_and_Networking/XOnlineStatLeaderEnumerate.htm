XOnlineStatLeaderEnumerate

Retrieves a contiguous set of users and their stats from a specified leaderboard. (Note that leaderboards are also referred to as scoreboards.) 

HRESULT XOnlineStatLeaderEnumerate(
    const XUID *pxuidPagePivot,
    DWORD dwPageStart,
    DWORD dwPageSize,
    DWORD dwLeaderboardID,
    DWORD dwNumStatsPerUser,
    const WORD *pStatsPerUser,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

pxuidPagePivot
[in] User whose rank should be as near as possible to the middle of the retrieved listing. The statistics service will attempt to retrieve a list of users in which the number of users with a rank higher than the specified user is the same as the number of users with a rank lower than the specified user. This parameter is optional and may be NULL. If this parameter is specified, the dwPageStart parameter is ignored.
dwPageStart
[in] Rank at which the listing of users should start. A value of one specifies the top of the leaderboard. If the pxuidPagePivot parameter is specified, this parameter is ignored.
dwPageSize
[in] Number of entries to retrieve. This value is equal to the number of elements in the array pointed to by the pUsers parameter of the XOnlineStatLeaderEnumerateGetResults call that is used to read the results.
dwLeaderboardID
[in] Leaderboard from which to retrieve the entries. Leaderboards are defined by the title developer.
dwNumStatsPerUser
[in] Number of statistics to retrieve per user. This value is equal to the number of elements in the array pointed to by the pStatsPerUser parameter.
pStatsPerUser
[in] Pointer to an array of IDs that specify stats. The number of IDs in this array should be equal to the value specified by the dwNumStatPerUser parameter. A stat may be defined by a title or by the Statistics Service. The following table lists the service-defined stats.
Identifier Description Type
XONLINE_STAT_RANK Rank stat. Ranks have a type of XONLINE_STAT_LONG (32-bits).
XONLINE_STAT_RATING Rating stat. Ratings have a type of XONLINE_STAT_LONGLONG (64-bits).
XONLINE_STAT_NICKNAME User nickname. Nicknames have a type of XONLINE_STAT_LPCWSTR (variable-length string).
XONLINE_STAT_LEADERBOARD_SIZE Number of entries on the leaderboard. The leaderboard. size has a type of XONLINE_STAT_LONG (32-bits).
XONLINE_STAT_ATTACHMENT_PATH Path to a stats attachment. Attachment paths have a type of XONLINE_STAT_LPCWSTR (variable-length string).
XONLINE_STAT_ATTACHMENT_SIZE Size of a stats attachment. Attachment sizes have a type of XONLINE_STAT_LONG (variable-length string).
XONLINE_STAT_UNIT_ACTIVITY_COUNTER Activity counter for a unit. Unit activity counters have a type of XONLINE_STAT_LONG (32-bits).
XONLINE_STAT_UNIT_LAST_ACTIVITY_DATE Last activity date for a unit. Unit last activity dates have a type of XONLINE_STAT_LONGLONG (64-bits).
hWorkEvent
[in, optional] Optional. Handle to a caller-created event object. This object will be signaled when there is work to do by calling XOnlineTaskContinue. The caller may pass in NULL instead, indicating that the caller will poll XOnlineTaskContinue itself.
phTask
[out] Address of an XONLINETASK_HANDLE that receives the stats enumeration task. This task sends the enumeration request to the Statistics Service and receives the enumeration response. Pump this handle to completion by calling XOnlineTaskContinue. When the task is complete, call XOnlineStatLeaderEnumerateGetResults to get the enumeration results. Call XOnlineTaskClose to close the task.

Note    To throttle requests sent to the Statistics Service, XOnlineTaskContinue will return XONLINE_E_TASK_ABORTED_BY_DUPLICATE if the task is still being pumped when a second operation is started.

Return Values

If XOnlineStatLeaderEnumerate succeeds, the return value is S_OK.

If XOnlineStatLeaderEnumerate fails, the return value is one of the following error codes.

Value Description
E_OUTOFMEMORY Insufficient memory. Free some memory.
Other See Online Errors for information about other possible errors.

The following table lists the possible values that XOnlineTaskContinue may return when called with the handle for the task created by XOnlineStatLeaderEnumerate.

Value Description
XONLINE_E_STAT_USER_NOT_FOUND The user specified by the pxuidPagePivot parameter does not exist on the leaderboard specified by the dwLeaderboardID parameter.

Remarks

This function must not be called more often than once a second between game sessions. During a game session, this function can only be called once at the beginning, once at the end and every five minutes if the game session lasts longer than five minutes. Note that this function will assert if you violate the usage constraints. Contact your Technical Game Manager or consult the Technical Certification Requirements for details.

This function can be called only with direct user input related to Statistics.

After the online task has completed, use the XOnlineStatLeaderEnumerateGetResults to determine the number of entries actually retrieved.

If the statistics for a user include a statistic of type XONLINE_STAT_ATTACHMENT_PATH, the title may download an "attachment" that contains title-specific content related to the users statistics. Use the XOnlineStorageDownload function to download the attachment.
