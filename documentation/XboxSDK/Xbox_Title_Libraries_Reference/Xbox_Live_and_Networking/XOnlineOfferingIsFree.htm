XOnlineOfferingIsFree

Macro that, given the fOfferingFlags value from an XONLINEOFFERING_INFO structure, determines whether the offering is free.

#define XOnlineOfferingIsFree(x) \
   (((x) & XONLINE_OFFERING_IS_NOT_FREE) == 0)

Parameters

x
[in] Type is WORD. The fOfferingFlags value from an XONLINEOFFERING_INFO structure.

Return Values

The return value is TRUE if the offering is free; FALSE otherwise.

The return value is of type BOOL.

Remarks

XOnlineOfferingIsFree accepts the entire fOfferingFlags from an XONLINEOFFERING_INFO structure and determines whether the offering is free.

Requirements

Header: Declared in Xonline.h.
