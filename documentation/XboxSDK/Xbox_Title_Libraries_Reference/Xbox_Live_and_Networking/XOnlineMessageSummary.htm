XOnlineMessageSummary

Retrieves the summary for a particular message.

HRESULT XOnlineMessageSummary(
    DWORD dwUserIndex,
    DWORD dwMessageID,
    XONLINE_MSG_SUMMARY *pMsgSummary
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the user whose message list is to be enumerated. The user must have been successfully logged on with a previous call to XOnlineLogon.
dwMessageID
[in] Unique ID of the message in the message list whose summary is to be retrieved.
pMsgSummary
[out] Pointer to the XONLINE_MSG_SUMMARY structure into which the message summary is copied.

Return Values

Possible return values for XOnlineMessageSummary are as follows:

Value Description
S_OK The function succeeded.
XONLINE_S_MESSAGE_PENDING_SYNC Returned when the message list is in the process of being retrieved (immediately after logon or when turning summary refresh back on). Pumping the logon task will eventually clear this state and S_OK will be returned.

Remarks

The XOnlineGetNotificationEx function will return the XONLINE_NOTIFICATION_STATE_FLAG_PENDING_SYNC flag while the message list is being retrieved. Titles can check that while pumping the logon task instead of waiting for XOnlineMessageSummary to return S_OK.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
