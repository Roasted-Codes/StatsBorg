XONLINESTORAGE_CONTENT_TYPE

typedef enum {
    XONLINESTORAGE_CONTENT_TYPE_PACKAGE = 0,
    XONLINESTORAGE_CONTENT_TYPE_BLOB = 1,
} XONLINESTORAGE_CONTENT_TYPE;

Constants

XONLINESTORAGE_CONTENT_TYPE_PACKAGE
Use the XOnlineStorageDownload function.
XONLINESTORAGE_CONTENT_TYPE_BLOB
Use the XOnlineStorageDownloadToMemory function.

Remarks

The content type is returned as part of storage enumeration and indicates which function must be used to download the content.
