ITitleUIPlugin::DestroyObject

Called by the Xbox Drop-in UI engine to destroy a UI object.

HRESULT DestroyObject(
    DWORD ObjectID
);

Parameters

ObjectID
[in] Object identifier of the UI object to destroy; must have been previously returned by a call to CreateObject.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

DestroyObject should free all resources associated with the object. A destroyed object's identifier is reusable.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
