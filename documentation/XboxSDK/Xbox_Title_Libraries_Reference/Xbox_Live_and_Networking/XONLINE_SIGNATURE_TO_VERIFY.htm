XONLINE_SIGNATURE_TO_VERIFY

Contains a signature/digest pair to be verified with XOnlineSignatureVerify.

typedef struct{
    DWORD cbDigest;
    PBYTE pbDigest;
    DWORD cbOnlineSignature;
    PBYTE pbOnlineSignature;
} XONLINE_SIGNATURE_TO_VERIFY;

Members

cbDigest
Size of the digest, in bytes, typically returned from XCalculateSignatureGetSize( XCALCSIG_FLAG_DIGEST ).
pbDigest
Pointer to the digest data, typically returned from XCalculateSignatureEnd.
cbOnlineSignature
Size of the online signature, in bytes, typically returned from XCalculateSignatureGetSize( XCALCSIG_FLAG_ONLINE ).
pbOnlineSignature
Pointer to the online signature data, typically returned from XCalculateSignatureEnd.
