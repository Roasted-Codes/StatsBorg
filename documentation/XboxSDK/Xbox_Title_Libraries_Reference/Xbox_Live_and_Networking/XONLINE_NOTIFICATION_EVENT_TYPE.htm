XONLINE_NOTIFICATION_EVENT_TYPE

Defines the values used to specify notification event types.

typedef enum {
    XONLINE_NOTIFICATION_EVENT_SERVICE,
    NUM_XONLINE_NOTIFICATION_EVENT_TYPES,
} XONLINE_NOTIFICATION_EVENT_TYPE;

Constants

XONLINE_NOTIFICATION_EVENT_SERVICE
Notification from the service.
NUM_XONLINE_NOTIFICATION_EVENT_TYPES
Value that specifies the total number of event notification types. This is not a valid value to be passed to any of the notification structures or functions.
