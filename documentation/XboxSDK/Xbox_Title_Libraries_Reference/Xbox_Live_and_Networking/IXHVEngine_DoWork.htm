IXHVEngine::DoWork

HRESULT DoWork();

Parameters

None.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function drives the data pump for the XHV library. It should be called in the main render loop, at least 30 times per second.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
