UIX_NOTIFICATION_PURPOSE

Specifies the purpose for which ILiveEngine::GetNotifications is being called.

typedef enum _UIX_NOTIFICATION_PURPOSE{
    UIX_NOTIFICATION_MENU = 0,
    UIX_NOTIFICATION_IN_GAME_FLASH = 1,
    UIX_NOTIFICATION_FORCE_DWORD = 0x7fffffff,
} UIX_NOTIFICATION_PURPOSE;

Constants

UIX_NOTIFICATION_MENU
Return notifications that must be indicated next to Friends in the main start menu.
UIX_NOTIFICATION_IN_GAME_FLASH
Return in-game notifications that must be indicated in the lower right corner of the screen for three seconds.
UIX_NOTIFICATION_FORCE_DWORD
This value is not used; it forces the enumeration to compile to 32 bits in size.

Remarks

UIX_NOTIFICATION_PURPOSE selects the purpose for which the title is calling ILiveEngine::GetNotifications. The UIX_NOTIFICATION_IN_GAME_FLASH flags are cleared every time ILiveEngine::GetNotifications is called with that notification purpose. The UIX_NOTIFICATION_MENU flags are not cleared until the user invokes the friends list and the title starts the friends feature.
