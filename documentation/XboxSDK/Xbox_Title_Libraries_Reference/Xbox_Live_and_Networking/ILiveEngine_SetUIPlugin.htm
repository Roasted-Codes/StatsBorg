ILiveEngine::SetUIPlugin

Sets the ITitleUIPlugin used by the Xbox Drop-in UI engine. This can be the default UI plug-in obtained with UIXCreateUIPlugin, or a custom plug-in implemented by the title.

HRESULT SetUIPlugin(
    ITitleUIPlugin *pUIPlugin
);

Parameters

pUIPlugin
[in] Pointer to an ITitleUIPlugin interface for the UI plug-in.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

After a UI plug-in has been exposed to the Xbox Drop-in UI engine with SetUIPlugin, the plug-in will begin receiving UI requests through its ITitleUIPlugin interface. The engine immediately calls ITitleUIPlugin::SetPluginSupport on the interface in order to provide UI plug-in support through an IPluginSupport interface.

Requirements

Import Library: Use Xonline.lib.
