XONLINE_STAT_ELO_UNIT

Contains information for calculating the ELO rating for a pair of units (teams) in a competition.

typedef struct{
    XUID xuidUnit1Members[XONLINE_STAT_MAX_MEMBERS_IN_UNIT];
    XUID xuidUnit2Members[XONLINE_STAT_MAX_MEMBERS_IN_UNIT];
    DWORD dwLeaderBoardID;
    DWORD dwConditionalUnitIndex;
    double W;
    double C1;
    double C2;
} XONLINE_STAT_ELO_UNIT, *PXONLINE_STAT_ELO_UNIT;

Members

xuidUnit1Members
Array containing the IDs of all players in unit 1.
xuidUnit2Members
Array containing the IDs of all players in unit 2.
dwLeaderBoardID

Leaderboard to be updated.

Note    Leaderboards are also referred to as scoreboards.

dwConditionalUnitIndex
One-based index of XONLINE_STAT_CONDITIONAL procedure that determines whether this update occurs, or zero if always updated.
W
Score of the winner.
C1
Weighting value of the range in which unit 1's rating currently falls.
C2
Weighting value of the range in which unit 2's rating currently falls.

Requirements

Header: Declared in Xonline.h.
