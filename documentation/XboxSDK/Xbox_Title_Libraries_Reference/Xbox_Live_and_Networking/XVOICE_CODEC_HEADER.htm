XVOICE_CODEC_HEADER

The XVOICE_CODEC_HEADER structure exposes internal Voice API statistics.

typedef struct{
    WORD bMsgNo : 4;
    WORD wSeqNo : 12;
} XVOICE_CODEC_HEADER, *LPXVOICE_CODEC_HEADER;

Members

bMsgNo
Value between 0 and 15 that represents the current talkspurt for a particular voice. When this value reaches 15 and is incremented, it rolls back to 0.
wSeqNo
Value between 0 and 4095 that represents the current frame for a particular voice. When this value is 4095 and is incremented, it rolls back to 0.

Remarks

The XVOICE_CODEC_HEADER structure exposes internal Voice API statistics about where a voice packet lies in the context of a conversation (bMsgNo) and also provides information about which frame the packet is a member of (wSeqNo). bMsgNo is incremented whenever a passage of speech is ended and silence occurs. wSeqNo is incremented with every encoding frame call to IXVoiceEncoder::ProcessMultiple.

Both variables roll back to zero whenever they are incremented past their maximum values (15 for bMsgNo and 4095 for wSeqNo).

Requirements

Header: Declared in Xvoice.h.
