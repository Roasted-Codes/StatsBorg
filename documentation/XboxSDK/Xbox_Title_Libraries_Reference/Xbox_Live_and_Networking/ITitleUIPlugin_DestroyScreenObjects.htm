ITitleUIPlugin::DestroyScreenObjects

Destroys all UI objects on the specified screen.

HRESULT DestroyScreenObjects(
    DWORD ScreenInstance
);

Parameters

ScreenInstance
[in] Screen instance. All objects that were created on the specified screen are destroyed.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

DestroyScreenObjects should free all resources associated with the destroyed objects. A destroyed object's identifier is reusable.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ITitleUIPlugin::CreateObject, ITitleUIPlugin::DestroyObject
