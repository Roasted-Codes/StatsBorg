UIX_EXIT_CODE_TYPE

Feature exit codes returned by ILiveEngine::GetExitInfo in the UIX_EXIT_INFO structure.

typedef enum _UIX_EXIT_CODE_TYPE{
    UIX_EXIT_NONE = 0,
    UIX_EXIT_LOGON_SUCCESSFUL = 1,
    UIX_EXIT_LOGON_FAILED = 2,
    UIX_EXIT_LOGON_USER_EXIT = 3,
    UIX_EXIT_FRIENDS_NORMAL_EXIT = 4,
    UIX_EXIT_FRIENDS_JOIN_GAME = 5,
    UIX_EXIT_FRIENDS_JOIN_GAME_CROSS_TITLE = 6,
    UIX_EXIT_FRIENDS_SIGNED_OUT = 7,
    UIX_EXIT_UITEST_NORMAL_EXIT = 8,
    UIX_EXIT_FORCE_DWORD = 0x7fffffff,
} UIX_EXIT_CODE_TYPE;

Constants

UIX_EXIT_NONE
The feature is still active (has not exited).
UIX_EXIT_LOGON_SUCCESSFUL
The logon feature logged on successfully.
UIX_EXIT_LOGON_FAILED
The logon feature exited but failed to log on. This value is also returned when the network connection is lost and the logon feature is not currently active.
UIX_EXIT_LOGON_USER_EXIT
The user exited the logon feature.
UIX_EXIT_FRIENDS_NORMAL_EXIT
The friends feature exited normally.
UIX_EXIT_FRIENDS_JOIN_GAME
The friends feature exited due to the user accepting a game invite.
UIX_EXIT_FRIENDS_JOIN_GAME_CROSS_TITLE
The friends feature exited due to the user accepting a cross-title game invite.
UIX_EXIT_FRIENDS_SIGNED_OUT
The friends feature exited due to the user selecting the Sign Out button on the friends screen. Before the friends feature exits and returns this code, it calls ILiveEngine::LogOff.
UIX_EXIT_UITEST_NORMAL_EXIT
The UI test feature exited normally. Available in debug builds only.
UIX_EXIT_FORCE_DWORD
This value is not used; it forces the enumeration to compile to 32 bits in size.

Remarks

After a successful logon (UIX_EXIT_LOGON_SUCCESSFUL) the title must call XOnlineGetServiceInfo for each service requested at logon (in the LogonServiceIDs member of UIX_LOGON_PARAMS) and check the returned HRESULTs to verify which services were successfully connected to. If a service critical to the title failed, the title should log off the Xbox Live service.

If a title wishes to determine the problem that caused the UIX_EXIT_LOGON_FAILED return value, the title can check the HRESULT returned in the UIX_EXIT_INFO structure.
