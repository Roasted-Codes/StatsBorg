XONLINE_STAT_CONDITIONAL

typedef struct _XONLINE_STAT_CONDITIONAL{
    XUID xuid;
    DWORD dwLeaderBoardID;
    BYTE bComparisonType;
    XONLINE_STAT StatToCompare;
} XONLINE_STAT_CONDITIONAL, *PXONLINE_STAT_CONDITIONAL;

Members

xuid
XUID structure that uniquely identifies the user.
dwLeaderBoardID
ID of the leaderboard as defined by the title developer.
bComparisonType
Value indicating the type of comparison to perform:
Value Description
XONLINE_STAT_COMPTYPE_EQUAL Current stored stat value equals the specified value.
XONLINE_STAT_COMPTYPE_GREATER Current stored stat value is greater than the specified value.
XONLINE_STAT_COMPTYPE_GREATER_OR_EQUAL Current stored stat value is greater than or equal to the specified value.
XONLINE_STAT_COMPTYPE_LESS Current stored stat value is less than the specified value.
XONLINE_STAT_COMPTYPE_LESS_OR_EQUAL Current stored stat value is less than or equal to the specified value.
XONLINE_STAT_COMPTYPE_EXISTS Current stored stat value exists (specified value is ignored).
XONLINE_STAT_COMPTYPE_NOT_EXISTS Current stored stat value does not exist (specified value is ignored).
StatToCompare
An XONLINE_STAT structure containing the stat information to compare.
