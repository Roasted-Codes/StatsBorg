XNetUnregisterKey

Unregisters session key information previously registered with XNetRegisterKey.

INT XNetUnregisterKey(
    const XNKID *pxnkid,
);

Parameters

pxnkid
Address of an XNKID structure that specifies the session key identifier to unregister.

Return Values

Returns zero if successful, an error code otherwise.

Remarks

The cfgKeyRegMax member of XNetStartupParams determines that maximum number of XNKID/XNKEY pairs that can be registered at one time (the default is 4 key pairs). Game session key identifiers should be unregistered when the game session is finished.

Requirements

Header: Declared in Winsockx.h, for include use Xtl.h.
