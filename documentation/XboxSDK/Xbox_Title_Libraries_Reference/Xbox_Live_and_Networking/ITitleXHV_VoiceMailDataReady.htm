ITitleXHV::VoiceMailDataReady

Notification indicating that the voice mail data is ready.

HRESULT VoiceMailDataReady(
    DWORD dwLocalPort,
    DWORD dwDuration,
    DWORD dwSize
);

Parameters

dwLocalPort
[in] Port of the local talker associated with the voice mail data.
dwDuration
[in] Duration of the recorded message in milliseconds.
dwSize
[in] Size of encoded voice data.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
