ITitleUIPlugin

ITitleUIPlugin implements the UI plug-in interface to the Xbox Drop-in UI engine. The engine calls ITitleUIPlugin methods to provide user interface elements and interaction. Titles can provide their own implementation or use the default ITitleUIPlugin obtained with UIXCreateUIPlugin.

ITitleUIPlugin methods Clear
CreateObject
DestroyObject
DestroyScreenObjects
DoWork
GetObjectState
InsertItem
PassInputToObject
RenderObject
SetObjectState
SetPluginSupport
SetRenderTarget
SetText

The ITitleUIPlugin interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.
