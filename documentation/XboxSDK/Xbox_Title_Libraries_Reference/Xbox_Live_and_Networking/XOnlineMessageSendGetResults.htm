XOnlineMessageSendGetResults

Retrieves the results of a message transmission.

HRESULT XOnlineMessageSendGetResults(
    XONLINETASK_HANDLE hTask,
    XONLINE_MSG_SEND_RESULT *pMsgSendResults
);

Parameters

hTask
[in] Task handle obtained from XOnlineMessageSend.
pMsgSendResults
[out] Address of a XONLINE_MSG_SEND_RESULT structure. Upon return, the XONLINE_MSG_SEND_RESULT structure contains the results of the message transmission.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
