XOnlineOfferingDetailsGetResults

Retrieves the details results requested with a call to XOnlineOfferingDetails.

HRESULT XOnlineOfferingDetailsGetResults(
    XONLINETASK_HANDLE hTask,
    PXONLINEOFFERING_DETAILS pDetails
);

Parameters

hTask
[in] Online task handle returned by the XOnlineOfferingDetails call that initiated the details request.
pDetails
[out] Pointer to an XONLINEOFFERING_DETAILS structure that recieves the details of the offering.

Return Values

If the function succeeds, the return value is S_OK.

Remarks

Call XOnlineOfferingDetailsGetResults only after XOnlineTaskContinue returns XONLINETASK_S_RESULTS_AVAIL for a task created with XOnlineOfferingDetails. If XOnlineTaskContinue returns any value other than XONLINETASK_S_RESULTS_AVAIL, no results are available and XOnlineOfferingDetailsGetResults should not be called.

When the details task is closed with XOnlineTaskClose, the details buffer returned in the pbDetailsBuffer member of pDetails will be deallocated and become invalid if the buffer was allocated by XOnlineOfferingDetails rather than passed in by the title.

Use XOnlineOfferingPriceFormat to convert an XONLINE_PRICE structure to a formatted text string.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
