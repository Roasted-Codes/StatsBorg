XOnlineGetUsers

Retrieves information about the online users stored on the Xbox hard disk and any attached Xbox Memory Units (MUs). 

HRESULT XOnlineGetUsers(
    PXONLINE_USER pUsers,
    DWORD *pdwUsers
);

Parameters

pUsers
[out] Address of an array of XONLINE_USER structures that will receive the online users. The array must be able to hold XONLINE_MAX_STORED_ONLINE_USERS structures of type XONLINE_USER.
pdwUsers
[out] Pointer to a variable of type DWORD that receives the number of XONLINE_USER structures written to pUsers.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

XInitDevices must be called before calling XOnlineGetUsers.

Note     To allow for proper initialization of the devices, do not call XOnlineGetUsers until XGetDeviceEnumerationStatus returns XDEVICE_ENUMERATION_IDLE.

The users returned by XOnlineGetUsers are automatically sorted by the most recent logon time. Users that have logged on more recently appear at the beginning of the array. The title should position the cursor on the most recently logged on user to make it easier for him/her to logon again.

If multiple copies of the user's information are found, the copy with the most recent logon time is returned, and the duplicates are automatically removed from the array of users returned. Additionally, if the user exists on both the hard disk and on an MU, the routine will report that the user came from the hard disk. To determine whether a user was found on the hard disk or on an MU, check the dwUserOptions member of the returned XONLINE_USER structure.

Note     Do not use the xuid.dwUserFlags members in the XONLINE_USER structures returned by XOnlineGetUsers to check for conditions such as voice banning or nickname banning. Instead, use the xuid.dwUserFlags members in the XONLINE_USER structures returned by XOnlineGetLogonUsers.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
