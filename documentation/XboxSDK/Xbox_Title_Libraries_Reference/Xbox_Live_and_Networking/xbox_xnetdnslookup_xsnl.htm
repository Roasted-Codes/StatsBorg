XNetDnsLookup

Initiates a DNS lookup.

INT XNetDnsLookup(
    const char *pszHost,
    WSAEVENT hEvent,
    XNDNS **ppxndns
);

Parameters

pszHost
String value that specifies the host name to lookup.
hEvent
Optional. Handle to an event object.
ppxndns
Address of a pointer to a XNDNS structure that will receive the IP addresses associated with the specified host name.

Return Values

Returns zero if successful, an error code otherwise.

Requirements

Header: Declared in Winsockx.h. for include use Xtl.h.

Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. For online titles, use the secure version of the library file (XOnlines.lib) for final testing (not available for system link play).
