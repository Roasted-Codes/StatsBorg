XNetRegisterKey

Registers the key information for a game session that a title will host or join.

INT XNetRegisterKey(
    const XNKID *pxnkid,
    const XNKEY *pxnkey
);

Parameters

pxnkid
Address of the XNKID structure that contains the session key identifier to register.
pxnkey
Address of the XNKEY structure that contains the session key-exchange key to register.

Return Values

Returns zero if successful, an error code otherwise.

Remarks

The cfgKeyRegMax member of XNetStartupParams determines the maximum number of XNKID/XNKEY pairs that can be registered at one time (the default is 4 key pairs). Game session key identifiers should be unregistered with XNetUnregisterKey when the game session is finished.

Requirements

Header: Declared in Winsockx.h, for include use Xtl.h.
