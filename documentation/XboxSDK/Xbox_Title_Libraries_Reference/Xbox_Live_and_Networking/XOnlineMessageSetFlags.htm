XOnlineMessageSetFlags

Sets the flags associated with a message.

HRESULT XOnlineMessageSetFlags(
    DWORD dwUserIndex,
    DWORD dwMessageID,
    DWORD dwMessageFlagsToSet,
    DWORD dwMessageFlagsToClear,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player. The player must have been successfully logged on with a previous call to XOnlineLogon.
dwMessageID
[in] Index number (zero-based) of the message in the message list whose details will be retrieved.
dwMessageFlagsToSet
[in] Message flags to set. XOnlineMessageSetFlags can set the flags XONLINE_MSG_FLAG_READ and XONLINE_MSG_FLAGS_TITLE_RESERVED. For more information about message flags, see XOnlineMessageCreate.
dwMessageFlagsToClear
[in] Message flags to clear. XOnlineMessageSetFlags can clear the XONLINE_MSG_FLAGS_TITLE_RESERVED flag.
hWorkEvent
[in, optional] Handle of the event you want to trigger every time there is work to be performed on this task. If you do not want to assign an event, set this parameter to NULL.
phTask
[out] Pointer to an XONLINETASK_HANDLE type. The handle is associated with the task started by this function.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
