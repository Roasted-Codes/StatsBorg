XOnlineOfferingDetailsMaxSize

Determines the details buffer size needed in calls to XOnlineOfferingDetails.

DWORD XOnlineOfferingDetailsMaxSize(
    DWORD dwTitleSpecificDataMaxSize
);

Parameters

dwTitleSpecificDataMaxSize
[in] Maximum size, in bytes, of the title-specific details 'blob' data that the title will use. If zero is specified, the maximum allowed blob size will be used in calculating the needed buffer size.

Return Values

The return value is the size, in bytes, of the buffer needed to process the retrieval of details results.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
