XOnlineTitleIdIsSameTitle

Tests whether a title ID specifies a title that is the same as the currently running title.

BOOL XOnlineTitleIdIsSameTitle(
    DWORD dwTitleID
);

Parameters

dwTitleID
[in] Title ID.

Return Values

XOnlineTitleIdIsSameTitle always succeeds with one of the following return codes.

Value Description
S_TRUE The specified title is the same as the current title.
S_FALSE The specified title is NOT the same as the current title.

Remarks

The titleID member of the XONLINE_FRIEND structure contains the ID of the title currently being played by that member of the friends list.

If a player on the local Xbox console accepts an invitation from a member of their friends list, the title should use the XOnlineTitleIdIsSameTitle function to determine whether the inviter is playing a different title. The situation in which the inviter is playing a different title is known as a "cross-title" invitation. For information about how to handle cross-title invitations, see the Friends Overview.

Requirements
