XOnlineQueryGetResultsBufferSize

Retrieves the buffer size required for the results of a search initiated by XOnlineQuerySearch.

DWORD XOnlineQueryGetResultsBufferSize(
    DWORD dwNumResults,
    DWORD dwNumSpecs,
    const XONLINE_ATTRIBUTE_SPEC *pSpecs
);

Parameters

dwNumResults
[in] Number of results per page.
dwNumSpecs
[in] Number of attribute specifiers.
pSpecs
[in] Array of XONLINE_ATTRIBUTE_SPEC structures that specify the type and size, in bytes, of each attribute.

Return Values

Returns the size, in bytes, required for the results buffer.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.

See Also

XOnlineQuerySearch, XOnlineQuerySearchGetResults
