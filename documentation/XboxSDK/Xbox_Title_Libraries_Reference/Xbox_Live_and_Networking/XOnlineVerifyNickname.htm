XOnlineVerifyNickname

Creates an asynchronous task to verify that the specified nickname is valid.

HRESULT XOnlineVerifyNickname(
    LPCWSTR lpNickname,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

lpNickname
[in] Pointer to a null-terminated Unicode string that specifies the nickname to verify.
hWorkEvent
[in, optional] Handle to an event that will be signaled when there is work to be done on the verification task. This parameter can be NULL if no such event is required.
phTask
[out] Address of an XONLINETASK_HANDLE that receives the verification task. This task sends the nickname verification request to the online service and receives the response. This handle must be pumped to completion by XOnlineTaskContinue. Call XOnlineTaskClose to close the task.

Return Values

If the name has been successfully verified with the Xbox Live service, then the task will return XONLINETASK_S_SUCCESS.

If the name failed verification with the Xbox Live service, then the task will return XONLINE_E_OFFERING_NAME_TAKEN.

See Online Errors for information about other possible errors.

Remarks

The returned task handle must be pumped with XOnlineTaskContinue until it returns a value other than XONLINETASK_S_RUNNING.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
