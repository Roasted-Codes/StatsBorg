XOnlineTeamEnumerate

Enumerates a list of teams.

HRESULT XOnlineTeamEnumerate(
    DWORD dwUserIndex,
    DWORD dwTeamCount,
    const XUID *pxuidTeams,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the user requesting the enumeration.
dwTeamCount
[in] Number of teams to enumerate.
pxuidTeams
[in] Pointer to an array of XUID structures containing the team information.
hWorkEvent
[in, optional] Handle to an event.
phTask
[out] Pointer to an XONLINETASK_HANDLE returned.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
