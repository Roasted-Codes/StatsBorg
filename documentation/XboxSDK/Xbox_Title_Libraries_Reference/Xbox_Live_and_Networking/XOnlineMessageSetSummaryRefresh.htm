XOnlineMessageSetSummaryRefresh

Enables or disables automatic summary refreshing, and returns the previous setting. 

BOOL XOnlineMessageSetSummaryRefresh(
    BOOL fEnable
);

Parameters

fEnable
[in] If this parameter is TRUE, automatic summary refreshing is enabled. If fEnable is FALSE, it is disabled.

Return Values

Returns the previous setting.

Remarks

By default, automatic summary refreshing is enabled. Titles can receive notifications that new messages have arrived if this is disabled, However, they will be unable to access any information about the summaries.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
