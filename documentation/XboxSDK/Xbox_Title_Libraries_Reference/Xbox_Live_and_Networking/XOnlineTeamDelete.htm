XOnlineTeamDelete

Deletes a team.

HRESULT XOnlineTeamDelete(
    DWORD dwUserIndex,
    XUID xuidTeam,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the user deleting the team.
xuidTeam
[in] XUID that uniquely identifies the team to be deleted.
hWorkEvent
[in, optional] Handle to an event.
phTask
[out] Pointer to an XONLINETASK_HANDLE returned.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
