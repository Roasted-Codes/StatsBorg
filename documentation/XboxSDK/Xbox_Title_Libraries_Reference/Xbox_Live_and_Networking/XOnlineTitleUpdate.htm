XOnlineTitleUpdate

Begins the download and installation of an available update for the game title.

HRESULT XOnlineTitleUpdate(
    DWORD dwContext
);

Parameters

dwContext
[in] DWORD value that is passed to the updated title. This value is passed to the updated title in the dwContext member of the LD_FROM_UPDATE structure returned by XGetLaunchInfo. The meaning of this value is defined by the title.

Return Values

If the function succeeds, the function does not return; the Xbox reboots into the updater (update.xbe).

Remarks

When an update is available for a title, the XOnlineLogon task will fail with XONLINE_E_LOGON_UPDATE_REQUIRED. The title will then need to perform an update before XOnlineLogon will succeed and allow the title to make other online calls.

XOnlineTitleUpdate reboots the Xbox console to download and install the update for the game title, and then reboots to the newly updated game title. The update persists on the Xbox console until the next update is available for the title.

Before calling XOnlineTitleUpdate, the title should inform the player that an update is about to occur and that the Xbox console will reboot in order to perform the update. The player should be given the option to forego updating, and, if desired, to proceed playing the game without invoking any online functionality.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
