ILivePlayersList::SetFilterFlags

Sets flags that filter which players are shown on the Players List.

HRESULT SetFilterFlags(
    CONST DWORD Flags
);

Parameters

Flags
[in] A bitmask of flags that determines the displayed players. The mask is ANDed with the player flags returned by ITitlePlayersListItem::GetBitflags, and players with a nonzero result are displayed. If Flags is zero, all players (regardless of flags) are displayed.

Return Values

Returns E_FAIL (even when successful).

Requirements

Header: Declared in Uix.h.

Import Library: Use Uix.lib.
