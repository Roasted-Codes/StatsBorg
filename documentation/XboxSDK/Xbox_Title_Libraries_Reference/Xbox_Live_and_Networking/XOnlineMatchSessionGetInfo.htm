XOnlineMatchSessionGetInfo

Obtains information for the specified online match session.

HRESULT XOnlineMatchSessionGetInfo(
    XONLINETASK_HANDLE hTask,
    XNKID *pSessionID,
    XNKEY *pKeyExchangeKey
);

Parameters

hTask
[in] Handle to the creation task for this session. Pass the handle generated by XOnlineMatchSessionCreate.
pSessionID
[out] Address of an XNKID structure to receive the session ID for the online-game session.
pKeyExchangeKey
[out] Address of an XNKEY structure to receive the key-exchange key for this online-game session.

Return Values

XOnlineMatchSessionGetInfo always succeeds with a return value of S_OK.

Remarks

Only the process that called XOnlineMatchSessionCreate should call XOnlineMatchSessionGetInfo. Call this function when the task returned by XOnlineMatchSessionCreate has been pumped to completion.

Use the returned session ID in subsequent calls to XOnlineMatchSessionUpdate.

This function is analagous to the system link function XNetCreateKey. For online sessions, the Matchmaking service creates the session ID and key-exchange key.

Use the returned XNKID and XNKEY with the XNetRegisterKey function to register a secure networking session.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
