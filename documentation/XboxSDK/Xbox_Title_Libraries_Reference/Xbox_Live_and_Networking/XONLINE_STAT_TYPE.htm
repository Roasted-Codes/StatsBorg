XONLINE_STAT_TYPE

The values of the XONLINE_STAT_TYPE enumeration type define the possible types for statistics used by the statistics service. 

typedef enum {
    XONLINE_STAT_NONE,
    XONLINE_STAT_LONG,
    XONLINE_STAT_LONGLONG,
    XONLINE_STAT_DOUBLE,
    XONLINE_STAT_LPCWSTR,
} XONLINE_STAT_TYPE;

Constants

XONLINE_STAT_NONE
Null Type. This type is not used.
XONLINE_STAT_LONG
Integer; 32-bits.
XONLINE_STAT_LONGLONG
Big integer; 64-bits.
XONLINE_STAT_DOUBLE
Real number; 64-bits.
XONLINE_STAT_LPCWSTR
Stores a variable-length string. This type should be used only for storing user nicknames and storing paths to statistics attachments.

Requirements

Header: Declared in XOnline.h.
