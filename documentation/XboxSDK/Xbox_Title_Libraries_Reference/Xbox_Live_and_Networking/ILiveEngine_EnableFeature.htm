ILiveEngine::EnableFeature

Enables an Xbox Drop-in UI feature.

HRESULT EnableFeature(
    UIX_FEATURE *FeatureID
);

Parameters

FeatureID

[in] Feature to enable. Must be one of the following UIX_FEATURE values.
Value Description
UIX_FRIENDS_FEATURE The Friends feature.
UIX_LOGON_FEATURE The logon feature.
UIX_PLAYERS_FEATURE The Players feature.
UIX_UITEST_FEATURE The UI test feature, for testing a custom ITitleUIPlugin. Available in debug builds only.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

A feature must be enabled in order to access its functionality and to start it. Typically, features are enabled at title initialization, then started later as needed. Enabling a feature links the appropriate code in with the title, but doesn't allocate any memory.
