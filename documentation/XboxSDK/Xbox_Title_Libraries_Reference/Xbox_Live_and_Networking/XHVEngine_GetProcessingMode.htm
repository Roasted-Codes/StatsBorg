IXHVEngine::GetProcessingMode

HRESULT GetProcessingMode(
    DWORD dwLocalPort,
    XHV_PROCESSING_MODE *pProcessingMode
);

Parameters

dwLocalPort
[in] Controller port of the local talker.
pProcessingMode
[in] An XHV_PROCESSING_MODE value returned. See the SetProcessingMode function for a list of possible values.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
