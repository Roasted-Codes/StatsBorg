XNetInAddrToString

Converts the specified Internet address to a string.

void XNetInAddrToString(
    const IN_ADDR ina,
    char *pchBuf,
    INT cchBuf
);

Parameters

ina
IN_ADDR structure that contains the Internet address to transform.
pchBuf
Address of a buffer that receives the string.
cchBuf
Number of characters in the buffer.

Return Values

Returns zero if successful, an error code otherwise.

Requirements

Header: Declared in Winsockx.h, for include use Xtl.h.

Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. Use the secure versions of the library file (Xnets.lib or XOnlines.lib) for final testing.
