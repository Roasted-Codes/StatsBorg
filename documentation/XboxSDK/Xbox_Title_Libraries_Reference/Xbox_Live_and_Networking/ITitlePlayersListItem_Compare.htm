ITitlePlayersListItem::Compare

Compares two ITitlePlayersListItem instances to determine sort order within the Players List.

INT Compare(
    CONST ITitlePlayersListItem *unnamed
);

Parameters

unnamed
Pointer to the ITitlePlayersListItem instance to compare against.

Return Values

Returns zero if the two instances are equal.

Returns 1 if the called instance is greater than the passed instance.

Returns −1 if the called instance is less than the passed instance.

Remarks

Titles can sort the Players List using whatever criteria desired (name, score, time in game, and so on).

Requirements

Header: Declared in Uix.h.

Import Library: Use Uix.lib.
