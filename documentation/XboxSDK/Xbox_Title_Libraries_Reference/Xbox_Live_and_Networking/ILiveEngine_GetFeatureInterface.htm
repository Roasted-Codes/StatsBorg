ILiveEngine::GetFeatureInterface

Retrieve a feature interface for a specified feature.

HRESULT GetFeatureInterface(
    UIX_FEATURE FeatureID,
    const VOID *pParam,
    VOID **ppFeatureInterface
);

Parameters

FeatureID

[in] Feature to retrieve an interface for. Must be one of the following UIX_FEATURE values.
Value Description
UIX_PLAYERS_FEATURE Retrieve an ILivePlayersList interface.

pParam
[in] Parameters to pass to the feature. Currenty unused; should be NULL.
ppFeatureInterface
[out] Pointer to a VOID pointer that will receive the feature interface.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Titles can use Xbox Drop-in UI Extensibility to implement custom features. GetFeatureInterface calls a feature's IUIXFeature::GetFeatureInterface method, in which case the meaning of pParam is determined by the custom feature.

Requirements

Header: Declared in Uix.h.

Import Library: Use Uix.lib.
