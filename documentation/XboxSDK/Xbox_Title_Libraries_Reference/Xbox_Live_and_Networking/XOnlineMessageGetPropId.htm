XOnlineMessageGetPropId

Extracts a property ID from a property tag.

#define XOnlineMessageGetPropId(tag) ((WORD)(tag) & 0xFF)

Parameters

tag
[in] Type is WORD. Property tag.

Return Values

This macro evaluates to the property ID.

Requirements

Header: Declared in Xonline.h.
