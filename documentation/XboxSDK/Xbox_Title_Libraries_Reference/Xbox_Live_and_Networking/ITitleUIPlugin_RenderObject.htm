ITitleUIPlugin::RenderObject

Renders a specified UI object to the current render target.

HRESULT RenderObject(
    DWORD ObjectID
);

Parameters

ObjectID
[in] Object identifier, previously returned by CreateObject.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The plug-in's current render target is set with SetRenderTarget.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ITitleUIPlugin::SetRenderTarget, ITitleUIPlugin::CreateObject
