ITitleXHV::SpeechRecognized

HRESULT SpeechRecognized(
    DWORD dwLocalPort,
    XHV_SR_ITEM *pSRArray,
    DWORD dwSRItemsCount
);

Parameters

dwLocalPort
[in] Port of the local talker associated with the speech recognition callback.
pSRArray
[in] Pointer to an array of XHV_SR_ITEM structures containing the word IDs and confidence levels of the recognized words.
dwSRItemsCount
[in] The number of structures in the pSRArray array (that is, the number of recognized words). If this value is zero, then no words were recognized.

Note    For version 1.0 of XHV, only a single recognized word will be returned.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
