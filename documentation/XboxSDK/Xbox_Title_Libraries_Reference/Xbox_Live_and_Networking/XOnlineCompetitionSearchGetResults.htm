XOnlineCompetitionSearchGetResults

Retrieves the results of the task begun by XOnlineCompetitionSearch.

HRESULT XOnlineCompetitionSearchGetResults(
    XONLINETASK_HANDLE hTask,
    DWORD *pdwTotalItemsInSearchResult,
    DWORD *pdwItemsReturned,
    DWORD *pdwResultBufferSize,
    PBYTE pbResultBuffer
);

Parameters

hTask
[in] Task handle received from XOnlineCompetitionSearch.
pdwTotalItemsInSearchResult
[out] Address of a DWORD. Upon return, the DWORD holds the total number of items in search result.
pdwItemsReturned
[out] Address of a DWORD. Upon return, the DWORD holds the number of items returned by the search.
pdwResultBufferSize
[in, out] Address of a DWORD that holds the size, in bytes, of the buffer pointed to by pbResultsBuffer. If XOnlineCompetitionSearchGetResults returns E_INSUFFICIENT_BUFFER, the DWORD contains the number of bytes necssary to hold the results list.
pbResultBuffer
[in, out] Array that contains the result list.

Return Values

Returns S_OK if successful, an error code otherwise. Possible error codes include E_INSUFFICIENT_BUFFER. This error indicates that the buffer size specified in pdwResultBufferSize is too small to hold the results.
