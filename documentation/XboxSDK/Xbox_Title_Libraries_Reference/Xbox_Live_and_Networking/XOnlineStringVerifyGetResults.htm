XOnlineStringVerifyGetResults

Retrieves the results from a call to XOnlineStringVerify.

HRESULT XOnlineStringVerifyGetResults(
    XONLINETASK_HANDLE hTask,
    WORD wNumResults,
    HRESULT *pResults
);

Parameters

hTask
[in] An XONLINETASK_HANDLE returned from a call to XOnlineStringVerify.
wNumResults
[in] The number of results to retrieve. This should be the same as the number of strings passed to XOnlineStringVerify.
pResults
[in, out] Array of HRESULT values indicating the success or failure of the verified strings. The order of HRESULTs returned corresponds exactly to the order of strings specified to XOnlineStringVerify.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
