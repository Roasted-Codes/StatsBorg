XOnlineFriendsStartup

Initializes the Friends List manager and allows the use of Friends Lists. 

HRESULT XOnlineFriendsStartup(
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

hWorkEvent
[in, optional] Handle to an event that will be signaled when there is work to be done on the asynchronous task. This parameter can be NULL if no such event is required.
phTask
[out] Pointer to a variable of type XONLINETASK_HANDLE that receives a task handle. Use this handle in calls to XOnlineTaskContinue to perform work on friend requests (both adding and removing), game invitations, and invitation revocations.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The task returned by XOnlineFriendsStartup is used to send and receive friend requests, game invitations, and invitation revocations. It is not used to enumerate and display the Friends List; for those operations, use the task returned by XOnlineFriendsEnumerate.

Because game invitations must be handled even during gameplay, the task returned by XOnlineFriendsStartup should be kept active throughout the game.

Unlike most asynchronous tasks, the task created by this function will never complete on its own, and will occassionally have work which must be done to process Friends List tasks such as handling game invitations, adding and removing friends from Friends Lists, and so on. The title should continue to perform work on this task by calling XOnlineTaskContinue for this task ( typically once per frame) until either the title no longer needs any Friends List functionality, or XOnlineTaskContinue returns an error code indicating that an error has occurred with this task.

When Friends List functionality is no longer needed (and no further Friends List functions will be called), pump the task handle until XOnlineTaskContinue returns XONLINETASK_S_RUNNING_IDLE, then close the asynchronous task by calling XOnlineTaskClose.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
