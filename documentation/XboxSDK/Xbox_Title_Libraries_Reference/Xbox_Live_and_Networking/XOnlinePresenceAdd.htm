XOnlinePresenceAdd

Adds one or more users to a presence subscription.

HRESULT XOnlinePresenceAdd(
    XONLINETASK_HANDLE hTask,
    DWORD dwGroupID,
    DWORD dwUserCount,
    XUID *pxuidUsers
);

Parameters

hTask
[in] An XONLINETASK_HANDLE returned from XOnlinePresenceInit.
dwGroupID
[in] ID of the group being added.
dwUserCount
[in] Number of users in the group being added.
pxuidUsers
[in] Pointer to the XUID information for the users being added.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
