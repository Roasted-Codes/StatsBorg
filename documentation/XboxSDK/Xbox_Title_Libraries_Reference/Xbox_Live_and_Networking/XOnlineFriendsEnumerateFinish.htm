XOnlineFriendsEnumerateFinish

Caches the current Friends list to the hard disk of the local Xbox console. 

HRESULT XOnlineFriendsEnumerateFinish(
    XONLINETASK_HANDLE hTask
);

Parameters

hTask
[in] Handle to the online task that was returned by XOnlineFriendsEnumerate.

Return Values

XOnlineFriendsEnumerateFinish always succeeds with a return value of S_OK.

Remarks

Call the XOnlineFriendsEnumerateFinish function when the title no longer needs to display the Friends list. Call XOnlineFriendsEnumerateFinish on the task handle returned by XOnlineFriendsEnumerate before calling XOnlineTaskClose on this handle.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
