ITitleUIPlugin::CreateObject

Called by the Xbox Drop-in UI engine to create a new UI object.

HRESULT CreateObject(
    UIX_OBJECT_TYPE ObjectType,
    DWORD ScreenInstance,
    DWORD ScreenResID,
    DWORD ObjectResID,
    DWORD *pObjectID
);

Parameters

ObjectType
[in] A UIX_OBJECT_TYPE value specifying the type of object to create.
ScreenInstance
[in] Screen instance on which the object is created. A unique screen instance is associated with all objects created for a screen, and is used in DestroyScreenObjects when destroying objects created for that screen.
ScreenResID
[in] Resource identifier of the screen on which the object is created.
ObjectResID
[in] Resource identifier for the object. The plug-in uses this identifier in a call to IPluginSupport::GetLayout to retrieve the object's layout from the skin file.
pObjectID
[out] Pointer to a DWORD variable that receives the object identifier of the newly created object.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ITitleUIPlugin::DestroyObject, ITitleUIPlugin::DestroyScreenObjects, IPluginSupport::GetLayout, UIX_OBJECT_TYPE
