XONLINE_PRICE

Contains the price of an online offering.

typedef struct{
    DWORD dwWholePart;
    DWORD dwFractionalPart;
    WCHAR rgwchISOCurrencyCode[3];
    BOOL fOfferingIsFree;
    XONLINE_TAX_TYPE Tax;
    BYTE bCurrencyFormat;
} XONLINE_PRICE, *PXONLINE_PRICE;

Members

dwWholePart
Value that contains the whole currency amounts of the price (for example, number of dollars in US currency).
dwFractionalPart
Value that contains the fractional portion of the price (for example, number of cents in US currency).
rgwchISOCurrencyCode
Array of wide characters that contains the currency code for the price.
fOfferingIsFree
Value indicating whether the offering is free or not (if the offering is not free, it needs to be purchased).
Tax

Type of tax that applies to the offering. Can be one of the following values:
Value Description
NO_TAX Net pre-tax price (tax will be added later)
DEFAULT Net price with default tax for the title's region included
GST Net price with general sales tax included
VAT Net price with value added tax included
TAX_NOT_APPLICABLE Net price (tax not applicable)

bCurrencyFormat
BYTE value that indicates the currency format of the price.

Requirements

Header: Declared in Xonline.h.
