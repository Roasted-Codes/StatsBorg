XONLINE_FEEDBACK_PARAMS

The XONLINE_FEEDBACK_PARAMS structure provides additional information about feedback sent using the XOnlineFeedbackSend function. 

typedef struct{
    LPCWSTR lpStringParam;
} XONLINE_FEEDBACK_PARAMS, *PXONLINE_FEEDBACK_PARAMS;

Members

lpStringParam
Pointer to a null-terminated Unicode string. The value of this string depends on the value of the FeebackType parameter to XOnlineFeedbackSend. If the feedback is XONLINE_FEEDBACK_POS_STATS_ATTACHMENT or XONLINE_FEEDBACK_NEG_STATS_ATTACHMENT, the string specifies the storage path of the attachment. For other feedback types, the value is the nickname of the user about whom feedback is being sent.

Requirements
