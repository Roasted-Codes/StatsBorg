XONLINE_NOTIFICATION_TYPE

Specifies the notification types that can be tested for using the XOnlineGetNotification function.

typedef enum {
    XONLINE_NOTIFICATION_FRIEND_REQUEST = 0,
    XONLINE_NOTIFICATION_GAME_INVITE = 1,
    XONLINE_NOTIFICATION_NEW_GAME_INVITE = 2,
    XONLINE_NOTIFICATION_GAME_INVITE_ANSWER = 3,
    XONLINE_NOTIFICATION_NUM = 4,
} XONLINE_NOTIFICATION_TYPE;

Constants

XONLINE_NOTIFICATION_FRIEND_REQUEST
A friend request has been received. Titles should use XOnlineFriendsEnumerate to determine more information. Titles can use this to detect the first new friends request received during game play or to indicate a new friends request is present when displaying a menu.
XONLINE_NOTIFICATION_GAME_INVITE
A game invitation has been received. Titles should use XOnlineFriendsEnumerate to determine more information. Titles can use this to detect the first new game invitation received during game play or to indicate that a game invitation is present when displaying a menu.
XONLINE_NOTIFICATION_NEW_GAME_INVITE
A new game invitation has been received. Titles should use XOnlineFriendsEnumerate to determine more information. Titles can use this to detect each invitation received throughout gameplay so the player can be notified via a glyph.
XONLINE_NOTIFICATION_GAME_INVITE_ANSWER
A friend has responded to a game invitation sent by the specified user.
XONLINE_NOTIFICATION_NUM
The number of public notification types.
