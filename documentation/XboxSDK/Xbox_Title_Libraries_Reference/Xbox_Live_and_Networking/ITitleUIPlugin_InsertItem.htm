ITitleUIPlugin::InsertItem

Inserts a new text into a list box.

HRESULT InsertItem(
    DWORD ObjectID,
    DWORD ItemIndex,
    DWORD *pReturnIndex
);

Parameters

ObjectID
[in] Object identifier of the list box, previously returned by CreateObject.
ItemIndex
[in] Index in the list box (zero-based) at which the new item should be inserted. Specify UIX_INVALID_VALUE, or a value greater than the number of items in the list box, to append the item to the end of the list box.
pReturnIndex
[out] Pointer to a DWORD variable that receives the index of the inserted item.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The newly inserted item has no text. To set the item's text, call SetText with the index returned in pReturnIndex.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ITitleUIPlugin::SetText, ITitleUIPlugin::CreateObject
