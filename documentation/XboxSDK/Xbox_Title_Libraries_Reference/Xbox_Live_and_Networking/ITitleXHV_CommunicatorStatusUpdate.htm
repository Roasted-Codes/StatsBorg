ITitleXHV::CommunicatorStatusUpdate

HRESULT CommunicatorStatusUpdate(
    DWORD dwLocalPort,
    XHV_VOICE_COMMUNICATOR_STATUS communicatorStatus
);

Parameters

dwLocalPort
[in] Port causing the communicator callback.
communicatorStatus
[in] An XHV_VOICE_COMMUNICATOR_STATUS value describing the status of the port.

Return Values

Returns S_OK if successful, an error code otherwise.
