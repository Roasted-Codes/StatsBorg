ILiveFriendsList::Request

Sends a request to another user asking that the two users be placed on each other's friends list.

HRESULT Request(
    XUID UserXUID
);

Parameters

UserXUID
[in] An XUID identifying the user to send the friend request to.

Return Values

Returns S_OK if the request was successfully sent or the user is already a friend; returns an error code otherwise.

For possible error codes, see XOnlineFriendsRequest.

Remarks

The user receiving the request must accept the request in order for the two users to be placed on each other's friends list.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
