XOnlineCompetitionTopologySingleEliminationGetResults

Retrieves the results of the task begun by XOnlineCompetitionTopologySingleElimination.

HRESULT XOnlineCompetitionTopologySingleEliminationGetResults(
    XONLINETASK_HANDLE hTask,
    PXONLINE_COMP_TOPOLOGY_SE_RESULTS *ppTopologyResults
);

Parameters

hTask
[in] Task handle obtained from XOnlineCompetitionTopologySingleElimination.
ppTopologyResults
[out] Address of an XONLINE_COMP_TOPOLOGY_SE_RESULTS structure. Upon return, the structure contains the results of the topology search.

Return Values

Returns S_OK if successful, an error code otherwise.
