XOnlineGameInviteSend

Sends a game invitation to a specified list of players.

HRESULT XOnlineGameInviteSend(
    DWORD dwUserIndex,
    DWORD dwPeerCount,
    const XUID *pxuidPeersToInvite,
    XNKID SessionID,
    DWORD dwFlags,
    XONLINE_MSG_HANDLE hMsg,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the user sending the game invitation. The user must have been successfully logged on with a previous call to XOnlineLogon.
dwPeerCount
[in] Number of peers to invite. This is the number of XUID structures represented by the pxuidPeersToInvite argument.
pxuidPeersToInvite
[in] Pointer to an array of XUID structures representing the peers to invite.
SessionID
[in] An XNKID structure representing the session.
dwFlags
[in] This value is currently unused and must be zero.
hMsg
[in] Pointer to an XONLINE_MSG_HANDLE specifying an XONLINE_MSG_TYPE_GAME_INVITE message, with an optional voice attachment. This parameter may be NULL if a custom invitation message is not needed.
hWorkEvent
[in, optional] Handle to an event that will be signaled when there is work to be done on this task. This parameter can be NULL if no such event is required.
phTask
[out] Address of an XONLINETASK_HANDLE that must be pumped to completion by XOnlineTaskContinue. Call XOnlineTaskClose to close the task.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function is similar to the XOnlineFriendsGameInvite function, with the exception that the sender can be a teammate or fellow peer in a competition.

The operation returned from XOnlineGameInviteSend may be passed to XOnlineMessageSendGetProgress to retrieve a numeric representation of the status of the invite message upload.

The title should not destroy the message object it specified through the hMsg parameter until the XOnlineGameInviteSend operation is complete.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
