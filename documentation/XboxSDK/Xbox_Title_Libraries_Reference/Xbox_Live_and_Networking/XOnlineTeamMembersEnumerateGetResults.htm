XOnlineTeamMembersEnumerateGetResults

Retrieves information resulting from a call to XOnlineTeamMembersEnumerate.

HRESULT XOnlineTeamMembersEnumerateGetResults(
    XONLINETASK_HANDLE hTask,
    DWORD *pdwTeamMemberCount,
    XUID *pxuidTeamMembers
);

Parameters

hTask
[in] An XONLINETASK_HANDLE returned from XOnlineTeamMembersEnumerate.
pdwTeamMemberCount
[out] Address of a DWORD containing the number of team members.
pxuidTeamMembers
[out] Address of the XUID(s) that uniquely identify the team members.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
