ILivePlayersList::Refresh

Refreshes the Players List displayed by the Players feature.

HRESULT Refresh();

Parameters

None.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The title should call Refresh to refresh the Players List when player information held by the title has changed (for example, a player's voice status or flags).

Requirements

Header: Declared in Uix.h.

Import Library: Use Uix.lib.
