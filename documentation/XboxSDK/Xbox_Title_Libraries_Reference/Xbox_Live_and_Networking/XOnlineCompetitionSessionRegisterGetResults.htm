XOnlineCompetitionSessionRegisterGetResults

Retrieves the results of the task started by the XOnlineCompetitionSessionRegister function.

HRESULT XOnlineCompetitionSessionRegisterGetResults(
    XONLINETASK_HANDLE hTask,
    DWORD dwRegistrantsBufferCount,
    XONLINE_ARB_REGISTRANT *pRegistrants,
    DWORD *pdwNumRegistrants
);

Parameters

hTask
[in] Task handle received from the XOnlineCompetitionSessionRegister function.
dwRegistrantsBufferCount
[in] Number of structures in the array specified by the pRegistrants parameter.
pRegistrants
[out] Array of XONLINE_ARB_REGISTRANT structures. Upon return, the structures in this array contain information that describes the registrants in the competition event.
pdwNumRegistrants
[out] Pointer to a DWORD. Upon return, the DWORD holds the number of registrants in the competition event.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
