XOnlineIsUserNicknameAllowed

Macro that, given the dwUserFlags value from an XUID structure, determines whether a user is permitted to choose nicknames.

#define XOnlineIsUserNicknameAllowed(dwUserFlags) \
   (((dwUserFlags) & XONLINE_USER_NICKNAME_NOT_ALLOWED) == 0)

Parameters

dwUserFlags
[in] Type is DWORD. The dwUserFlags value from an XUID structure contained in an XONLINE_USER structure.

Return Values

The return value is TRUE if the user is permitted to choose nicknames; FALSE otherwise.

The return value is of type BOOL.

Remarks

XOnlineIsUserNicknameAllowed accepts the entire dwUserFlags from an XUID structure and determines whether that user is permitted to choose custom nicknames. Titles must enforce this restriction by not allowing restricted users to set custom nicknames. This restriction is placed on the user by the Xbox Live service.

Requirements

Header: Declared in Xonline.h.
