ILiveFriendsList::IsReady

Indicates whether the friends list has initialized and is ready to service requests.

BOOL IsReady();

Parameters

None.

Return Values

Returns TRUE if the friends list has initialized and is ready, FALSE otherwise.

Remarks

After an ILiveFriendsList is created with ILiveEngine::GetFriendsForUser, IsReady will continue to return FALSE until initialization is complete and the friends list has been enumerated. After initialization is complete, other ILiveFriendsList methods may then be called.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
