XOnlineArbitrationCreateRoundID

Creates an arbitration round ID.

HRESULT XOnlineArbitrationCreateRoundID(
    ULONGLONG *pqwRoundID
);

Parameters

pqwRoundID
[out] Address of a ULONGLONG variable. Upon return, this variable contains the arbitration round ID.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The other arbitration functions require the round ID that this function generates. Therefore, your title needs to call this function before invoking the other arbitration functions.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
