XONLINE_LOGON_STATE

Contains a logon state returned by a call to XOnlineSaveLogonState. This structure is designed to be opaque information that is saved or passed as an argument to a function or executable being launched; to retrieve state information from the structure, pass the structure to XOnlineRetrieveLogonState.

typedef struct{
    BYTE bType;
    BYTE bVersion;
    WORD cbSize;
    BYTE Data[XONLINE_LOGON_STATE_SIZE];
} XONLINE_LOGON_STATE, *PXONLINE_LOGON_STATE;

Members

bType
Type of state data.
bVersion
Version of the state data.
cbSize
Size of the entire XONLINE_LOGON_STATE structure.
Data
Opaque data representing the logon state, including logged on users and services, at the time that XOnlineSaveLogonState was called.

Requirements

Header: Declared in Xonline.h.
