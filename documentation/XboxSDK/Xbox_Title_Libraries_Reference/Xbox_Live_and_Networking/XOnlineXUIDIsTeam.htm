XOnlineXUIDIsTeam

Macro that, given a pointer to an XUID structure, determines whether that XUID is a team XUID (as opposed to a player XUID).

#define XOnlineXUIDIsTeam(pxuid) \
	(((pxuid)->qwUserID & 0xFF00000000000000) == 0xFE00000000000000)

Parameters

pxuid
[in] Type is XUID*. A pointer to a valid XUID structure.

Return Values

The return value is TRUE if the user XUID is a team identifier; FALSE otherwise.

The return value is of type BOOL.

Requirements

Header: Declared in Xonline.h.
