ILiveFriendsList::HasChanged

Indicates whether the friends list has changed since Refresh was last called.

BOOL HasChanged();

Parameters

None.

Return Values

Returns TRUE if the friends list has changed since Refresh was last called; returns FALSE otherwise.

Remarks

HasChanged indicates that the friends list has changed on the server, and Refresh should be called to update the local copy cached by ILiveFriendsList.

Requirements
