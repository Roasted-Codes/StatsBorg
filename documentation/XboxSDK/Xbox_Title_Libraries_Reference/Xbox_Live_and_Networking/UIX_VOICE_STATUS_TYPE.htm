UIX_VOICE_STATUS_TYPE

Communicator and voice status returned by ITitlePlayersListItem::GetVoiceStatus.

typedef enum _UIX_VOICE_STATUS_TYPE{
    UIX_VOICE_STATUS_COMMUNICATOR = 0,
    UIX_VOICE_STATUS_SPEAKERS = 1,
    UIX_VOICE_STATUS_NONE = 2,
    UIX_VOICE_STATUS_FORCE_DWORD = 0x7fffffff,
} UIX_VOICE_STATUS_TYPE;

Constants

UIX_VOICE_STATUS_COMMUNICATOR
Voice through communicator.
UIX_VOICE_STATUS_SPEAKERS
Voice through speakers.
UIX_VOICE_STATUS_NONE
No voice.
UIX_VOICE_STATUS_FORCE_DWORD
This value is not used; it forces the enumeration to compile to 32 bits in size.
