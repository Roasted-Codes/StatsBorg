XVoiceCreateOneToOneDecoder

Returns an IXVoiceDecoder interface for a one-to-one decoder. 

HRESULT XVoiceCreateOneToOneDecoder(
    LPXVOICEDECODER *ppMediaObject
);

Parameters

ppMediaObject
[out] Address of a pointer of type LPXVOICEDECODER. This pointer receives the address of the IXVoiceDecoder interface.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The interface is returned with a reference count of one. If only a single thread is accessing the decoder, the title need not call the AddRef method on the interface.

When the title no longer needs the decoder it should call the Release method on the interface. Calling the Release method deallocates the memory associated with the decoder.

The IXVoiceDecoder::ProcessMultiple method for the one-to-one decoder decompresses a single input channel, then outputs the decompressed result to a single channel.

Requirements

Header: Declared in Xvoice.h.

Import Library: Use Xvoice.lib.
