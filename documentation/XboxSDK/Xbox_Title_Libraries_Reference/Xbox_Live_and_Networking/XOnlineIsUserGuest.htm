XOnlineIsUserGuest

Macro that, given the dwUserFlags value from an XUID structure, determines whether a user is a guest.

#define XOnlineIsUserGuest(dwUserFlags) \
   (((dwUserFlags) & XONLINE_USER_GUEST_MASK) != 0)

Parameters

dwUserFlags
[in] Type is DWORD. The dwUserFlags value from an XUID structure contained in an XONLINE_USER structure.

Return Values

The return value is TRUE if the user is a guest; FALSE otherwise.

The return value is of type BOOL.

Remarks

XOnlineIsUserGuest accepts the entire dwUserFlags from an XUID structure and determines whether that user is a guest.

Requirements

Header: Declared in Xonline.h.
