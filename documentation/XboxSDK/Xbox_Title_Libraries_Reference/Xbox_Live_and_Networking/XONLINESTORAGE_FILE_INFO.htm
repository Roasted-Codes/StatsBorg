XONLINESTORAGE_FILE_INFO

typedef struct{
    DWORD dwTitleID;
    DWORD dwTitleVersion;
    ULONGLONG qwOwnerPUID;
    BYTE bCountryID;
    ULONGLONG qwReserved;
    DWORD dwContentType;
    DWORD dwStorageSize;
    DWORD dwInstalledSize;
    FILETIME ftCreated;
    FILETIME ftLastModified;
    WORD wAttributesSize;
    WORD cchPathName;
    LPCWSTR wszPathName;
    PBYTE pbAttributes;
} XONLINESTORAGE_FILE_INFO, *PXONLINESTORAGE_FILE_INFO;

Members

dwTitleID
Title identifier of the content.
dwTitleVersion
Title version of the content.
qwOwnerPUID
Player unique ID (PUID) of the uploaded file.
bCountryID
Country identifier of the user who created this content.
qwReserved
Reserved for future expansion.
dwContentType
Indicates the content type. See the XONLINESTORAGE_CONTENT_TYPE enumeration to determine which function should be used for the content download.
dwStorageSize
Size of the file in bytes on the server.
dwInstalledSize
For packages (when dwContentType is XONLINESTORAGE_CONTENT_TYPE_PACKAGE), this is the size in bytes, but in multiples of HD blocks. For memory blobs (when dwContentType is XONLINESTORAGE_CONTENT_TYPE_BLOB), this is the size of the exact memory footprint, in bytes.
ftCreated
Timestamp indicating creation time of the file in UTC format.
ftLastModified
Timestamp indicating the last modification of the file in UTC format.
wAttributesSize
This value must be zero.
cchPathName
Length of the full path name to the content.
wszPathName
UNICODE string of the storage path identifying this content.
pbAttributes
This value must be NULL.
