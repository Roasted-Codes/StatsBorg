ITitlePlayersListItem::GetVoiceStatus

Returns the voice capabilities of the player.

UIX_VOICE_STATUS_TYPE GetVoiceStatus();

Parameters

None.

Return Values

A UIX_VOICE_STATUS_TYPE value indicating the voice capabilities of the player.

Remarks

For titles using Xbox High Level Voice (XHV), local player voice status can be obtained by calling IXHVEngine::GetLocalTalkerStatus. Remote player voice status will need to be communicated through game data packets.

Requirements

Header: Declared in Uix.h.

Import Library: Use Uix.lib.

See Also

IXHVEngine::GetLocalTalkerStatus, UIX_VOICE_STATUS_TYPE
