XOnlineStatUnitRead

Initiates a task to retrieve stats for the specified units.

HRESULT XOnlineStatUnitRead(
    const XUID *pxuidUnitMembers,
    DWORD dwNumStatSpecUnits,
    const XONLINE_STAT_SPEC_UNIT *pStatSpecUnits,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

pxuidUnitMembers
[in] Pointer to an array of XUID structures representing the unit members. This array must always contain XONLINE_STAT_MAX_MEMBERS_IN_UNIT entries. Units with fewer than XONLINE_STAT_MAX_MEMBERS_IN_UNIT members must place the valid member IDs starting at index 0, and must zero out the unused entries.
dwNumStatSpecUnits
[in] Number of XONLINE_STAT_SPEC_UNIT structures in the array pointed to by the pStatSpecUnits parameter.
pStatSpecUnits
[in] Pointer to an array of XONLINE_STAT_SPEC_UNIT structures. Each structure specifies stats to retrieve for that unit.
hWorkEvent
[in, optional] Handle to a caller-created event object. This object will be signaled when there is work to do by calling XOnlineTaskContinue. The caller may pass in NULL instead, indicating that the caller will poll XOnlineTaskContinue itself.
phTask
[out] Address of an XONLINETASK_HANDLE that receives the stats retrieval task. This task sends the stats request to the Statistics Service and receives the response. This handle must be pumped to completion by XOnlineTaskContinue. The task is complete when XOnlineTaskContinue returns XONLINETASK_S_SUCCESS. After the task completes, call XOnlineStatUnitReadGetResult to retrieve the stats. After retrieving the stats, call XOnlineTaskClose to close the task.

Note    To throttle requests sent to the Statistics Service, XOnlineTaskContinue will return XONLINE_E_TASK_ABORTED_BY_DUPLICATE if the task is still being pumped when a second operation is started.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function can be called only with direct user input related to Statistics.

This function must not be called more often than once a second between game sessions. During a game session, this function can only be called once at the beginning, once at the end and every five minutes if the game session lasts longer than five minutes. Contact your Technical Game Manager or consult the Technical Certification Requirements for details.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
