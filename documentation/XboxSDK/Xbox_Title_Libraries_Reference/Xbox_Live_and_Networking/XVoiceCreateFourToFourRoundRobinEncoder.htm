XVoiceCreateFourToFourRoundRobinEncoder

The XVoiceCreateFourToFourRoundRobinEncoder function returns an IXVoiceEncoder interface for a four-to-four round-robin encoder. 

HRESULT XVoiceCreateFourToFourRoundRobinEncoder(
    LPXVOICEENCODER *ppMediaObject
);

Parameters

ppMediaObject
[out] Address of a pointer of type LPXVOICEENCODER. This pointer receives the address of the IXVoiceEncoder interface.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The interface is returned with a reference count of one. If only a single thread is accessing the encoder, the title need not call the AddRef method on the interface.

When the title no longer needs the encoder it should call the Release method on the interface. Calling the Release method deallocates the memory associated with the encoder.

The IXVoiceEncoder::ProcessMultiple method for the four-to-four round-robin encoder takes four input channels. The encoder selects two of these input channels to compress. It sends the compressed results to the corresponding output channels. On the following frame, the two input channels that were not compressed previously are given priority over the two that were compressed. However, if a previously uncompressed channel does not contain any voice data, the encoder again compresses data from one of the channels that it compressed on the previous frame.

Requirements

Header: Declared in Xvoice.h.

Import Library: Use Xvoice.lib.
