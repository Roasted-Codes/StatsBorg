XONLINE_ARB_REPORT_DATA

Contains data about lost connections or suspicious activity during an arbitrated session.

typedef struct{
    BYTE bNumLostConnectivityAddresses;
    XNADDR *pxnaddrLostConnectivityAddresses;
    XONLINE_ARB_SUSPICIOUS_INFO *pSuspiciousInfoType1;
    XONLINE_ARB_SUSPICIOUS_INFO *pSuspiciousInfoType2;
    XONLINE_ARB_SUSPICIOUS_INFO *pSuspiciousInfoType3;
} XONLINE_ARB_REPORT_DATA, *PXONLINE_ARB_REPORT_DATA;
