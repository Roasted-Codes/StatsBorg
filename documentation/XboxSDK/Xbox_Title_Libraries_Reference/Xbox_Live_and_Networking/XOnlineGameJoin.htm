XOnlineGameJoin

Enables a player to join a game.

HRESULT XOnlineGameJoin(
    DWORD dwUserIndex,
    const XONLINE_GAME_JOIN_INFO *pGameJoinInfo
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player. The player must have been successfully logged on with a previous call to XOnlineLogon.
pGameJoinInfo
[in] Address of a XONLINE_GAME_JOIN_INFO structure containing the information required to join a game.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
