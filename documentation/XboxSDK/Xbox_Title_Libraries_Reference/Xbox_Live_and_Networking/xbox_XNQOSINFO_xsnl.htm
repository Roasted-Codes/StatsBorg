XNQOSINFO

Contains the results from a quality of service (QoS) probe to a particular host. 

typedef struct{
    BYTE bFlags;
    BYTE bReserved;
    WORD cProbesXmit;
    BYTE cProbesRecv;
    WORD cbData;
    BYTE *pbData;
    WORD wRttMinInMsecs;
    WORD wRttMedInMsecs;
    DWORD dwUpBitsPerSec;
    DWORD dwDnBitsPerSec;
} XNQOSINFO, ;

Members

bFlags
Combination of the following flags.
Value Description
XNET_XNQOSINFO_COMPLETE This flag is set when the QoS values in this structure are final
XNET_XNQOSINFO_TARGET_CONTACTED This flag is set as soon as the QoS probe successfully contacts the remote host. If this flag is set, the title is able to open a socket to the remote host because the QoS mechanism uses the same NAT traversal algorithm as the key-exchange mechanism. If this flag is not set, the QoS probe did not receive any response from the remote host. Possible reasons are that the host is down, has released its QoS listener (see XNetQosListen), has unregistered its session ID, or is behind a NAT that is not allowing the connection.
XNET_XNQOSINFO_TARGET_DISABLED This flag is set if the QoS probe received a "go away" response from the remote host.
XNET_XNQOSINFO_DATA_AVAILABLE This flag is set when more than half of the required responses have been received. Although the final QoS values are not yet available, the data received so far may be useful to the title. The system continues to update the XNQOSINFO structure until all responses have been received.
XNET_XNQOSINFO_DATA_RECEIVED This flag is set if the remote host returns a title-specific data block. The size of the data returned is specified by the cbData member. The pbData member points to the returned data.
bReserved
This member is reserved.
cProbesXmit
Number of probes transmitted.
cProbesRecv
Number of probe responses received.
cbData
Number of bytes in the data block pointed to by the pbData member. This member is zero if the remote host did not return a data block.
pbData
Pointer to a title-specific data block that was returned by the remote host in the probe response packet. This member is NULL if the remote host did not return a data block.
wRttMinInMsecs
Minimum roundtrip time in milliseconds.
wRttMedInMsecs
Median roundtrip time in milliseconds.
dwUpBitsPerSec
Average upstream bandwidth in bits per second.
dwDnBitsPerSec
Average downstream bandwidth in bits per second.
