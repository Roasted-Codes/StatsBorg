XOnlineTitleIdIsSamePublisher

Tests whether a title ID specifies a title by the same publisher as the currently-running title.

BOOL XOnlineTitleIdIsSamePublisher(
    DWORD dwTitleID
);

Parameters

dwTitleID
[in] Title ID.

Return Values

XOnlineTitleIdIsSamePublisher always succeeds with one of the following return codes.

Value Description
S_TRUE The specified title has the same publisher as the current title.
S_FALSE The specified title has a different publisher than the current title.

Remarks

The titleID member of the XONLINE_FRIEND structure contains the ID of the title currently being played by that member of the friends list.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
