XOnlineStatUnitReadGetResult

Retrieves the stats data requested by calling XOnlineStatUnitRead.

HRESULT XOnlineStatUnitReadGetResult(
    XONLINETASK_HANDLE hTask,
    DWORD dwNumStatSpecUnits,
    XONLINE_STAT_SPEC_UNIT *pStatSpecUnits
);

Parameters

hTask
[in] Handle to the task returned by XOnlineStatUnitRead.
dwNumStatSpecUnits
[in] Number of XONLINE_STAT_SPEC_UNIT structures pointed to by the pStatSpecUnits parameter.
pStatSpecUnits
[out] Pointer to an array of XONLINE_STAT_SPEC_UNIT structures. Each structure receives the stats for a single user.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
