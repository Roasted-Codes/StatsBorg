TOPOLOGY_ID

Builds a topology ID from a round number and an event number.

#define TOPOLOGY_ID(roundNumber,eventNumber) (((roundNumber) << 16) + (eventNumber))

Parameters

roundNumber
[in] Type is WORD. Round number.
eventNumber
[in] Type is WORD. Event number.

Return Values

This macro evaluates to a topology ID.
