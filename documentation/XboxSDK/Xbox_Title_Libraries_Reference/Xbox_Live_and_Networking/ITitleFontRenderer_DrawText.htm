ITitleFontRenderer::DrawText

Draws the specified text onto a surface.

HRESULT DrawText(
    LPCWSTR pText,
    DWORD X,
    DWORD Y,
    DWORD MaxWidth
);

Parameters

pText
[in] Pointer to a null-terminated Unicode string that specifies the text to draw.
X
[in] X pixel coordinate of the text.
Y
[in] Y pixel coordinate of the text.
MaxWidth
[in] Maximum width to draw. Text extending beyond the specified width should be clipped.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The text specified should be drawn to the title's current render target. DrawText uses the current text height and color as set with SetHeight and SetColor.

The X and Y coordinates specify the upper-left corner of the text cell.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ITitleFontRenderer::SetHeight, ITitleFontRenderer::SetColor
