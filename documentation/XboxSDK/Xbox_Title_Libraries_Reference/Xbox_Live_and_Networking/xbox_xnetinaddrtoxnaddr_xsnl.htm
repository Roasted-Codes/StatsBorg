XNetInAddrToXnAddr

Converts the specified address to an Xbox network complete address.

INT XNetInAddrToXnAddr(
    const IN_ADDR ina,
    XNADDR *pxna,
    XNKID *pxnkid
);

Parameters

ina
IN_ADDR structure containing the address to convert.
pxna
Address of the XNADDR structure that receives the Xbox network complete address information. Can be set to NULL if the title does not need this information.
pxnkid
Address of the XNKID structure that receives the key identifier associated with the address. Can be set to NULL if the title does not need this information.

Return Values

Returns zero if successful, an error code otherwise.

Requirements

Header: Declared in Winsockx.h, for include use Xtl.h.

Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. Use the secure versions of the library file (Xnets.lib or XOnlines.lib) for final testing.
