ITitlePlayersListItem::GetXUID

Returns the XUID associated with the player.

XUID* GetXUID();

Parameters

None.

Return Values

Returns a pointer to an XUID structure representing the player.

Remarks

A player's XUID is obtained when the player joins a game session.

Requirements

Header: Declared in Uix.h.

Import Library: Use Uix.lib.
