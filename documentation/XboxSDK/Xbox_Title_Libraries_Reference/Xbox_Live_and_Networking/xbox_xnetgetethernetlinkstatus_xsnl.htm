XNetGetEthernetLinkStatus

Retrieves the current status of the Ethernet link.

DWORD XNetGetEthernetLinkStatus();

Parameters

None.

Return Values

Returns a combination of the following values:

Value Description
XNET_ETHERNET_LINK_ACTIVE The Ethernet cable is connected and active.
XNET_ETHERNET_LINK_100MBPS The Ethernet link is set to 100 Mbps.
XNET_ETHERNET_LINK_10MBPS The Ethernet link is set to 10 Mbps.
XNET_ETHERNET_LINK_FULL_DUPLEX The Ethernet link is in full duplex mode.
XNET_ETHERNET_LINK_HALF_DUPLEX The Ethernet link is in half duplex mode.

Remarks

XNetGetEthernetLinkStatus can be called before the network stack has been initialized by either XNetStartup or WSAStartup.

If the XNET_ETHERNET_LINK_ACTIVE flag is zero, all other flags are zero as well. To check whether the ethernet cable is connected, test whether the return value is zero.

Requirements

Header: Declared in Winsockx.h, for include use Xtl.h.

Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. Use the secure versions of the library file (Xnets.lib or XOnlines.lib) for final testing.
