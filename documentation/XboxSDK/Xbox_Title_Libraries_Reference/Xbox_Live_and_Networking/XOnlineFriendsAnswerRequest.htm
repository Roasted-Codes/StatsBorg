XOnlineFriendsAnswerRequest

Responds to a request from a player that the player and the current user be added to each other's Friends lists.

HRESULT XOnlineFriendsAnswerRequest(
    DWORD dwUserIndex,
    const XONLINE_FRIEND *pToFriend,
    XONLINE_REQUEST_ANSWER_TYPE Answer
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the user answering the request. The user must have been successfully logged on with a previous call to XOnlineLogon.
pToFriend
[in] Pointer to an XONLINE_FRIEND structure specifying the player to whom the answer should be sent—that is, the player who made the request.
Answer

[in] An XONLINE_REQUEST_ANSWER_TYPE value that specifies the answer to the request.

Return Values

If XOnlineFriendsAnswerRequest succeeds, the return value is one of the following return codes.

Value Description
S_OK The system sent the specified response to the Friend request.
S_FALSE The user has already responded to this Friend request.

If XOnlineFriendsAnswerRequest fails, the return value is one of the following error codes.

Value Description
XONLINE_E_NOTIFICATION_NO_TASK The Friends startup task is not active. You must first call XOnlineFriendsStartup to create the startup task.
XONLINE_E_NOTIFICATION_USER_NOT_FOUND The specified friend is not in list. This error should occur only if the title is currently enumerating the Friends list. You can accept Friend requests only from users that are on the friend's list.
XONLINE_E_NOTIFICATION_FRIEND_PENDING The current user previously sent a Friend request to the specified user, but has not yet received a response. Because the specified user is not yet on the current user's Friends list, the current user should not be answering requests from the specified user.
Other See Online Errors for information about other possible errors.

Remarks

The task returned by the XOnlineFriendsEnumerate function must be active at the time the title calls the XOnlineFriendsAnswerRequest function.

If the request is accepted, the two users will be placed on each other's Friends lists.

Pump the task returned by XOnlineFriendsStartup in order to send the answer.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
