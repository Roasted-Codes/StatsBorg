XOnlineMessageEnumerate

Retrieves a list of message summaries from the Xbox Live service.

HRESULT XOnlineMessageEnumerate(
    DWORD dwUserIndex,
    XONLINE_MSG_SUMMARY *pMsgSummaries,
    DWORD *pdwNumMsgSummaries
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the user whose message list is to be enumerated. The user must have been successfully logged on with a previous call to XOnlineLogon.
pMsgSummaries
[out] Pointer to the list of message summaries. The maximum number of XONLINE_MSG_SUMMARY structures is XONLINE_MAX_NUM_MESSAGES.
pdwNumMsgSummaries
[out] Pointer to a DWORD that receives the number of message summaries in the list.

Return Values

Possible return values for XOnlineMessageEnumerate are as follows:

Value Description
S_OK The function succeeded.
XONLINE_S_MESSAGE_PENDING_SYNC Returned when the message list is in the process of being retrieved (immediately after logon or when turning summary refresh back on). Pumping the logon task will eventually clear this state and S_OK will be returned.

Remarks

If a user does not have any messages, XOnlineMessageEnumerate will return S_OK and place zero in pdwNumMsgSummaries.

The XOnlineGetNotificationEx function will return the XONLINE_NOTIFICATION_STATE_FLAG_PENDING_SYNC flag while the message list is being retrieved. Titles can check that while pumping the logon task instead of waiting for XOnlineMessageEnumerate to return S_OK.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
