ITitleUIPlugin::PassInputToObject

Called by the Xbox Drop-in UI engine to forward input to a UI object so that the object can respond.

HRESULT PassInputToObject(
    DWORD ObjectID,
    UIX_INPUT_TYPE InputKey
);

Parameters

ObjectID
[in] Object identifier, previously returned by CreateObject.
InputKey
[in] A UIX_INPUT_TYPE specifying the type of input received.

Return Values

Returns S_OK if successful, S_FALSE if the input is inapplicable or out of range, or an error code otherwise.

Remarks

A list box object should change the selected item in response to directional input. A list box should return S_FALSE if directional input attempts to select an item out of the range of the list box (an UP input when the first item is already selected, or a DOWN input when the last item is already selected).

Requirements
