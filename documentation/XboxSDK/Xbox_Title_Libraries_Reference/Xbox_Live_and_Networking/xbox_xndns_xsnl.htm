XNDNS

Contains DNS lookup information.

typedef struct {
    INT iStatus;  // WSAEINPROGRESS if pending; 0 if success; error if failed
    UINT cina;  // count of IP addresses for the given host
    IN_ADDR aina[8];  // vector of IP addresses for the given host
} XNDNS;

Members

iStatus
Value that specifies the current status.
Value Description
0 The lookup was successful.
WSAEINPROGRESS The request is still pending.
error code The request failed. Possible error codes include WSAETIMEDOUT and WSAHOST_NOT_FOUND.
cina
Value that specifies the IP address for the given host.
aina
Array of values that specifies the IP address for the given host.

Requirements

Header: Declared in Winsockx.h. for include use Xtl.h.
