XOnlineStatUnitEnumerate

Retrieves a contiguous set of units from a specified leaderboard that a user belongs to.

HRESULT XOnlineStatUnitEnumerate(
    XUID xuidUnitMember,
    DWORD dwLeaderboardID,
    XONLINE_STAT_SORTORDER SortOrder,
    DWORD dwMaxUnitsToReturn,
    DWORD dwNumStatsPerUnit,
    const WORD *pStatsPerUnit,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

xuidUnitMember
[in] XUID structure containing the member identifier to use for the enumeration.
dwLeaderboardID
[in] Leaderboard from which to retrieve the entries. Leaderboards are defined by the title developer.
SortOrder

[in] Value indicating the sort order of the data as specified by XONLINE_STAT_SORTORDER. Can be the following:
Identifier Description
XONLINE_STAT_SORTORDER_LASTACTIVITY Stats data should be sorted by most recent activity.
XONLINE_STAT_SORTORDER_RATING Stats data is sorted by rating in descending order.

dwMaxUnitsToReturn
[in] Maximum number of units that should be returned from the enumeration. This can be up to XONLINE_STAT_MAX_UNITS.
dwNumStatsPerUnit
[in] Number of stats to retrieve per unit. This value is equal to the number of elements in the array pointed to by the pStatsPerUnit parameter.
pStatsPerUnit
[in] Pointer to an array of IDs that specify stats (see the XONLINE_STAT_SPEC_UNIT structure). The number of IDs in this array should be equal to the value specified by the dwNumStatPerUnit parameter.
hWorkEvent
[in, optional] Handle to a caller-created event object. This object will be signaled when there is work to do by calling XOnlineTaskContinue. The caller may pass in NULL instead, indicating that the caller will poll XOnlineTaskContinue itself.
phTask
[out] Address of an XONLINETASK_HANDLE that receives the stats enumeration task. This task sends the enumeration request to the Statistics Service and receives the enumeration response. Pump this handle to completion by calling XOnlineTaskContinue. When the task is complete, call XOnlineStatUnitEnumerateGetResults to get the enumeration results. Call XOnlineTaskClose to close the task.

Note    To throttle requests sent to the Statistics Service, XOnlineTaskContinue will return XONLINE_E_TASK_ABORTED_BY_DUPLICATE if the task is still being pumped when a second operation is started.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function can be called only with direct user input related to Statistics.

This function must not be called more often than once a second between game sessions. During a game session, this function can only be called once at the beginning, once at the end and every five minutes if the game session lasts longer than five minutes. Contact your Technical Game Manager or consult the Technical Certification Requirements for details.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
