IXVoiceEncoder::ProcessMultiple

Compresses the data packets in the specified input array, and outputs the compressed data to the specified output array. 

HRESULT ProcessMultiple(
    DWORD dwInputArraySize,
    LPCXMEDIAPACKET pInputPacketArray,
    DWORD dwOutputArraySize,
    LPCXMEDIAPACKET pOutputPacketArray
);

Parameters

dwInputArraySize
[in] Number of elements in the array pointed to by the pInputPacketArray parameter.
pInputPacketArray
[in] Pointer to an array of XMEDIAPACKET structures. The elements of this array contain the voice data to compress.
dwOutputArraySize
[in] Number of elements in the array pointed to by the pOutputPacketArray parameter.
pOutputPacketArray
[out] Pointer to an array of XMEDIAPACKET structures. The elements of this array receive the compressed output data.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Use the IXVoiceEncoder::GetCodecBufferSize method to determine the size of the buffers in the elements of the output array.

Requirements

Header: Declared in Xvoice.h.

Import Library: Use Xvoice.lib.
