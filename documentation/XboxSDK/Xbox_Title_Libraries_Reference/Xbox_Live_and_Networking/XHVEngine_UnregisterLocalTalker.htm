IXHVEngine::UnregisterLocalTalker

Unregisters a local talker previously registered with RegisterLocalTalker.

HRESULT UnregisterLocalTalker(
    DWORD dwLocalPort
);

Parameters

dwLocalPort
[in] Controller port of the local talker to unregister.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xhv.h.
