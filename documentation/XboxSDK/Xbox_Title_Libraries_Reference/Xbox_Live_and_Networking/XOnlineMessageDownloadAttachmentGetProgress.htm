XOnlineMessageDownloadAttachmentGetProgress

Reports the progress of downloading an attachment.

HRESULT XOnlineMessageDownloadAttachmentGetProgress(
    XONLINETASK_HANDLE hDownloadTask,
    DWORD *pdwPercentDone,
    ULONGLONG *pqwNumerator,
    ULONGLONG *pqwDenominator
);

Parameters

hDownloadTask
[in] The task handle obtained from XOnlineMessageDownloadAttachmentToDirectory or XOnlineMessageDownloadAttachmentToMemory.
pdwPercentDone
[out, optional] Address of a DWORD. Upon return, the DWORD contains a two-digit integer indicating the percentage of the download that has been completed.
pqwNumerator
[out, optional] Address of a ULONGLONG variable. XOnlineMessageDownloadAttachmentGetProgress can optionally report the percentage of the download that has been completed as a fraction. If the address in pqwNumerator is not NULL, XOnlineMessageDownloadAttachmentGetProgress stores the numerator of the fraction in the ULONGLONG variable pointed to by pqwNumerator.
pqwDenominator
[out, optional] Address of a ULONGLONG variable. XOnlineMessageDownloadAttachmentGetProgress can optionally report the percentage of the download that has been completed as a fraction. If the address in pqwDenominator is not NULL, XOnlineMessageDownloadAttachmentGetProgress stores the denominator of the fraction into the ULONGLONG variable pointed to by pqwDenominator.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.

See Also

XOnlineMessageDownloadAttachmentToDirectory, XOnlineMessageDownloadAttachmentToMemory
