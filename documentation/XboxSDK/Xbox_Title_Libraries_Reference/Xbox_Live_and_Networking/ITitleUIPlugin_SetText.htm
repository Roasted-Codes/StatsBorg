ITitleUIPlugin::SetText

Sets the text for a UI object.

HRESULT SetText(
    DWORD ObjectID,
    DWORD ItemIndex,
    LPCWSTR pText,
    DWORD IconCount,
    const UIX_SKIN_ICON_INFO *pIconInfo
);

Parameters

ObjectID
[in] Object identifier, previously returned by CreateObject.
ItemIndex
[in] If the object specified in ObjectID is a list box, specifies the index of the item within the list box. Otherwise, ItemIndex must be zero.
pText
[in] Pointer to a null-terminated string specifying the new text.
IconCount
[in] Count of structures in pIconInfo. At most UIX_MAX_ICONS_IN_TEXT structures will be specified.
pIconInfo
[in] Array of UIX_SKIN_ICON_INFO structures containing information for icons included with the text.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The array of UIX_SKIN_ICON_INFO structures passed in pIconInfo contains icons that are included as part of the text. Each structure contains the icon's character position location within the text string along with the resource identifier for the icon. The icon(s) must be rendered along with the text when the object is rendered.

Icons may also be specified as being right-justified within the object, in which case they are drawn on the object's right side (based on the object's layout). Multiple icons may be specified as right justified, in which case they are rendered next to each other on the right side, with the first such icon in the array leftmost, and the last icon rightmost.

Icons are not drawn with any inter-icon spacing. Spacing between icons must be part of the icons themselves or be due to spacing in adjacent character cells.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
