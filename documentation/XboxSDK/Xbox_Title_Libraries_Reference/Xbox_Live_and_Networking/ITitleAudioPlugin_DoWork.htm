ITitleAudioPlugin::DoWork

Perform periodic work required by the audio plug-in. DoWork is called by the Xbox Drop-in UI engine every time ILiveEngine::DoWork is called (typically once per frame).

HRESULT DoWork();

Parameters

None.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The default ITitleAudioPlugin implements DoWork by simply calling XACTEngineDoWork.

The Xbox Drop-in UI engine currently ignores the return value of DoWork.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ILiveEngine::DoWork, XACTEngineDoWork
