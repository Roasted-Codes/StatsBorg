ITitleAudioPlugin::PlaySound

Play a named sound. PlaySound is called by the Xbox Drop-in UI engine when an action in the user interface requires that an accompanying sound be played.

HRESULT PlaySound(
    LPCSTR SoundName
);

Parameters

SoundName
Pointer to a null-terminated string that specifies the name of the sound to play.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The mapping of sound events to sound names is defined in the Xbox Drop-in UI skin file. The default ITitleAudioPlugin implements PlaySound by playing the sound bank sound named SoundName.

The Xbox Drop-in UI engine currently ignores the return value of PlaySound.
