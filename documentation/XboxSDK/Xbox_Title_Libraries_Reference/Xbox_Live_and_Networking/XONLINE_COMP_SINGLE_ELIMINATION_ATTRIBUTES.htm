XONLINE_COMP_SINGLE_ELIMINATION_ATTRIBUTES

Contains information describing a single-elimination competition.

typedef struct{
    DWORD dwPrivateSlots;
    DWORD dwPublicSlots;
    DWORD dwMinimumPlayers;
    FILETIME ftRegistrationOpen;
    FILETIME ftRegistrationClose;
    FILETIME ftCompetitionStart;
    FILETIME ftRoundOneStart;
    FILETIME ftRoundOneEnd;
    DWORD dwMatchReminderAdvanceMinutes;
    XONLINE_COMP_INTERVAL_UNIT Interval;
    XONLINE_COMP_UNITS_OR_MASK UnitOrMask;
    BOOL fTeamCompetition;
    DWORD dwTeamSize;
} XONLINE_COMP_SINGLE_ELIMINATION_ATTRIBUTES, *PXONLINE_COMP_SINGLE_ELIMINATION_ATTRIBUTES;

Members

dwPrivateSlots
Number of private slots in the competition.
dwPublicSlots
Number of public slots in the competition.
dwMinimumPlayers
Minimum number of players required for the competition.
ftRegistrationOpen
Time and date when registration for the competition begins.
ftRegistrationClose
Time and date when registration for the competition ends.
ftCompetitionStart
Time and date when the competition begins. Initial slotting takes place and matches start.
ftRoundOneStart
This time and date designates the official starting of matches in the first round. All future rounds are scheduled as offsets from this time and date.
ftRoundOneEnd
Time and date when round one of the competition ends. All round one match scores should be submitted before this time.
dwMatchReminderAdvanceMinutes
Number of minutes before the start of each round when a reminder is sent to all participants. Set to 0 if no reminders should be sent.
Interval
Value from the XONLINE_COMP_INTERVAL_UNIT enumeration that sets the time units used for the round interval.
UnitOrMask
Amount of time between competition rounds. Can be a literal numeric value or a constant from the following table. To use one of the following constants, set the Interval member to XONLINE_COMP_INTERVAL_DAILY.
Constant Description
XONLINE_COMP_DAY_MASK_ALL Competition rounds are scheduled at the same time every day of the week.
XONLINE_COMP_DAY_MASK_SUNDAY Competition rounds are scheduled at the same time every Sunday.
XONLINE_COMP_DAY_MASK_MONDAY Competition rounds are scheduled at the same time Monday.
XONLINE_COMP_DAY_MASK_TUESDAY Competition rounds are scheduled at the same time Tuesday.
XONLINE_COMP_DAY_MASK_WEDNESDAY Competition rounds are scheduled at the same time Wednesday.
XONLINE_COMP_DAY_MASK_THURSDAY Competition rounds are scheduled at the same time Thursday.
XONLINE_COMP_DAY_MASK_FRIDAY Competition rounds are scheduled at the same time Friday.
XONLINE_COMP_DAY_MASK_SATURDAY Competition rounds are scheduled at the same time Saturday.
fTeamCompetition
Set to TRUE if the participants in the competition are teams; FALSE if not.
dwTeamSize
Number of players per team. Ignored if fTeamCompetition is FALSE.
