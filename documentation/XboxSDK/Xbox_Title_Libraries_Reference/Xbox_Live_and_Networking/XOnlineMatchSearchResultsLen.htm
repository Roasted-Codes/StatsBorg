XOnlineMatchSearchResultsLen

Returns the size in bytes of the results returned by the XOnlineMatchSearch function. Pass this size as the value of the dwResultsLen parameter to the XOnlineMatchSearch function. 

DWORD XOnlineMatchSearchResultsLen(
    DWORD dwNumResults,
    DWORD dwNumSessionAttributes,
    const XONLINE_ATTRIBUTE_SPEC *pSessionAttributeSpec
);

Parameters

dwNumResults
[in] Number of search results to retrieve.
dwNumSessionAttributes
[in] Number of XONLINE_ATTRIBUTE_SPEC structures pointed to by the pSessionAttributeSpec parameter.
pSessionAttributeSpec
[in] Pointer to an array of XONLINE_ATTRIBUTE_SPEC structures that contain the attributes returned by the stored procedure on the matchmaking server.

Return Values

The return value is the size in bytes of the results returned by the XOnlineMatchSearch function.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
