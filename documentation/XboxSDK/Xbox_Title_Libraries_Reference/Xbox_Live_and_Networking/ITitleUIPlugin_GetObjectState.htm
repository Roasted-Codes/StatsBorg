ITitleUIPlugin::GetObjectState

Gets the state of a specified UI object.

HRESULT GetObjectState(
    DWORD ObjectID,
    DWORD ItemIndex,
    UIX_OBJSTATE_TYPE State,
    DWORD *pValue
);

Parameters

ObjectID
[in] Object identifier, previously returned by CreateObject.
ItemIndex
[in] If the object specified in ObjectID is a list box, specifies the index of the item within the list box. Otherwise, ItemIndex must be zero.
State
[in] An UIX_OBJSTATE_TYPE value specifying the state to retrieve.
pValue
[out] Pointer to a DWORD variable that receives the current object state. The meaning of the state value varies depending on State. See UIX_OBJSTATE_TYPE for details.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ITitleUIPlugin::SetObjectState, ITitleUIPlugin::CreateObject, UIX_OBJSTATE_TYPE
