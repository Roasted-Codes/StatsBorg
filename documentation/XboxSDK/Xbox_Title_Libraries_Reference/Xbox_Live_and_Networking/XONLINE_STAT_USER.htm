XONLINE_STAT_USER

Specifies a particular user of the statistics service.

typedef struct{
    XUID xuidUser;
    union {
        CHAR szGamertag[XONLINE_GAMERTAG_SIZE];
        WCHAR wszTeamName[XONLINE_MAX_TEAM_NAME_SIZE];
    } ;
} XONLINE_STAT_USER, *PXONLINE_STAT_USER;

Members

xuidUser
The user's XUID.
szGamertag

Null-terminated string that contains the user's Gamertag. The format of the Gamertag must adhere to the following rules:
  • Minimum of one character in length.
  • Maximum of 15 characters in length.
  • Can include only the following characters: A–Z, a–z, 0–9, and [space].
  • Cannot begin with a [space] character.
  • Cannot have multiple contiguous [space] characters.
  • Must begin with an alphabetic character (A–Z, a–z).
  • Cannot end with space.

The Gamertag is case insensitive for comparisons, but case should be preserved when stored. For example, Ubba and UBBA are the same and hence cannot both exist as accounts in the system, but which ever one was entered into the system, the case is preserved.

wszTeamName
Null-terminated wide character string that contains the user's team name.
