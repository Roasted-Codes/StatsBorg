IXHVEngine::GetRemoteTalkers

HRESULT GetRemoteTalkers(
    DWORD *pdwRemoteTalkersCount,
    XUID *pxuidRemoteTalkers
);

Parameters

pdwRemoteTalkersCount
[out] Pointer to a DWORD variable that receives the count of currently registered remote talkers.
pxuidRemoteTalkers
[out] Pointer to an array of XUID structures that receives the currently registered remote talkers. At most XHV_MAX_REMOTE_TALKERS structures of type XUID will be written to the pxuidRemoteTalkers buffer.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

To ensure that the pxuidRemoteTalkers buffer is large enough to contain the returned XUIDs, either the buffer should be large enough to contain XHV_MAX_REMOTE_TALKERS structures of type XUID, or the title should match the buffer size to the maximum number of remote talkers it registers with RegisterRemoteTalker.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
