XOnlineStorageSetFamilyTitleID

Sets the family title ID for storage.

HRESULT XOnlineStorageSetFamilyTitleID(
    DWORD dwTitleID
);

Parameters

dwTitleID
[in] Family title ID to use for storage. Passing zero as the family title ID will set the family title ID back to the original value.

Return Values

Returns S_OK if successful, an error code otherwise.

XONLINE_E_INVALID_TITLE_ID is returned when title identifier supplied is neither a valid title identifier or one of the family title identifier.

Remarks

Setting and resetting the family title ID will generate debug output for informational purposes.

Attempting to use this function to enable a family title ID without first including that ID in the title's certificate will result in an error message displayed in the debug output as well as the function not working correctly. A family title ID can be included in a title's certificate by opening the project properties dialog box in Visual Studio, and adding the family title ID to the alternate title IDs.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
