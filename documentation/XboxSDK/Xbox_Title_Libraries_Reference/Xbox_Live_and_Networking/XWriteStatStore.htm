XWriteStatStore

Stores the specified statistic for the specified user on the local Xbox hard disk. This statistic is later retrieved by the Xbox Statistics Manager and stored on the specified Xbox Live leaderboard.

Note    Leaderboards are also referred to as scoreboards.

HRESULT XWriteStatStore(
    LPCWSTR lpLocalUserName,
    DWORD dwLeaderBoardIndex,
    WORD wAttributeIndex,
    XOFFLINE_STAT_TYPE Type,
    VOID *pStatValue
);

Parameters

lpLocalUserName
[in] Pointer to a Unicode string that contains the title-defined identifier for the user. This string is not the user's gamertag on the Xbox Live service.
dwLeaderBoardIndex
[in] Leaderboard on which to store the statistic.
wAttributeIndex
[in] Title-defined identifier for the statistic.
Type
Type of the attribute (statistic) to store. This parameter is a value from the enumeration type XOFFLINE_STAT_TYPE:
Value Meaning
XOFFLINE_STAT_LONG The statistic is an integer (32-bit).
XOFFLINE_STAT_LONGLONG The statistic is a big integer (64-bit).
XOFFLINE_STAT_DOUBLE The statistic is an real number (64-bit).
pStatValue
Pointer to the value to store for this statistic.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

XWriteStatStore does not digitally sign the statistics that it writes to the Xbox hard disk.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
