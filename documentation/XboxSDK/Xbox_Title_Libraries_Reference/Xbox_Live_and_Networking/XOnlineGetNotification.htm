XOnlineGetNotification

The XOnlineGetNotification function tests whether the specified notification exists for the specified user.

BOOL XOnlineGetNotification(
    DWORD dwUserIndex,
    XONLINE_NOTIFICATION_TYPE NotificationType
);

Parameters

dwUserIndex
[in] Specifies the controller index of the user for whom to check for notifications.
NotificationType

[in] Specifies a value of type XONLINE_NOTIFICATION_TYPE. See the XONLINE_NOTIFICATION_TYPE page for a list of possible values.

Return Values

If the specified notification exists for the specified user, the function returns TRUE. Otherwise, the function returns false.

Remarks

To use the XOnlineGetNotification function, only the XOnlineLogon task need be active. The XOnlineFriendsStartup and XOnlineFriendsEnumerate tasks need not be active.

To take action on any of these notifications, the title needs to enumerate the friends list using XOnlineFriendsEnumerate.

The XOnlineGetNotification function supersedes the no-longer-supported XOnlineNotificationIsPending function.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
