XOnlineGetNatType

The XOnlineGetNatType function returns a value that describes the "strictness" of the Network Address Translation (NAT) device between the Xbox console and the Xbox Live Service. 

XONLINE_NAT_TYPE XOnlineGetNatType();

Parameters

None.

Return Values

The XOnlineGetNatType function returns a value from the XONLINE_NAT_TYPE enumeration type.

Remarks

Calling this function requires that the title has successfully logged on to the XONLINE_NAT_TYPE_DETECTION_SERVICE.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
