XNetGetDebugXnAddr

Retrieves the debugger's Xbox network complete address.

DWORD XNetGetDebugXnAddr(
    XNADDR *pxna
);

Parameters

pxna
Address that receives the XNADDR structure that contains the debugger's Xbox network complete address.

Return Values

Returns a combination of one or more of the following flags:

Value Description
XNET_GET_XNADDR_DHCP The host has a dynamic IP address via DHCP.
XNET_GET_XNADDR_DNS The host has one or more DNS servers configured.
XNET_GET_XNADDR_ETHERNET The host has an Ethernet address.
XNET_GET_XNADDR_GATEWAY The host has one or more Internet gateways configured.
XNET_GET_XNADDR_NONE Either the network stack is not initialized or no debugger was found.
XNET_GET_XNADDR_ONLINE The host is currently connected to the online service.
XNET_GET_XNADDR_PENDING The address aquisition is not yet complete.
XNET_GET_XNADDR_PPPOE Host has a PPPoE assigned IP address.
XNET_GET_XNADDR_STATIC The host has a static IP address.
XNET_GET_XNADDR_TROUBLESHOOT The network configuration requires troubleshooting.

Remarks

The debugger's Xbox network complete address is the address used for communications with the debugger. Communications related to the title must use the title's Xbox network complete address.

If XNetGetDebuggerXnAddr returns XNET_GET_XNADDR_PENDING, the title should perodically call the function until it receives a different return value.

Requirements

Header: Declared in Winsockx.h, for include use Xtl.h.

Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. Use the secure versions of the library file (Xnets.lib or XOnlines.lib) for final testing.
