XOnlineStatWriteGetResult

Retrieves a handle to a remote file reference. Use this handle to upload file attachments to accompany player statistics written with the XOnlineStatWrite function. 

HRESULT XOnlineStatWriteGetResult(
    XONLINETASK_HANDLE hTask,
    HANDLE *phServerFileReference,
    PXONLINE_STAT_ATTACHMENT_REFERENCE *prgReferences,
    DWORD *pdwReferences
);

Parameters

hTask
[in] Handle to the task returned by the XOnlineStatWrite function.
phServerFileReference
[out] Pointer to a HANDLE variable that on successful return receives a handle to a remote file reference. Pass this handle as the hServerFileReference parameter to XOnlineStorageUpload.
prgReferences
[out] Pointer to an array of XONLINE_STAT_ATTACHMENT_REFERENCE structures for the users that qualify for attachment upload.
pdwReferences
[out] Pointer to a DWORD variable that receives the number of structures pointed to by the prgReferences parameter.

Return Values

If XOnlineStatWriteGetResult succeeds, the return value is S_OK.

If XOnlineStatWriteGetResult fails, the return value is one of the following error codes.

Value Description
E_INVALIDARG The specified task handle is invalid.
Other See Online Errors for information about other possible errors.

Remarks

Call XOnlineStatWriteGetResult if the XOnlineStatWrite function returns XONLINE_S_STAT_CAN_UPLOAD_ATTACHMENT

Do not call XOnlineTaskClose on the task handle returned by XOnlineStatWrite until after calling XOnlineStorageUpload.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
