ITitlePlayersListItem::IsTalking

Indicates whether the player is currently talking (sending voice packets this frame).

BOOL IsTalking();

Parameters

None.

Return Values

Returns TRUE if the player is currently talking, FALSE otherwise.

Remarks

For titles using Xbox High Level Voice (XHV), the functions IXHVEngine::GetLocalTalkerStatus and IXHVEngine::IsTalking indicate whether a player is currently talking.

Requirements

Header: Declared in Uix.h.

Import Library: Use Uix.lib.

See Also

IXHVEngine::GetLocalTalkerStatus, IXHVEngine::IsTalking
