XONLINE_FRIENDS_RESULTS_ERROR

Contains information about a player list change.

typedef struct {
    XONLINE_FRIENDS_RESULTS_TYPE type;
    union {
        struct {
            XUID xuid;
            HRESULT hr;
        } requestError;
        struct {
            XUID xuid;
            HRESULT hr;
        } removeError;
    } info;
} XONLINE_FRIENDS_RESULTS_ERROR, *PXONLINE_FRIENDS_RESULTS_ERROR;

Members

type
XONLINE_FRIENDS_RESULTS_TYPE value specifying the error type.
xuid
XUID of the player the user attempted to add.
hr
HRESULT value that was returned.
xuid
XUID of the friend the user attempted to remove.
