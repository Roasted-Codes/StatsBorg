sockaddr

Xbox Winsock always interprets a sockaddr* parameter as a pointer to a sockaddr_in structure:

struct sockaddr_in {
        short   sin_family;
        u_short sin_port;
        struct  in_addr sin_addr;
        char    sin_zero[8];
};

The sockaddr structure is defined as:

struct sockaddr {
  u_short    sa_family;
  char       sa_data[14];
};

Requirements

Header: Declared in winsockx.h, for include use Xtl.h.
