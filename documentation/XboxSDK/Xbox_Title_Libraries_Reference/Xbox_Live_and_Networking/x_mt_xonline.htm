Xbox Live Library

The Xbox Live Library provides functionality for Xbox Live multiplayer games, including authentication, billing, matchmaking, and friends lists.

Networking Samples and PartnerNet

With the exception of the voice samples and WinsockPeer, all Networking samples require access to PartnerNet. To log onto PartnerNet, you must configure your Xbox with a machine account and user accounts. Use the Xbox Live Setup Tool in the XDK Launcher to configure your Xbox.

Obtaining Additional Title IDs for Xbox Live Testing

Many of the Xbox Live services return results based on title ID. For instance, the XBOS (Xbox Billing and Offerings) servers enumerate content based on title ID. Approved Xbox Live title developers receive temporary title IDs that may be used for testing against PartnerNet. To obtain additional title IDs, please contact Developer Support at xboxds@xbox.com.

Sample Content Packages on PartnerNet

When calling XOnlineOfferingEnumerate, several sample packages created by Microsoft will be returned. A few of these will indicate that there is a fee for downloading and installing the package. Executing an XOnlineOfferingDetails call on one of the non-free packages will return a sample price and sample tax for the package. However, these samples are provided for completeness and no fee is actually collected. You can call XOnlineOfferingPurchase to complete a fake transaction and download and install the content package without incurring any charge.

Credit Card Numbers for Test User Accounts

Starting with the April 2002 XDK, the billing server requires a credit card number for every user account. We set up a temporary solution on our servers to work with this new requirement.

To create an account using XDash, enter a credit card type of "Visa", and enter the sequence "4111 1111 1111 1111" in the Credit Card # field. Do NOT enter a valid credit card number.

When creating an account from the XDK Launcher, all the required fields (including address and credit card number) are entered for you.
