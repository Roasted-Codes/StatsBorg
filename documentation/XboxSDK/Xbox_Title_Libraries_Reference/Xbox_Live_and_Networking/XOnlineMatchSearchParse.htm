XOnlineMatchSearchParse

Parses the search results returned by the XOnlineMatchSearchGetResults function.

HRESULT XOnlineMatchSearchParse(
    const XONLINE_MATCH_SEARCHRESULT *pSearchResult,
    DWORD dwNumSessionAttributes,
    const XONLINE_ATTRIBUTE_SPEC *pSessionAttributeSpec,
    PVOID pQuerySession
);

Parameters

pSearchResult
[in] Pointer to an XONLINE_MATCH_SEARCHRESULT structure that represents a search result. This structure is returned by the XOnlineMatchSearchGetResults function.
dwNumSessionAttributes
[in] Number of XONLINE_ATTRIBUTE_SPEC structures in the array specified by the pSessionAttributeSpec parameter. This is also the number of attributes appended to the end of the XONLINE_MATCH_SEARCHRESULT structure specified by the pSearchResult parameter. The number of attributes is available from the dwNumAttributes member of the XONLINE_MATCH_SEARCHRESULT structure.
pSessionAttributeSpec
[in] Address of an array of XONLINE_ATTRIBUTE_SPEC structures that specify how to parse the attributes that are appended to the end of the XONLINE_MATCH_SEARCHRESULT structure.
pQuerySession
[out] Address of a buffer to receive the attributes for the search result.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The XOnlineMatchSearchParse function is called from an Xbox that wishes to join an online-match session that is hosted by a different Xbox. For information on how to connect to a particular online-game session, see the Xbox Secure Network Library white paper.
