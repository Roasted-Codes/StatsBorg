IPluginSupport::GetScreenImage

Retrieves the image with the specified screen and image resource identifiers from the skin file. The image is returned as an IDirect3DTexture8 texture.

HRESULT GetScreenImage(
    DWORD ScreenResID,
    DWORD ImageResID,
    IDirect3DTexture8 **ppTexture
);

Parameters

ScreenResID
[in] Resource identifier of the screen.
ImageResID
[in] Resource identifier of the image.
ppTexture
[out] Address of an IDirect3DTexture8 pointer that receives the texture.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The returned texture should not be modified or freed.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
