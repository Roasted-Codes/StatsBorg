XOnlineCompetitionTopologyGetResults

Retrieves the results of the topology search started by XOnlineCompetitionTopology.

HRESULT XOnlineCompetitionTopologyGetResults(
    XONLINETASK_HANDLE hTask,
    DWORD *pdwTotalItemsInSearchResult,
    DWORD *pdwItemsReturned,
    DWORD *pdwResultBufferSize,
    PBYTE pbResultBuffer
);

Parameters

hTask
[in] Handle received from XOnlineCompetitionTopology.
pdwTotalItemsInSearchResult
[out] Address of a DWORD. Upon return, the DWORD contains the total number of items in the search result.
pdwItemsReturned
[out] Address of a DWORD. Upon return, the DWORD contains the number of items in the current page of results.
pdwResultBufferSize
[in, out] Address of a DWORD. Upon return, the DWORD contains the size, in bytes, of the results buffer.
pbResultBuffer
[in, out] Starting address of the results buffer.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

When retrieving the results of a competition topology search, the output information is paged. The number of results in the current page of output is returned through the pdwItemsReturned parameter. The total number of results produced by the search is passed to the calling program through the pdwTotalItemsInSearchResult parameter.

Any structure that you use for returning data from this function must be defined with the following syntax.

    // Members here
