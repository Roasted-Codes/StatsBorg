XOnlineGameInviteAnswer

Responds to a game invitation received from another player.

HRESULT XOnlineGameInviteAnswer(
    DWORD dwUserIndex,
    const XONLINE_GAMEINVITE_ANSWER_INFO *pGameInviteAnswerInfo,
    XONLINE_PEER_ANSWER_TYPE GameInviteAnswer,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the user answering the game invitation. The user must have been successfully logged on with a previous call to XOnlineLogon.
pGameInviteAnswerInfo
[in] Pointer to a XONLINE_GAMEINVITE_ANSWER_INFO structure containing information about invitation answer.
GameInviteAnswer
[in] XONLINE_PEER_ANSWER_TYPE value indicating the response. Can be one of the following values:
Value Description
XONLINE_PEER_ANSWER_NO Rejects the request.
XONLINE_PEER_ANSWER_YES Accepts the request.
XONLINE_PEER_ANSWER_NEVER Rejects the request indefinitely and prevents the recruiting player from sending to the user again.
hWorkEvent
[in, optional] Handle to an event that will be signaled when there is work to be done on this task. This parameter can be NULL if no such event is required.
phTask
[out] Address of an XONLINETASK_HANDLE that must be pumped to completion by XOnlineTaskContinue. Call XOnlineTaskClose to close the task.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function is similar to the XOnlineFriendsAnswerGameInvite function, with the exception that the sender can be a teammate or fellow peer in a competition.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
