XOnlineIsUserPurchaseAllowed

Macro that, given the dwUserFlags value from an XUID structure, determines whether a user is permitted to purchase Xbox Live premium content or subscription offerings.

#define XOnlineIsUserPurchaseAllowed(dwUserFlags) \
   (((dwUserFlags) & XONLINE_USER_PURCHASE_NOT_ALLOWED) == 0)

Parameters

dwUserFlags
[in] Type is DWORD. The dwUserFlags value from an XUID structure contained in an XONLINE_USER structure.

Return Values

The return value is TRUE if the user is permitted to purchase premium offerings; FALSE otherwise.

The return value is of type BOOL.

Remarks

XOnlineIsUserPurchaseAllowed accepts the entire dwUserFlags from an XUID structure and determines whether that user is permitted to purchase premium offerings. Titles must enforce this restriction by not allowing restricted users to purchase premium offerings or to cancel previously purchased subscriptions. This restriction reflects the parental control setting in the Xbox Dashboard.

This macro must be used on XUID structure obtained from a call to XOnlineGetLogonUsers. An XUID structure obtained from a XOnlineGetUsers does not have updated information.

Requirements

Header: Declared in Xonline.h.
