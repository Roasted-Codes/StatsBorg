XOnlineMutelistGet

Retrieves the specified user's mutelist.

HRESULT XOnlineMutelistGet(
    DWORD dwUserIndex,
    DWORD dwMutelistUserBufferCount,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask,
    PXONLINE_MUTELISTUSER pMutelistUsersBuffer,
    DWORD *pdwNumMustlistUsers
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the user whose mutelist will be retrieved. The user must have been successfully logged on with a previous call to XOnlineLogon.
dwMutelistUserBufferCount
[in] Number of XONLINE_MUTELISTUSER structures that will fit into the buffer pointed to by pMutelistUserBuffer. This value should always equal MAX_MUTELISTUSERS.
hWorkEvent
[in, optional] Handle to an event object, created by the title. This event is signaled when there is work to do on the asynchronous task that retrieves the mutelist. Perform work on the task by calling XOnlineTaskContinue with the handle returned in the phTask parameter. This parameter is optional and may be NULL. If this parameter is NULL, the title should call XOnlineTaskContinue periodically to perform work on the task.
phTask
[out] Pointer to a variable of type XONLINETASK_HANDLE that receives a handle to the task that retrieves the mutelist. Pump this task to completion using XOnlineTaskContinue. When the task completes, XOnlineTaskContinue returns XONLINETASK_S_SUCCESS. After the task completes, call XOnlineTaskClose to close the task.
pMutelistUsersBuffer
[out] Points to a buffer that receives an array of XONLINE_MUTELISTUSER structures. The buffer must be large enough to hold at dwMutelistUserBufferCount structures of type XONLINE_MUTELISTUSER.
pdwNumMustlistUsers
[out] Pointer to a DWORD variable that receives the number of XONLINE_MUTELISTUSER structures in the buffer pointed to by the pMutelistUsersBuffer parameter.

Return Values

If XOnlineMutelistGet succeeds, the return value is S_OK.

If XOnlineMutelistGet fails, the return value is one of the following error codes.

Value Description
E_OUTOFMEMORY Insufficient memory. Free some memory.
Other See Online Errors for information about other possible errors.

Remarks

XOnlineMutelistGet does not depend on the task returned by XOnlineMutelistStartup.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
