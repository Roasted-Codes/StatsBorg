XOnlineMessageDetailsGetResultsProperty

Retrieves the specified property from a message's details.

HRESULT XOnlineMessageDetailsGetResultsProperty(
    XONLINETASK_HANDLE hTask,
    WORD wPropTag,
    DWORD dwPropValueBufferSize,
    PVOID pPropValue,
    DWORD *pdwPropValueSize,
    DWORD *pdwAttachmentFlags
);

Parameters

hTask
[in] Task handle obtained from the XOnlineMessageDetails function.
wPropTag

[in] Unique identifier for the property.

Note    The upper byte of a property tag specifies the property's type. Retrieve it with the XOnlineMessageGetPropType macro.

dwPropValueBufferSize
[in] Size, in bytes, of the memory block that receives the property's value.
pPropValue
[in, out, optional] Address of the memory block that receives the property's value.
pdwPropValueSize
[out, optional] Address of a DWORD. Upon return, the DWORD contains the number of bytes of memory the property value actually requires.
pdwAttachmentFlags
[out, optional] One or more property attachment flags from the following table.
Attachment Flag Description
XONLINE_MSG_ATTACHMENT_FLAG_NON_EXPORTABLE The attachment should not be visible when the message is displayed on the Web.
XONLINE_MSG_ATTACHMENT_FLAG_DIRECTORY The property value points to an ANSI path string to a local directory that will be uploaded to storage.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Use the XOnlineMessageGetPropType macro to retrieve the property type from the property tag. If the property type is not XONLINE_MSG_PROP_TYPE_ATTACHMENT, the pdwAttachmentFlags parameter is always zero.

If the property type is XONLINE_MSG_PROP_TYPE_ATTACHMENT, the attachment size is placed in the pdwPropValueSize parameter and the attachment flags are in pdwAttachmentFlags. In this case, the XOnlineMessageDetailsGetResultsProperty function returns XONLINE_E_MESSAGE_PROPERTY_DOWNLOAD_REQUIRED. The title should then call XOnlineMessageDownloadAttachmentToDirectory or XOnlineMessageDownloadAttachmentToMemory to retrieve the attachment.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
