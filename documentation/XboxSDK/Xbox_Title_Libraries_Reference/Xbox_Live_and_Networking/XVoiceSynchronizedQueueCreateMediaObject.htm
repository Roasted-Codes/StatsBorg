XVoiceSynchronizedQueueCreateMediaObject

Creates an instance of the Xbox Media Object (XMO) that handles the queuing and decoding of voice data.

HRESULT XVoiceSynchronizedQueueCreateMediaObject(
    LPXVOICE_QUEUE_XMO_CONFIG pConfig,
    LPXMEDIAOBJECT *ppMediaObject
);

Parameters

pConfig
[in] Address of a XVOICE_QUEUE_XMO_CONFIG structure that is used to initialize the XMO.
ppMediaObject
[out] Address of a pointer to the XMO created.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The created queue XMO is synchronized with other queues created through the same API. These queues provide output at the same time and no external sync is necessary.

Requirements

Header: Declared in XVoice.h.

Import Library: Use XVoice.lib.
