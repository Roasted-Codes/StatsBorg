timeval

The timeval structure is used to specify time values. It is associated with the BSD file Time.h.

struct timeval {
  long    tv_sec;         // seconds 
  long    tv_usec;        // and microseconds 
};

Members

tv_sec
Time value, in seconds.
tv_usec
Time value, in microseconds.

Requirements

Header: Declared in winsockx.h, for include use Xtl.h.
