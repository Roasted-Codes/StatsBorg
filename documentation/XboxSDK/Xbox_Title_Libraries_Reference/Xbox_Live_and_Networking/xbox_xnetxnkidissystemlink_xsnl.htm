XNetXnKidIsSystemLink

Determines if the key identifier is from a system link session.

BOOL XNetXnKidIsSystemLink(
    XNKID *pxnkid
);

Parameters

pxnkid
Address of the XNKID structure to test.

Return Values

Returns TRUE if the XNKID represents an Xbox on the same local subnet who is not signed into the Xbox Live service; otherwise returns FALSE.

Requirements

Header: Declared in Winsockx.h.
