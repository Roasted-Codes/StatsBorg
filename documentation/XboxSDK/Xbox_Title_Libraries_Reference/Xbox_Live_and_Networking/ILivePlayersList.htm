ILivePlayersList

Titles use the methods of the ILivePlayersList interface to add and remove players in the list displayed by the Players feature.

The ILivePlayersList interface is obtained by calling ILiveEngine::GetFeatureInterface with UIX_PLAYERS_FEATURE.

ILivePlayersList methods Refresh
RegisterPlayer
SetFilterFlags
UnregisterPlayer

The ILivePlayersList interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

Requirements

Header: Declared in Uix.h.
