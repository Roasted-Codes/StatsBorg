XOnlineFriendsRemove

Removes a friend from the specified user's Friends List.

HRESULT XOnlineFriendsRemove(
    DWORD dwUserIndex,
    const XONLINE_FRIEND *pFriend
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the user whose Friends List will be modified. The user must have been successfully logged on with a previous call to XOnlineLogon.
pFriend
[in] Pointer to an XONLINE_FRIEND structure specifying the friend that will be removed from the user's Friends List.

Return Values

If XOnlineFriendsRemove succeeds, it returns S_OK.

If XOnlineFriendsRemove fails, it returns one of the following error codes.

Value Description
XONLINE_E_NOTIFICATION_NO_TASK The Friends startup task is not active. You must first call XOnlineFriendsStartup to create the startup task.
S_FALSE The user tried to delete his friend and then tried again before the list was updated. This error should occur only if the title is currently enumerating the Friends List. This return code could also be the result of a developer error if the game allows the player to remove users that aren't on their Friends List.
Other See Online Errors for information about other possible errors.

Remarks

Pump the task handle returned by XOnlineFriendsStartup to submit the change to the Xbox Live service. Pump the task handle returned by XOnlineFriendsEnumerate to update the Friends List with the result of the removal.

The XOnlineFriendsRemove function may also be used to cancel a pending friend request.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
