XONLINE_GAMEINVITE_ANSWER_TYPE

Defines the values used to specify answers to a game invitation.

typedef enum {
    XONLINE_GAMEINVITE_NO = 0,
    XONLINE_GAMEINVITE_YES = 1,
    XONLINE_GAMEINVITE_REMOVE = 2,
} XONLINE_GAMEINVITE_ANSWER_TYPE;

Constants

XONLINE_GAMEINVITE_NO
Declines the game invitation.
XONLINE_GAMEINVITE_YES
Accepts the game invitation.
XONLINE_GAMEINVITE_REMOVE
Does not accept or decline. Instead, removes the inviter from the friends list of the user that received the invitation.
