ITitleUIPlugin::SetPluginSupport

Receives an IPluginSupport interface from the Xbox Drop-in UI engine. The IPluginSupport interface is used by the UI plug-in to access skin file and word-wrap support functions.

HRESULT SetPluginSupport(
    IPluginSupport *pPluginSupport
);

Parameters

pPluginSupport
[in] Pointer to an IPluginSupport interface that provides support functions.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The Xbox Drop-in UI engine calls SetPluginSupport when the UI plug-in is registered with ILiveEngine::SetUIPlugin. If SetPluginSupport returns a failure code, the engine fails the ILiveEngine::SetUIPlugin call.

Requirements
