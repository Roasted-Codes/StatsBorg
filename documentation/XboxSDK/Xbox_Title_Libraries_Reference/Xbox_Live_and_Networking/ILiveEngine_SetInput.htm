ILiveEngine::SetInput

Forwards controller input received by the title on to the Xbox Drop-in UI engine.

HRESULT SetInput(
    DWORD Port,
    const XINPUT_STATE *pInputState
);

Parameters

Port
[in] The controller port from which input was received.
pInputState
[in] Pointer to an XINPUT_STATE structure containing the controller input state. Specify NULL to indicate that there is no controller in Port.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Titles must forward controller input to the engine by calling SetInput within the title's render loop (typically once per frame). The XINPUT_STATE structure is obtained by calling XInputGetState.

There is no explicit support within the Xbox Drop-in UI engine for nonstandard input values generated by alternative input devices (such as light guns, steering wheels, boards, and so on). To be processed by the engine, nonstandard input must be mapped to standard buttons and directional controls in the XINPUT_STATE structure.
