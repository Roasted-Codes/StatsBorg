XOnlineCleanup

Performs required cleanup for the Xbox Live functions.

HRESULT XOnlineCleanup();

Parameters

None.

Return Values

If the function succeeds, the return value is S_OK.

Remarks

Call this function when the title is finished using the Xbox Live functions and wishes to clean up associated resources. After calling this function, no additional Xbox Live functions (except XOnlineStartup) can be called.

Do not call XOnlineCleanup, if you use XLaunchNewImage with reason code, XLD_LAUNCH_DASHBOARD_ACCOUNT_MANAGEMENT, to boot to the Xbox Dashboard.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
