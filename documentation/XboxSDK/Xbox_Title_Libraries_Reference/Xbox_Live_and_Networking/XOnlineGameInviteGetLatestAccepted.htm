XOnlineGameInviteGetLatestAccepted

Retrieves invitation data that was stored locally on the Xbox console when a player accepted a cross-title invitation.

HRESULT XOnlineGameInviteGetLatestAccepted(
    XONLINE_LATEST_ACCEPTED_GAMEINVITE *pLatestAcceptedGameInvite
);

Parameters

pLatestAcceptedGameInvite
[out] Pointer to an XONLINE_LATEST_ACCEPTED_GAMEINVITE structure that receives information about the accepted cross-title invitation.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function is similar to the XOnlineFriendsGetAcceptedGameInvite function, with the exception that it can apply to teammates or fellow peers in a competition.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
