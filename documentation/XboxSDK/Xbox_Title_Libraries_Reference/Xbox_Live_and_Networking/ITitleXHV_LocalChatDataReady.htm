ITitleXHV::LocalChatDataReady

HRESULT LocalChatDataReady(
    DWORD dwLocalPort,
    DWORD dwSize,
    VOID *pData
);

Parameters

dwLocalPort
[in] Port of the local talker associated with the voice data callback.
dwSize
[in] Size of encoded voice data.
pData
[in] Pointer to encoded voice data.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
