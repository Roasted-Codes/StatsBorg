XOnlinePresenceGetTitleName

Retrieves a title name.

HRESULT XOnlinePresenceGetTitleName(
    XONLINETASK_HANDLE hTask,
    DWORD dwTitleID,
    DWORD dwLanguage,
    DWORD dwTitleNameSize,
    LPWSTR wszTitleName
);

Parameters

hTask
[in] An XONLINETASK_HANDLE returned from XOnlinePresenceInit.
dwTitleID
[in] ID of the title.
dwLanguage
[in] Language that should be used for the title name.
dwTitleNameSize
[in] Maximum length of the title name string.
wszTitleName
[out] Wide character string to contain the returned title name.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
