ILiveFriendsList

Titles use the methods of the ILiveFriendsList interface to access a user's friends list when the friends feature is enabled but not actively displayed.

The ILiveFriendsList interface is obtained by calling ILiveEngine::GetFriendsForUser.

ILiveFriendsList methods Count
GetFriendByIndex
GetFriendByXUID
HasChanged
IsReady
Refresh
Remove
Request

The ILiveFriendsList interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.
