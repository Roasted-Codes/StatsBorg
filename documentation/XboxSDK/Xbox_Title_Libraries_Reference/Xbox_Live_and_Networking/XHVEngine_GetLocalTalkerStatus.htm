IXHVEngine::GetLocalTalkerStatus

HRESULT GetLocalTalkerStatus(
    DWORD dwLocalPort,
    XHV_LOCAL_TALKER_STATUS *pLocalTalkerStatus
);

Parameters

dwLocalPort
[in] Controller port of the local talker.
pLocalTalkerStatus
[out] Pointer to an XHV_LOCAL_TALKER_STATUS structure that receives the status of the local talker.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
