Xbox Network Subsystem Memory Management

Two of the top design goals for the Xbox Title Libraries are predictability and speed. In line with these design goals, the Xbox network subsystem differs in a number of ways from its desktop counterparts. One of the more significant changes is in how it allocates memory. The topics in this section highlight some of the differences in the network memory architecture and discuss some of the potential impacts that this may have on development using the Xbox network subsystem libraries.
