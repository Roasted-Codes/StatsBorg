XOnlineMessageDelete

Deletes a message received from the server.

HRESULT XOnlineMessageDelete(
    DWORD dwUserIndex,
    DWORD dwMessageID,
    BOOL fBlockSender
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player that received the message to be deleted.
dwMessageID
[in] Unique integer that identifies the message.
fBlockSender
[in] If this parameter is TRUE, all future communication from this message's sender is deleted automatically. The default is FALSE.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
