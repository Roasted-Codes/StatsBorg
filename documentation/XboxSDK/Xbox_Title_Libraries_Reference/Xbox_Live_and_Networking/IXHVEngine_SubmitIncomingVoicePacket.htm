IXHVEngine::SubmitIncomingVoicePacket

HRESULT SubmitIncomingVoicePacket(
    XUID xuidRemoteTalker,
    VOID *pvData,
    DWORD dwSize
);

Parameters

xuidRemoteTalker
[in] XUID defining the remote player associated with the incoming voice packet.
pvData
[in] Pointer to the data packet.
dwSize
[in] Size, in bytes, of the data packet.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
