IXVoiceDecoder::GetVoiceProperties

Retrieves an array of XVOICE_VOICE_PROPERTIES structures that contain voice properties (lip-synchronization information) for this decoder's input streams. 

HRESULT GetVoiceProperties(
    DWORD dwArraySize,
    LPXVOICE_VOICE_PROPERTIES pPropertiesArray
);

Parameters

dwArraySize
[in] Number of XVOICE_VOICE_PROPERTIES structures in the array pointed to by the pPropertiesArray parameter. The number of structures is equal to the number of input streams for this decoder. For example, if the decoder is a four-to-one decoder, the number of structures is equal to four.
pPropertiesArray
[out] Pointer to an array of XVOICE_VOICE_PROPERTIES structures.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The decoder buffers the voice properties for the last set of decoded packets ("frame"). Call GetVoiceProperties after each call to IXVoiceDecoder::ProcessMultiple to obtain the most recent frame's voice properties.

Requirements

Header: Declared in Xvoice.h.

Import Library: Use Xvoice.lib.
