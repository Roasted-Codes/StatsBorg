XOnlineMatchSessionDelete

Creates a delete session request.

HRESULT XOnlineMatchSessionDelete(
    XNKID SessionID,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

SessionID
[in] Session identifier.
hWorkEvent
[in, optional] Optional. Handle to a caller-created event object. This object will be signaled when there is work to do by calling XOnlineTaskContinue. The caller may pass in NULL instead, indicating that the caller will poll XOnlineTaskContinue itself.
phTask
[out] Address of an XONLINETASK_HANDLE that uniquely identifies this delete session request.

Return Values

Returns S_OK if successful.

Otherwise, returns one of the following error codes.

Value Description
E_OUTOFMEMORY Insufficient memory. Free some memory.
WSAEADDRNOTAVAIL See Winsock Error Codes for more information.
WSACONNRESET See Winsock Error Codes for more information.
Other See Online Errors for information about other possible errors.

Remarks

Call the XOnlineMatchSessionDelete function to stop advertising the online-game session. Only the Xbox that called XOnlineMatchSessionCreate can call XOnlineMatchSessionDelete.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
