XVoiceGetEncodedPacketEnergy

Provides an energy measurement of a given encoded media packet. 

HRESULT XVoiceGetEncodedPacketEnergy(
    LPCXMEDIAPACKET pPacket,
    FLOAT *pfEnergy
);

Parameters

pPacket
[in] Address of an XMEDIAPACKET structure containing voice information. The structure should contain an encoded packet (not PCM data).
pfEnergy
[out] Pointer to a FLOAT, containing the returned energy value.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
