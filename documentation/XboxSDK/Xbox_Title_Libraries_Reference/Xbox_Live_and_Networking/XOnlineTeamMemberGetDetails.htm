XOnlineTeamMemberGetDetails

Retrieves information about a particular team member.

HRESULT XOnlineTeamMemberGetDetails(
    XONLINETASK_HANDLE hTask,
    XUID xuidTeamMember,
    XONLINE_TEAM_MEMBER *pTeamMemberInfo
);

Parameters

hTask
[in] An XONLINETASK_HANDLE.
xuidTeamMember
[in] XUID that uniquely identifies the team member.
pTeamMemberInfo
[out] Address of an XONLINE_TEAM_MEMBER structure to contain the team member information.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
