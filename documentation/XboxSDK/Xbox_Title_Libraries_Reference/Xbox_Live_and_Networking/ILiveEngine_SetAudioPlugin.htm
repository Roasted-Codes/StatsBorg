ILiveEngine::SetAudioPlugin

Sets the ITitleAudioPlugin used by the Xbox Drop-in UI engine. This can be the default audio plug-in obtained with UIXCreateAudioPlugin, or a custom plug-in implemented by the title.

HRESULT SetAudioPlugin(
    ITitleAudioPlugin *pAudioPlugin
);

Parameters

pAudioPlugin
[in] Pointer to an ITitleAudioPlugin interface for the audio plug-in.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

After an audio plug-in has been exposed to the Xbox Drop-in UI engine with SetAudioPlugin, the plug-in will begin receiving audio requests through its ITitleAudioPlugin interface.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
