IXHVEngine::RegisterRemoteTalker

HRESULT RegisterRemoteTalker(
    XUID xuidRemoteTalker
);

Parameters

xuidRemoteTalker
[in] The XUID of the user to register as a remote talker.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

If the specified user is already registered as a remote talker, the function does nothing and returns S_OK.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
