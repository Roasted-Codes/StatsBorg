IXHVEngine::SetVoiceMask

Sets the input voice mask parameters for the specified local talker.

HRESULT SetVoiceMask(
    DWORD dwLocalPort,
    const XHV_VOICE_MASK *pVoiceMask
);

Parameters

dwLocalPort
[in] Controller port of the local talker.
pVoiceMask
[in] Pointer to an XHV_VOICE_MASK structure that specifies the voice mask parameters to apply.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Specifying a value of XHV_VOICE_MASK_PARAM_DISABLED for any member of the XHV_VOICE_MASK structure disables the masking effects controlled by that member.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
