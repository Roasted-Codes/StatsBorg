ITitlePlayersListItem::GetName

Returns the display name for the player as it should appear in the Players List.

WCHAR* GetName();

Parameters

None.

Return Values

Returns a pointer to a null-terminated Unicode string containing the player's display name.

Remarks

The display name can contain information in addition to the player's name—for example, whether the player is a guest, the player's score, status, or other text.

Requirements

Header: Declared in Uix.h.

Import Library: Use Uix.lib.
