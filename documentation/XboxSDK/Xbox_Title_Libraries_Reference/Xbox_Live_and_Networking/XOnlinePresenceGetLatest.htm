XOnlinePresenceGetLatest

Retrieves the most recent presence session information.

HRESULT XOnlinePresenceGetLatest(
    XONLINETASK_HANDLE hTask,
    DWORD dwGroupID,
    DWORD dwUserPresenceBufferCount,
    XONLINE_PRESENCE *pUserPresence
);

Parameters

hTask
[in] An XONLINETASK_HANDLE returned from XOnlinePresenceInit.
dwGroupID
[in] ID of the group requested.
dwUserPresenceBufferCount
[in] Size of the array of XONLINE_PRESENCE structures to be returned.
pUserPresence
[out] Pointer to the XONLINE_PRESENCE data returned.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
