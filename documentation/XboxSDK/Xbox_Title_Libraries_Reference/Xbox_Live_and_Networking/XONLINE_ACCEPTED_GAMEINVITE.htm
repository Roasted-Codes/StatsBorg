XONLINE_ACCEPTED_GAMEINVITE

Contains details about an accepted game invitation.

typedef struct{
    XONLINE_FRIEND InvitingFriend;
    XUID xuidAcceptedFriend;
    FILETIME InviteAcceptTime;
    XUID xuidLogonUsers[XONLINE_MAX_LOGON_USERS];
} XONLINE_ACCEPTED_GAMEINVITE, *PXONLINE_ACCEPTED_GAMEINVITE;

Members

InvitingFriend
XONLINE_FRIEND structure containing information about the user that sent the invitation.
xuidAcceptedFriend
XUID structure containing the user identifier of the Friend that accepted the invitation.
InviteAcceptTime
FILETIME value specifying the time that the Friend accepted the invitation. An invitation should be considered expired after 15 minutes.
xuidLogonUsers
Array of XUID structures that specify the users that were logged in at the time the cross-title invitation was accepted. Use the XOnlineGetUsers function to retrieve information for these users (see Remarks below). The index of a XUID in this array is not necessarily equal to the controller index of the corresponding user.

Remarks

The XOnlineGetUsers function returns an array of XONLINE_USER structures for the users accounts stored on the Xbox console hard disk and connected memory units.

Requirements
