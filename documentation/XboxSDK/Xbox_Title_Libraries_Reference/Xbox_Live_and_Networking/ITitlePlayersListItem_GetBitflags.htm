ITitlePlayersListItem::GetBitflags

Returns a DWORD bitmask representing title-specific flags associated with the player.

DWORD GetBitflags();

Parameters

None.

Return Values

Returns a DWORD bitmask containing the player's flags.

Remarks

A player's bit flags are used in conjunction with ILivePlayersList::SetFilterFlags to determine the set of players displayed in the Players List. Flags could represent, for example, specific teams, players with certain abilities, or whatever other filter capabilities the title wishes to enable.

Requirements

Header: Declared in Uix.h.

Import Library: Use Uix.lib.
