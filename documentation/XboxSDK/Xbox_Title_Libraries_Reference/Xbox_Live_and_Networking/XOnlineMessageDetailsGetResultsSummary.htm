XOnlineMessageDetailsGetResultsSummary

Retrieves the summary for a particular message after XOnlineMessageDetails finishes.

HRESULT XOnlineMessageDetailsGetResultsSummary(
    XONLINETASK_HANDLE hTask,
    XONLINE_MSG_SUMMARY *pMsgSummary,
    DWORD *pdwNumProperties,
    ULONGLONG *pqwAttachmentsSize
);

Parameters

hTask
[in] The task handle obtained from the XOnlineMessageDetails function.
pMsgSummary
[out, optional] Address of an XONLINE_MSG_SUMMARY structure. Upon return, the structure contains the message summary.
pdwNumProperties
[out, optional] Address of a DWORD. If the address is not NULL, XOnlineMessageDetailsGetResultsSummary stores the number of properties in the message into the DWORD. If the address is NULL, this parameter is ignored.
pqwAttachmentsSize
[out, optional] Holds the address of a ULONGLONG variable. If the address is not NULL, XOnlineMessageDetailsGetResultsSummary stores the total size of the message's attachments into the ULONGLONG variable. If the address is NULL, this parameter is ignored.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

You can also use the XOnlineMessageEnumerate or XOnlineMessageSummary function to retrieve a message summary.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.

See Also

XOnlineMessageDetails, XOnlineMessageEnumerate, XOnlineMessageSummary
