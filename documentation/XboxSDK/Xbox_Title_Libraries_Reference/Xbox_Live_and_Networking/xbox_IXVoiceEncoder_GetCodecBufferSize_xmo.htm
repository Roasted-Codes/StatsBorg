IXVoiceEncoder::GetCodecBufferSize

Returns the size of the output buffer required to store the compressed data that results from the specified amount of input data. 

HRESULT GetCodecBufferSize(
    DWORD dwPCMBufferSize,
    DWORD *pdwCodecBufferSize
);

Parameters

dwPCMBufferSize
[in] Amount of input data.
pdwCodecBufferSize
[out] Pointer to a DWORD variable that receives the size of the output buffer required to store the compressed data.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The title typically calls this method once for a given encoder after it has established the size of the input buffer it will use.

Requirements

Header: Declared in Xvoice.h.

Import Library: Use Xvoice.lib.
