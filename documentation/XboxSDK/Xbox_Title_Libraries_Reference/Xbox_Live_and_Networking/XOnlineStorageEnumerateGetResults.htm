XOnlineStorageEnumerateGetResults

Returns the results of a call to XOnlineStorageEnumerate.

HRESULT XOnlineStorageEnumerateGetResults(
    XONLINETASK_HANDLE hTask,
    DWORD *pdwTotalResults,
    DWORD *pdwResultsReturned,
    PXONLINESTORAGE_FILE_INFO **prgpStorageFileInfo
);

Parameters

hTask
[in] Pointer to a variable of type XONLINETASK_HANDLE that receives a task handle. Use this handle in calls to XOnlineTaskContinue to process the file enumeration.
pdwTotalResults
[out] Number of files in global storage.
pdwResultsReturned
[out] Number of structures in the array of results returned.
prgpStorageFileInfo
[out] Array of pointers to XONLINESTORAGE_FILE_INFO structures containing the results of the enumeration.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function should be called after the pumping of the XOnlineStorageEnumerate task returns a success code other than XONLINETASK_S_RUNNING.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
