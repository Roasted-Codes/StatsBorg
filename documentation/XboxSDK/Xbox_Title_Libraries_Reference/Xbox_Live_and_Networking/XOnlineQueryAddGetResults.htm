XOnlineQueryAddGetResults

Retrieves the results of the task started by XOnlineQueryAdd. This is a low-level function. In gecneral, developers find it easier to use XOnlineCompetitionCreateGetResults.

HRESULT XOnlineQueryAddGetResults(
    XONLINETASK_HANDLE hTask,
    XENTITY_ID *pEntityId
);

Parameters

hTask
[in] Task handle received from XOnlineQueryAdd.
pEntityId
[out] Pointer to an XENTITY_ID variable. Upon return, this variable contains the results of the competition creation task.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function should only be called after successful competition of the task started by XOnlineQueryAdd.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.

See Also

XOnlineQueryAdd, XOnlineCompetitionCreateGetResults
