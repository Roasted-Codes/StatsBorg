ITitleUIPlugin::SetRenderTarget

Sets the plug-in's render target to the specified surface.

HRESULT SetRenderTarget(
    IDirect3DSurface8 *pSurface
);

Parameters

pSurface
[in] Pointer to an IDirect3DSurface8 surface that becomes the new render target.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The surface specified in pSurface becomes the render target for subsequent RenderObject calls.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
