XONLINE_STAT_ELO

Contains information for calculating the ELO rating for a pair of competitors.

typedef struct{
    XUID xuid1;
    XUID xuid2;
    DWORD dwLeaderBoardID;
    DWORD dwConditionalIndex;
    double W;
    double C1;
    double C2;
} XONLINE_STAT_ELO, *PXONLINE_STAT_ELO;

Members

xuid1
ID of player 1.
xuid2
ID of player 2.
dwLeaderBoardID

Leaderboard to be updated.

Note    Leaderboards are also referred to as scoreboards.

dwConditionalIndex
One-based index of XONLINE_STAT_CONDITIONAL procedure that determines whether this update occurs, or zero if always updated.
W
Score of the winner.
C1
Weighting value of the range in which player 1's rating currently falls.
C2
Weighting value of the range in which player 2's rating currently falls.

Requirements

Header: Declared in Xonline.h.
