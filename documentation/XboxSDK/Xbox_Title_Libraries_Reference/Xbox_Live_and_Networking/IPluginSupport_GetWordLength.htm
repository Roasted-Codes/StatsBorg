IPluginSupport::GetWordLength

Determines the next word length in a string by calculating the number of characters up to (and including) the first break (wrappable) character in the string.

HRESULT GetWordLength(
    LPCWSTR pString,
    DWORD *pWordLength
);

Parameters

pString
[in] Pointer to a null-terminated Unicode string.
pWordLength
[out] Pointer to a DWORD variable that receives the length, in characters, up to and including the first break character in String.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

If the string contains no break characters, then the length of the entire string is returned.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
