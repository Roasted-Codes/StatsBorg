XOnlineArbitrationReport

Files an arbitration report. This updates players' reliability ratings.

HRESULT XOnlineArbitrationReport(
    const XONLINE_ARB_ID *pArbID,
    DWORD dwNumStatProcs,
    const XONLINE_STAT_PROC *pStatProcs,
    const XONLINE_ARB_REPORT_DATA *pReportData,
    DWORD dwFlags,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

pArbID
[in] Arbitration session ID.
dwNumStatProcs
[in, optional] Number of statistical entries in the array specified by the pStatProcs parameter.
pStatProcs
[in, optional] Array of XONLINE_STAT_PROC structures that are used to report the stats for each player.
pReportData
[in, optional] Array of XONLINE_ARB_REPORT_DATA structures that are used to report the round's results for each player.
dwFlags
[in] Set of zero or more of the following flags combined with a bitwise OR operation. If these flags are not needed, set this parameter to zero.
Flag Value Description
XONLINE_ARB_REPORT_FLAG_WAS_HOST 00000001 Caller was the host for the arbitrated round.
XONLINE_ARB_REPORT_FLAG_VOLUNTARILY_QUITTING 00000002 Caller accepts disconnect penalty and stats submitted by other participants, but wants to report connectivity or suspicious information before leaving.
hWorkEvent
[in, optional] Handle to an event that will be signaled when there is work to be done on this task. This parameter can be NULL if no such event is required.
phTask
[out] Address of an XONLINETASK_HANDLE variable. Upon return, this variable contains the handle of the task initiated by this function.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function reports the results of a specified round. If stats are specified, every participant must pass the stats for all participants. The arbitration server matches up and analyzes the stats, connectivity problems, and suspicious information reported by every participant to determine the outcome of the round. Discrepancies may affect a user's reliability in future arbitration.

For competitions, call XOnlineCompetitionSubmitResults.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
