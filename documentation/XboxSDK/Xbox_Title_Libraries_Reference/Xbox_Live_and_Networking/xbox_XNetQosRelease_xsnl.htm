XNetQosRelease

Releases the resources associated with a set of QoS probes initiated by XNetQosLookup. 

INT XNetQosRelease(
    XNQOS * pxnqos
);

Parameters

pxnqos
Pointer to the XNQOS structure returned by XNetQosLookup.

Return Values

Returns zero if successful, an error code otherwise.

Remarks

To cancel the lookup before it is complete, the title may call XNetQosRelease before all the QoS probes have returned results. This terminates the lookup immediately without waiting for any further network traffic, and there are no side effects. However, your program cannot dereference the XNQOS structure returned through the XNetQosLookup function's ppxnqos parameter after calling XNetQosRelease.

Requirements

Header: Declared in Winsockx.h.

Import Library: Use Xonline.lib or xnets.lib.
