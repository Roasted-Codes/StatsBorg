XONLINE_TAX_TYPE

The values of the XONLINE_TAX_TYPE enumeration type define the possible types for tax used by the online offering pricing.

typedef enum {
    NO_TAX = 0,
    DEFAULT,
    GST,
    VAT,
    TAX_NOT_APPLICABLE,
} XONLINE_TAX_TYPE;

Constants

NO_TAX
Tax not calculated
DEFAULT
Default tax for the title's region
GST
General sales tax
VAT
Value added tax
TAX_NOT_APPLICABLE
Tax not applicable
