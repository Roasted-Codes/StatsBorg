ILiveEngine::GetFriendsForUser

Retrieves an ILiveFriendsList interface representing the friends list for a specified user, and begins friends list enumeration for that user.

HRESULT GetFriendsForUser(
    DWORD Port,
    ILiveFriendsList **ppFriendsList
);

Parameters

Port
[in] The controller port index (zero-based) of the user for which the friends list will be retrieved.
ppFriendsList
[out] Address of an ILiveFriendsList pointer that receives the friends list interface.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Titles can use ILiveFriendsList to retrieve friends information for a user only while the friends feature is not currently displaying the friends list. Titles can use the friends list information to indicate which players in the players list are friends. Additionally, when displaying statistics, titles may wish to show a comparison between player stats and friends stats.

When the interface is no longer needed, call Release on it to stop friends list enumeration for that user and to free associated resources.

When the friends feature is active (displaying the friends list), the title cannot call any ILiveFriendsList methods. Because of this, and because an extant ILiveFriendsList consumes resources, the title should free all of its ILiveFriendsList interfaces before starting the friends feature. The title should also not call any direct Xbox Live friends functions (such as XOnlineFriendsGetLatest), while the friends feature or ILiveFriendsList is active.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
