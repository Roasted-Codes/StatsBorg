IPluginSupport

IPluginSupport provides Xbox Drop-in UI engine support functions to custom ITitleUIPlugin plug-ins. UI plug-ins use the support functions primarily to obtain skin file information.

The IPluginSupport interface is passed to ITitleUIPlugin::SetPluginSupport by the engine when the UI plug-in is registered with ILiveEngine::SetUIPlugin.

IPluginSupport methods GetImage
GetLayout
GetScreenImage
GetString
GetWordLength

The IPluginSupport interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.
