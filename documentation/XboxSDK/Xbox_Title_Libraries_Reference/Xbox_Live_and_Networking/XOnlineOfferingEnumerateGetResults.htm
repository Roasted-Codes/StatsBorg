XOnlineOfferingEnumerateGetResults

Retrieves the results of an offering enumeration initiated with a call to XOnlineOfferingEnumerate.

HRESULT XOnlineOfferingEnumerateGetResults(
    XONLINETASK_HANDLE hTask,
    PXONLINEOFFERING_INFO **prgpOfferingInfo,
    DWORD *pdwReturnedResults,
    BOOL *pfMoreResults
);

Parameters

hTask
[in] Online task handle returned by the call to XOnlineOfferingEnumerate that began the enumeration.
prgpOfferingInfo
[out] Pointer to a variable that receives the address of an array of pointers to XONLINEOFFERING_INFO structures. Each pointer in the returned array will point to an XONLINEOFFERING_INFO structure representing a single offering in the enumeration.
pdwReturnedResults
[out] Pointer to a variable that receives the number of structure pointers returned in the prgpOfferingInfo array.
pfMoreResults
[out] Pointer to a variable that receives TRUE if more records are available for enumeration, and FALSE otherwise.

Return Values

If the function succeeds, the return value is S_OK.

Remarks

If pfMoreResults is set to TRUE on return, the function should continue the offering enumeration by calling XOnlineTaskContinue until more results or task completion is indicated, then making another call to XOnlineOfferingEnumerateGetResults.

The title does not need to free the data pointed to by prgpOfferingInfo, but should, when done accessing that data and when enumeration is complete, call XOnlineTaskClose with the enumeration task handle in order to free the resources associated with the enumeration.

The following code snippet demonstrates calling XOnlineOfferingEnumerateGetResults and displaying to the debugger the type of each offering returned:


PXONLINEOFFERING_INFO *rgpOfferingInfo;
DWORD dwResults, i;
CHAR szBuf[15];
HRESULT hr;
BOOL fMore;


hr = XOnlineOfferingEnumerateGetResults( hTask, &rgpOfferingInfo, &dwResults, &fMore );
if( !SUCCEEDED( hr ) )
{
	return hr;
}

for( i = 0; i < dwResults; i++ )
{
    wsprintf( szBuf, "%u\n", rgpOfferingInfo[i]->dwOfferingType );
    OutputDebugString( szBuf );
};
