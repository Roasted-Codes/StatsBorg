XHVEngineCreate

Creates the Xbox High Level Voice engine and returns an IXHVEngine interface.

Note    Call XInitDevices and XOnlineLogon before calling XHVEngineCreate, or the console may fault and create a crash dump.

HRESULT XHVEngineCreate(
    PXHV_RUNTIME_PARAMS pParams,
    PXHVENGINE *ppEngine
);

Parameters

pParams
[in] Pointer to an XHV_RUNTIME_PARAMS structure that specifies initialization parameters for the XHV engine.
ppEngine
[out] Pointer to a PXHVENGINE variable that receives the created IXHVEngine interface.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

To free the XHV engine and clean up the associated resources, call Release on the returned IXHVEngine pointer. The title must free the XHV engine before calling XOnlineChangeLogonUsers, and then (if desired), re-initialize the XHV engine to restore voice capability.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
