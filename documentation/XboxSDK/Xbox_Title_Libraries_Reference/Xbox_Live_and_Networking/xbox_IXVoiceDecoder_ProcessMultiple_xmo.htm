IXVoiceDecoder::ProcessMultiple

Decompresses the data packets in the specified input array, and outputs the decompressed data to the specified output array. 

HRESULT ProcessMultiple(
    DWORD dwInputArraySize,
    LPCXMEDIAPACKET pInputPacketArray,
    DWORD dwOutputArraySize,
    LPCXMEDIAPACKET pOutputPacketArray
);

Parameters

dwInputArraySize
[in] Number of elements in the array pointed to by the pInputPacketArray parameter.
pInputPacketArray
[in] Pointer to an array of XMEDIAPACKET structures. The elements of this array contain the compressed voice data.
dwOutputArraySize
[in] Number of elements in the array pointed to by the pOutputPacketArray parameter.
pOutputPacketArray
[out] Pointer to an array of XMEDIAPACKET structures. The elements of this array receive the decompressed output data.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in Xvoice.h.

Import Library: Use Xvoice.lib.
