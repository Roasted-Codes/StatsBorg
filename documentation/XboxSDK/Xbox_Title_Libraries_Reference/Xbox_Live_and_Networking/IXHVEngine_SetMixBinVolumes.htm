IXHVEngine::SetMixBinVolumes

HRESULT SetMixBinVolumes(
    XUID xuidRemoteTalker,
    const DSMIXBINS *pMixBins
);

Parameters

xuidRemoteTalker
[in] XUID defining the remote player associated with the mixbin volume assignment.
pMixBins
[in] Pointer to a DSMIXBINS structure that specifies the mixbins and the volumes they should be set to.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

SetMixBinVolumes is limited in that it doesn't support separate sends to the same mixbin. It can send more than one channel of a multichannel sound to the same mixbin, but it cannot control the volume of each send separately. Additionally, SetMixBinVolumes generates a warning if passed an array which contains the same mixbin in more than one entry of that array.

Requirements

Import Library: Use Xvoice.lib.
