XONLINE_PASSCODE_TYPE

Defines values that allow users to enter passcodes with an Xbox controller.

typedef enum {
    XONLINE_PASSCODE_DPAD_UP = 1,
    XONLINE_PASSCODE_DPAD_DOWN,
    XONLINE_PASSCODE_DPAD_LEFT,
    XONLINE_PASSCODE_DPAD_RIGHT,
    XONLINE_PASSCODE_GAMEPAD_X,
    XONLINE_PASSCODE_GAMEPAD_Y,
    XONLINE_PASSCODE_GAMEPAD_LEFT_TRIGGER = 9,
    XONLINE_PASSCODE_GAMEPAD_RIGHT_TRIGGER,
} XONLINE_PASSCODE_TYPE;

Constants

XONLINE_PASSCODE_DPAD_UP
The user pressed the up direction on the directional pad.
XONLINE_PASSCODE_DPAD_DOWN
The user pressed the down direction on the directional pad.
XONLINE_PASSCODE_DPAD_LEFT
The user pressed the left direction on the directional pad.
XONLINE_PASSCODE_DPAD_RIGHT
The user pressed the right direction on the directional pad.
XONLINE_PASSCODE_GAMEPAD_X
The user pressed the X button.
XONLINE_PASSCODE_GAMEPAD_Y
The user pressed the Y button.
XONLINE_PASSCODE_GAMEPAD_LEFT_TRIGGER
The user pulled the left trigger on the controller.
XONLINE_PASSCODE_GAMEPAD_RIGHT_TRIGGER
The user pulled the right trigger on the controller.
