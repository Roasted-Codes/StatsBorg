XONLINE_STAT_CUSTOM

Contains the parameters for a title-specific statistics update procedure.

typedef struct{
    DWORD dwNumParams;
    XONLINE_STAT_CUSTOM_PARAM *pParams;
} XONLINE_STAT_CUSTOM, *PXONLINE_STAT_CUSTOM;

Members

dwNumParams
Number of parameters in the array specified by the pParams member.
pParams
Array of XONLINE_STAT_CUSTOM_PARAM structures containing the parameters to the custom statistics update procedure.

Requirements

Header: Declared in Xonline.h.
