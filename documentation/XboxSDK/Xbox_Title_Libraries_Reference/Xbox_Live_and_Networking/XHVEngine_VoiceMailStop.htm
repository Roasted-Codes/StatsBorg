IXHVEngine::VoiceMailStop

Stops the current voice mail operation on the local talker.

HRESULT VoiceMailStop(
    DWORD dwLocalPort
);

Parameters

dwLocalPort
[in] Controller port of the local talker.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This will generate a ITitleXHV::VoiceMailStopped callback notification. In the case where the voice mail is recording, this will also generate a ITitleXHV::VoiceMailDataReady callback notification.

Requirements

Header: Declared in Xhv.h.

Import Library: Use Xvoice.lib.
