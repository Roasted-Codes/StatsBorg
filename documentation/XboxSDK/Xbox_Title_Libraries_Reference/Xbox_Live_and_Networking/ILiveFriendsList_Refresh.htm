ILiveFriendsList::Refresh

Retrieves the latest, up-to-date friends list.

HRESULT Refresh();

Parameters

None.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Refresh should be called only if HasChanged returns TRUE (indicating that changes have occurred that must be updated).

The friends feature, while active, automatically checks and updates the friends list as appropriate. When the friends feature is active (displaying the friends list), the title cannot call any ILiveFriendsList methods (including Refresh). Because of this, and because an extant ILiveFriendsList consumes resources, the title should free all of its ILiveFriendsList interfaces before starting the friends feature.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
