XOnlineStatWriteEx

Sets statistical information for one or more specified users. 

HRESULT XOnlineStatWriteEx(
    DWORD dwNumStatProcs,
    const XONLINE_STAT_PROC *pStatProcs,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwNumStatProcs
[in] Number of XONLINE_STAT_PROC structures in the array pointed to by the pStatProcs parameter.
pStatProcs
[in] Array of XONLINE_STAT_PROC structures that are used to report stats for each player.
hWorkEvent
[in, optional] Handle to a caller-created event object. This object will be signaled when there is work to do by calling XOnlineTaskContinue. The caller may pass in NULL instead, indicating that the caller will poll XOnlineTaskContinue itself.
phTask
[out] Address of an XONLINETASK_HANDLE that receives the task that stores the stats. This task sends the storage request to the Statistics Service and receives the response. This handle must be pumped to completion by XOnlineTaskContinue. Call XOnlineTaskClose to close the task.

Note    To throttle requests sent to the Statistics Service, XOnlineTaskContinue will return XONLINE_E_TASK_ABORTED_BY_DUPLICATE if the task is still being pumped when a second operation is started.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function must not be called more often than once every 5 minutes between game sessions. During a game session, this function can only be called once at the beginning, once at the end and every five minutes if the game session lasts longer than five minutes. This function should not be used to write to more than five leaderboards per user. Contact your Technical Game Manager or consult the Technical Certification Requirements for details.

The XOnlineStatWriteEx function is dynamically throttled by the server depending on the server's load. The default throttling is one second.

This function can be called only with direct user input related to Statistics.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
