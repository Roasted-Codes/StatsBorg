ILiveEngine::Render

Renders active Xbox Drop-in UI screens to the specified surface.

HRESULT Render(
    IDirect3DSurface8 *pSurface
);

Parameters

pSurface
[in] Pointer to an IDirect3DSurface8 interface representing the surface to render to.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Titles should call Render within their render loop after performing game rendering to the specified surface. Render need be called only when DoWork indicates that rendering is required.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.

See Also

ILiveEngine::DoWork, IDirect3DSurface8
