UIX_SKIN_LAYOUT_INFO

Describes the layout of a UI object. This structure is retrieved from the skin file with IPluginSupport::GetLayout.

typedef struct _UIX_SKIN_LAYOUT_INFO{
    WORD X;
    WORD Y;
    WORD Width;
    WORD Height;
    DWORD ImageOffset;
    D3DCOLOR BackColor;
    D3DCOLOR TextColor;
    D3DCOLOR DisabledTextColor;
    D3DCOLOR SelectionBackColor;
    D3DCOLOR HighlightedTextColor;
    WORD FontHeight;
    WORD Flags;
    WORD XOffset;
    WORD YOffset;
    DWORD CustomParam;
} UIX_SKIN_LAYOUT_INFO, *PUIX_SKIN_LAYOUT_INFO;

Members

X
X coordinate of the object, in pixels.
Y
Y coordinate of the object, in pixels.
Width
Width, in pixels.
Height
Height, in pixels.
ImageOffset
For a background object only, specifies an image that should be used for the background, or UIX_INVALID_VALUE if the background color in BackColor should be used. See Remarks below.
BackColor
D3DCOLOR that specifies the background color.
TextColor
D3DCOLOR that specifies the text color.
DisabledTextColor
D3DCOLOR that specifies the disabled text color.
SelectionBackColor
D3DCOLOR that specifies the selection text color.
HighlightedTextColor
D3DCOLOR that specifies the highlighted text color.
FontHeight
Font height, in pixels.
Flags
Unused.
XOffset
The X offset of the object.
YOffset
The Y offset of the object.
CustomParam
A custom parameter specific to the ITitleUIPlugin implementation. The custom parameter value is set in the skin file using the CustomParam attribute.

Remarks

For background objects (UIX_OBJECT_BACKGROUND), ImageOffset is UIX_INVALID_VALUE if the background should be rendered using the color specified in BackColor. If ImageOffset is not UIX_INVALID_VALUE, the background should be rendered with an image, and ImageOffset is an image resource identifier that can be passed as the ImageResID parameter to IPluginSupport::GetScreenImage to retrieve the image.
