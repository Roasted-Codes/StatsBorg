ILiveEngine::GetExitInfo

Returns exit information for the most recently exited feature.

HRESULT GetExitInfo(
    UIX_EXIT_INFO *pExitInfo
);

Parameters

pExitInfo
[out] Pointer to a UIX_EXIT_INFO structure that receives the exit information.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The returned exit information is for the feature that most recently exited (and thus caused the UIX_DOWORK_FEATURE_EXIT flag to be set).

If no feature has exited, the function returns failure code HRESULT_FROM_WIN32(ERROR_NOT_FOUND), which is 0x80070490.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
