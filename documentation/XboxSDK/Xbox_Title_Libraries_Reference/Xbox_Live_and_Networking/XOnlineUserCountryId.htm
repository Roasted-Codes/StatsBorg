XOnlineUserCountryId

Macro that, given the dwUserFlags value from an XUID structure, returns the user's country identifier.

#define XOnlineUserCountryId(dwUserFlags) \
   ((BYTE)(((dwUserFlags) & XONLINE_USER_COUNTRY_MASK) >> 8))

Parameters

dwUserFlags
[in] Type is DWORD. The dwUserFlags value from an XUID structure contained in an XONLINE_USER structure.

Return Values

The return value is the country identifier.

The return value is of type BYTE.

Remarks

XOnlineUserCountryId accepts the entire dwUserFlags from an XUID structure and returns the user's country identifier. The country identifier will be one of the XONLINE_COUNTRY_ values in xonline.h.

Requirements

Header: Declared in Xonline.h.
