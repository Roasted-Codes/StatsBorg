htonl

Converts a u_long from host to TCP/IP network byte order (which is big-endian).

u_long htonl(
  u_long hostlong  
);

Parameters

hostlong
[in] 32-bit number in host byte order.

Return Values

The htonl function returns the value in TCP/IP's network byte order.

Remarks

The htonl function takes a 32-bit number in host byte order and returns a 32-bit number in the network byte order used in TCP/IP networks.

Requirements

Header: Declared in winsockx.h, for include use Xtl.h.
Import Library: Use Xnet.lib for titles that only support system link play, XOnline.lib for titles that support online play. Use the secure versions of the library file (Xnets.lib or XOnlines.lib) for final testing.
