XONLINE_ATTRIBUTE

Contains a single attribute for a matchmaking search or session.

typedef struct _XONLINE_ATTRIBUTE{
    DWORD dwAttributeID;
    BOOL fChanged;
    union {
        struct {
            ULONGLONG qwValue;
        } integer;
        struct {
            LPWSTR lpValue;
        } string;
        struct {
            PVOID pvValue;
            DWORD dwLength;
        } blob;
    } info;
} XONLINE_ATTRIBUTE, *PXONLINE_ATTRIBUTE;

Members

dwAttributeID
Value that specifies the attribute scope, type, and identifier. This value is calculated by taking the bit-wise OR of a value from each of the following tables together with a unique developer-specified value between one and X_ATTRIBUTE_ID_MASK.
Scope Description
X_ATTRIBUTE_SCOPE_GLOBAL Specifies a cross-title attribute.
X_ATTRIBUTE_SCOPE_TITLE_SPECIFIC Specifies a title-specific attribute.
Type Description
X_ATTRIBUTE_DATATYPE_BLOB Specifies that this attribute contains binary data.
X_ATTRIBUTE_DATATYPE_INTEGER Specifies that this attribute is an integer.
X_ATTRIBUTE_DATATYPE_NULL Specifies an optional query parameter that has been omitted.
X_ATTRIBUTE_DATATYPE_STRING Specifies that this attribute is a string.
fChanged
Boolean value that indicates whether this value has been updated. This member is ignored when calling XOnlineMatchSessionCreate.
qwValue
Integer attribute value.
lpValue
Wide character string attribute value. Must be less than or equal to X_MAX_STRING_ATTRIBUTE_LEN (400) characters.
pvValue
Address of a buffer containing the binary data. Must be less than or equal to X_MAX_BLOB_ATTRIBUTE_LEN (800) bytes
dwLength
Size, in bytes, of the binary data.
