ITitleUIPlugin::Clear

Destroys all items in the specified list box object.

HRESULT Clear(
    DWORD ObjectID,
    BOOL ResetSelectionIndex
);

Parameters

ObjectID
[in] Object identifier of the list box UI object to clear; must have been previously returned by a call to CreateObject.
ResetSelectionIndex
[in] Value indicating whether the selection index for the list box should be reset. If this value is TRUE, the plug-in should call SetObjectState to set the UIX_OBJSTATE_LIST_SELECTION_INDEX state to zero. If FALSE, the selection index for the list box is retained.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

To remove an item from a list, the engine clears all items in the list (by calling the Clear method), and then adds back all items except for the one removed. The selection index is retained (unless ResetSelectionIndex is TRUE), so that when the list is displayed again the selection point in the list remains at the same index.

If the selection index is greater than the number of items in the list, the UI plug-in should place the selection on the last item in the list.

Requirements

Header: Declared in Uix.h.

Import Library: Use Xonline.lib.
