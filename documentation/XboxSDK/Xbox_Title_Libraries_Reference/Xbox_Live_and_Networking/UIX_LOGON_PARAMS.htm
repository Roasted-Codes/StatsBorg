UIX_LOGON_PARAMS

Logon parameters for the logon feature. This structure is passed in pFeatureParams when calling ILiveEngine::StartFeature with UIX_LOGON_FEATURE.

typedef struct _UIX_LOGON_PARAMS{
    SIZE_T StructSize;
    UIX_LOGON_TYPE LogonType;
    DWORD LogonUserCount;
    DWORD LogonServiceIDs[UIX_MAX_LOGON_SERVICES];
    XONLINE_LOGON_STATE *pLogonState;
    XONLINE_ACCEPTED_GAMEINVITE *pGameInvite;
} UIX_LOGON_PARAMS, *PUIX_LOGON_PARAMS;

Members

StructSize
Size of the structure. Must be set to the structure size before calling ILiveEngine::StartFeature.
LogonType
UIX_LOGON_TYPE value specifying the type of logon to perform.
LogonUserCount

Maximum number of simultaneously logged on users (1, 2, or 4). Logon feature will select the correct screen layouts based on this value. For UIX_LOGON_TYPE_SILENT, LogonUserCount must be one (1).

LogonServiceIDs
Array of DWORD values specifying the service identifiers of services to log on to. Service identifiers must be placed in the array starting at index zero and be consecutively specified, with no gaps. A value of zero (XONLINE_INVALID_SERVICE) signifies the end of the requested services. For the list of valid service identifiers, see XOnlineLogon.
pLogonState
Pointer to an XONLINE_LOGON_STATE structure containing a saved logon state to use for the logon. Valid only with LogonType UIX_LOGON_TYPE_RETRIEVED_STATE; otherwise, this member should be set to NULL.
pGameInvite
Pointer to an XONLINE_ACCEPTED_GAMEINVITE structure containing a game invitation to use for the logon. Valid only with LogonType UIX_LOGON_TYPE_RETRIEVED_GAME_INVITE; otherwise, this member should be set to NULL.

Remarks

Initialize the structure to zero and set StructSize to the size of the structure before calling ILiveEngine::StartFeature.

Communication tickets are obtained for the services specified in LogonServiceIDs, enabling subsequent secure communications with those services (and those services only).

Requirements

Header: Declared in Uix.h.
