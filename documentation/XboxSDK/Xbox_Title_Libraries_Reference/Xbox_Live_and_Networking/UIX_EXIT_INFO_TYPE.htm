UIX_EXIT_INFO

Feature exit information obtained with ILiveEngine::GetExitInfo

typedef struct _UIX_EXIT_INFO{
    UIX_FEATURE FeatureID;
    UIX_EXIT_CODE_TYPE ExitCode;
    HRESULT hr;
    PVOID pExitData;
} UIX_EXIT_INFO, *PUIX_EXIT_INFO;

Members

FeatureID
The feature for which exit information was returned. The exit information will be for the feature that most recently exited (and thus caused the UIX_DOWORK_FEATURE_EXIT flag to be set).
ExitCode
A UIX_EXIT_CODE_TYPE value containing the exit code for the feature.
hr
The success or failure result code returned by the feature.
pExitData

Exit data returned by the feature. This data varies depending on the feature and exit code.
FeatureID ExitCode pExitData
UIX_LOGON_FEATURE UIX_EXIT_LOGON_SUCCESSFUL A pointer to an array of XONLINE_USER structures representing the logged on users.
UIX_FRIENDS_FEATURE UIX_EXIT_FRIENDS_JOIN_GAME A pointer to the XONLINE_FRIEND structure of the friend that sent the game invite.

pExitData is NULL for any other combination of FeatureID and ExitCode.
