XOnlineCompetitionCreateGetResults

Retrieves the results of the task started by XOnlineCompetitionCreate or XOnlineCompetitionCreateSingleElimination.

HRESULT XOnlineCompetitionCreateGetResults(
    XONLINETASK_HANDLE hTask,
    PXONLINE_COMP_CREATE_RESULTS pCompResults
);

Parameters

hTask
[in] Task handle received from XOnlineCompetitionCreate or XOnlineCompetitionCreateSingleElimination.
pCompResults
[out] Pointer to an XONLINE_COMP_CREATE_RESULTS structure that contains the ID of the newly created competition.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Call XOnlineCompetitionCreateGetResults to determine whether the competition or tournament was properly created, and to obtain the competition ID.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.

See Also

XOnlineCompetitionCreate, XOnlineCompetitionCreateSingleElimination
