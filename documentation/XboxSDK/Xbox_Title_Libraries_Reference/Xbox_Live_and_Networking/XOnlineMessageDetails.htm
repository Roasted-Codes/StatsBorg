XOnlineMessageDetails

Retrieves the details of a particular message.

HRESULT XOnlineMessageDetails(
    DWORD dwUserIndex,
    DWORD dwMessageID,
    DWORD dwMessageFlagsToSet,
    DWORD dwMessageFlagsToClear,
    HANDLE hWorkEvent,
    PXONLINETASK_HANDLE phTask
);

Parameters

dwUserIndex
[in] Controller index (zero-based) of the player. The player must have been successfully logged on with a previous call to XOnlineLogon.
dwMessageID
[in] Unique identifier of the message whose details will be retrieved.
dwMessageFlagsToSet
[in] Message flags to set. For more information about message flags, see XOnlineMessageCreate. For a list of flags that can be set, see XOnlineMessageSetFlags.
dwMessageFlagsToClear
[in] Message flags to clear.For a list of flags that can be cleared, see XOnlineMessageSetFlags.
hWorkEvent
[in, optional] Handle of the event you want triggered every time there is work to be performed on this task. If you do not want to assign an event, set this parameter to NULL.
phTask
[out] Pointer to an XONLINETASK_HANDLE type. The handle is associated with the task started by this function.

Return Values

Returns S_OK if successful; an error code otherwise, which can be one of the following values.

Error Code Description
XONLINE_E_MESSAGE_INVALID_MESSAGE_ID The value in the dwMessageID parameter does not indicate a valid message number.
XONLINE_E_MESSAGE_NO_MESSAGE_DETAILS There are no details associated with this message.

Remarks

Titles typically enumerate the player's messages first by calling XOnlineMessageEnumerate. If the player wants to open a particular message, the title calls XOnlineMessageDetails to get all of the message's details.

Requirements

Header: Declared in Xonline.h.

Import Library: Use Xonline.lib.
