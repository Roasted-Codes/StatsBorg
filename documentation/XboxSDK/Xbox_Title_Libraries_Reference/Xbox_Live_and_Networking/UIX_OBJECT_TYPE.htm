UIX_OBJECT_TYPE

Object type of an object handled by ITitleUIPlugin. Specified in the ObjectType parameter of ITitleUIPlugin::CreateObject when a request is made to create a UI object.

typedef enum _UIX_OBJECT_TYPE{
    UIX_OBJECT_NONE = 0,
    UIX_OBJECT_TEXTBOX = 1,
    UIX_OBJECT_LISTBOX = 2,
    UIX_OBJECT_BACKGROUND = 3,
    UIX_OBJECT_FORCE_DWORD = 0x7fffffff,
} UIX_OBJECT_TYPE;

Constants

UIX_OBJECT_NONE
An invalid object.
UIX_OBJECT_TEXTBOX
A static text box.
UIX_OBJECT_LISTBOX
A list box.
UIX_OBJECT_BACKGROUND
The background.
UIX_OBJECT_FORCE_DWORD
This value is not used; it forces the enumeration to compile to 32 bits in size.

Remarks

UIX_OBJECT_NONE will not be passed as an object type to create (but can be used internally by a plug-in to signify an invalid or deleted object).

Requirements
