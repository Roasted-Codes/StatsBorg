XHV_LOCAL_TALKER_STATUS

Describes the current status of a local talker. This structure is returned by IXHVEngine::GetLocalTalkerStatus.

typedef struct _XHV_LOCAL_TALKER_STATUS{
    XHV_VOICE_COMMUNICATOR_STATUS communicatorStatus;
    BOOL bIsTalking;
} XHV_LOCAL_TALKER_STATUS, *PXHV_LOCAL_TALKER_STATUS;

Members

communicatorStatus
An XHV_VOICE_COMMUNICATOR_STATUS structure that describes the current status of the local talker's voice communicator.
bIsTalking
Indicates whether the local talker is currently talking (that is, whether sound is currently detected in the local talker's microphone).

Requirements

Header: Declared in Xhv.h.
