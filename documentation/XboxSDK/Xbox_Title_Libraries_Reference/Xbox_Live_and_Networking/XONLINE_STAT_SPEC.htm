XONLINE_STAT_SPEC

Specifies a user and a set of statistics associated with that user on a particular leaderboard.

Note    Leaderboards are also referred to as scoreboards.

typedef struct{
    XUID xuidUser;
    DWORD dwLeaderBoardID;
    DWORD dwNumStats;
    PXONLINE_STAT pStats;
} XONLINE_STAT_SPEC, *PXONLINE_STAT_SPEC;

Members

xuidUser
The XUID of the user.
dwLeaderBoardID
Leaderboard that lists the user specified by the xuidUser member. Leaderboards are defined by the title developer. A leaderboard cannot have an ID of zero because XOnlineStatReset uses an ID of zero to specify all leaderboards for the title.
dwNumStats
Number of XONLINE_STAT structures pointed to by the pStats member.
pStats
Pointer to an array of XONLINE_STAT structures. The number of structures in the array is specified by the dwNumStats member. These structures specify statistics associated with the specified user on the specified leaderboard.

Requirements

Header: Declared in Xonline.h.
