IXACTSoundBank::PlayEx

Plays a sound cue. This method supersedes IXACTSoundBank::Play.

HRESULT PlayEx(
    PCXACT_PREPARE_SOUNDCUE pPrepareData,
    PXACTSOUNDCUE *ppSoundCue
);

Parameters

pPrepareData
[in] Pointer to an XACT_PREPARE_SOUNDCUE structure containing information about the sound cue to play.
ppSoundCue
[out] Pointer to an IXACTSoundCue to be returned.

Return Values

Returns DS_OK if successful, an error code otherwise.

Remarks

Titles must call Stop to release sound cue pointers returned by PlayEx, Play, PrepareEx, or Prepare. For sound cues that play to completion, call Stop when the cue's stop notification arrives. Sound cue pointers are not returned (and need not be released) when XACT_FLAG_SOUNDCUE_AUTORELEASE is specified.

Requirements
