XACT_SOUNDBANK_SELECT_VARIATION

typedef struct _XACT_SOUNDBANK_SELECT_VARIATION{
    DWORD dwFlags;
    union {
        DWORD dwIndex;
        FLOAT flValue;
    } Sound;
    union {
        DWORD dwIndex;
        FLOAT flValue;
    } Wave;
} XACT_SOUNDBANK_SELECT_VARIATION, *PXACT_SOUNDBANK_SELECT_VARIATION;

Members

dwFlags
Flags indicating which values to set:
Value Description
XACT_FLAG_SELECT_VARIATION_SOUND_INDEX Set the sound variation index. This flag is mutually exclusive with the XACT_FLAG_SELECT_VARIATION_SOUND_WEIGHT flag.
XACT_FLAG_SELECT_VARIATION_SOUND_VALUE Set the sound value. This flag is mutually exclusive with the XACT_FLAG_SELECT_VARIATION_SOUND_INDEX flag.
XACT_FLAG_SELECT_VARIATION_WAVE_INDEX Set the wave variation index. This flag is mutually exclusive with the XACT_FLAG_SELECT_VARIATION_WAVE_WEIGHT flag.
XACT_FLAG_SELECT_VARIATION_WAVE_VALUE Set the wave value. This flag is mutually exclusive with the XACT_FLAG_SELECT_VARIATION_WAVE_INDEX flag.
dwIndex
Index of sound variation.
flValue
Weight of sound variation.
dwIndex
Index of wave variation.
flValue
Weight of wave variation.

Remarks

XACT_FLAG_SELECT_VARIATION_WAVE_VALUE and XACT_FLAG_SELECT_VARIATION_SOUND_VALUE are valid only for cues with parameter-controlled wave and sound variation lists. Those flags have no effect on other variation list types.
