IDirectSoundStream::SetMixBins

Assigns mixbins to a DirectSound stream.

HRESULT SetMixBins(
    LPCDSMIXBINS pMixBins
);

Parameters

pMixBins
Address of a DSMIXBINS structure that specifies the mixbins to assign to this stream. This parameter can be NULL, in which case the default mixbins will be used.

Remarks

The SetFormat call resets the mixbin assignments and should be called before calling SetMixBins. See the SetFormat function for more information.
