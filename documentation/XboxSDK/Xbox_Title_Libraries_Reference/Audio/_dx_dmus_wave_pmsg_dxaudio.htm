DMUS_WAVE_PMSG

Contains message data for a wave sound.

typedef struct _DMUS_WAVE_PMSG {
    DMUS_PMSG_PART
    REFERENCE_TIME rtDuration;
    long lOffset;
    long lVolume;
    long lPitch;
    DWORD dwStartOffset;
    DWORD dwLoopStart;
    DWORD dwLoopEnd;
    BYTE bFlags;
} DMUS_WAVE_PMSG;
DMUS_PMSG_PART
Macro for common message members. See DMUS_PMSG. The punkUser member contains the address of the IUnknown interface of the voice object associated with the wave.
rtDuration
Value that specifies the duration of the wave. If DMUS_PMSGF_LOCKTOREFTIME is present in the dwFlags member of DMUS_PMSG_PART, this value is in reference time units. Otherwise it is in music time.
lOffset
Value that specifies the offset from actual time to logical time, in either reference or music time.
lVolume
Value that specifies the initial volume in hundredths of a decibel.
lPitch
Value that specifies the transposition of the pitch in hundredths of a semitone.
dwStartOffset
Value that specifies how far into the wave to start, in samples.
dwLoopStart
Value that specifies the starting loop point, in samples.
dwLoopEnd
Value that specifies the ending loop point, in samples.
bFlags
Value that contains any applicable flags. Can be zero or a combination of the following values:
DMUS_WAVEF_NOINVALIDATE
Do not invalidate this wave.
DMUS_WAVEF_NOPREROLL
Do not preroll any wave data.
DMUS_WAVEF_OFF
This message is stopping playback of the wave.
DMUS_WAVEF_STREAMING
Wave is streaming.

Remarks

Applications cannot send messages of this type by using IDirectMusicPerformance8::SendPMsg, because they have no way of obtaining a pointer to a wave object. However, tools can process wave messages.
