IDirectSoundBuffer8::GetStatus

Retrieves the status of the sound buffer.

HRESULT GetStatus(
  LPDWORD pdwStatus
);

Parameters

pdwStatus
Address of a variable to contain the status of the sound buffer. The status can be a combination of the following flags:
Flag Description
DSBSTATUS_PLAYING The buffer is playing. If this value is not set, the buffer is stopped or paused.
DSBSTATUS_PAUSED The buffer is paused. If this value is not set, the buffer is stopped or playing.
DSBSTATUS_LOOPING The buffer is being looped. If this value is not set, the buffer will stop when it reaches the end of the sound data. This value is returned only in combination with DSBSTATUS_PLAYING or DSBSTATUS_PAUSED.

Return Values

Returns DS_OK if successful, an error code otherwise.
