DSEFFECTIMAGELOC

Contains the indexes of the crosstalk and reverberation effects in the current effects image.

typedef struct _DSEFFECTIMAGELOC{
    DWORD dwI3DL2ReverbIndex;
    DWORD dwCrosstalkIndex;
} DSEFFECTIMAGELOC, *LPDSEFFECTIMAGELOC;

Members

dwI3DL2ReverbIndex
Value that contains the Interactive 3-D Audio Level 2 (I3DL2) reverberation effect index. Can be set to DSFX_IMAGELOC_UNUSED if the effect isn't used in this image.
dwCrosstalkIndex
Value that contains the crosstalk cancellation effect index. Can be set to DSFX_IMAGELOC_UNUSED if the effect isn't used in this image.

Remarks

When a DSP Builder image is generated, the header file contains fixed names for I3DL2 and crosstalk effect indices. These are referenced as GraphI3DL2_I3DL2Reverb and GraphXTalk_XTalk respectively.
