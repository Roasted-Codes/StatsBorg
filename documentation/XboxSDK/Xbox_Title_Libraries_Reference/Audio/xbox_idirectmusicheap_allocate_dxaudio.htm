IDirectMusicHeap::Allocate

Allocates the specified amount of memory.

HRESULT Allocate(
    DWORD cbSize,
    PVOID *pData
);

Parameters

cbSize
Value that specifies the number of bytes to allocate.
pData
Address of a pointer to the memory allocated.

Return Values

Returns S_OK if successful, an error code otherwise.
