DSFX_CHORUS_MONO_PARAMS

Contains the parameters used by the mono chorus effect.

typedef struct _DSFX_CHORUS_MONO_PARAMS{
    DSFX_CHORUS_MONO_STATE  State;
    DWORD                   dwGain;
    DWORD                   dwModScale;
} DSFX_CHORUS_MONO_PARAMS, *LPDSFX_CHORUS_MONO_PARAMS;

Members

State
DSFX_CHORUS_MONO_STATE structure that contains the state information for this effect.
dwGain
Value that specifies the gain for this effect. Can be a value between 0 and 2^23 (which is 0.0 to 1.0 in 24 bit).
dwModScale
Value that specifies how much of the delay line to tap. Can be a value between 0 and 2^23 (which is 0.0 to 1.0 in 24 bit).
