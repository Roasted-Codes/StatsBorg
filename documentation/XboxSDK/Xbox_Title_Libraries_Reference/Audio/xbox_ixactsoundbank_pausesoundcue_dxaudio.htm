IXACTSoundBank::PauseSoundCue

Pauses or resumes playback of a sound cue.

HRESULT PauseSoundCue(
    PXACTSOUNDCUE pSoundCue,
    BOOL fPause
);

Parameters

pSoundCue
[in] Pointer to a sound cue instance to pause or resume.
fPause
[in] Value indicating whether to pause or resume the cue. A value of TRUE will pause the cue. A value of FALSE will resume the paused cue.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
