DirectSound Playback Overview

The DirectSound object represents the Xbox Media Communications Processor (MCP) and is used to manage and create sound buffers.

The DirectSound buffer object represents a buffer containing sound data. Buffer objects are used to start, stop, and pause sound playback, as well as to set attributes such as frequency and format. The DirectSound stream object represents an audio data stream.

A short sound can be loaded into a buffer in its entirety and played at any time by a simple method call. Longer sounds have to be streamed. Applications should use the IDirectSoundStream interface.

It is your responsibility to put data in the correct format into the sound buffers. DirectSound does not include methods for parsing a wave file or resource. However, there is code in the XDK samples that helps with this task.
