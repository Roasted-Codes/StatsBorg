IDirectSound8::SetMixBinHeadroom

Adjusts the headroom of a specified mixbin.

HRESULT SetMixBinHeadroom(
    DWORD dwMixBin,
    DWORD dwHeadroom
);

Parameters

dwMixBin
Value that contains the identifier of the mixbin to set. Can be one of the Mixbin Constants.
dwHeadroom
Value that specifies the headroom attenuation in bits, from zero to seven. The value seven specifies seven bits (approximately –42 dB). The mixbin headroom defaults to one, which specifies 1 bit (–6 dB).

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Mixbin headroom refers to the amount of additional attenuation that is applied to a DirectSound buffer or stream when is is mixed into the mixbin.
