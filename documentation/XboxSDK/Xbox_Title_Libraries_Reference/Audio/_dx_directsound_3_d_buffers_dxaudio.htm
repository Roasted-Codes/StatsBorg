DirectSound 3D Buffers

Each sound source in a 3D environment is represented by an IDirectSoundBuffer8 interface. This interface is supported only by sound buffers created with the DSBCAPS_CTRL3D flag. Methods of the interface are used to set and retrieve parameters of a single sound source.

Applications must supply monaural sound sources when using the 3D capabilities of DirectSound. If you attempt to create a buffer with the DSBCAPS_CTRL3D flag set and a wave format with more than one channel, an error results.
