DSFILTERDESC

Describes the characteristics of a filter.

typedef struct _DSFILTERDESC{
    DWORD dwMode;
    DWORD dwQCoefficient;
    DWORD adwCoefficients[4];
} DSFILTERDESC, *LPDSFILTERDESC;

Members

dwMode
Value that specifies the filter mode. Can be one of the following values:
DSFILTER_MODE_BYPASS
Bypass all filtering. Not valid for 3D buffers.
DSFILTER_MODE_DLS2
Perform DLS2 filtering. On 3D buffers, it should also perform I3DL2 filtering.
DSFILTER_MODE_PARAMEQ
Perform parametric EQ processing. On 3D buffers, it should also perform I3DL2 filtering.
DSFILTER_MODE_MULTI
Perform DLS2 and parametric EQ filtering on mono 2D buffers. Perform only I3DL2 filtering on mono 3D buffers.
dwQCoefficient
Value that corresponds to how wide a range of frequencies are affected. Can be a value from zero (narrowest) to seven (widest).
adwCoefficients
Array of values that contains the filter coefficients.

Remarks

For stereo buffers: 1) the first and second coefficients specified in adwCoefficients handle the left channel, and 2) the third and fourth coefficients handle the right channel.

For 3D buffers, you must have at least the I3DL2 filtering enabled.

Mathematically, the Q coefficient is the peaking curve of the filter, defined as a ratio of the center frequency to the bandwidth (as defined by the +/- 3dB point). Thus, using the same Q coefficient, doubling the center frequency will also double the width in Hz. The math used for determining these coefficients can be found in the SetFilter XDK sample.

The following table shows how the dwQCoefficient value corresponds to the Q coefficient.

dwQCoefficient Q
0 1
1 .5
2 .25
3 .125
4 .0625
5 .03125
6 .015625
7 .007813
