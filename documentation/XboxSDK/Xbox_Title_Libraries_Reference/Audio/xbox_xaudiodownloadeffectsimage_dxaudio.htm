XAudioDownloadEffectsImage

Downloads a DSP effects image from a file or an XBE section to the global processor in the Xbox Media Communications Processor (MCP).

HRESULT XAudioDownloadEffectsImage(
    LPCSTR pszImageName,
    LPCDSEFFECTIMAGELOC pImageLoc,
    DWORD dwFlags,
    LPDSEFFECTIMAGEDESC *ppImageDesc
);

Parameters

pszImageName
String value that specifies the file or section name that contains the DSP effects image.
pImageLoc
Address of a DSEFFECTIMAGELOC structure that specifies the location of the reverberation and crosstalk effects in the effects image being downloaded. Can be set to NULL if none of the effects used by other DirectSound methods (currently reverb and crosstalk) appear in the image.
dwFlags
Value that specifies the flags used to control this function. Can be one of the following values:
Value Description
XAUDIO_DOWNLOADFX_EXTERNFILE The DSP effects image is located in a separate file.
XAUDIO_DOWNLOADFX_XBESECTION The DSP effects image is located in a section of the XBE.
ppImageDesc
Address of a pointer to the DSEFFECTIMAGEDESC structure that is returned by the method, which describes the DSP scratch image.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Before calling this function, the DirectSound object must be initialized by calling either DirectSoundCreate or IDirectMusicPerformance8::InitAudioX.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
