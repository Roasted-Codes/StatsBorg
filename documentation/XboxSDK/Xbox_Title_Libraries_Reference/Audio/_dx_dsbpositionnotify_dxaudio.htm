DSBPOSITIONNOTIFY

Describes a notification position. Used by the IDirectSoundBuffer8::SetNotificationPositions method.

typedef struct {
    DWORD    dwOffset;
    HANDLE   hEventNotify;
} DSBPOSITIONNOTIFY, *LPDSBPOSITIONNOTIFY;
 
typedef const DSBPOSITIONNOTIFY *LPCDSBPOSITIONNOTIFY;

Members

dwOffset
Offset from the beginning of the buffer where the notify event is to be triggered, or DSBPN_OFFSETSTOP.
hEventNotify
Handle to the event to be signaled when the offset has been reached.

Remarks

The DSBPN_OFFSETSTOP value in the dwOffset member causes the event to be signaled when playback stops, either because the end of the buffer has been reached (and playback is not looping) or because the application called the IDirectSoundBuffer8::Stop method.
