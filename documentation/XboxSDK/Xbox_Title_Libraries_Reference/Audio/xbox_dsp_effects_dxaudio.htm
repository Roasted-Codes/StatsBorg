DSP Effects

This section covers information on using the global processor on the Xbox Media Communications Processor (MCP) to do sound effects.

  • DSP Effects Processing
  • Creating a DSP Effects Image
  • Using XGPImage
  • Downloading Images to the Global Processor

For information on the tools used to create DSP effect images, see the Audio Tools section of the XDK.

Note   All titles using the 3D sound features of DirectSound and DirectMusic must download a DSP effects image that contains at least the Crosstalk cancellation effect image. Without this effect image, 3D sounds will not be outputted to the speakers.
