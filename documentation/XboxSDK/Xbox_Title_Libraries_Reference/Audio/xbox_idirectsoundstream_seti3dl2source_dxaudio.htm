IDirectSoundStream::SetI3DL2Source

Sets the Interactive 3-D Audio Level 2 (I3DL2) source properties.

HRESULT SetI3DL2Source(
    LPCDSI3DL2BUFFER pds3db,
    DWORD dwApply
);

Parameters

pds3db
Address of a DSI3DL2BUFFER structure that specifies the properties of the sound source.
dwApply
Value that specifies when the setting should be applied. Can be one of the following values:
DS3D_IMMEDIATE
Apply the values immediately.
DS3D_DEFERRED
Defer the values until CommitDeferredSettings is called.

Return Values

Returns S_OK if successful; returns an error code otherwise.

Remarks

This function is only valid for streams created using the DSBCAPS_CTRL3D flag.
