Getting Started with DirectMusic

Creating your first DirectMusic application does not require a lot of setup. The necessary libs are in the Microsoft Xbox SDK.

Apart from that, all you need are an Xbox development kit, speakers or headphones, and some sounds. Any of the following will do:

  • Wave (.wav) files.
  • Segment (.sgt) files from DirectMusic Producer for Xbox. Sample files are installed along with that application, and others are included with the Xbox sample applications.

For now, you can ignore the other kinds of files you'll find with DirectMusic Producer for Xbox. Chordmaps, styles, bands, and DLS files are building blocks for playable segments but aren't playable themselves. Segments can also be obtained from container files.
