DirectMusicInitializeEx

Initializes DirectMusic using the specified normal heap, physical heap, and object factory.

HRESULT DirectMusicInitializeEx(
    IDirectMusicHeap *pNormalHeap, 
    IDirectMusicHeap *pPhysicalHeap,
    LPDIRECTMUSICFACTORYFN pFactory
);

Parameters

pNormalHeap
Address of the normal heap that DirectMusic should use.
pPhysicalHeap
Address of the physical heap that DirectMusic should use.
pFactory
Address of the factory function. Can be set to &DirectMusicDefaultFactory or to a custom factory generated by the Audio Analyzer tool.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

DirectMusicInitializeEx, DirectMusicInitialize, or DirectMusicInitializeFixedSizeHeaps must be called before making calls to DirectMusic. Titles are only allowed to call one of these initialization functions once per XBE. Titles must reboot before you are allowed to call any of them a second time.

The normal heap can use any threadsafe method of allocating memory (like LocalAlloc) and the physical heap should use XPhysicalAlloc, which is also threadsafe.
