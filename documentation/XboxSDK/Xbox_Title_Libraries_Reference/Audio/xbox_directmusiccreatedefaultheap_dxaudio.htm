DirectMusicCreateDefaultHeap

Creates an instance of the default normal heap used by Microsoft® DirectMusic®.

HRESULT DirectMusicCreateDefaultHeap(
    IDirectMusicHeap **ppHeap
);

Parameters

ppHeap
Address of a pointer to the IDirectMusicHeap interface of the default normal heap created.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

Header: Declared in DMusicI.h.

Import Library: Use DMusic.lib.
