IDirectMusicScript8

The IDirectMusicScript8 interface represents a script that contains variables that can be set and retrieved by the title, and routines that can be called by the title.

Typically the interface is obtained by using IDirectMusicLoader8::LoadObjectFromFile or IDirectMusicLoader8::GetObject to load a script file. The title then calls IDirectMusicScript8::Init to associate the script with the DirectMusicPerformance object that performs the actual playback.

The methods of the IDirectMusicScript8 interface can be grouped as follows:

Initialization Init
Calling routines CallRoutine
Getting variables GetVariableNumber
  GetVariableObject
  GetVariableString
Setting variables SetVariableNumber
  SetVariableObject
  SetVariableString

The IDirectMusicScript8 interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

Remarks

Loading a script with IDirectMusicLoader8 and then releasing the loader will cause the script to be garbage collected. You must keep the loader object allocated while the script is still in use.
