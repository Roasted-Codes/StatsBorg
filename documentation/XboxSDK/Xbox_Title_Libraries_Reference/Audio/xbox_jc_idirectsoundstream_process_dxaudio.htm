IDirectSoundStream::Process

Submits a buffer to the stream.

HRESULT Process(
    XMEDIAPACKET *pInputBuffer, 
    XMEDIAPACKET *pOutputBuffer
);

Parameters

pInputBuffer
Address of the XMEDIAPACKET structure that specifies the buffer to submit.
pOutputBuffer
Reserved. Must be set to NULL.

Return Values

Process returns DS_OK if successful.

Requirements

  Header: Declared in dsound.h.
