DSFX_FLANGE_STEREO_PARAMS

Contains the parameters used by the stereo flange effect.

typedef struct _DSFX_FLANGE_STEREO_PARAMS{
    DSFX_FLANGE_STEREO_STATE State;
    DWORD dwFeedback;
    DWORD dwModScale;
} DSFX_FLANGE_STEREO_PARAMS, *LPDSFX_FLANGE_STEREO_PARAMS;

Members

State
DSFX_FLANGE_STEREO_STATE structure that contains the state information for this effect.
dwFeedback
Value that specifies how much of the effect's output should be mixed with the dry signal. Can be a value between 0 and 2^23 (which is 0.0 to 1.0 in 24 bit).
dwModScale
Value that specifies how much of the delay line to tap. Can be a value between 0 and 2^23 (which is 0.0 to 1.0 in 24 bit). Should be left at its default value.
