IDirectSound8::GetTime

Gets the current reference time.

HRESULT GetTime(
  REFERENCE_TIME *prtCurrent
);

Parameters

prtCurrent
Pointer to a REFERENCE_TIME value to be filled in with the current reference time.

Remarks

The time returned by the master clock is a 64-bit value defined as type REFERENCE_TIME. Reference time is measured in units of approximately 100 nanoseconds, so the clock ticks about 10 million times each second.

Note    GetTime does not retrieve it's time from the Audio Processor's (Xbox MCP) sample clock.

Requirements

  Header: Declared in dsound.h.
