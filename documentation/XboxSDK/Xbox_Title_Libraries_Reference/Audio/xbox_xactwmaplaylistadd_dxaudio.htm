XACT_WMA_PLAYLIST_ADD

Contains XACT WMA playlist add information.

typedef struct _XACT_WMA_PLAYLIST_ADD{
    DWORD dwType;
    PCSTR pszFileName;
    DWORD dwSoundtrackId;
    DWORD dwSongId;
    DWORD dwSongIndex;
} XACT_WMA_PLAYLIST_ADD, *PXACT_WMA_PLAYLIST_ADD;

typedef const XACT_WMA_PLAYLIST_ADD *PCXACT_WMA_PLAYLIST_ADD;

Members

dwType
Type of playlist add information. Can be one of the following values:
Value Description
eXACTWmaPlayListAdd_Directory

Indicates adding all WMA files from a given directory to a WMA playlist.

Note    In the June 2003 and later XDK releases, if the IXACTWmaPlayList::Add method is used with this flag to add a directory that contains only one WMA soundtrack, the cue for the WMA playlist does not loop. However, if the directory contains more than one soundtrack, the cue loops correctly.
eXACTWmaPlayListAdd_File Indicates adding a single WMA file to a WMA playlist.
eXACTWmaPlayListAdd_Soundtrack Indicates adding all files from a soundtrack to a WMA playlist.
eXACTWmaPlayListAdd_SoundtrackSong Indicates adding a single song from a soundtrack to a WMA playlist.
pszFileName
Name of the file to add. The IXACTWmaPlayList::Add function makes a copy of the file name, so maintaining the string memory after calling the function is unnecessary.
dwSoundtrackId
ID of the soundtrack to be added.
dwSongId
ID of the song to be added.
dwSongIndex
Index of the song to be added. This member is only required when specifying the eXACTWmaPlayListAdd_SoundtrackSong flag.

Remarks

WMA files added to a playlist must be in 44Khz stereo format.

Requirements
