IDirectMusicGraph8

The IDirectMusicGraph8 interface manages the loading and message flow of tools.

The IDirectMusicGraph8 interface has the following methods:

Routing StampPMsg
Tools GetTool
  InsertTool
  RemoveTool

The IDirectMusicGraph8 interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

Remarks

Graphs can occur in audiopaths, performances, and segments. The graph of tools in a performance is global in nature; it processes messages from all segments. A graph in a segment exists only for playback of that segment.

To get a reference to the appropriate IDirectMusicGraph8 interface, call QueryInterface method on the appropriate object (audiopath, segment, or performance). This way, the complete routing can be managed, not just the routing within the graph.
