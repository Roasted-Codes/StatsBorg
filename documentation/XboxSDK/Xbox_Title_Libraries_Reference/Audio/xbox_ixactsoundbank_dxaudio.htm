IXACTSoundBank

Applications use the methods of the IXACTSoundBank interface to manipulate sound bank objects.

An IXACTSoundBank object is obtained by calling the IXACTEngine::CreateSoundBank function.

IXACTSoundBank methods CreateWMAPlayList
GetSoundCueIndexFromFriendlyName
GetSoundCueProperties
PauseSoundCue
Play
PlayEx
Prepare
PrepareEx
SelectVariation
Stop

The IXACTSoundBank interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
