IXACTEngine::GlobalPause

Pause and mute audio output from XACT engine.

HRESULT GlobalPause(
    WORD wCategory,
    BOOL fPause
);

Parameters

wCategory
[in] Value specifying the category of sounds to pause. XACT allows the sound designer to define a category for each sound. For example, Music, Sound Effects, and Voice may be three categories. This allows the game to have a separate control for each of these game sound elements. To pause for all categories, the special category XACT_CATEGORY_INDEX_UNUSED may be used. XACT supports up to 255 categories.
fPause
[in] A Boolean value specifying whether the output should be paused. A value of true pauses and mutes output. A value of false resumes playing from a paused state.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
