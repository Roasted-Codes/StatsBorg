DirectSoundDumpMemoryUsage

Dumps the current memory usage to the debugger.

void DirectSoundDumpMemoryUsage(
    BOOL fAssertNone
);

Parameters

fAssertNone
Value that specifies if it should assert if there is no memory allocated. Set to TRUE to assert that there is no memory allocated.

Return Values

None.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
