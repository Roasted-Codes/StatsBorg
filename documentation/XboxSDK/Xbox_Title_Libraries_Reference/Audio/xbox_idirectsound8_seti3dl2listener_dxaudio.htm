IDirectSound8::SetI3DL2Listener

Sets the Interactive 3-D Audio Level 2 (I3DL2) listener properties.

HRESULT SetI3DL2Listener(
    LPCDSI3DL2LISTENER pds3dl, 
    DWORD dwApply
);

Parameters

pds3dl
Address of a DSI3DL2LISTENER structure that specifies listener properties.
dwApply
Specifies when the setting should be applied. Can be one of the following values:
DS3D_IMMEDIATE
Apply the values immediately.
DS3D_DEFERRED
Defer the values until CommitDeferredSettings is called.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

SetI3DL2Listener should not be called on every frame, because this will result in unnecessary DSP recalculations and data writes to the DSP. SetI3DL2Listener should be called only when necessary (parameter changes, for example).

For convienience, a set of pre-defined listener parameter constants are defined. See Dsound.h for the parameter values associated with each preset.

Note:   Although DirectSoundI3DL2ListenerPreset_Default is used by default, DirectSoundI3DL2ListenerPreset_Default2 is recommended as a better set of default parameters.
