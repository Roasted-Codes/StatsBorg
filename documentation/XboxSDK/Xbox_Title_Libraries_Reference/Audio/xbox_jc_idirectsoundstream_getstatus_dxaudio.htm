IDirectSoundStream::GetStatus

Retrieves the status of the sound stream.

HRESULT GetStatus(
  LPDWORD pdwStatus
);

Parameters

pdwStatus
Pointer to a variable that receives the current object status. Can be a combination of the following values:
Flag Description
DSSTREAMSTATUS_PLAYING Indicates that the stream is playing. This function will return either DSSTREAMSTATUS_PLAYING or DSSTREAMSTATUS_PAUSED, but never both.
DSSTREAMSTATUS_PAUSED Indicates that the stream has data queued, but playback is paused. This function will return either DSSTREAMSTATUS_PLAYING or DSSTREAMSTATUS_PAUSED, but never both.
DSSTREAMSTATUS_READY Indicates that the stream is ready to accept input data.
DSSTREAMSTATUS_STARVED Indicates that the stream has been paused because no data is available to be played.

Return Values

Returns DS_OK if successful, an error code otherwise.
