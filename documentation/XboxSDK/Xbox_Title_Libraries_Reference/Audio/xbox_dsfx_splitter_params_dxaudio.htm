DSFX_SPLITTER_PARAMS

Contains the parameters used by the splitter effect, which takes an input and sends it to up to 8 other mixbins.

typedef struct _DSFX_SPLITTER_PARAMS{
    DSFX_SPLITTER_STATE State;
    DWORD dwNumOutputs;
    DWORD dwGains[8];
} DSFX_SPLITTER_PARAMS, *LPDSFX_SPLITTER_PARAMS;

Members

State
DSFX_SPLITTER_STATE structure that contains the state information for this effect.
dwNumOutputs
Value that specifies the number of output mixbins send the audio data from the input mixbin to.
dwGains
Array of values that allows a title to adjust the gain for each output.

Requirements

Header: Declared in DSFXParm.h.
