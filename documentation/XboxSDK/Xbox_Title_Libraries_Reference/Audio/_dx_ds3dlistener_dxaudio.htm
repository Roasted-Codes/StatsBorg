DS3DLISTENER

Describes the 3D world parameters and position of the listener. This structure is used with the IDirectSound8::SetAllParameters method.

typedef struct { 
    DWORD      	 dwSize; 
    D3DXVECTOR3  vPosition; 
    D3DXVECTOR3  vVelocity; 
    D3DXVECTOR3  vOrientFront; 
    D3DXVECTOR3  vOrientTop; 
    FLOAT	 flDistanceFactor; 
    FLOAT	 flRolloffFactor; 
    FLOAT	 flDopplerFactor; 
} DS3DLISTENER, *LPDS3DLISTENER;
 
typedef const DS3DLISTENER *LPCDS3DLISTENER;

Members

dwSize
Size of the structure, in bytes. This member must be initialized before the structure is used.
vPosition, vVelocity, vOrientFront, vOrientTop
D3DXVECTOR3 structures that describe the listener's position, velocity, front orientation, and top orientation, respectively.
flDistanceFactor, flRolloffFactor, flDopplerFactor
The current distance, rolloff, and Doppler factors, respectively.
