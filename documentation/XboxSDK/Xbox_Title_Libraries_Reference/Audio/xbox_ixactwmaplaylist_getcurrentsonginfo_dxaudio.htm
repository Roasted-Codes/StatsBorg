IXACTWmaPlayList::GetCurrentSongInfo

Retrieves information about the current song in an XACT WMA playlist.

HRESULT GetCurrentSongInfo(
    PDWORD pdwSongLength,
    PWCHAR pszNameBuffer,
    DWORD dwBufferSize,
    PXACTWMASONG *ppSong
);

Parameters

pdwSongLength
[out] Pointer to a DWORD indicating the length of the song in milliseconds. This parameter is optional and can be NULL.
pszNameBuffer
[out] String to contain the title of the song.
dwBufferSize
[in] Value indicating the length of pszNameBuffer in WCHARs.
ppSong
[out] Pointer to an IXACTWmaSong to be returned.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

GetCurrentSongInfo can be called at any time, even when the playlist is not playing. However, calling this function while the song is stopped may cause a block. Subsequent calls to this function will not block as long as the current song does not change.

Note    Of the calls in the XACT API, GetCurrentSongInfo and GetCurrentSongInfoEx are the only calls that may block. All other calls return immediately or are asynchronous.

At least one song must be added to the playlist for this call to succeed.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
