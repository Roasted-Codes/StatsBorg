XACT_PARAMETER_CONTROL_SCOPE

Specifies the scope of a parameter control.

typedef enum _XACT_PARAMETER_CONTROL_SCOPE{
    eXACTParameterControlScope_Global = 0,
    eXACTParameterControlScope_SoundCueInstance,
    eXACTParameterControlScope_Max,
} XACT_PARAMETER_CONTROL_SCOPE;

Constants

eXACTParameterControlScope_Global
Indicates the parameter control scope is global for the category specified in the wCategoryIndex member of the XACT_PARAMETER_ENTRY structure (see Remarks below).
eXACTParameterControlScope_SoundCueInstance
Indicates the parameter control scope is a sound cue instance.
eXACTParameterControlScope_Max
Do not use. This is not a valid control scope; it is used internally to determine the total number of parameter control scopes.

Remarks

Parameter control scope is specified in the wScope member of the XACT_PARAMETER_ENTRY structures used with IXACTEngine::SetParameterControl.

The eXACTParameterControlScope_Global constant is used only with a pszFriendlyName of XACT_PARAMETER_CONTROL_MASTER_VOLUME to set the category master volume for the category specified in wCategoryIndex. It is equivalent to calling IXACTEngine::SetMasterVolume with the specified category and computed control volume. Other than for category master volumes, eXACTParameterControlScope_Global cannot be used to apply parameters to identically named run-time parameter controls (RPCs).
