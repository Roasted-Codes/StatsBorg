IDirectMusicScript8::GetVariableString

Retrieves the specified string value in a DirectMusic script.

HRESULT GetVariableString(
    char *pszVariableName,
    char *pszValue,
    LONG lLength,
    LONG *plConverted, 
    DMUS_SCRIPT_ERRORINFO *pErrorInfo
);

Parameters

pszVariableName
Address of string value that contains the variable name.
pszValue
Address of a string buffer that receives the value of the variable. Can be set to NULL to get the size of the string value, to make sure you have a large enough buffer.
lLength
Value that contains the size of the string buffer.
plConverted
Address of a value that receives the number of bytes (including the NULL terminator) returned in pszValue. If pszValue was set to NULL, this parameter receives the size, in bytes, of the buffer needed to receive the string and its NULL terminator.
pErrorInfo
Address of a DMUS_SCRIPT_ERRORINFO structure.

Return Values

Returns S_OK if successful, an error code otherwise. Will return DMUS_E_SCRIPT_VARIABLE_NOT_FOUND if the specified variable doesn't exist in the script.
