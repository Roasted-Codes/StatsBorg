IXACTSoundSource::SetI3DL2Source

Sets the Interactive 3-D Audio Level 2 (I3DL2) source properties of the sound source.

HRESULT SetI3DL2Source(
    LPCDSI3DL2BUFFER pds3db,
    DWORD dwApply
);

Parameters

pds3db
[in] A pointer to a DSI3DL2BUFFER structure.
dwApply
[in] Value that specifies when the setting should be applied. This value must be one of the following:
Value Description
DS3D_DEFERRED Settings are not applied until the application calls the IXACTEngine::CommitDeferredSettings method. This allows the application to change several settings and generate a single recalculation.
DS3D_IMMEDIATE Settings are applied immediately, causing the system to recalculate the 3D coordinates for all 3D sound buffers.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

This function is only valid for sound sources created using the XACT_FLAG_SOUNDSOURCE_3D flag.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
