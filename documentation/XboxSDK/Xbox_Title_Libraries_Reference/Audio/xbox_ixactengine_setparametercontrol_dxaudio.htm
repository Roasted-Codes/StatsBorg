IXACTEngine::SetParameterControl

Sets the parameter control(s) for the XACT engine.

HRESULT SetParameterControl(
    PCXACT_PARAMETER_CONTROL_DESC pParams
);

Parameters

pParams
A pointer to an XACT_PARAMETER_CONTROL_DESC structure describing the control(s) to be set.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

For the settings to take effect immediately, XACTEngineDoWork must be called after calling SetParameterControl.

Requirements

Header: Declared in Xact.h.
