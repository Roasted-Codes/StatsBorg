DSFX_RMS_PARAMS

Contains the parameters used by the root-mean-square (RMS) effect.

typedef struct _DSFX_RMS_PARAMS{
    DSFX_RMS_STATE State;
    DWORD dwNumMixBins;
    DWORD dwRMSValues[6];
    DWORD dwPeakValues[6];
} DSFX_RMS_PARAMS, *LPDSFX_RMS_PARAMS;

Members

State
DSFX_RMS_STATE structure that contains the state information for this effect.
dwNumMixBins
Value that specifies the number of mixbins that are being monitored. Can be a value from one to six.
dwRMSValues
Array of values that contains the average value of each 32 sample frame for each of the mixbins being monitored.
dwPeakValues
Array of values that contains the maximum sample value for each of the mixbins being monitored.

Remarks

The RMS effect calculates the average value of each 32 sample frame for up to 6 mixbins. This effect also tracks the maximum sample value that has been processed for the mixbins you are monitoring.

To reset the maximum sample values, use the IDirectSound8::SetEffectData method on the dwPeakValues array.

This effect can be used to produce real-time peak level meters.
