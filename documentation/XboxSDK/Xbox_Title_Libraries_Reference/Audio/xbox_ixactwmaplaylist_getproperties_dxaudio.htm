IXACTWmaPlayList::GetProperties

Retrieves information about the XACT WMA playlist.

HRESULT GetProperties(
    PXACT_WMA_PLAYLIST_PROPERTIES pProperties
);

Parameters

pProperties
[out] Pointer to an XACT_WMA_PLAYLIST_PROPERTIES structure to be returned.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
