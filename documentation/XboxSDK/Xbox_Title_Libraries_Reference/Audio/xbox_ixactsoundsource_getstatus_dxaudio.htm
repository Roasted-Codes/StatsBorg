IXACTSoundSource::GetStatus

Retrieves the status of an IXACTSoundSource object.

HRESULT GetStatus(
    PDWORD pdwStatus
);

Parameters

pdwStatus
[out] Address of a DWORD to contain the status of the sound buffer. Can be zero or the following:
Value Description
XACT_FLAG_SOUNDSOURCE_STATUS_ACTIVE Indicates that the sound source is currently in use.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

Sound sources managed manually can be freed when they are no longer active.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
