DSCAPS

Describes the capabilities of a DirectSound device. Used by the IDirectSound8::GetCaps method.

typedef struct _DSCAPS{                                                       
    DWORD dwFree2DBuffers;
    DWORD dwFree3DBuffers;
    DWORD dwFreeBufferSGEs;
    DWORD dwMemoryAllocated;
} DSCAPS, *LPDSCAPS;

Members

dwFree2DBuffers
Value that contains the number of 2D sound buffers that are available.
dwFree3DBuffers
Value that contains the number of 3D sound buffers that are available.
dwFreeBufferSGEs
Value that contains the number of buffer scatter/gather entries that are available.
dwMemoryAllocated
Value that contains the total amount of memory that has been allocated by DirectSound.

Remarks

DirectSound for the Xbox has drastically changed the DSCAPS structure to reflect the nature of the Xbox Media Communications Processor (MCP).
