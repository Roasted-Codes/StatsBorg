Processing Mode

Sound buffers have three processing modes: normal, head-relative, and disabled.

In normal mode, the sound source is positioned and oriented absolutely in world space. This is the default mode.

In head-relative mode, the buffer is automatically repositioned in world space as the listener moves and turns. Values set through methods such as IDirectSoundBuffer8::SetPosition and IDirectSoundBuffer8::SetVelocity are all relative to the current position, velocity, and orientation of the listener.

In disabled mode, 3D sound processing is disabled and the sound seems to originate from the center of the listener's head.

An application sets the mode for a 3D sound buffer by using the IDirectSoundBuffer8:SetMode method.
