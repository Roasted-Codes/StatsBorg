Batch Parameters for 3D Buffers

Applications can retrieve or set a 3D sound buffer's parameters individually or in batches. To set individual values, your application can use the applicable IDirectSoundBuffer8 interface method. However, applications often must set all the values at once. You can do this with the IDirectSoundBuffer8::SetAllParameters method.

Parameter changes can also be made more efficiently by flagging them as deferred and then executing them all at once. For more information, see Deferred Settings.
