XACTEngineSetFileIOCallbacks

Registers title-defined callback functions used by XACT when streaming data in from disk.

void XACTEngineSetFileIOCallbacks(
    PCXACT_FILEIO_CALLBACKS pFileIOCallbacks
);

Parameters

pFileIOCallbacks
Pointer to an XACT_FILEIO_CALLBACKS structure that specifies the callback functions to register.

Return Values

None.

Remarks

A title can register file input callbacks with the XACT engine. XACT will stream data in from disk by calling the title callbacks rather than calling system routines directly. The title is responsible for retreiving and returning the data requested. By providing these callbacks, the title can prioritize XACT disk input with other disk I/O in the game.

Titles must call XACTEngineSetFileIOCallbacks before creating the XACT engine with XACTEngineCreate.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
