IXACTSoundSource::SetFilter

Sets the sound source programmable filter block.

HRESULT SetFilter(
    LPCDSFILTERDESC pFilterDesc
);

Parameters

pFilterDesc
[in] Address of a DSFILTERDESC structure that contains the filter settings.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

Each XACT sound source has a programmable filter block. The filter can be set to one of four modes specified by the dwMode member of the DSFILTERDESC structure. The filter block is used to provide DLS2-style filtering, a single band of parametric EQ, or occlusion/obstruction filtering for I3DL2 sources.

For an example of the math used in setting the DSFILTERDESC coefficients, see the XDK SetFilter sample.
