IDirectMusicSegment8::SetLength

Sets the length, in music time, of the segment. This method is usually called by the loader, which retrieves the segment length from the file and passes it to the segment object.

HRESULT SetLength(
    MUSIC_TIME mtLength
);

Parameters

mtLength
Desired length. Must be greater than zero.

Return Values

Returns S_OK if successful, an error code otherwise. Can return one of the following error values:

E_INVALIDARG

Remarks

Titles must call SetLength before playing the segment. This allows the title to set the length, play the segment, change the length, and then play the segment again.

Neglecting to set a primary segment length can cause problems when queueing other primary segments with the DMUS_SEGF_QUEUE flag.
