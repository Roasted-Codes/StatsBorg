IDirectSoundBuffer8::SetLoopRegion

SetLoopRegion allows an application to set loop points for the DirectSound buffer.

HRESULT SetLoopRegion(
    DWORD dwLoopStart,
    DWORD dwLoopLength
);

Parameters

dwLoopStart
Value that specifies the number of bytes from beginning of sound to the loop start point.
dwLoopLength
Value that specifies over how many bytes the sound will loop.

Return Values

If the method succeeds, the return value is DS_OK.

Remarks

When played, a sound will play from the beginning of the sound up to the end of the loop, at which point it will loop back to dwLoopStart. When StopEx (DSXB_STOP_IMMEDIATE) is called, the sound will stop immediately. If StopEx (DSXB_STOP_ENVELOPE) is called, the sound may continue past the end of the loop, through to the end of the buffer as determined by the dwBufferBytes member of SetBufferData. If dwLoopLength is zero, playback will loop to the end of the buffer.

The length is exclusive (this means that a loop with a length of 1 will only play at dwLoopStart.) For example, when playing a sound with a dwLoopStart value of 10 and a dwLoopLength value of 1, the audio at position 10 will be heard. When playing a sound with a dwLoopStart value of 10 and a dwLoopLength value of 2, the audio at position 10 and 11 will be heard.
