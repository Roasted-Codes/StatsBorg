IXACTSoundSource::SetPitch

Adjusts the pitch of all sounds playing on the sound source.

HRESULT SetPitch(
    LONG lPitch
);

Parameters

lPitch
Value that specifies the pitch bend, in 1/4096ths of an octave or 3/1024ths of a semitone, relative to 48 kHz (not the actual frequency of the voice). Can be a value from -8192 to 8191.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This function is identical to the IDirectSoundBuffer8::SetPitch function.

SetPitch should not be called on 3D sound sources. For 2D sound sources it should only be used for implementing non-HRTF 3D (using SetMixBinVolumes and Pan3D as an example).

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
