IXACTEngine::GetVariable

Retrieves the current value of a specified global XACT trigger variable.

HRESULT GetVariable(
    DWORD dwVariable,
    PWORD pwValue
);

Parameters

dwVariable
[in] Trigger variable to query. Valid values are 0 to 31 (XACT_GLOBAL_VARIABLE_MIN_INDEX to XACT_GLOBAL_VARIABLE_MAX_INDEX).
pwValue
[out] Pointer to a WORD variable that receives the trigger variable's current value.

Return Values

Returns DS_OK if successful, an error code otherwise.

Remarks

The returned trigger variable value represents the last value set with SetVariable, regardless of whether DS3D_IMMEDIATE or DS3D_DEFERRED was specified. That is, even deferred changes not yet committed will be reflected in the value returned by GetVariable.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
