IDirectMusicHeap

This interface controls the heap used by DirectMusic.

The IDirectMusicHeap interface contains the following methods.

IDirectMusicHeap Methods Allocate
  Free
  GetSize

The IDirectMusicHeap interface also includes the AddRef and Release methods. Unlike many of the other interfaces on Xbox, the QueryInterface method is also included.

Remarks

Currently, the Release and QueryInterface methods are both unused by DirectMusic's heap implementations.

This interface can be implemented by titles that want to control the normal and/or physical heaps used by DirectMusic.

The normal heap can use any threadsafe method of allocating memory (like LocalAlloc) and the physical heap should use XPhysicalAlloc, which is also threadsafe.
