Listener Position and Velocity

An application can set a listener's position in 3D space by using the IDirectSound8::SetPosition method.

To set the velocity value that DirectSound uses to calculate Doppler-shift effects for a moving listener, use the IDirectSound8::SetVelocity method. Velocity is measured in distance units per second—by default, meters per second.

As with buffers, a listener's position and its velocity are completely independent. It is up to the application to set an appropriate velocity for the listener.
