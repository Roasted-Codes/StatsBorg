g_pXACTEngineDebugOutputCallback

Provides the ability for a game to intercept all debug output calls from the XACT engine.

PFNXACTDEBUGOUTPUTCALLBACK g_pXACTEngineDebugOutputCallback;

Remarks

The g_pXACTEngineDebugOutputCallback global callback variable provides a way to intercept all debug output calls from the XACT engine. When this callback is set, XACT will not send debug information to the debugger. To use this global, define your own callback function (see XACT.H for definition) and assign the function as in the following example:

extern "C" PFNXACTDEBUGOUTPUTCALLBACK g_pXACTEngineDebugOutputCallback;
VOID CALLBACK DebugPrint(DWORD dwLevel, PCSTR pszString)
{
    DbgPrint("Debug: %s", pszString);
}

VOID __cdecl main()
{
    g_pXACTEngineDebugOutputCallback = DebugPrint;
    ...
}

Requirements

Header: Declared in Xact.h.

Import Library: Use Xactengd.lib or Xactengid.lib.
