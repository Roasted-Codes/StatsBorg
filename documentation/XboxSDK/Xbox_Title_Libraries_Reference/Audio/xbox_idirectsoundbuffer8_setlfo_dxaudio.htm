IDirectSoundBuffer8::SetLFO

Sets one of the two low frequency oscillators on the buffer.

HRESULT SetLFO(
    LPCDSLFODESC pLFODesc,
);

Parameters

pLFODesc
Address of a DSLFODESC structure that contains the common low frequency oscillator settings.

Return Values

Returns S_OK if successful, an error code otherwise.
