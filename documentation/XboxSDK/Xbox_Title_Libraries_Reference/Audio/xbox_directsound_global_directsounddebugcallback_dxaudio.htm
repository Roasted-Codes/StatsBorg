g_pfnDirectSoundDebugCallback

Provides the ability for a game to intercept all debug output calls from DirectSound.

LPFNDSDPFCALLBACK g_pfnDirectSoundDebugCallback;

Remarks

This global callback variable provides a way to intercept all debug output calls from DirectSound. When this callback is set, DirectSound will not send debug information to the debugger, unless the callback function returns TRUE. To use this global, define your own callback function (see Dsound.H for definition) and assign the function as in the following example:

extern "C" LPFNDSDPFCALLBACK g_pfnDirectSoundDebugCallback;
BOOL CALLBACK DebugPrint(DWORD dwLevel, LPCSTR pszString)
{
    DbgPrint("Debug: %s", pszString);
    return FALSE;
}

VOID __cdecl main()
{
    g_pfnDirectSoundDebugCallback = DebugPrint;
    ...
}

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
