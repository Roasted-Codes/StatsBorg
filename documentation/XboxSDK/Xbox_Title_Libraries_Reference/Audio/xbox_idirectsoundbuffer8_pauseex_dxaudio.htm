IDirectSoundBuffer8::PauseEx

Sets the pause state of the sound buffer.

HRESULT PauseEx(
    REFERENCE_TIME rtTimestamp,
    DWORD dwPause
);

Parameters

rtTimestamp
REFERENCE_TIME value that specifies the time in the future to resume playing the buffer.
dwPause
Value that specifies the pause state. Can be one of the following values:
Value Description
DSBPAUSE_PAUSE Pause the buffer.
DSBPAUSE_RESUME Resume playing the buffer.
DSBPAUSE_SYNCHPLAYBACK Synchronize playback of multiple buffers and streams. See the IDirectSound8::SynchPlayback function for more information on using this flag.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

The buffer will sit idle until the timestamp elapses if the buffer is paused.

This function only works on a buffer that is already playing. IDirectSoundBuffer8::SetCurrentPosition should not be called on a buffer that is paused.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
