DirectSoundCreateStream

Creates an IDirectSoundStream object.

HRESULT DirectSoundCreateStream(
    LPCDSSTREAMDESC pdssd,
    LPDIRECTSOUNDSTREAM *ppStream,
);

Parameters

pdssd
Address of a DSSTREAMDESC structure that describes the sound stream to create.
ppStream
Address of a variable that receives the IDirectSoundStream interface of the new stream object.

Return Values

Returns DS_OK if successful, an error code otherwise.

Requirements

Header: Declared in dsound.h.
