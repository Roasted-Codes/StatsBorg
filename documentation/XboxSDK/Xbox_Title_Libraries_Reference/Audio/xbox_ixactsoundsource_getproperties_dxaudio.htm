IXACTSoundSource::GetProperties

Retrieves the properties of an IXACTSoundSource object.

HRESULT GetProperties(
    PXACT_SOUNDSOURCE_PROPERTIES pProperties
);

Parameters

pProperties
[out] Pointer to an XACT_SOUNDSOURCE_PROPERTIES structure containing the information to retrieve.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
