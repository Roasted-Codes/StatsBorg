XAudioCreateAdpcmFormat

Creates an ADPCM wave format structure.

void XAudioCreateAdpcmFormat(
    WORD nChannels,
    DWORD nSamplesPerSec,
    LPXBOXADPCMWAVEFORMAT pwfx
);

Parameters

nChannels
[in] Value that specifies the number of channels in the wave.
nSamplesPerSec
[in] Value that specifies the sampling rate.
pwfx
[out] Address of the XBOXADPCMWAVEFORMAT structure created.

Return Values

None.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
