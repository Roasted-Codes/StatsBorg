Using DirectMusic with Xbox

The DirectMusic library set is drastically cut for Xbox. DirectMusic on Windows is an immense library designed for much more than game audio, consisting of well over 300 methods on 29 interfaces! For Xbox, only those libraries that make sense for game audio playback will be supported, and only DirectX 8 style DirectMusic is supported.

The philosophy that the Xbox team is taking is that we will initially provide only those libraries that we believe are truly useful for game development. As noted above, this is a very small subset of the libraries present on DirectMusic for Windows. Our desire is to be small and lean. However, if there are libraries that exist on the Windows DirectMusic implementation you need but are not listed as supported on the Xbox game system, please let your account manager know. We will then try to include those libraries in a future release of Xbox System Software. We would rather start too small and grow than load up the code and docs with a lot of libraries you do not need.

Note    The IReferenceClock interface is not supported on the Xbox implementation of DirectMusic. Instead, the timing routines provided by IDirectMusicPeformance8 should be used.
