XACTEngineDoWork

Updates the XACT audio engine.

void XACTEngineDoWork();

Parameters

None.

Return Values

None.

Remarks

This function is required to update the XACT engine and must be called continuously. It is responsible for the following:

  • Processing notifications.
  • Processing audio streams.
  • Sequencing events for all active tracks.
  • Calling DirectSoundDoWork.

XACTEngineDoWork should be called at least once before entering the main code loop. This allows the XACT engine to begin processing on any primed sound banks by copying the audio data into sound memory from the DVD or hard disk.

The granularity of processing notifications, audio streams, and sequencing events depends on the frequency of calls to this function. To help minimize latency and provide more timely notification in the game code, this function should be called as often as reasonably possible (such as during the main game loop).

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
