WAVEBANKENTRY

Contains the entry information from an XACT wave bank.

typedef struct _WAVEBANKENTRY{
    DWORD dwFlags;
    WAVEBANKMINIWAVEFORMAT Format;
    WAVEBANKREGION PlayRegion;
    WAVEBANKREGION LoopRegion;
} WAVEBANKENTRY, *LPWAVEBANKENTRY;

typedef const WAVEBANKENTRY *LPCWAVEBANKENTRY;

Members

dwFlags
Flags for the wave bank entry. Can be the following values:
Value Description
WAVEBANKENTRY_FLAGS_READAHEAD Indicates stream read-ahead is enabled.
WAVEBANKENTRY_FLAGS_LOOPCACHE Indicates one or more looping sounds use this wave.
WAVEBANKENTRY_FLAGS_REMOVELOOPTAIL Indicates removal of data after the end of the loop region.
WAVEBANKENTRY_FILTERS_ADPCM Indicates ADPCM conversion.
WAVEBANKENTRY_FILTERS_8BITPCM Indicates 8-bit PCM conversion.
Format
A WAVEBANKMINIWAVEFORMAT structure describing the mini-wave format for the wave bank entry.
PlayRegion
A WAVEBANKREGION structure that describes the region within the wave data segment that contains this entry.
LoopRegion
A WAVEBANKREGION structure that describes the region within the wave data that should loop.

Requirements

Header: Declared in Xactwb.h.
