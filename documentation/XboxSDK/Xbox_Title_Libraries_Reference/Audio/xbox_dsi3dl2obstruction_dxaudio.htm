DSI3DL2OBSTRUCTION

Contains the parameters for an obstruction effect on the DirectSound path.

typedef struct _DSI3DL2OBSTRUCTION{
    LONG lHFLevel;
    FLOAT flLFRatio;
} DSI3DL2OBSTRUCTION, *LPDSI3DL2OBSTRUCTION;

Members

lHFLevel
Value that specifies the attenuation applied at high frequencies in millibels. Can be a value between –10000 and zero. The default value is 0 mB.
flLFRatio
Value that specifies the ratio of attenuation at low frequencies to the attenuation at high frequencies. Can be a value between 0.0 and 1.0. The default value is 0.0.
