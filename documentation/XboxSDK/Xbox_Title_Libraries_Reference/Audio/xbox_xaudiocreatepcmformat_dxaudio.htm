XAudioCreatePcmFormat

Creates a PCM wave format structure.

void XAudioCreatePcmFormat(
    WORD nChannels,
    DWORD nSamplesPerSec,
    WORD wBitsPerSample,
    LPWAVEFORMATEX pwfx
);

Parameters

nChannels
[in] Value that specifies the number of channels in the wave.
nSamplesPerSec
[in] Value that specifies the sampling rate.
wBitsPerSample
[in] Value that specifies the number of bits per sample.
pwfx
[out] Address of the WAVEFORMATEX structure created.

Return Values

None.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
