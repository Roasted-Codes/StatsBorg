DirectSoundUseFullHRTF

Enables the azimuth-plus-elevation version of the 3D filter table.

DirectSoundUseFullHRTF();

Parameters

None.

Return Values

None.

Remarks

All titles using 3D sound must first call either DirectSoundUseFullHRTF or DirectSoundUseLightHRTF before creating a 3D buffer.

The DirectSoundUseFullHRTF version of the 3D filter tables adds 75968 bytes of code and data to your XBE.
