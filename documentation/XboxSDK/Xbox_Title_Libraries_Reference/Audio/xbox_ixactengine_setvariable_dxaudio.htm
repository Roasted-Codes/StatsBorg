IXACTEngine::SetVariable

Sets the current value of a specified global XACT trigger variable. Changing the value of a trigger variable causes transitions in XACT interactive audio cues that are based on that variable.

HRESULT SetVariable(
    DWORD dwVariable,
    WORD wValue,
    DWORD dwApply
);

Parameters

dwVariable
[in] Trigger variable to set. Valid values are 0 to 31 (XACT_GLOBAL_VARIABLE_MIN_INDEX to XACT_GLOBAL_VARIABLE_MAX_INDEX).
wValue
[in] Value to set. Valid values are 0 to 65535, but note that interactive audio cues only use the values 0 to 10000 (XACT_GLOBAL_VARIABLE_MIN_VALUE to XACT_GLOBAL_VARIABLE_MAX_VALUE).
dwApply
[in] Specifies when the new value should be applied. Must be one of the following values:
Value Description
DS3D_IMMEDIATE The variable's value is changed immediately.
DS3D_DEFERRED The variable's value change is deferred until a subsequent call to SetVariable (for a different variable) specifies DS3D_IMMEDIATE.

Return Values

Returns DS_OK if successful, an error code otherwise.

Remarks

Each XACT interactive audio cue is dependent on a particular trigger variable, as set in the cue's Interactive Audio tab. Run-time, programmatic changes to the value of the trigger variable cause transitions in the interactive audio cue based on weights assigned to the cue. The wValue values 0 to 10000 map to the interactive audio cue weights 0.00 to 100.00. Any value greater than 10000 (or otherwise not in the range of weights as set for the cue) is out of range and causes a stop transition.

To change multiple variables at once (without causing cue transitions each time a variable is changed), specify DS3D_DEFERRED in dwApply for each variable change except the final one. Specify DS3D_IMMEDIATE for the final variable change to cause that variable change, and all deferred changes, to be committed at once.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
