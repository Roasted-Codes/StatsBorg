DirectSound Buffers

DirectSound buffer objects control the delivery of wave data from a source to a destination. The source might be a synthesizer, another buffer, a file, or a resource.

The information in this section is of interest mainly for applications originally written for previous versions of DirectSound, or applications that need to have low-level control over buffers. If your application is playing sounds by using the DirectMusic performance object, buffer creation and management is done automatically. You can, however, obtain buffer objects in order to gain access to parameters for 3D effects and effects in DirectX Media Objects. For more information, see 3D Sound.
