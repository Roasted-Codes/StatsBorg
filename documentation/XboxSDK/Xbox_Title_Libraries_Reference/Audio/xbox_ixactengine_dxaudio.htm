IXACTEngine

Applications use the methods of the IXACTEngine interface to manage wave banks, sound banks, and notification.

The IXACTEngine interface is obtained by calling the XACTEngineCreate function.

The IXACTEngine interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.
