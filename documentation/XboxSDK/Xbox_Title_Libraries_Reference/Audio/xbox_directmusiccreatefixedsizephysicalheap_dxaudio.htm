DirectMusicCreateFixedSizePhysicalHeap

Creates an instance of a fixed-size physical heap.

HRESULT DirectMusicCreateFixedSizePhysicalHeap(
    DWORD cbSize,
    IDirectMusicHeap **ppHeap
);

Parameters

cbSize
Value that specifies the maximum number of bytes the physical heap can allocate. Can be set to zero if the title does not stream any ADPCM-compressed waves.
ppHeap
Address of a pointer to the IDirectMusicHeap of the heap created.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The function will pre-allocate the number of bytes of physical memory specified in cbSize rounded up to a 4-KB (page size) boundary.

This heap is implemented as a simple first-fit heap.

Initially, titles should allocate approximately 512 KB for the physical heap and then use the xbperfmon tool to calculate the correct values for your title.
