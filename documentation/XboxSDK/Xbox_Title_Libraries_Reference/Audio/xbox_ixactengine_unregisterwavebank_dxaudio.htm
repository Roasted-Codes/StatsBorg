IXACTEngine::UnRegisterWaveBank

Unregisters a wave bank.

HRESULT UnRegisterWaveBank(
    PXACTWAVEBANK pWaveBank
);

Parameters

pWaveBank
[in] A pointer to an IXACTWaveBank to be unregistered. An IXACTWaveBank can be obtained by calling IXACTEngine::RegisterWaveBank or IXACTEngine::RegisterStreamedWaveBank.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

When unregistered, the wave bank will no longer be available for sounds that reference it. The wave bank may be streaming or non-streaming.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.

See Also

IXACTEngine::RegisterWaveBank, IXACTEngine::RegisterStreamedWaveBank
