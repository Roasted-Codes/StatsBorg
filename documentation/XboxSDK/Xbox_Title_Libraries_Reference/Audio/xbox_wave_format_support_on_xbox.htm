Wave Format Support on Xbox

Wave files are the preferred format used for DirectSound playback of audio data. The Xbox supports 8-bit and 16-bit PCM data, 4-bit adaptive differential pulse code modulation (ADPCM) data (proprietary format), and WMA-compressed wave data.

The following table contains the wave formats and specifications that are supported:

Specification Formats Supported Comments
Wave format (uncompressed) Linear PCM
Wave format (compressed) ADPCM, WMA For DLS-2, ADPCM support.
Channels Mono, stereo, 4.0, 5.1 (up to six discrete channels). Dolby® Digital–encoded source waves not supported as source content within the Xbox MCP.
Sampling rate 188 Hz – 48 kHz Up-sampled in hardware to 48 kHz (multi-point interpolation).
Bits per sample 8 bits, 16 bits (PCM/WMA), 4 bits (ADPCM) Xbox MCP uses 24-bit resolution internally, outputs 20-bit.
