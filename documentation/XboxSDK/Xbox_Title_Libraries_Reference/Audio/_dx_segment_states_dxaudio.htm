Segment States

When you play a segment, parameters for that segment such as the audiopath, repetitions, and start point are stored in a segment state, represented by an IDirectMusicSegmentState8 interface. The parameters are valid only for that instance of the segment. Changes subsequently made to the segment by using methods of IDirectMusicSegment8 are reflected in new segment states created by calls to IDirectMusicPerformance8::PlaySegmentEx.
