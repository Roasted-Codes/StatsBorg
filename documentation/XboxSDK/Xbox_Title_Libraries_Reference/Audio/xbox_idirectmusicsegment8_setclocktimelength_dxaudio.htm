IDirectMusicSegment8::SetClockTimeLength

Sets the length of the segment in REFERENCE_TIME.

HRESULT SetClockTimeLength(
    REFERENCE_TIME rtLength,
    BOOL fClockTime
);

Parameters

rtLength
REFERENCE_TIME value that specifies the length of the segment.
fClockTime
Boolean value that indicates whether the REFERENCE_TIME length should be used. If TRUE, the value specified by rtLength will be used. If FALSE, the MUSIC_TIME length will be used.

Return Values

Returns S_OK.
