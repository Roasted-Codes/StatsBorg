IXACTEngine::UnRegisterNotification

Cancel registration of an event notification.

HRESULT UnRegisterNotification(
    PCXACT_NOTIFICATION_DESCRIPTION pNotificationDesc
);

Parameters

pNotificationDesc
[in] A pointer to an XACT_NOTIFICATION_DESCRIPTION structure describing the notification to unregister.

Note    Attempting to unregister a notification that has not been registered has no effect.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
