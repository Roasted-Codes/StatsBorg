LPFNDSDPFCALLBACK

typedef BOOL (CALLBACK *LPFNDSDPFCALLBACK)(
    DWORD dwLevel,
    LPCSTR pszString
);

Parameters

dwLevel
[in] Value indicating the current debug output level. See g_dwDirectSoundDebugLevel for more information.
pszString
[in] Null-terminated string containing the debug output text.

Return Values

Returning TRUE indicates that the message will be printed to the debugger. Returning FALSE prevents it from being printed.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
