IXACTSoundSource::StopSoundCues

Stops all playing or queued sound cues on a sound source.

HRESULT StopSoundCues();

Parameters

None.

Return Values

Returns DS_OK if successful, an error code otherwise.

Remarks

StopSoundCues stops all sound cues (queued or playing) on the specified sound source.

All sounds are stopped immediately—the release envelope is not used. To use release envelopes, call IXACTSoundBank::Stop on individual cues rather than on the sound source.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
