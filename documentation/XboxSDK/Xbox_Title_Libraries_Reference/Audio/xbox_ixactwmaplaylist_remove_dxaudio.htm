IXACTWmaPlayList::Remove

Removes a song from an XACT WMA playlist.

HRESULT Remove(
    PXACTWMASONG pSong
);

Parameters

pSong
[in] Pointer to an IXACTWmaSong to be removed. This pointer is obtained from calling the IXACTWmaPlayList::Add function.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
