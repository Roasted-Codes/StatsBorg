DirectSoundGetSampleTime

Retrieves the current sample clock time.

DWORD DirectSoundGetSampleTime();

Parameters

None.

Return Values

Returns the number of samples that have been processed by the hardware.

Remarks

Because the Xbox console processes data at a fixed rate (48kHz), the sample counter can be used as a sub-millisecond accurate clock. Any titles using this function must be prepared for the sample count to reset unexpectedly due to rollover and certain hardware conditions that require the counter to be reset.

DirectSoundGetSampleTime will return an undetermined value if it is called before a DirectSound object is created.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
