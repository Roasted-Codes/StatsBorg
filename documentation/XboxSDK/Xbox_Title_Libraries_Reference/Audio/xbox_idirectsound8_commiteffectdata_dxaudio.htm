IDirectSound8::CommitEffectData

Commits deferred effects settings.

HRESULT CommitEffectData();

Parameters

None.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This method is used to update and set individual DSP effect program parameters whose update has been deferred by setting the DSFX_DEFERRED flag when calling the IDirectSound8::SetEffectData method. This is useful when parameters (such as a set of filter coefficients) are designed to be updated as a set.
