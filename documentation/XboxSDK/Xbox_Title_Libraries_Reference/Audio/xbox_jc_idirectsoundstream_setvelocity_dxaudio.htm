IDirectSoundStream::SetVelocity

Sets the velocity of the sound source. 

HRESULT SetVelocity(
    FLOAT x,
    FLOAT y,
    FLOAT z,
    DWORD dwApply
);

Parameters

x, y, and z
Values that specify the coordinates of the velocity vector. DirectSound may adjust these values to prevent floating-point overflow.
dwApply
Value that specifies when the setting should be applied. This value must be one of the following:
DS3D_DEFERRED
Settings are not applied until the application calls the IDirectSound8::CommitDeferredSettings method. This allows the application to change several settings and generate a single recalculation.
DS3D_IMMEDIATE
Settings are applied immediately, causing the system to recalculate the 3D coordinates for all 3D sound buffers.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

Velocity is used only for calculating Doppler effect. It does not change the position of the buffer. For more information, see Doppler Factor.

The default unit of measurement is meters per second, but this can be changed by using the IDirectSound8::SetDistanceFactor method.

This function is only valid for streams created using the DSBCAPS_CTRL3D flag.
