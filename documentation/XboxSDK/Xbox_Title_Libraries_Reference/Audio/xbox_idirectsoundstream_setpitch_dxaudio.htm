IDirectSoundStream::SetPitch

Adjusts the pitch of all sounds playing on this stream.

HRESULT SetPitch(
    LONG lPitch
);

Parameters

lPitch
Value that specifies the pitch bend, in 1/4096ths of an octave or 3/1024ths of a semitone, relative to 48 kHz (not the actual frequency of the voice). Can be a value from DSSTREAMPITCH_MIN to DSSTREAMPITCH_MAX.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The following examples demonstrate calls to the SetPitch method.

// pStream is a valid pointer to an IDirectSoundStream interface

// Play the sound at 48 kHz (no pitch bend)
pStream->SetPitch(0);   

// Play the sound two octaves lower (12 kHz)
pStream->SetPitch(-8192);

// Play the sound one octave higher (96 kHz)
pStream->SetPitch(4096);

Another way to figure out the value for lPitch would be to calculate the base pitch of the voice (log2(nSamplesPerSecond/48000)), add (or subtract) the number of octaves you want to change (for example, -1 if you want to go down one octave), and convert the value to register format (multiply by 4096).

Internally, SetFrequency calls SetPitch, so they are equivalent methods and will override each other. Calling SetPitch with the value zero is the same as calling SetFrequency with the value of 48000.
