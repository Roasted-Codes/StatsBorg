XACTEngineCreate

Creates and initializes the XACT audio engine.

HRESULT XACTEngineCreate(
    PCXACT_RUNTIME_PARAMETERS pParams,
    PXACTENGINE *ppEngine
);

Parameters

pParams
[in] A pointer to an XACT_RUNTIME_PARAMETERS structure used to initialize the XACT audio engine.
ppEngine
[out] A pointer to the XACT engine returned.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

When waves and sounds are auditioned, they use voices from the same pool the XACT engine uses for in-game sounds. It is important to provide enough voices to support both. When creating the engine, the number of voices available are specified in the XACT_RUNTIME_PARAMETERS structure. See the XACT_RUNTIME_PARAMETERS section for more information.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
