DSMIXBINS

Contains mixbin assignment information.

typedef struct _DSMIXBINS{
    DWORD dwMixBinCount;  // Count of mixbins to assign the voice to or mixbins to set volume on
    LPCDSMIXBINVOLUMEPAIR lpMixBinVolumePairs;  // MixBin identifier/volume pairs
} DSMIXBINS, *LPDSMIXBINS;

Members

dwMixBinCount
Value that specifies the number of mixbins specified in this structure. Can be set to zero if you want to set a voice to zero mixbins.
lpMixBinVolumePairs
Address of an array of DSMIXBINVOLUMEPAIR structures.

Requirements

Header: Declared in DSound.h.
