DirectMusicSetDebugLevel

Sets the priority level of the debug output and determines what level of trace message causes a break in the debugger.

void DirectMusicSetDebugLevel(
    int iDebugLevel,
    int iRIPLevel,
);

Parameters

iDebugLevel
Integer value that determines what level of trace message is output. Can be any value from zero to five. The highest priority information occurs at level zero, lowest at level five. The default value is one.
iRIPLevel
Integer value that determines what level of trace message causes a break in the debugger. Can be any value from zero to five. Must be less than or equal to the value set for iDebugLevel. The default value is zero.

Return Values

None.

Remarks

DirectMusicSetDebugLevel must be called after the first call to DirectMusicCreateInstance to be effective.
