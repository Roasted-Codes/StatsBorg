IDirectSoundBuffer8::SetMixBinVolumes

Sets the individual mixbin volumes.

HRESULT SetMixBinVolumes(
    LPCDSMIXBINS pMixBins
);

Parameters

pMixBins
Address of a DSMIXBINS structure that specifies the mixbins and the volumes they should be set to.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

SetMixBinVolumes only changes the volumes, it does not reassign mixbins.

When a DirectSound buffer mixes its content into its mixbins, it has control over each individual mixbin volume.

Once the output mixbins have been defined, the individual mixbin volumes are set with SetMixBinVolumes.

SetMixBinVolumes is limited in that it doesn't support separate sends to the same mixbin. It can send more than one channel of a multichannel sound to the same mixbin, but it cannot control the volume of each send separately. Additionally, SetMixBinVolumes generates a warning if passed an array which contains the same mixbin in more than one entry of that array.

Note    Volumes set by SetMixBinVolumes are cumulative with SetVolume.
