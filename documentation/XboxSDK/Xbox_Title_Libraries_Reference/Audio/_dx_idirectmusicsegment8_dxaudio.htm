IDirectMusicSegment8

The IDirectMusicSegment8 interface represents a segment, which is a piece of music made up of multiple tracks. Most titles deal with musical data at the segment level. For an overview, see Using Segments.

The methods of the IDirectMusicSegment8 interface can be grouped as follows:

Instruments Download
  Unload
Timing and looping GetClockTimeLength
  GetLength
  GetLoopPoints
  GetRepeats
  GetStartPoint
  SetClockTimeLength
  SetLength
  SetLoopPoints
  SetStartPoint
  SetRepeats
Miscellaneous AddNotificationType
  Compose
  RemoveNotificationType
  SetWavePlaybackParams

The IDirectMusicSegment8 interface also supports the AddRef and Release methods. There is also minimal support for the QueryInterface method.
