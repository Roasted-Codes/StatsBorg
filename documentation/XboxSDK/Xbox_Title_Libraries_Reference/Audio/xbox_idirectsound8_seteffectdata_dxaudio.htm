IDirectSound8::SetEffectData

Updates part or all of the effect's data.

HRESULT SetEffectData(
    DWORD dwEffectIndex,
    DWORD dwOffset,
    LPCVOID pvData,
    DWORD dwDataSize,
    DWORD dwApply
);

Parameters

dwEffectIndex
DWORD value that contains the absolute index of the effect.
dwOffset
DWORD value that contains the offset, in bytes, of the start of the update region.
pvData
Address of the data buffer containing the update.
dwDataSize
DWORD value that contains the size of the data buffer in bytes.
dwApply
DWORD value that specifies when the effect data should be updated. Can be one of the following values:
Value Description
DSFX_DEFERRED Defer updating the effect data until IDirectSound8::CommitEffectData is called.
DSFX_IMMEDIATE Apply the effect data immediately.

Return Values

Returns S_OK if successful, an error code otherwise.
