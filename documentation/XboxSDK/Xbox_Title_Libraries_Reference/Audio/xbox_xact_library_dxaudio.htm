Xbox Audio Creation Tool Library

To use the Xbox Audio Creation tool (XACT) on Microsoft® Windows® 2000, you need to install Microsoft® Windows Media® Player 7.1 (WMP7.1). You can download WMP7.1 from the following location:

http://www.microsoft.com/windows/windowsmedia/download

You do not need to install Media Player to run XACT on Windows XP.
