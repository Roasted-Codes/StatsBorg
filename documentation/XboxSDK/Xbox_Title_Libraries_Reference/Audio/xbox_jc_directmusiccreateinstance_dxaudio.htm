DirectMusicCreateInstance

Creates a DirectMusic object.

HRESULT DirectMusicCreateInstance(
  REFCLSID  clsid,
  LPUNKNOWN  pUnkOuter,
  REFIID  *iid,
  LPVOID  *ppvInterface
);

Parameters

clsid
Class ID of the instance to be created.
pUnkOuter
Pointer to unknown. Must be NULL.
iid
Pointer to IID.
ppvInterface
Pointer to interface returned.

Remarks

DirectMusicCreateInstance provides a replacement for the COM function CoCreateInstance, which is not supported on the Xbox.
