DirectMusicCreateFixedSizeHeap

Creates an instance of a fixed size normal heap.

HRESULT DirectMusicCreateFixedSizeHeap(
    DWORD dwSize,
    IDirectMusicHeap **ppHeap
);

Parameters

dwSize
Value that specifies the maximum number of bytes the normal heap can allocate.
ppHeap
Address of a pointer to the IDirectMusicHeap of the heap created.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The heap created will not allocate more memory than specified by the dwSize parameter, rounded up to a 4-KB (page size) boundary.

Initially, titles should allocate approximately 4 MB for the normal heap and then use the xbperfmon tool to calculate the correct values for your title.
