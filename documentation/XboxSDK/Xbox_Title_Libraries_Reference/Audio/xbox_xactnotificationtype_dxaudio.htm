XACT_NOTIFICATION_TYPE

XACT notification types.

typedef enum _XACT_NOTIFICATION_TYPE{
    eXACTNotification_Start = 0,
    eXACTNotification_Stop,
    eXACTNotification_Marker,
    eXACTNotification_StreamDataAvailable,
    eXACTNotification_SoundCueProcessing,
    eXACTNotification_Max,
} XACT_NOTIFICATION_TYPE;

Constants

eXACTNotification_Start
Notification issued when the object has started.
eXACTNotification_Stop
Notification issued when the object has stopped.
eXACTNotification_Marker
Notification issued when a marker has been reached.
eXACTNotification_StreamDataAvailable
Notification issued when the object's data stream is available (testing for a primed sound bank, for example). This notification can be generated by cues or wave banks. For a zero-latency streaming cue, notification does not fire on a sound cue when the sound bank is first loaded, but does fire when the cue is primed. For non-zero-latency streaming cues, notification does fire on a sound cue when the sound bank is first loaded. For a wave bank containing no zero-latency waves, notification fires immediately after the wave bank is loaded. For a wave bank containing any zero-latency waves, notification does not fire until the read-ahead waves are primed.
eXACTNotification_SoundCueProcessing
Notification issued the first time a sound cue is activated.
eXACTNotification_Max
Maximum number of XACT notification types (this is not a valid notification type).
