DSFX_CHORUS_MONO_STATE

Contains the state information for the mono chorus effect.

typdef struct _DSFX_CHORUS_MONO_STATE{
    DWORD dwScratchOffset;
    DWORD dwScratchLength;
    DWORD dwYMemoryOffset;
    DWORD dwYMemoryLength;
    DWORD dwFlags;
    DWORD dwInMixbinPtrs[2];
    DWORD dwOutMixbinPtrs[1];
} DSFX_CHORUS_MONO_STATE, *LPDSFX_CHORUS_MONO_STATE;

Members

dwScratchOffset
Value that contains the offset in bytes of the scratch area for this effect.
dwScratchLength
Value that contains the length, in DWORDs, of the scratch area for this effect.
dwYMemoryOffset
Value that contains the offset, in DSP WORDS, of the Y memory area for this effect.
dwYMemoryLength
Value that contains the length, in DSP WORDS, of the Y memory for this effect.
dwFlags
Value that contains the flags for this effect. Can be a combination of the following values:
DSFX_STATE_INITIALIZED
Indicates that the parameters for this effect have been changed. Use this flag to reset the effect state and force the effect to use any new parameters that you set using SetEffectData.
DSFX_STATE_GLOBAL
Indicates that the outputs for this effect should be mixed with the content of the output mixbins.
dwInMixbinPtrs
Array that contains references to the input mixbins.
dwOutMixbinPtrs
Array that contains references to the output mixbins.
