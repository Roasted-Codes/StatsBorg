Compressing a Wave File using the Xbox ADPCM Codec

This article demonstrates how to compress a wave file using the Xbox adaptive delta pulse code modulation (ADPCM) codec.

  • Prerequisites and Requirements
  • Implementation Steps
  • Related Topics

Prerequisites and Requirements

To compress wave files using the Xbox ADPCM codec, you must first install the Xbox ADPCM codec if you haven't installed the Xbox Development Kit (XDK) or installed the Xbox ADPCM codec manually. For instructions on how to install the codec, see Installing the Xbox ADPCM Codec. You will also need a stereo or mono PCM wave file to compress and an application that can utilize the audio codecs installed in Windows.

This tutorial will use the Windows logon sound (Windows Logon Sound.wav) and the sound recorder (SndRec32.exe), since these are both included with Windows. Other tools, like DirectMusic Producer, are also capable of using the Xbox ADPCM codec, but aren't discussed in this article.

Implementation Steps

  • From the Windows desktop, click on the Start menu.
  • Click on Run.
  • Type in "SndRec32" and press the Enter key. This will bring up the Sound Recorder.
  • Click on File to bring up the File menu.
  • Click on the Open Menu item and select the wave file you want to compress. For this example, I'm going to use the Windows logon sound (C:\WINNT\Media\Windows Logon Sound.wav).
  • Click on the Open button to load the wave file.
  • Click on File to bring up the File menu again.
  • Click on Save As to bring up the Save As dialog box.
  • Click on the Change button.
  • Click on the arrow for the Format list box to bring up the list of formats available.
  • Click on Xbox ADPCM to select the Xbox ADPCM codec.
  • Click on the arrow for the Attributes list box to bring up the list of possible sample rates, bits per sample (which is restricted to 4 bits per sample for this codec), and channels (mono or stereo).
  • Click on one of the attribute options. For this example, lets use 22.050 kHz, 4 bit, Stereo.
  • Click OK.
  • Select a folder to save your compressed wave file in. For this example, lets just use C:\WINNT\Media\.
  • Enter a file name. For this example, lets use ADPCMTest.wav.
  • Click OK.
