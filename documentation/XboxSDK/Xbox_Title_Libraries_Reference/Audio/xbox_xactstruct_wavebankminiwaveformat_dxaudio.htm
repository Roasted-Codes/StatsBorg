WAVEBANKMINIWAVEFORMAT

Contains the mini-wave format for an XACT wave bank.

typedef struct _WAVEBANKMINIWAVEFORMAT{
    DWORD wFormatTag : 2;
    DWORD nChannels : 3;
    DWORD nSamplesPerSec : 26;
    DWORD wBitsPerSample : 1;
} WAVEBANKMINIWAVEFORMAT, *LPWAVEBANKMINIWAVEFORMAT;

typedef const WAVEBANKMINIWAVEFORMAT *LPCWAVEBANKMINIWAVEFORMAT;

Members

wFormatTag
Format of the wave file. Can be one of the following values:
Value Description
WAVEBANKMINIFORMAT_TAG_PCM Specifies PCM wave format.
WAVEBANKMINIFORMAT_TAG_ADPCM Specifies ADPCM wave format.
WAVEBANKMINIFORMAT_TAG_WMA Specifies WMA format.
nChannels
Number of channels in the wave file. Can be a value from one to six.
nSamplesPerSec
Sampling rate of the wave file.
wBitsPerSample
Bit depth of the wave file. Can be one of the following values:
Value Description
WAVEBANKMINIFORMAT_BITDEPTH_8 Specifies 8-bit data. (PCM only)
WAVEBANKMINIFORMAT_BITDEPTH_16 Specifies 16-bit data. (PCM only)

Requirements

Header: Declared in Xactwb.h.
