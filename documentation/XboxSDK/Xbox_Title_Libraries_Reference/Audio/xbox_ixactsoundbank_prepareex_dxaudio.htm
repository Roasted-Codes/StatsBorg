IXACTSoundBank::PrepareEx

Primes a sound for playback with zero latency. This method overrides the IXACTSoundBank::Prepare method.

HRESULT PrepareEx(
    PCXACT_PREPARE_SOUNDCUE pPrepareData,
    PXACTSOUNDCUE *ppSoundCue
);

Parameters

pPrepareData
[in] Pointer to an XACT_PREPARE_SOUNDCUE structure containing information about the sound cue to prepare.

Note    The pSoundSource member of the XACT_PREPARE_SOUNDCUE structure must be NULL when calling this method.

ppSoundCue
[out] Pointer to an IXACTSoundCue to be returned. This parameter is passed to the IXACTSoundBank::Play or IXACTSoundBank::PlayEx method and cannot be NULL.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

For information on priming, see the XACT Concepts section. Also, see the XACTParamControl sample for an example of using priming.

Requirements
