XACT_DEBUG_OUTPUT_LEVELS

Debug output callback support.

typedef enum _XACT_DEBUG_OUTPUT_LEVELS{
    eXACTDebugLevel_Undefined = 0,
    eXACTDebugLevel_Error,
    eXACTDebugLevel_Warning,
    eXACTDebugLevel_Info,
    eXACTDebugLevel_Spam,
} XACT_DEBUG_OUTPUT_LEVELS;

Constants

eXACTDebugLevel_Undefined
Debug level undefined.
eXACTDebugLevel_Error
Debug level error.
eXACTDebugLevel_Warning
Debug level warning.
eXACTDebugLevel_Info
Debug level information.
eXACTDebugLevel_Spam
Debug level verbose.

Remarks

These debug levels are used in conjunction with the XACT Global Variables to control output from the XACT engine.
