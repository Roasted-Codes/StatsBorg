XBOXADPCMWAVEFORMAT

Contains information about waveform-audio data.

typedef struct xbox_adpcmwaveformat_tag {
    WAVEFORMATEX wfx;
    WORD wSamplesPerBlock;
} XBOXADPCMWAVEFORMAT, *PXBOXADPCMWAVEFORMAT, *LPXBOXADPCMWAVEFORMAT;

Members

wfx
WAVEFORMATEX structure containing data about the wave.
wSamplesPerBlock
Value that contains the number of samples in each encoded block. Must be 64.

Requirements

  Header: Declared in Dsound.h.
