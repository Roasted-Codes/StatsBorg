IDirectSoundBuffer8::SetHeadroom

Allows an application to specify an additional amount of attenuation for the voice.

HRESULT SetHeadroom(
    DWORD dwHeadroom,
);

Parameters

dwHeadroom
Value that specifies a millibel attenuation value. Can be a value between 0 and 10,000. The default value is zero for 3D voices and submixes, while the default value for 2D voices is 600.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This method allows a title to ensure that mixing multiple voices does not result in saturation during voice mixing.

The values for SetHeadroom and SetVolume are additive. To figure out the true volume, use the following equation:

true_volume = mixbin_volume + 3D_volume + volume - headroom
