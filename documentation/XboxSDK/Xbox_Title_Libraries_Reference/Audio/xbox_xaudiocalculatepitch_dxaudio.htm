XAudioCalculatePitch

Converts frequency to a Xbox Media Communications Processor (MCP) pitch value.

LONG XAudioCalculatePitch(
    DWORD dwFrequency
);

Parameters

dwFrequency
[in] Value that specifies the frequency in hertz.

Return Values

Returns the Xbox MCP pitch value.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
