IDirectMusicHeap::GetSize

Retrieves the size of allocated memory.

HRESULT GetSize(
    PVOID pData,
    LPDWORD pcbSize
);

Parameters

cbSize
Address of memory that was previously allocated by IDirectMusicHeap::Allocate.
pcbSize
Address of a DWORD value that receives the size of allocated memory in bytes.

Return Values

Returns S_OK if successful, an error code otherwise.
