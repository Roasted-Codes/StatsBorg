IXACTWmaPlayList::Add

Adds one or more songs to an XACT WMA playlist.

HRESULT Add(
    PCXACT_WMA_PLAYLIST_ADD pDesc,
    PXACTWMASONG *ppSong
);

Parameters

pDesc
[in] Pointer to an XACT_WMA_PLAYLIST_ADD structure that describes the songs to be added.

Note    WMA files added to a playlist must be in 44Khz stereo format.

ppSong
[out] Receives an IXACTWmaSong pointer to the added song (or to the last song when more than one is added). This parameter can be NULL if the song pointer is not desired.

Return Values

Returns DS_OK if successful, an error code otherwise.

Returns E_FAIL if pDesc specifies a soundtrack that contains no songs.

Remarks

Add all desired songs to the playlist before playing it or calling any of the following IXACTWmaPlayList methods on it: Next, Previous, SetCurrent, SetPlaybackBehavior, or GetProperties.

The IXACTWmaSong pointer returned through ppSong can be used with SetCurrent and Remove.

In the June 2003 and later XDK releases, if the this method is used with the XACTWmaPlayListAdd_Directory value to add a directory that contains only one WMA soundtrack, the cue for the WMA playlist does not loop. However, if the directory contains more than one soundtrack, the cue loops correctly.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
