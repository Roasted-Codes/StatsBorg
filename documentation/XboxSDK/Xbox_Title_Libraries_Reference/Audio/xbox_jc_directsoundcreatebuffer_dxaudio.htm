DirectSoundCreateBuffer

Creates an IDirectSoundBuffer8 object.

HRESULT DirectSoundCreateBuffer(
    LPCDSBUFFERDESC pdsbd,
    LPDIRECTSOUNDBUFFER *ppBuffer,
);

Parameters

pdsbd
Address of a DSBUFFERDESC structure that describes the sound buffer to create.
ppBuffer
Address of a variable that receives the IDirectSoundBuffer8 interface of the new buffer object.

Return Values

Returns DS_OK if successful, an error code otherwise.

Requirements

  Header: Declared in Dsound.h.
