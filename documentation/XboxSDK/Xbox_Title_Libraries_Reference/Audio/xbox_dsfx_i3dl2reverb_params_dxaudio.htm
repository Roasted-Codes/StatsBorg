Remarks

Titles should not attempt to directly manipulate the parameters for the I3DL2 reverberation effect. Settings for the I3DL2 reverberation effect should be controlled using the IDirectSound8::SetI3DL2Listener, IDirectSoundBuffer8::SetI3DL2Source, and IDirectSoundStream::SetI3DL2Source methods.
