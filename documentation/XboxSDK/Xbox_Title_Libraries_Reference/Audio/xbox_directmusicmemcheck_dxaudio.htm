DirectMusicMemCheck

Retrieves the number of items allocated for the given object type.

long DirectMusicMemCheck(
    DWORD dwMemType,
    char **ppName,
);

Parameters

dwMemType
Value that specifies the object type, starting with zero.

Note    Specific type IDs are subject to change with each release of the library. Do not hard-code this value.

ppName
Address of a pointer to a string value that receives the name of the object type.

Return Values

Returns the number of items currently allocated, –1 if it is at the end of the type list.

Remarks

To find out the names and identifiers for all of the objects currently tracked by the system, titles should iterate though the entire list until –1 is returned.

Requirements
