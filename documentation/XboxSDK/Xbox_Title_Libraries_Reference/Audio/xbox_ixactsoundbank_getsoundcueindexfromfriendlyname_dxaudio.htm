IXACTSoundBank::GetSoundCueIndexFromFriendlyName

Retrieve a sound cue index from the friendly name.

HRESULT GetSoundCueIndexFromFriendlyName(
    PCSTR pFriendlyName,
    PDWORD pdwSoundCueIndex
);

Parameters

pFriendlyName
[in] Null-terminated string representing the friendly name for a cue in the sound bank.
pdwSoundCueIndex
[out] Pointer to DWORD returning the sound cue index for the given friendly name. If pFriendlyName cannot be found, the value is set to 0xffffffff and E_FAIL is returned.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
