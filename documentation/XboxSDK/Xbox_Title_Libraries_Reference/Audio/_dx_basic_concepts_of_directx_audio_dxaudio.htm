Basic Concepts of DirectMusic

This section is an introduction to some of the key concepts of DirectMusic and the objects that implement them.

DirectMusic is based on the Component Object Model (COM), so all functionality is performed by objects. In addition, source data is encapsulated in objects.

The objects discussed in this section are exposed in every DirectMusic application. Many other objects are at work behind the scenes and can be accessed by more advanced applications, but they are not discussed here.

The following topics are discussed:

  • Loader
  • Segments and Segment States
  • Performance
  • Messages
  • Performance Channels
  • DLS Synthesizer
  • Instruments and Downloading
  • Audiopaths
