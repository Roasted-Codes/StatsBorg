IDirectSound8::CreateSoundBuffer

Creates a DirectSoundBuffer object to hold a sequence of audio samples.

HRESULT CreateSoundBuffer(
    LPCDSBUFFERDESC pdsbd,
    LPDIRECTSOUNDBUFFER *ppBuffer,
    LPUNKNOWN pUnkOuter
);

Parameters

pdsbd
Address of a DSBUFFERDESC structure that describes the sound buffer to create.
ppBuffer
Address of a variable that receives the IDirectSoundBuffer interface of the new buffer object.
pUnkOuter
Must be NULL.

Return Values

If the method succeeds, the return value is DS_OK.

If the method fails, the return value may be one of the following error values:

DSERR_CONTROLUNAVAIL
DSERR_INVALIDCALL
DSERR_NOAGGREGATION
DSERR_OUTOFMEMORY
DSERR_UNSUPPORTED

Remarks

DirectSound does not initialize the contents of the buffer, and the application cannot assume that it contains silence.
