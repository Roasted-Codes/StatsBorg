IDirectSoundStream::GetVoiceProperties

Returns the voice properties of the sound stream.

HRESULT GetVoiceProperties(
    LPDSVOICEPROPS pVoiceProps
);

Parameters

pVoiceProps
Pointer to a DSVOICEPROPS structure that returns the properties of the voice.

Return Values

Returns DS_OK if successful; otherwise returns an error code.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
