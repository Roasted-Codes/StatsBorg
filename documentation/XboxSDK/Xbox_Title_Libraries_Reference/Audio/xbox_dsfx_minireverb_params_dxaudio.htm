DSFX_MINIREVERB_PARAMS

Contains the parameters used by the mini reverberation effect.

typedef struct _DSFX_MINIREVERB_PARAMS{
    DSFX_MINIREVERB_STATE State;
    DWORD dwDelayLineLengths[8];
    DWORD dwReflectionTaps[8];
    DWORD dwReflectionGains[8];
    DWORD dwInputIIRCoefficients[2];
    DWORD dwInputIIRDelay;
    DWORD dwOutputIIRCoefficients[2];
    DWORD dwOutputIIRDelay;
} DSFX_MINIREVERB_PARAMS, *LPDSFX_MINIREVERB_PARAMS;

Members

State
DSFX_MINIREVERB_STATE structure that contains the state information of the effect.
dwDelayLineLengths
Array of values that specify the length of the delay lines for each reflection. The values cannot exceed the scratch size specified for the effect.
dwReflectionTaps
Array of values that specify the absolute position in the delay line of the reflection.
dwReflectionGains
Array of values that specify the gains for each reflection.
dwInputIIRCoefficients
Array of values that specify the IIR coefficients.
dwInputIIRDelay
Value that specifies the IIR delay.
dwOutputIIRCoefficients
Array of values that specify the output IIR coefficients.
dwOutputIIRDelay
Value that specifies the output IIR delay.
