IXACTEngine::GetRealtimeData

Returns data indicating the current memory usage, buffer status, and capabilities of the XACT engine.

HRESULT GetRealtimeData(
    XACT_REALTIME_AUDIO_DATA *pData
);

Parameters

pData
[out] Pointer to an XACT_REALTIME_AUDIO_DATA structure that receives XACT engine status and capabilities.

Return Values

Returns DS_OK if successful, an error code otherwise.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
