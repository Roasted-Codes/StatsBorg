DSI3DL2BUFFER

Contains the information related to the source for Interactive 3-D Audio Level 2 (I3DL2) standard support.

typedef struct _DSI3DL2BUFFER{
    LONG lDirect;
    LONG lDirectHF;
    LONG lRoom;
    LONG lRoomHF;
    FLOAT flRoomRolloffFactor;
    DSI3DL2OBSTRUCTION Obstruction;
    DSI3DL2OCCLUSION Occlusion;
} DSI3DL2BUFFER, *LPDSI3DL2BUFFER;

Members

lDirect
Value that specifies the attenuation of the direct path's intensity, in decibels. Can be a value between –10000 mB and 1000 mB. The default value is 0 mB.
lDirectHF
Value that specifies the attenuation of the direct path's low-pass filter, in decibels. Can be a value between –10000 mB and 0 mB. The default value is 0 mB.
lRoom
Value that adjusts the intensity level and low-pass filter for the room effect. Can be a value between –10000 mB and 1000 mB. The default value is 0 mB.
lRoomHF
Value that specifies the attenuation at high frequencies relative to the intensity at low frequencies. Can be a value between –10000 mB and 0 mB. The default value is 0 mB.
flRoomRolloffFactor
Value that controls the rolloff effect intensity vs. distance. Can be a value between 0.0 and 10.0. The default value is 0.0.
Obstruction
DSI3DL2OBSTRUCTION structure that specifies the parameters for an obstruction effect on the direct path.
Occlusion
DSI3DL2OCCLUSION structure that specifies the parameters for an occlusion effect on the direct path.
