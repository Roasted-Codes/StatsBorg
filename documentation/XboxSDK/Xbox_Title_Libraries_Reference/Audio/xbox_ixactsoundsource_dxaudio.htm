IXACTSoundSource

Applications use the methods of the IXACTSoundSource interface to manipulate sound source objects.

An IXACTSoundSource object is obtained by calling the XACTEngine::CreateSoundSource function.

IXACTSoundSource methods GetProperties
GetStatus
SetConeOrientation
SetFilter
SetI3DL2Source
SetMixBins
SetMixBinVolumes
SetMode
SetPitch
SetPosition
SetVelocity
StopSoundCues

The IXACTSoundSource interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
