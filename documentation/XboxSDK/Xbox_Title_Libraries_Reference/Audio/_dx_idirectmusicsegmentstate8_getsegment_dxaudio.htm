IDirectMusicSegmentState8::GetSegment

Returns a pointer to the segment that owns this segment state.

HRESULT GetSegment(
    IDirectMusicSegment **ppSegment
);

Parameters

ppSegment
Address of a variable that receives a pointer to the IDirectMusicSegment interface.

Return Values

Returns S_OK if successful, otherwise it will return one of the following error values.

DMUS_E_NOT_FOUND
E_POINTER

Remarks

The pointer returned in ppSegment must be released by the application.

DMUS_E_NOT_FOUND will be returned if GetSegment is called on an object whose segment is done playing.
