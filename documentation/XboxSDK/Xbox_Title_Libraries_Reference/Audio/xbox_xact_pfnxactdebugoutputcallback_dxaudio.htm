PFNXACTDEBUGOUTPUTCALLBACK

typedef VOID (CALLBACK *PFNXACTDEBUGOUTPUTCALLBACK)(
    DWORD dwDebugOutputLevel,
    PCSTR pszDebugOutput
);

Parameters

dwDebugOutputLevel
[in] Value indicating the current debug output level. See g_dwXactDebugLevel for more information.
pszDebugOutput
[in] Null-terminated string containing the debug output text.

Return Values

None.

Requirements

Header: Declared in Xact.h.

Import Library: Use Xactengd.lib or Xactengid.lib.

See Also

XACT Callback Functions, g_pXACTEngineDebugOutputCallback
