DSFX_MIXER_2x1_PARAMS

Contains the parameters used by the mixer 2x1 effect.

typedef struct _DSFX_MIXER_2x1_PARAMS{
    DSFX_MIXER_2x1_STATE State;
    DWORD dwGains[2];
} DSFX_MIXER_2x1_PARAMS, *LPDSFX_MIXER_2x1_PARAMS;

typedef const DSFX_MIXER_2x1_PARAMS *LPCDSFX_MIXER_2x1_PARAMS;

Members

State
DSFX_MIXER_2x1_STATE structure that contains the state information for this effect.
dwGains[2]
Value that specifies the gain for input channel one and input channel two. Can be a value between 0 and 2^23 (which is 0.0 to 1.0 in 24 bit).

Requirements

Header: Declared in DSFXParm.h.
