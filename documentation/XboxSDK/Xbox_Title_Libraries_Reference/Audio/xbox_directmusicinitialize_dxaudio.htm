DirectMusicInitialize

Initializes DirectMusic using its default settings.

HRESULT DirectMusicInitialize();

Parameters

None.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

DirectMusicInitialize, DirectMusicInitializeEx, or DirectMusicInitializeFixedSizeHeaps must be called before making calls to DirectMusic. Titles are only allowed to call one of these initialization functions once per XBE. Titles must reboot before you are allowed to call any of them a second time.
