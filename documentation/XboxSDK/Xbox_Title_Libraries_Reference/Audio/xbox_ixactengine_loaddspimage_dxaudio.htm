IXACTEngine::DownloadEffectsImage

Downloads an effects image to the global processor in the Xbox Media Communications Processor (MCP).

HRESULT DownloadEffectsImage(
    PVOID pvData,
    DWORD dwSize,
    LPCDSEFFECTIMAGELOC pEffectLoc,
    LPDSEFFECTIMAGEDESC *ppImageDesc
);

Parameters

pvData
Address of the DSP image data to load and run. This data is a scrambled binary DSP image produced by the XGPImage tool and represents the final DSP image.
dwSize
DWORD value that contains the size of the DSP image data in bytes.
pEffectLoc
Address of the DSEFFECTIMAGELOC structure that specifies the location of the reverberation and crosstalk effects in the effects image being downloaded. Can be set to NULL if none of the effects used by other DirectSound methods (currently reverb and crosstalk) appear in the image.
ppImageDesc
Address of a pointer to the DSEFFECTIMAGEDESC structure that is returned by the method, which describes the DSP scratch image.

Note     This buffer is maintained by the GP manager and should not be freed by the caller. The memory is valid until the DirectSound object is released.

Return Values

Returns DS_OK if successful; returns an error code otherwise.

Remarks

Replacing a DSP effects image while audio is being played will cause a barely noticeable glitch (approximately 40 ms). Titles should plan accordingly if they need to swap images.

This function is identical to IDirectSound8::DownloadEffectsImage.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
