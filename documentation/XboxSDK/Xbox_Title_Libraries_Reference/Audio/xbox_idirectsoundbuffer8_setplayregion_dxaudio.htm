IDirectSoundBuffer8::SetPlayRegion

Sets the buffer play region.

HRESULT SetPlayRegion(
    DWORD dwPlayStart,
    DWORD dwPlayLength
);

Parameters

dwPlayStart
Value that specifies the starting position of the play region.
dwPlayLength
Value that specifies the length, in bytes, of the play region.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Data alignment restrictions apply when using this function. PCM data must be DWORD aligned (4 bytes), and ADPCM data must be block aligned (36 bytes).

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
