Playback Controls

To set the volume at which a buffer is played, your application can use the IDirectSoundBuffer8::SetVolume method.

To set the frequency at which audio samples play, call the IDirectSoundBuffer8::SetFrequency method.

To use most of these controls, you must set the appropriate flags when creating the buffer. The volume and frequency control bits are automatically set on every buffer and stream. See Buffer Control Options for more information on control options.
