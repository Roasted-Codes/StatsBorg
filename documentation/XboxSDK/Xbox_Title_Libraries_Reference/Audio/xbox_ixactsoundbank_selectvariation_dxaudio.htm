IXACTSoundBank::SelectVariation

Selects a sound and/or wave variation before a cue has been prepared or played.

HRESULT SelectVariation(
    DWORD dwSoundCueIndex,
    PCXACT_SOUNDBANK_SELECT_VARIATION pVariation
);

Parameters

dwSoundCueIndex
[in] A value indicating the index of the sound cue to select the variations for.
pVariation
[in] Optional pointer to an XACT_SOUNDBANK_SELECT_VARIATION structure defining the variations to set. If NULL, the engine selects the variation normally.

Return Values

Returns DS_OK if successful, an error code otherwise.

Requirements

Header: Declared in XACT.h.

Import Library: Use XACTEng.lib.
