DSFX_HIGH_LEVEL_EFFECT_DESCRIPTION

typedef struct _DSFX_HIGH_LEVEL_EFFECT_DESCRIPTION{
    DSFX_EFFECT_TYPE effectType;
    union {
        struct {
            FLOAT flFrequency;
            FLOAT flGain;
            FLOAT flQ;
        } IIR2;
        struct {
            FLOAT flGain;
            FLOAT flPreFilterFrequency;
            FLOAT flPreFilterGain;
            FLOAT flPreFilterQ;
            FLOAT flPostFilterFrequency;
            FLOAT flPostFilterGain;
            FLOAT flPostFilterQ;
        } Distortion;
        DSI3DL2LISTENER I3DL2Reverb;
    } ;
} DSFX_HIGH_LEVEL_EFFECT_DESCRIPTION, *LPDSFX_HIGH_LEVEL_EFFECT_DESCRIPTION,
  *PDSFX_HIGH_LEVEL_EFFECT_DESCRIPTION;

typedef const DSFX_HIGH_LEVEL_EFFECT_DESCRIPTION *LPCDSFX_HIGH_LEVEL_EFFECT_DESCRIPTION;

Members

effectType
A DSFX_EFFECT_TYPE identifying the appropriate sub-structure to use.
flFrequency
Frequency value in Hertz. Ranges from 0.0 to 20000.0.
flGain
Gain value in Db. Ranges from -30.0 to 30.0.
flQ
Q value. Ranges from 0.0 to 30.0.

Note    This value can be approximately 0.0, but not exactly 0.0. An absolute 0.0 value will cause a divide by zero to occur.

flGain
Gain value in Db. Ranges from -30.0 to 30.0.
flPreFilterFrequency

Note    This value can be approximately 0.0, but not exactly 0.0. An absolute 0.0 value will cause a divide by zero to occur.

flPostFilterFrequency
Post-filter frequency value in Hertz. Ranges from 0.0 to 20000.0.
flPostFilterGain

flPostFilterQ
Post-filter Q value. Ranges from 0.0 to 30.0.

Note    This value can be approximately 0.0, but not exactly 0.0. An absolute 0.0 value will cause a divide by zero to occur.
