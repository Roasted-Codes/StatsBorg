Unsupported Functions

The following C run-time functions are not supported by the C run-time libraries on the Xbox game system.

Note   This list is not comprehensive and is subject to change.

Current Working Directory and Current Drive

  • _getdrive
  • _chdrive
  • _chdir
  • _getcwd
  • _getdcwd
  • _fullpath

Process Execution and Control

  • _spawnl, _spawnle, _spawnlp, _spawnlpe
  • _spawnv, _spawnve, _spawnvp, _spawnvpe
  • _execl, _execle, _execlp, _execlpe
  • _execv, _execve, _execvp, _execvpe
  • _cwait
  • system
  • _getpid

Environment

  • getenv
  • _putenv

Pipes

  • _pipe
  • _popen
  • _pclose

Command-Prompt Window

  • _cgets
  • _cprintf
  • _cputs
  • _cscanf
  • _getch
  • _getche
  • _kbhit
  • _putch
  • _ungetch

Miscellaneous

  • _getdiskfree
  • _getdrives
  • _setsystime
  • _locking

Obsolete Functions

  • _seterrormode
  • _beep
  • _loaddll
  • _unloaddll
