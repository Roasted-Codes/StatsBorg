XMVDecoder::GetAudioStream

Returns an audio stream from an Xbox Media Video (XMV) file.

void GetAudioStream(
    DWORD AudioStream,
    IDirectSoundStream **ppStream
);

Parameters

AudioStream
[in] Number of the stream in the XMV file.
ppStream
[out] Address of a variable that will receive a pointer to the DOCERROR: name is not a valid attribute...IDirectSoundStream interface, which represents the stream indicated in AudioStream. On return, this will be NULL if the stream isn't enabled.

Return Values

None.

Requirements

Header: Declared in Xmv.h.

Import Library: Use Xmv.lib.

See Also

XMVDecoder::EnableAudioStream, IDirectSoundStream
