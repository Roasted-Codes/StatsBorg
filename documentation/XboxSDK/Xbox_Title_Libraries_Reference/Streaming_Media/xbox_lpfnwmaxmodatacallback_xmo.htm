LPFNWMAXMODATACALLBACK

Supplies more data to the decoder.

typedef DWORD (CALLBACK *LPFNWMAXMODATACALLBACK)(
    LPVOID pvContext,
    DWORD dwOffset,
    DWORD dwByteCount,
    LPVOID *ppvData
);

Parameters

pvContext
Address of a buffer containing the callback context.
dwOffset
Value that specifies the offset into the WMA file to get the data from.
dwByteCount
Value that specifies the number of bytes to get.
ppvData
Address of a pointer to the returned buffer with more data.

Return Values

Should return S_OK if successful, an error code otherwise.

Requirements

  Header: Declared in DSound.h.
