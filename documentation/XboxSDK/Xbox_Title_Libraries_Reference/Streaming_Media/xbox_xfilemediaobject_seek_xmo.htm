XFileMediaObject::Seek

Determines the present position in the file.

HRESULT Seek(
    LONG lOffset,
    DWORD dwOrigin,
    LPDWORD pdwAbsolute
);

Parameters

lOffset
[in] Number of bytes to move the file pointer.
dwOrigin
[in] Starting point for the file pointer move. Can be one of the following values:
Value Definition
FILE_BEGIN The starting point is zero or the beginning of the file.
FILE_CURRENT The starting point is the current value of the file pointer.
FILE_END The starting point is the current end-of-file position.
pdwAbsolute
[out, retval] Pointer to a DWORD value that is filled with the current position in the file.

Return Values

Returns S_OK if successful, an error code otherwise.

Requirements

  Header: Declared in Dsound.h.

  Import Library: Use Dsound.lib.
