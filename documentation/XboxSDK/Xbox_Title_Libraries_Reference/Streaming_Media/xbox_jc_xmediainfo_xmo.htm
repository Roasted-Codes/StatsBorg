XMEDIAINFO

Provides information about how a particular Xbox Media Object can read or output data.

typedef struct {
    DWORD  dwFlags;
    DWORD  dwInputSize;
    DWORD  dwOutputSize;
    DWORD  dwMaxLookahead;
} XMEDIAINFO, *PXEIDIAINFO, *LPXMEDIAINFO;

Members

dwFlags
Can be one of the following values.
XMO_STREAMF_FIXED_PACKET_ALIGNMENT
The Xbox Media Object supports only a fixed packet alignment.
XMO_STREAMF_FIXED_SAMPLE_SIZE
The Xbox Media Object supports input streams that have a fixed sample size.
XMO_STREAMF_INPUT_ASYNC
The Xbox Media Object supports receiving input data asynchronously.
XMO_STREAMF_IN_PLACE
The Xbox Media Object processes all data in place (with a single buffer).
XMO_STREAMF_OUTPUT_ASYNC
The supports providing output data asynchronously.
dwInputSize
Required alignment of input packet data. The dwMaxSize member of all input XMEDIAPACKET structures must be aligned to this value.
dwOutputSize
Required alignment of output packet data. The dwMaxSize member of all output XMEDIAPACKET structures must be aligned to this value.
dwMaxLookahead
Minimum amount of data, in bytes, needed to process the input data. Valid only for asynchronous Xbox Media Objects.
