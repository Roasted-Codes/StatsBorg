XFileCreateMediaObjectEx

Creates and initializes the file Xbox Media Object (XMO).

HRESULT XFileCreateMediaObjectEx(
    HANDLE hFile,
    XFileMediaObject **ppMediaObject
);

Parameters

hFile
Handle to the file.
ppMediaObject
Address of a pointer to the XFileMediaObject that is created.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The title is responsible for freeing the XFileMediaObject that is created by using the Release method.

For wave files, this function will create an XWaveFileMediaObject.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
