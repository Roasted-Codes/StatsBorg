XWaveFileMediaObject::GetLoopRegion

Retrieves the loop region within the wave file.

HRESULT GetLoopRegion(
    LPDWORD pdwLoopStart,
    LPDWORD pdwLoopLength
);

Parameters

pdwLoopStart
Address of a DWORD value that receives the starting point of the loop.
pdwLoopLength
Address of a DWORD value that receives the length of the loop.

Return Values

Returns DS_OK if successful, an error code otherwise.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
