XWmaFileMediaObject::SeekToTime

Sets the present position in an open Microsoft® Windows Media® Audio (WMA) file.

HRESULT SeekToTime(
    DWORD dwSeek,
    LPDWORD pdwActualSeek
);

Parameters

dwSeek
[in] An offset in milliseconds from the beginning of the file.
pdwActualSeek
[out] Address of a DWORD containing the actual seek position in the file.

Note    The actual point is rarely the requested point because the seek must be aligned to a packet boundary. The deviation is based on the encoding, and can vary from 0 to 1000 milliseconds.

Return Values

On success, the function returns a non-negative number.

Remarks

Each seek operation generally forces the WMA decoder to re-read its look-ahead buffer. This is a costly operation and if performed too frequently can cause the stream to starve.

This method is intended for arbitrary seeking within a file, and will not perform well for fast forward or reverse playback.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
