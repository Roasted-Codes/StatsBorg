WMAXMODECODERPARAMETERS

Contains the parameters for creating an Xbox Media Object (XMO).

typedef struct _WMAXMODECODERPARAMETERS{
    LPCSTR pszFileName;
    HANDLE hFile;
    DWORD dwFileOffset;
    DWORD dwLookaheadBufferSize;
} WMAXMODECODERPARAMETERS, *LPWMAXMODECODERPARAMETERS;

typedef const WMAXMODECODERPARAMETERS *LPCWMAXMODECODERPARAMETERS;

Members

pszFileName
Null-terminated string value that specifies the file path. If this value is specified, the hFile member must be NULL.
hFile
A valid file handle. If this value is specified, the pszFileName member must be NULL.
dwFileOffset
File offset to start reading within the specified file. This is for embedded WMA files packed in large game files.
dwLookaheadBufferSize
Indicates the size of the lookahead buffer. This value should be in the order of the PacketSize*8. So if you use 8192 byte packets in Process() calls, set this to 64K. This value must be at least 16K.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
