XMVDecoder::Reset

Resets the movie playback to the beginning (where the file was opened). 

void Reset();

Parameters

None.

Return Values

None.

Remarks

Reset will not disable any audio streams or reset the audio synchronization stream.

If the file was opened on a buffer (using XMVDecoder_CreateDecoderForPackets), the buffer manager will need to send the very first packet at the next call to pfnGetNextPacket.

Requirements

Header: Declared in Xmv.h.

Import Library: Use Xmv.lib.
