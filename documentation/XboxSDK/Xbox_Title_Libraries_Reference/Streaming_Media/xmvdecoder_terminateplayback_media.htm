XMVDecoder::TerminatePlayback

Tells the decoder to end the playback of an Xbox Media Video (XMV) file as soon as possible. Termination may not happen immediately.

void TerminatePlayback();

Parameters

None.

Return Values

None.

Remarks

This function can be safely called from a different thread than the one the decoder is running on.

Requirements

Header: Declared in Xmv.h.

Import Library: Use Xmv.lib.

See Also

XMVDecoder::Play, XMVDecoder::TerminateLoop, XMVDecoder::TerminateImmediately
