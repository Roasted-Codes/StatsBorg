XMVDecoder::GetVideoDescriptor

Retrieves the general attributes of an Xbox Media Video (XMV) file.

void GetVideoDescriptor(
    XMVVIDEO_DESC *pVideoDescriptor
);

Parameters

pVideoDescriptor
[out] Address of a XMVVIDEO_DESC structure that will receive the description of the XMV file.

Return Values

None.

Requirements

Header: Declared in Xmv.h.

Import Library: Use Xmv.lib.
