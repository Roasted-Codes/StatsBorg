XMVDecoder::GetAudioDescriptor

Retrieves information about a specific audio stream in an Xbox Media Video (XMV) file.

void GetAudioDescriptor(
    DWORD AudioStream,
    XMVAUDIO_DESC *pAudioDescriptor
);

Parameters

AudioStream
[in] Number of the specific audio stream.
pAudioDescriptor
[out] Address of an XMVAUDIO_DESC structure that will receive the description of the audio stream.

Return Values

None.

Requirements

Header: Declared in Xmv.h.

Import Library: Use Xmv.lib.
