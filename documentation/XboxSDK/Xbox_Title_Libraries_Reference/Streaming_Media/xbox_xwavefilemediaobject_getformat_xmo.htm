XWaveFileMediaObject::GetFormat

Retrieves the format of the wave file.

HRESULT GetFormat(
    LPCWAVEFORMATEX *ppwfxFormat
);

Parameters

ppwfxFormat
Address of a pointer to a WAVEFORMATEX structure that contains the wave format information.

Return Values

Returns DS_OK if successful, an error code otherwise.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
