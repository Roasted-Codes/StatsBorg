PFNXMVGETNEXTPACKET

Used in XMVDecoder_CreateDecoderForPackets to supply the decoder with packets.

typedef HRESULT (CALLBACK *PFNXMVGETNEXTPACKET)(
    DWORD Context,
    void **ppPacket,
    DWORD *pOffsetToNextPacket
);

Parameters

Context
[in] Context passed to the callback.
ppPacket
[out] Address of pointer to the next packet to retrieve from the buffer.
pOffsetToNextPacket
[out] Address of a value that will receive the bytes offset to the next packet.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

Assuming that Context is a pointer to a decoder object, a function that implements this callback should follow the pseudocode below:

if (Context->NextPacketHasFinishedLoading)
{
    Context->PendingPacket  = Context->DecodingPacket;
    Context->DecodingPacket = Context->LoadingPacket;
    Context->LoadingPacket  = NULL;
    *ppPacket = Context->DecodingPacket;
    *pOffsetToNextPacket = Context->DecodingPacketSize;
}
else
{
    *ppPacket = NULL;
    *pOffsetToNextPacket = 0;
}
