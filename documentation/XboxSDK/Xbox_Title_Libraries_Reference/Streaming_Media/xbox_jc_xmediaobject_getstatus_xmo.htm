XMediaObject::GetStatus

Checks if the Xbox Media Object can accept new data.

HRESULT GetStatus(
    LPDWORD pdwStatus
);

Parameters

pdwStatus
Address of a DWORD value that receives the status of the Xbox Media Object. Can receive a combination of the following values:
XMO_STATUSF_ACCEPT_INPUT_DATA
Xbox Media Object is ready to accept the input data structure.
XMO_STATUSF_ACCEPT_OUTPUT_DATA
Xbox Media Object is ready to accept the output data structure.

Return Values

Returns DS_OK if successful, or an error specific to the Xbox Media Object otherwise.
