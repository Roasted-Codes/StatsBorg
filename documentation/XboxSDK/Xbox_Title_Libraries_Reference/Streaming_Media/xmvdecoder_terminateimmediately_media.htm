XMVDecoder::TerminateImmediately

Tells the decoder to end the playback of an Xbox Media Video (XMV) file immediately.

void TerminateImmediately();

Parameters

None.

Return Values

None.

Remarks

This routine must be called on the same thread that the decoder is running on and therefore cannot be used with XMVDecoder::Play. Additionally, this routine is guaranteed to make the next call to XMVDecoder::GetNextFrame to return an XMVRESULT of XMV_ENDOFFILE.

Requirements

Header: Declared in Xmv.h.

Import Library: Use Xmv.lib.

See Also

XMVDecoder::TerminateLoop, XMVDecoder::TerminatePlayback
