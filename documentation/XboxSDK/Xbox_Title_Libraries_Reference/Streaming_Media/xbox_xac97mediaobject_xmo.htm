XAc97MediaObject

This is the interface for the AC-97 Xbox Media Object.

The XAc97MediaObject interface contains the following methods.

XMediaObject Methods Discontinuity
  Flush
  GetInfo
  GetStatus
  Process
XAc97MediaObject Methods GetCurrentPosition
  SetMode

The XAc97MediaObject interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

Requirements

  Import Library: Use Dsound.lib.
