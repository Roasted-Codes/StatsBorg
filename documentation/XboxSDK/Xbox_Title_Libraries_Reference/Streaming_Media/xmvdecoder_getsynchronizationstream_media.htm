XMVDecoder::GetSynchronizationStream

Returns the current sychronization stream in an Xbox Media Video (XMV) file.

DWORD GetSynchronizationStream();

Parameters

None.

Return Values

Returns the number of the audio stream used to synchronize the video. If no audio stream is being used to synchronize the video, –1 will be returned.

Requirements

Header: Declared in Xmv.h.

Import Library: Use Xmv.lib.
