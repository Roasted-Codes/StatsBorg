XMVAUDIO_DESC

Describes an audio stream in an Xbox Media Video (XMV) file.

typedef struct _XMVAUDIO_DESC{
    DWORD WaveFormat;
    DWORD ChannelCount;
    DWORD SamplesPerSecond;
    DWORD BitsPerSample;
    DWORD Flags;
} XMVAUDIO_DESC;

Members

WaveFormat
Describes how the audio data in the stream is encoded. This member must be either WAVE_FORMAT_PCM or WAVE_FORMAT_ADPCM.
ChannelCount
Number of channels in the audio stream. Number of channels is either 1, 2, 4, or 6.
SamplesPerSecond
Number of samples per second (Hz) in the audio stream.
BitsPerSample
Number of bits in each sample.
Flags
Indicates which 5.1 audio stream set this stream belongs to, if any. The value will be zero or one of the following:
Value Description
XMVAUDIODESC_51_ADPCM_FRONT_LEFT_RIGHT Indicates front left and right channels.
XMVAUDIODESC_51_ADPCM_CENTER_LOW_FREQUENCY Indicates center and low frequency channels.
XMVAUDIODESC_51_ADPCM_REAR_LEFT_RIGHT Indicates rear left and right channels.
