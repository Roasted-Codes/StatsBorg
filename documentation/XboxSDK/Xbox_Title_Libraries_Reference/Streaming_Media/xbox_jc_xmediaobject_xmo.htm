XMediaObject

This is the standard interface for media transforms on the Xbox.

The XMediaObject interface contains the following methods.

XMediaObject Methods Discontinuity
  Flush
  GetInfo
  GetStatus
  Process

The XMediaObject interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.
