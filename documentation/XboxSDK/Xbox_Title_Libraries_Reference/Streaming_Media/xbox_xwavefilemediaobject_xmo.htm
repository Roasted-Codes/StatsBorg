XWaveFileMediaObject

Provides access to the Xbox Media Object (XMO), which is used to handle wave files.

The XWaveFileMediaObject supports the following methods.

XMediaObject Methods Discontinuity
Flush
GetInfo
GetStatus
Process
XFileMediaObject Methods DoWork
GetLength
Seek
XWaveFileMediaObject Methods GetFormat
GetLoopRegion

The XWaveFileMediaObject interface also supports the AddRef and Release methods. The QueryInterface method is not supported on Xbox.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
