XMVDecoder_CreateDecoderForFile

Loads a Xbox Media Video (XMF) file into the decoder.

HRESULT __stdcall XMVDecoder_CreateDecoderForFile(
    DWORD Flags,
    LPCSTR szFileName,
    XMVDecoder **ppDecoder
);

Parameters

Flags
[in] May be zero or one of the following flags ORed together:
Value Description
XMVFLAG_UNSYNCHRONIZED_PLAYBACK Disables synchronization between the audio and the video. This flag is useful only when using XMVDecoder:GetNextFrame to play back the video. When this flag is set, regardless of the timing, the decoder will return a frame with every call to XMVDecoder::GetNextFrame (except when it's waiting for data to load).
XMVFLAG_FULL_LOOP The movie will loop until either XMVDecoder::TerminateLoop or XMVDecoder::TerminatePlayback is called.
XMVFLAG_SYNC_ON_NEXT_VBLANK Indicates that the decoder should synchronize playback on the next vertical blank. XMVDecoder::GetNextFrame will return the frame that is due before the next vertical blank, so that the frame can be prepared and displayed early. The default behavior is to return only the frame that is currently due.
szFileName
[in] Name of the XMV file to load.
ppDecoder
[out] Address of a variable that receives a pointer to the XMVDecoder interface that will decode and play the file.

Return Values

Returns S_OK if successful; returns an error code otherwise.

Remarks

This function will allocate two buffers: one to load portions of the file, and another to contain all other internal data structures.

Requirements

Header: Declared in Xmv.h.

Import Library: Use Xmv.lib.
