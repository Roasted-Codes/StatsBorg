XWmaDecoderCreateMediaObject

Creates and initializes a WMA Xbox Media Object (XMO).

HRESULT XWmaDecoderCreateMediaObject(
    LPCWMAXMODECODERPARAMETERS pParameters,
    XWmaFileMediaObject **ppMediaObject
);

Parameters

pParameters
Address of a WMAXMODECODERPARAMETERS structure that specifies how the XMO should be created.
ppMediaObject
Address of a pointer to the XWmaFileMediaObject that is created.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

This XMO is asynchronous. After XWmaDecoderCreateMediaObject is called to instantiate the XMO, you must call DoWork() on the object on a regular interval.

// Create the XMO
hr = XWmaDecoderCreateMediaObject(&Params, &pTrack->StreamData.pSourceXMO);

Do the following once per frame (or in a tight loop if you don't care to block):

pTrack->StreamData.pSourceXMO->DoWork();
pTrack->StreamData.pSourceXMO->GetStatus(&dwStatus);
bReady = (dwStatus & XMO_STATUSF_ACCEPT_OUTPUT_DATA) != 0;

Once XMediaObject::GetStatus indicates that the XMO is ready, XMediaObject::Process can be called.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
