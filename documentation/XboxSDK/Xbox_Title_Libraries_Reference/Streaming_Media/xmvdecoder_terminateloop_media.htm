XMVDecoder::TerminateLoop

Tells the decoder to stop the playback of an Xbox Media Video (XMV) file at the end of the current loop.

void TerminateLoop();

Parameters

None.

Return Values

None.

Remarks

This function can be safely called from a different thread than the one the decoder is running on.

Requirements

Header: Declared in Xmv.h.

Import Library: Use Xmv.lib.

See Also

XMVDecoder::Play, XMVDecoder::TerminatePlayback, XMVDecoder::TerminateImmediately
