WmaCreateInMemoryDecoderEx

Creates and initializes an Xbox Media Object that performs Microsoft® Windows Media® Audio (WMA) decoding on data stored in memory, and provides access to information embedded in the WMA file.

HRESULT WmaCreateInMemoryDecoderEx(
    LPFNWMAXMODATACALLBACK pfnCallback,
    LPVOID pvContext,
    DWORD dwYieldRate,
    LPWAVEFORMATEX pwfxCompressed,
    XWmaFileMediaObject **ppMediaObject
);

Parameters

pfnCallback
Address of a callback function.
pvContext
Address of a buffer containing the callback context.
dwYieldRate
Value that specifies how often the decoder should yield execution while decoding data. The value zero indicates it should never yield, the value one indicates it should always yield, and a value two or higher indicates it should yield once every n iterations (where n = dwYieldRate).
pwfxCompressed
Address of a WAVEFORMATEX structure that describes the input format.
ppMediaObject
Address of a pointer to the XWmaFileMediaObject interface of the decoder.

Return Values

Returns S_OK if successful, an error code otherwise.
