WMAXMOFileContDesc

Contains information from a Microsoft® Windows Media® Audio (WMA) file.

typedef struct _WMAXMOFileContDesc{
    WORD wTitleLength;
    WORD wAuthorLength;
    WORD wCopyrightLength;
    WORD wDescriptionLength;
    WORD wRatingLength;
    WCHAR* pTitle;
    WCHAR* pAuthor;
    WCHAR* pCopyright;
    WCHAR* pDescription;
    WCHAR* pRating;
} WMAXMOFileContDesc, *LPWMAXMOFileContDesc;

Members

wTitleLength
Value that specifies the size of the buffer, in bytes, that is being passed to the pTitle member. Receives the size of the actual string returned, in bytes, including the zero WCHAR at the end of the string.
wAuthorLength
Value that specifies the size of the buffer, in bytes, that is being passed to the pAuthor member. Receives the size of the actual string returned, in bytes, including the zero WCHAR at the end of the string.
wCopyrightLength
Value that specifies the size of the buffer, in bytes, that is being passed to the pCopyright member. Receives the size of the actual string returned, in bytes, including the zero WCHAR at the end of the string.
wDescriptionLength
Value that specifies the size of the buffer, in bytes, that is being passed to the pDescription member. Receives the size of the actual string returned, in bytes, including the zero WCHAR at the end of the string. Rarely used.
wRatingLength
Value that specifies the size of the buffer, in bytes, that is being passed to the pRating member. Receives the size of the actual string returned, in bytes, including the zero WCHAR at the end of the string. Rarely used.
pTitle
Address of the buffer that receives the title.
pAuthor
Address of the buffer that receives the author.
pCopyright
Address of the buffer that receives the copyright information.
pDescription
Address of the buffer that receives the description.
pRating
Address of the buffer that receives the rating.
