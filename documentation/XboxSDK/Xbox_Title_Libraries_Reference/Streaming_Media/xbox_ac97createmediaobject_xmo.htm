Ac97CreateMediaObject

Creates an Xbox Media Object that provides direct access to an Audio Codec '97 (AC-97) audio device.

HRESULT Ac97CreateMediaObject(
    DWORD dwChannel,
    LPFNXMEDIAOBJECTCALLBACK pfnCallback,
    LPVOID pvContext,
    LPAC97MEDIAOBJECT *ppMediaObject
);

Parameters

dwChannel
Value that contains a channel identifier.
pfnCallback
Address of the callback function.
pvContext
Address of the callback context.
ppMediaObject
Address of a pointer to the Xbox Media Object that is returned.

Return Values

Returns DS_OK if successful, an Xbox Media Object–specific error code otherwise.

Requirements
