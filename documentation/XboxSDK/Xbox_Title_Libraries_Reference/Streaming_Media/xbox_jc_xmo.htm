Xbox Media Objects Libraries

Xbox Media Objects are a standard interface for media transforms on the Xbox. The Xbox Media Object interface can be used for decoders, audio or video filter transforms, codecs, decompressors, and the like.

Xbox Media Objects are designed to be very lightweight and not to add unnecessary overhead beyond the transform process itself. This section describes the Xbox Media Object format and describes the methods Xbox Media Objects should support. It is primarily designed to help those creating their own Xbox Media Objects understand the format and to help those using Xbox Media Objects understand the methods supported.

Each Xbox Media Object will have its own unique creation function. The exact nature and parameters of the creation function will vary with each Xbox Media Object, but all will take as a parameter a pointer to the Xbox Media Object that has been created.
