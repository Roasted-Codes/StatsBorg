XMediaObject::GetInfo

Gets information about the data to be processed by the Xbox Media Object.

HRESULT GetInfo(
    LPXMEDIAINFO pInfo
);

Parameters

pInfo
Pointer to an XMEDIAINFO structure to contain Xbox Media Object input and output information.

Return Values

GetInfo should return DS_OK if successful.
