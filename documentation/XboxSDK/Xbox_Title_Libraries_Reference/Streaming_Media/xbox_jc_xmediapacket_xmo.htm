XMEDIAPACKET

Data is sent to Xbox media objects using the XMEDIAPACKET structure.

typedef struct {
	LPVOID pvBuffer;
	DWORD dwMaxSize;
	PDWORD pdwCompletedSize;
	PDWORD pdwStatus;
	union {
		HANDLE hCompletionEvent;
		PVOID pContext;
	};
	PREFERENCE_TIME prtTimestamp;
} XMEDIAPACKET, *PXMEDIAPACKET, *LPXMEDIAPACKET;

Members

pvBuffer
Pointer to packet data to process.
dwMaxSize
Size of packet to process, in bytes. This value should be aligned to the dwInputSize or dwOutputSize member of the XMEDIAINFO structure obtained by calling XMediaObject::GetInfo.
pdwCompletedSize
Pointer to the amount of data processed by the Xbox media object. When calling an asynchronous Xbox media object, this value initially will be set to 0, then updated as the Xbox media object continues processing. When calling a synchronous Xbox media object, this value will be set during the call to XMediaObject::Process.
pdwStatus
Status of packet. States will be one of the following values:
Value Description
XMEDIAPACKET_STATUS_SUCCESS Packet was processed successfully by the Xbox media object.
XMEDIAPACKET_STATUS_PENDING Packet has been submitted but not yet processed by the Xbox media object.
XMEDIAPACKET_STATUS_FLUSHED Packet has been flushed from the packet queue prior to processing.
XMEDIAPACKET_STATUS_FAILURE Packet processing has failed for some unknown reason.
hCompletionEvent
Event to trigger upon completion of the packet. An Xbox media object may use either completion events or callbacks, but never both. Refer to the documentation for each specific Xbox media object for more information.
pContext
Context to be passed to the callback function.
prtTimestamp
Not supported.
