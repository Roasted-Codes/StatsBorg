XWaveFileCreateMediaObjectEx

Creates and initializes the wave loader Xbox Media Object (XMO).

HRESULT XWaveFileCreateMediaObjectEx(
    LPCSTR pszFileName,
    HANDLE hFile,
    XWaveFileMediaObject **ppMediaObject
);

Parameters

pszFileName
String value that specifies the file path.
hFile
Handle to the file.
ppMediaObject
Address of a pointer to the XWaveFileMediaObject that is created.

Return Values

Returns S_OK if successful, an error code otherwise.

Remarks

The title is responsible for freeing the XWaveFileMediaObject that is created by using the Release method.

Requirements

Header: Declared in DSound.h.

Import Library: Use DSound.lib.
