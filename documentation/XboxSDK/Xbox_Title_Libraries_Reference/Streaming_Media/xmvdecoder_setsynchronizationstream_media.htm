XMVDecoder::SetSynchronizationStream

Uses a specific audio stream to synchronize the video in an Xbox Media Video (XMV) file.

void SetSynchronizationStream(
    DWORD AudioStream
);

Parameters

AudioStream
[in] Number of the audio stream to use for synchronization This stream must have been enabled. If AudioStream is –1, all synchronization will be disabled.

Return Values

None.

Requirements

Header: Declared in Xmv.h.

Import Library: Use Xmv.lib.

See Also

XMVDecoder::EnableAudioStream, XMVDecoder::GetSynchronizationStream
