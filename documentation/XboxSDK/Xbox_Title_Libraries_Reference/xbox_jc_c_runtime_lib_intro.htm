C Run-Time Libraries

The Xbox Title Libraries (XTL) include an implementation of a subset of the Microsoft C run-time libraries. The features and functionality details unique to this implementation are presented here.

For complete documentation of the Microsoft C run-time libraries, refer to the Run-Time Library Reference provided in the Microsoft Visual C++ documentation.

This section includes:

  • Implementation Information: general implementation information and unique features of the Xbox C run-time libraries.
  • Unsupported Functions: a list of C run-time functions that are not supported by the XTL.
