XINPUT_LIGHTGUN_CALIBRATION_OFFSETS

The XINPUT_LIGHTGUN_CALIBRATION_OFFSETS structure is used by the XInputSetLightgunCalibration function to send upper-left and center calibration offsets to a lightgun.

typedef struct{
    WORD wCenterX;
    WORD wCenterY;
    WORD wUpperLeftX;
    WORD wUpperLeftY;
} XINPUT_LIGHTGUN_CALIBRATION_OFFSETS, *PXINPUT_LIGHTGUN_CALIBRATION_OFFSETS;

Members

wCenterX
The X calibration offset for the center calibration point.
wCenterY
The Y calibration offset for the center calibration point.
wUpperLeftX
The X calibration offset for the upper-left calibration point.
wUpperLeftY
The Y calibration offset for the upper-left calibration point.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
