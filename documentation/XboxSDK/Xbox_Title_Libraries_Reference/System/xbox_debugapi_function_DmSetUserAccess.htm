DmSetUserAccess

Programmatically sets the privileges of a user in the list of authorized users.

DMHRAPI DmSetUserAccess(
    LPCSTR szUserName,
    DWORD dwAccess
);

Parameters

szUserName
Username whose privileges will be set.
dwAccess
Access privileges for the specified username.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

For users to establish secure connections, they must be added to the list of authorized users. For more information, see DmAddUser. The DmSetUserAccess function provides a way to set a user's access privileges programmatically.

Requirements
