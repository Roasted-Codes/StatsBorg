DM_D3DPALETTE

The DM_D3DPALETTE structure, returned by the DmGetPalette function, contains the palette of a Direct3D texture stage.

typedef struct{
    DWORD dwNumEntries;
    DWORD color[256];
} DM_D3DPALETTE, *DM_D3DPALETTE;

Members

dwNumEntries
Number of entries in the palette. Valid values for this member are 32, 64, 128, and 256.
color
Array of up to 256 colors.

Requirements

Header: Declared in XboxDbg.h.
