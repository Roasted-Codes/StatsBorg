DmGetD3DState

Retrieves the current Direct3D render state.

DMHRAPI DmGetD3DState(
    LPDM_D3DSTATE pState
);

Parameters

pState
Pointer to a DM_D3DSTATE structure. Upon return, this structure holds the Direct3D render state information.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
