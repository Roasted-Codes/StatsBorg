XGetSectionHandle

The XGetSectionHandle function returns a section handle for the specified XBE image section.

HANDLE XGetSectionHandle(
  LPCSTR pSectionName                       
);

Parameters

pSectionName
[in] Pointer to a null-terminated string that specifies the section to retrieve a handle to.

Return Values

If the function succeeds, the return value is a handle to the specified section.

If the function fails, the return value is INVALID_HANDLE_VALUE. To get extended error information, call GetLastError.

Remarks

This function does not load the section and does not allocate any resources.

The section does not need to be currently loaded in order for this function to succeed.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
