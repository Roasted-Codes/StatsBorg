XMemSizeDefault

Default memory replacement function that determines the size of an allocated memory block.

SIZE_T WINAPI XMemSizeDefault(
    PVOID pAddress,
    DWORD dwAllocAttributes
);

Parameters

pAddress
[in] Base address of the memory block for which the size is to be determined. This parameter must be the address returned by the XMemAlloc function when the memory was allocated.
dwAllocAttributes
[in] An XALLOC_ATTRIBUTES value that specifies the allocation attributes of the memory block for which the size is to be determined. These attributes should match those that were specified when the memory was allocated.

Return Values

If the function succeeds, the return value is the size, in bytes, of the allocated memory block.

If the function fails, the return value is -1.

Remarks

XMemSizeDefault is the default memory callback function used by the XTL libraries for size queries. All memory size queries performed by the XTL libraries call XMemSize. If the title does not provide an overriden XMemSize, then XMemSizeDefault is used. A title can provide an overriden XMemSize, and within its implementation, call XMemSizeDefault in order to perform the appropriate actions. Thus the title can track calls without needing to fully implement XMemSize functionality.

Requirements

Header: Declared in Xbox.h; for include use xtl.h.
