DmEnableGPUCounter

Provides a way to enable or disable the GPU performance counter. When the GPU performance counter is enabled, CPU performance is affected.

DMHRAPI DmEnableGPUCounter(
  BOOL  b
  );

Parameters

b
A BOOL that determines whether the GPU performance counter is enabled (TRUE) or disabled (FALSE).

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: XbDm.lib for target system code; XboxDbg.lib for development system code.
