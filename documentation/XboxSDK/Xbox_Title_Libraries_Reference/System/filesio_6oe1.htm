RemoveDirectory

The RemoveDirectory function deletes an existing empty directory. 

BOOL RemoveDirectory(
  LPCSTR lpPathName   // directory name
);

Parameters

lpPathName
[in] Pointer to a null-terminated string that specifies the path of the directory to be removed. The path must specify an empty directory.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero. To get extended error information, call GetLastError.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
