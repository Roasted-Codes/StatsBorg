XCloseContentSignatures

Closes a content signature handle.

VOID XCloseContentSignatures(
    HANDLE hSignature
);

Parameters

hSignature
[in] Handle to signature data opened with XLoadContentSignatures.

Return Values

This function does not return a value.

Remarks

XCloseContentSignatures closes a content signature handle that was previously opened with XLoadContentSignatures, and then frees associated resources.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
