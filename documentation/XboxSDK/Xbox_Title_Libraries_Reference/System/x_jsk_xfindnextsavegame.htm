XFindNextSaveGame

The XFindNextSaveGame function continues a save game enumeration begun with a call to XFindFirstSaveGame.

BOOL XFindNextSaveGame(
  HANDLE hFindGame,                      
  PXGAME_FIND_DATA pFindGameData         
);

Parameters

hFindGame
[in] A search handle returned by a previous call to the XFindFirstSaveGame function.
pFindGameData
[out] Pointer to the XGAME_FIND_DATA structure that receives information about the found save game. The structure can be used in subsequent calls to XFindNextSaveGame to refer to the found save game.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call GetLastError. If the enumeration has been completed, the function returns FALSE and GetLastError will return ERROR_NO_MORE_FILES.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
Unicode: The returned structure contains both Unicode and ANSI members.
