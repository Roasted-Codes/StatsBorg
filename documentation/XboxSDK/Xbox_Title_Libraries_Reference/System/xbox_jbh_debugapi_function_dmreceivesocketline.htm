DmReceiveSocketLine

Retrieves a single line of text from the specified connection.

DMHRAPI DmReceiveSocketLine(
  PDM_CONNECTION pConnection, 
  LPSTR          szResponse, 
  LPDWORD        lpdwResponseSize
  );

Parameters

pConnection
Pointer to a DM_CONNECTION structure that contains information about the connection.
szResponse
Pointer to a buffer that will receive the response.
lpdwResponseSize
Pointer to a DWORD that, on entry, contains the size of the response buffer. On return, the DWORD contains the number of bytes actually received.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
