LocalSize

The LocalSize function retrieves the current size, in bytes, of the specified memory block.

Note  All Globalxxx and Localxxx memory management functions use the title's default heap, and provide less flexibility than the heap functions. Titles should use the heap functions instead.

UINT LocalSize(
  HLOCAL  hMem   // handle to local memory object
);

Parameters

hMem
[in] Handle to the memory block. This handle is returned by the LocalAlloc, LocalReAlloc, or LocalHandle function.

Return Values

If the function succeeds, the return value is the size, in bytes, of the specified memory block.

If the specified handle is not valid, or if the specified block has been freed, the function fails and returns -1. The function does not call SetLastError, so a title cannot call GetLastError to get extended error information.

Remarks

The size of a memory block may be larger than the size requested when the memory was allocated.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
