XGetAttributesOnHeapAlloc

Gets the allocation attribute flags associated with a specified heap entry. Used with the memory allocation replacement functions (XMemAlloc, XMemFree, and XMemSize).

DWORD XGetAttributesOnHeapAlloc(
    PVOID pBaseAddress
);

Parameters

pBaseAddress
[in] Specifies the starting address of the heap entry.

Return Values

The return value is the XALLOC_ATTRIBUTES value associated with the heap entry.

Remarks

XGetAttributesOnHeapAlloc returns the attribute flags associated with the specified heap entry, as set by XSetAttributesOnHeapAlloc.

To detect a mismatch between heap allocations and deallocations, a custom memory deallocation routine can check the allocation attributes specified in the XMemFree call against those stored with the heap entry. The default XMemFree implementation performs this check in debug builds only.

The custom allocation routine should have stored allocation attributes along with allocations if the dwHeapTracksAttributes member of the XALLOC_ATTRIBUTES structure was specified when the heap entry was allocated.

Requirements

Header: Declared in Xbox.h; for include use xtl.h.
