XInputClose

The XInputClose function is called when a title is done using a specific XID device. This function cleans up all of the additional memory allocated by XInputOpen.

VOID XInputClose(
  HANDLE hDevice  
);

Parameters

hDevice
[in] Handle to the device to be closed.

Return Values

The XInputClose function has no return value.

Remarks

Although the Xbox game system allocates enough memory to properly enumerate and identify devices attached to the system, it must call XInputOpen to allocate the memory necessary to make full use of the specified device. Once use of the device is finished, XInputClose is required to free up the allocated memory.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
