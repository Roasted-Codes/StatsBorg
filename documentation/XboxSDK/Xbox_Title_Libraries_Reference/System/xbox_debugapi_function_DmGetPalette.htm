DmGetPalette

Retrieves the palette from a Direct3D texture stage.

DMHRAPI DmGetPalette(
    DWORD dwStage,
    LPDM_D3DPALETTE lpPalette
);

Parameters

dwStage
Texture stage. Must be in the range 0–3, inclusive.
lpPalette
Pointer to a DM_D3DPALETTE structure. Upon return, this structure holds the palette information.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
