DmIsBreakpoint

Tests whether a breakpoint is set at the specified address.

DMHRAPI DmIsBreakpoint(
    PVOID addr,
    LPDWORD pdwType
);

Parameters

addr
Pointer to the address to test.
pdwType
Type of breakpoint for which to test. For further information, see DmSetDataBreakpoint

Return Values

Returns XBDM_NOERR if the address contains a breakpoint of the specified type. Otherwise, the function returns an error value.

Requirements

Header: Declared in XboxDbg.h, XbDm.h.

Import Library: Use XboxDbg.lib.

See Also

DmSetBreakpoint, DmSetDataBreakpoint
