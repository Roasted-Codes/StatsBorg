XINPUT_DEBUG_KEYQUEUE_PARAMETERS

The XINPUT_DEBUG_KEYQUEUE_PARAMETERS structure is used in the XInputDebugInitKeyboardQueue function to specify configuration and notification options for keyboard input.

typedef struct _XINPUT_DEBUG_KEYQUEUE_PARAMETERS {
  DWORD dwFlags;
  DWORD dwQueueSize;
  DWORD dwRepeatDelay;
  DWORD dwRepeatInterval;
} XINPUT_DEBUG_KEYQUEUE_PARAMETERS, *PXINPUT_DEBUG_KEYQUEUE_PARAMETERS;

Members

dwFlags
Keyboard events that the title wants to receive. This member can be any combination of the following flags (it is recommended that at least one be set):
Flag Event
XINPUT_DEBUG_KEYQUEUE_FLAG_KEYDOWN Queue key down events.
XINPUT_DEBUG_KEYQUEUE_FLAG_KEYREPEAT Queue key repeat events.
XINPUT_DEBUG_KEYQUEUE_FLAG_KEYUP Queue key up events.
XINPUT_DEBUG_KEYQUEUE_FLAG_ASCII_ONLY Queue only keys that correspond to valid ASCII values.
dwQueueSize
Size of the input queue to create. This is the maximum number of keyboard input events that the input queue can hold. Once the queue is full, additional keystrokes are lost.
dwRepeatDelay
Initial delay (in milliseconds) before repeating a key that is held down.
dwRepeatInterval
Delay (in milliseconds) between repeated keystrokes of a key that is held down.

Requirements

Header: Declared in xkbd.h, for include use xtl.h.
