DmGetUtilityDriveInfo

Retrieves information about the utility drive of the Xbox console.

DMHRAPI DmGetUtilityDriveInfo(
    PDM_UTILITY_DRIVE_INFO pdmUtilityDriveInfo
);

Parameters

pdmUtilityDriveInfo
Pointer to a DM_UTILITY_DRIVE_INFO structure. Upon return, the structure contains the utility drive information.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.

Import Library: Use XboxDbg.lib.
