DmSetThreadContext

Sets the CONTEXT for a given thread.

DMHRAPI DmSetThreadContext(
  DWORD dwThreadId, 
  PCONTEXT pdmcr
  );

Parameters

dwThreadId
Thread ID.
pdmcr
Pointer to the context.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

A CONTEXT structure contains processor-specific register data. The system uses CONTEXT structures to perform various internal operations. Refer to the header files for a definition of this structure.

CONTEXT Flags

The Flags member of the CONTEXT structure contains one or more of the following values:

Attribute Meaning
CONTEXT_INTEGER Eax, Ebx, Ecx, Edx, Esi, Edi are valid.
CONTEXT_CONTROL Esp, Ebp, Eip, SegCs, SegSs, EFlags are valid.
CONTEXT_FULL A combination of CONTEXT_INTEGER and CONTEXT_CONTROL.
CONTEXT_FLOATING_POINT The FloatSave area is valid.
CONTEXT_EXTENDED_REGISTERS The ExtendedRegisters data is valid.
CONTEXT_SEGMENTS Not used on Xbox.
CONTEXT_DEBUG_REGISTERS Not used on Xbox.

Only the registers specified by CONTEXT.Flags are actually updated. Only the TF bit of EFlags can be changed. Esp should never be modified; modifying Esp is likely to cause the Xbox to completely halt and become undebuggable until it is restarted.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
