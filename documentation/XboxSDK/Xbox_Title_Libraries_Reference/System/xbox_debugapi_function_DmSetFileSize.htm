DmSetFileSize

Sets the size of a file and creates it if it doesn't exist.

DMHRAPI DmSetFileSize(
    LPCSTR szFileName,
    DWORD dwFileOffset,
    DWORD dwCreateDisposition
);

Parameters

szFileName
String that specifies the name of the file.
dwFileOffset
New file size, in bytes
dwCreateDisposition
Can be any one of the following values.
Constant Description
OPEN_EXISTING Open an existing file and reset its size. Do not creat it if it doesn't exist.
CREATE_NEW Create the file and set its size. Overwrite it if it already exists.
CREATE_ALWAYS Create the file if it does not exist, and set its size. If it already exists, reset its size.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
