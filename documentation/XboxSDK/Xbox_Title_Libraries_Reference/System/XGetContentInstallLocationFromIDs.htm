XGetContentInstallLocationFromIDs

Determines the destination directory where a content offering was installed.

BOOL XGetContentInstallLocationFromIDs(
    DWORD dwTitleID,
    XOFFERING_ID xOfferingID,
    LPSTR lpInstallDirectory
);

Parameters

dwTitleID
[in] Title identifier of the title that installed the content. If zero is specified for TitleID, the title identifier of the calling title is used.
xOfferingID
[in] Offering identifier of the content.
lpInstallDirectory
[out] Pointer to a buffer that receives the installation directory of the content offering. A buffer of size MAX_PATH bytes will always be large enough to contain the returned directory.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call GetLastError.

Remarks

XGetContentInstallLocationFromIDs determines the directory where a content offering (either online or offline) has been installed.

XGetContentInstallLocationFromIDs can also be used to determine where a specified content offering will be installed. However, XGetContentInstallLocationFromIDs is typically used to determine the location of content that has already been (or should have been) installed, in order to verify, access, or remove that content.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
