DmQueryMemoryStatistics

Retrieves various statistics about the number of memory pages currently in use for different purposes on the Xbox console.

DMHRAPI DmQueryMemoryStatistics(
    PDM_MEMORY_STATISTICS MemStat
);

Parameters

MemStat
Upon return, DmQueryMemoryStatistics fills this parameter with a pointer to a DM_MEMORY_STATISTICS structure that contains memory usage statistics.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

You must set the cbSize member of the DM_MEMORY_STATISTICS structure before calling DmQueryMemoryStatistics; failure to do so causes the function to return the E_INVALIDARG error value. The following code demonstrates how to set the cbSize member:

MemStat.cbSize = sizeof MemStat;
hr = DmQueryMemoryStatistics(&MemStat);

Requirements

Header: Declared in XbDm.h.

Import Library: Use Xapilib.lib.
