DeleteFile

The DeleteFile function deletes an existing file. 

BOOL DeleteFile(
  LPCSTR lpFileName    // file name
);

Parameters

lpFileName
[in] Pointer to a null-terminated string that specifies the file to be deleted.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero. To get extended error information, call GetLastError.

Remarks

If an application attempts to delete a file that does not exist, the DeleteFile function fails.

The DeleteFile function fails if an application attempts to delete a file that is currently open.

To close an open file, use the CloseHandle function.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
