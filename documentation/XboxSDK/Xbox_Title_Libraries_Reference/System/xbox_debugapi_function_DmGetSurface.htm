DmGetSurface

Retrieves information about a DirectX surface.

DMHRAPI DmGetSurface(
    DWORD dwSurfaceID,
    LPDWORD lpdwData,
    LPDWORD lpdwFormat,
    LPDWORD lpdwSize,
    LPDWORD lpdwPushBufferPut
);

Parameters

dwSurfaceID
Identifier of the surface to be retrieved.
lpdwData
Pointer to a DWORD. Upon return, the DWORD contains descriptive information about the surface.
lpdwFormat
Pointer to a DWORD. Upon return, the DWORD contains the surface format information.
lpdwSize
Pointer to a DWORD. Upon return, the DWORD contains the size, in bytes, of the surface.
lpdwPushBufferPut
Pointer to a DWORD. Upon return, the DWORD contains information that should be passed to other debug manager functions.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
