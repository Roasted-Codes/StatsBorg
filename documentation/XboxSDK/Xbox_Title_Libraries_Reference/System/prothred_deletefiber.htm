DeleteFiber

The DeleteFiber function deletes an existing fiber.

VOID DeleteFiber(
  LPVOID lpFiber   // pointer to the fiber to delete
);

Parameters

lpFiber
[in] Specifies the address of the fiber to delete.

Return Values

This function does not return a value.

Remarks

The DeleteFiber function deletes all data associated with the fiber, including the fiber's stack.

DeleteFiber cannot be used to delete the currently executing fiber, and it cannot be used to delete a fiber created with ConvertThreadToFiber. Either action will cause an assertion breakpoint in debug builds and access violations in retail builds.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
