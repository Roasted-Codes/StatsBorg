DmStopProfile

Unregisters a profiler interrupt handler that was registered using DmStartProfile.

DMHRAPI DmStopProfile(
  HANDLE handle
  );

Parameters

handle
Handle of the profiler interrupt handler that should be stopped.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h
Import Library: XbDm.lib
