XTL String Functions

Xbox Title Libraries functions that use strings or characters are implemented in one of four ways:

  • A generic version that can be compiled for either ANSI or Unicode.
  • An ANSI-only version.
  • A Unicode-only version.
  • A mixed parameter function that requires both ANSI and Unicode parameters.

Here are the XTL functions that have string parameters, grouped by category:


Mixed Parameters

  • XCreateSaveGame
  • XDeleteSaveGame
  • XFindFirstSaveGame
  • XFindNextSaveGame

Unicode Only

  • XSetNickname
  • XFindFirstNickname
  • XFindNextNickname
