XGetGameRegion

The XGetGameRegion function returns the game region for the Xbox.

DWORD XGetGameRegion();

Parameters

None.

Return Values

The function will return one of the values below:

Value Meaning
XC_GAME_REGION_NA The Xbox was built for the North American game region.
XC_GAME_REGION_JAPAN The Xbox was built for the Japanese game region.
XC_GAME_REGION_RESTOFWORLD The Xbox was built for a game region outside of North America or Japan.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
