DmEnableStackTrace

Enables or disables stack tracing.

DMHRAPI DmEnableStackTrace(
    BOOL fEnable
);

Parameters

fEnable
[in] Enables stack tracing when set to TRUE. Disables stack tracing when set to FALSE.

Return Values

Returns XBDM_NOERR if successful, an error code otherwise.

Requirements

Header: Declared in XbDm.h.

Import Library: Use XboxDbg.lib.
