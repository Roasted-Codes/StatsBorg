DmGetThreadList

Retrieves an array containing all of the thread IDs from the console.

DMHRAPI DmGetThreadList(
  LPDWORD rgdwThreads, 
  LPDWORD pcThreads
  );

Parameters

rgdwThreads
Array of DWORDs into which the thread IDs will be copied.
pcThreads
Pointer to a DWORD that contains the size of the array. The pcThreads parameter supports a maximum of 256 threads.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise. If there are more threads than there are array entries in rgdwThreads, DmGetThreadList returns XBDM_BUFFER_TOO_SMALL.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
