XGetFilePhysicalSortKey

Returns a physical sort key for a file. The sort key represents the physical location of the file on the volume, enabling the sorting of file access requests in order to minimize seek time.

DWORD XGetFilePhysicalSortKey(
    HANDLE hFile
);

Parameters

hFile
[in] Handle of the file to obtain a sort key for.

Return Values

If the function succeeds, the return value is the physical sort key of the file.

If the function fails, the return value is 0xFFFFFFFF. To get extended error information, call GetLastError.

Remarks

By using the physical sort keys, I/O operations on a volume can be ordered based on the physical location of the files. This enables optimizations such as ordering file I/O in order to minimize seek time.

For the DVD drive, the returned value can also be used as a measurement of distance between files. The returned key is the starting sector number of the file.

Requirements

Header: Declared in Xbox.h; for include use xtl.h.
