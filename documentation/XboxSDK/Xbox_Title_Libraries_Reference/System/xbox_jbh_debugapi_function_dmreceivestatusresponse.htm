DmReceiveStatusResponse

Receives a single line of text over an open connection and translates the error code contained at the front of the string.

DMHRAPI DmReceiveStatusResponse(
  PDM_CONNECTION pConnection, 
  LPSTR          szResponse, 
  LPDWORD        lpdwResponseSize
  );

Parameters

pConnection
Pointer to a DM_CONNECTION structure that contains information about the connection.
szResponse
Pointer to a buffer that will receive the response. This parameter can be NULL. If it is NULL, lpdwResponseSize should be NULL too.
lpdwResponseSize
Pointer to a DWORD that, on entry, contains the size of the response buffer. On return, the DWORD contains the number of bytes actually received.

Return Values

The return value is dependent on the line of text received. See Remarks.

Remarks

DmReceiveStatusResponse will receive a single line of text over an open connection (in szResponse) and will translate the error code at the front of the string. For example, DmSendCommand(conn, szCommand, szResponse, lpdwResponseSize) roughly expands to the following:

   DmSendBinary(conn, szCommand, strlen(szCommand));
   DmSendBinary(conn, "\r\n", 2);
   return DmReceiveStatusResponse(conn, szResponse, lpdwResponseSize);

Requirements
