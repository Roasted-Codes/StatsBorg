DmRemoveAllocationEntry

Removes a memory allocation type from the tracking database. The XbMemDump tool only reports allocation statistics for memory types that are in the tracking database.

HRESULT DmRemoveAllocationEntry(
  PVOID   AllocPtr,
  SIZE_T  AllocSize,
  USHORT  AllocType
  );

Parameters

AllocPtr
Pointer to the block of allocated memory that should be removed from the tracking database.
AllocSize
Size (in bytes) of the allocated memory to be removed. This parameter is optional. This parameter allows memory allocation tracking to detect bad deallocations when the deallocation routine requires a size to be specified.
AllocType
Type of memory allocation. This is a number in the range 14–65535, inclusive.

Return Values

If DmRemoveAllocationEntry successfully removes the memory type from the tracking database, it returns XBDM_NOERR. If the AllocType specified has not been assigned, DmRemoveAllocationEntry returns E_INVALIDARG.
