GlobalHandle

The GlobalHandle function retrieves the handle associated with the specified pointer to a global memory block. Because all memory allocated with the global or local memory functions is fixed, a handle to a memory block and the address of the memory block are the same.

Note  All Globalxxx and Localxxx memory management functions use the title's default heap, and provide less flexibility than the heap functions. Titles should use the heap functions instead.

HGLOBAL GlobalHandle(
  LPCVOID pMem   // first byte of global memory block
);

Parameters

pMem
[in] Pointer to the first byte of the global memory block. This pointer is returned by the GlobalLock function.

Return Values

The return value is a handle to the specified global memory object. Because all memory allocated with the global or local memory functions is fixed, the handle returned is the same as the address passed in.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
