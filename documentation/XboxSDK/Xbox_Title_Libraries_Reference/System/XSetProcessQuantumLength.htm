XSetProcessQuantumLength

Sets the quantum length used for preemptive multitasking.

VOID XSetProcessQuantumLength(
    DWORD dwMilliseconds
);

Parameters

dwMilliseconds
[in] Specifies the new desired quantum, in milliseconds. This value can be from 1 to 1000 milliseconds.

Return Values

This function does not return a value.

Remarks

The quantum is the maximum length of time that the scheduler will allow a thread to execute before switching to another thread that has the same priority level and is ready to run. For more information about threads, quantums, and scheduling, see the Thread Overview.

The default quantum is 20 milliseconds.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
