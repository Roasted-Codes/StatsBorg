DmOpenNotificationSession

Opens a new notification session into which you may register notification handlers for debugging events.

DMHRAPI DmOpenNotificationSession(
  DWORD dwFlags, 
  PDMN_SESSION *pSession
);

Parameters

dwFlags

This parameter can be zero, or a combination of the following values.

Value Meaning
DM_PERSISTENT The debugging subsystem will persist the connection. Handlers will continue to receive notifications if the console is rebooted.
DM_DEBUGSESSION The session is for a remote debugger. Calls on the target system to DmIsDebuggerPresent will return TRUE, indicating that a debugger is present.
DM_ASYNCSESSION Return immediately with a valid session pointer, but establish the notification connection asynchronously. Once the connection has been established, a DM_EXEC notification will be sent.
pSession
Pointer to a PDMN_SESSION to receive the new session pointer.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
