DmTranslateError

Converts an error code into a string.

DMHRAPI DmTranslateError(
  HRESULT hr,
  LPTSTR lpBuffer,
  int nBufferMax
  );

Parameters

hr
Error to be translated.
lpBuffer
Buffer to receive the string representation of the error.
nBufferMax
Size of the buffer.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Note   DmTranslateError does not return XBDM_BUFFER_TOO_SMALL if lpBuffer is too small for the incoming data.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
