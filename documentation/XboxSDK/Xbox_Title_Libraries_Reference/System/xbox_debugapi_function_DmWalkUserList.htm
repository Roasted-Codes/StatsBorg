DmWalkUserList

Enumerates the entries in the list of authorized users. Call DmWalkUserList repeatedly to iterate over the entries in the list.

DMHRAPI DmWalkUserList(
    PDM_WALK_USERS * ppdmwu,
    PDM_USER lpdwAccess
);

Parameters

ppdmwu
Address of a pointer to the user list.
lpdwAccess
Pointer to a DM_USER structure. Upon return, the structure holds an entry from the user list.

Return Values

Returns XBDM_NOERR if successful, or XBDM_ENDOFLIST when it reaches the end of the user list. Otherwise, this function returns an error value.

Remarks

For users to establish secure connections, they must be added to the list of authorized users. For more information, see DmAddUser. The DmWalkUserList enables programmatic retrieval of each entry in the list of authorized users.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.

Import Library: Use XboxDbg.lib.
