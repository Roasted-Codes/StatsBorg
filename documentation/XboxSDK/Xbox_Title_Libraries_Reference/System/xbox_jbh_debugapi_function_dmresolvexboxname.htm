DmResolveXboxName

Resolves the currently selected console name to an IP address. The IP address retrieved is the debug IP. See DmGetAltAddress for more information about the relationship between the title IP and the debug IP.

DMHRAPI DmResolveXboxName(
  LPDWORD lpdwAddress
  );

Parameters

lpdwAddress
Pointer to a DWORD to receive the IP address of the console. The address is returned in host byte order.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
