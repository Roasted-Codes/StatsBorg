DmEnableSecurity

Enables or disables secure connections.

DMHRAPI DmEnableSecurity(
    BOOL fEnable
);

Parameters

fEnable
Enables security if set to TRUE, disables security if set to FALSE.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.

Import Library: Use XboxDbg.lib.
