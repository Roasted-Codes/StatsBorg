DmGetMemory2

Copies the contents of memory from the Xbox console to the development PC.

DMHRAPI DmGetMemory2(
    const LPBYTE lpbAddr,
    DWORD cb,
    LPBYTE lpbBuf,
    LPDWORD pcbRet
);

Parameters

lpbAddr
Memory address on the Xbox console of the first byte of memory to be copied.
cb
Number of bytes to be copied.
lpbBuf
Memory address on the development PC of the buffer that receives the data.
pcbRet
Pointer to a DWORD that receives the number of bytes actually copied. This value can be NULL. If pcbRet is NULL and the number of bytes requested does not equal the number of bytes received, DmGetMemory2 returns an error.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

If DmGetMemory2 does not return an error and the DWORD pointed to by pcbRet is zero, the memory pointed to by lpbAddress is not accessible.

Under the following circumstances, DmGetMemory2 can be faster than DmGetMemory.

  • lpbAddr is aligned on a 16-byte boundary.
  • lpbAddr does not point to code.
  • cb is a multiple of 16 bytes.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.

Import Library: Use XboxDbg.lib.
