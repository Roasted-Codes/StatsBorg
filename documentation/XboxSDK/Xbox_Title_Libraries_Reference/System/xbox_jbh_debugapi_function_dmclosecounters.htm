DmCloseCounters

Frees the memory allocated by DmWalkPerformanceCounters.

DMHRAPI DmCloseCounters(
  PDM_WALK_COUNTERS pWalkCounters
  );

Parameters

pWalkCounters
Pointer to the DM_WALK_COUNTERS structure used by DmWalkPerformanceCounters.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
