DmGetDiskFreeSpace

Retrieves information about the free space of a mounted Xbox drive.

DMHRAPI DmGetDiskFreeSpace(
	char            *szDrive,
	PULARGE_INTEGER pnFreeBytesAvailableToCaller,
	PULARGE_INTEGER pnTotalNumberOfBytes,
	PULARGE_INTEGER pnTotalNumberOfFreeBytes
  );

Parameters

szDrive
Drive to query.
pnFreeBytesAvailableToCaller
Pointer to a LARGE_INTEGER to receive the free bytes available.
pnTotalNumberOfBytes
Pointer to a LARGE_INTEGER to receive the size of the drive.
pnTotalNumberOfFreeBytes
Pointer to a LARGE_INTEGER to receive the total number of free bytes.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
