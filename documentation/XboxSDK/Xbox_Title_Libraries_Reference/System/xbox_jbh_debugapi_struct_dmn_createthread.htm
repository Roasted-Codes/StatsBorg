DMN_CREATETHREAD

Contains information relevant to a DM_CREATETHREAD notification type. For a complete list of notification types, see DmNotify.

typedef struct _DMN_CREATETHREAD {
    DWORD ThreadId;
    PVOID StartAddress;
} DMN_CREATETHREAD, *PDMN_CREATETHREAD;

Members

ThreadId
ID of the thread that was created.
StartAddress
Code address where the thread will begin execution.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
