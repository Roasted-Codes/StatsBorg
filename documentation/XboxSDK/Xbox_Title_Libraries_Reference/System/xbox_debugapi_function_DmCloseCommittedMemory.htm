DmCloseCommittedMemory

Frees memory allocated by DmWalkCommittedMemory.

DMHRAPI DmCloseCommittedMemory(
    PDM_WALK_MEMORY * ppDmWalkMemory
);

Parameters

ppDmWalkMemory
Pointer to a structure that DmWalkCommittedMemory used to iterate over the list of PDM_MEMORY_REGION structures.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.

Import Library: Use XboxDbg.lib.
