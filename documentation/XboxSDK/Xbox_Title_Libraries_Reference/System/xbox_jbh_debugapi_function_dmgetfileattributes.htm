DmGetFileAttributes

Retrieves the file attributes of a file on the Xbox console.

DMHRAPI DmGetFileAttributes(
  LPCSTR szFileName, 
  PDM_FILE_ATTRIBUTES pdmFileAttributes
  );

Parameters

szFileName
Name of the remote file that is being queried.
pdmFileAttributes
Pointer to a DM_FILE_ATTRIBUTES structure.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
