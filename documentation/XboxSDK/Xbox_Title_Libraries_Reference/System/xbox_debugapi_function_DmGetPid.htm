DmGetPid

Retrieves a title's process ID.

DMHRAPI DmGetPid(
    LPDWORD pdwPid
);

Parameters

pdwPid
[out] Address of a DWORD, which, upon return, contains the title's process ID.

Return Values

Returns XBDM_NOERR if successful, an error code otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
