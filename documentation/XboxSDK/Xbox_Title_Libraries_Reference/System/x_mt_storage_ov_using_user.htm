Using the User Data Region

In the user data region of the hard disk, there are subdirectories representing each title's specific user data region on the device. The first time a title is launched on the console, subdirectories dedicated to that title are created within the global user data region of the hard disk. At initialization time, the U:\ logical drive is mapped to the title's specific user data region directory, and the title uses that drive to access its user data region from that point on.

The user data region is used primarily to store save games, as discussed in Xbox Save Games.

User data regions also exist on Xbox memory units, where save games are often stored. Like the hard disk, a memory unit has subdirectories that represent each title's specific user data region on that memory unit. The Using Memory Units section describes the procedures to detect, mount, and use the memory units.

In some circumstances, it is also possible to access the user data of another, related title. Details of this process are given in the Alternate Title Data topic.
