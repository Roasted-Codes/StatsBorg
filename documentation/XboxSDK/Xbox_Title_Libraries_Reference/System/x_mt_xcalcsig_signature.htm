XCALCSIG_SIGNATURE

The XCALCSIG_SIGNATURE structure contains a signature returned by the XCalculateSignatureEnd function.

typedef struct _XCALCSIG_SIGNATURE {
    BYTE Signature[XCALCSIG_SIGNATURE_SIZE];
} XCALCSIG_SIGNATURE, *PXCALCSIG_SIGNATURE;

Members

Signature
A byte array of size XCALCSIG_SIGNATURE_SIZE that receives the signature calculated over a data set.

Remarks

XCalculateSignatureEnd fills the XCALCSIG_SIGNATURE structure with the signature calculated over a set of data.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
