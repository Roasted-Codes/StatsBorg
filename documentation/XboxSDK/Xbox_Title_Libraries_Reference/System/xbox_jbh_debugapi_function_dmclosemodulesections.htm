DmCloseModuleSections

Frees the memory allocated by DmWalkModuleSections.

DMHRAPI DmCloseModuleSections(
  PDM_WALK_MODSECT pWalkModSect
  );

Parameters

pWalkModSect
Pointer to the DM_WALK_MODSECT structure used by DmWalkModuleSections.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
