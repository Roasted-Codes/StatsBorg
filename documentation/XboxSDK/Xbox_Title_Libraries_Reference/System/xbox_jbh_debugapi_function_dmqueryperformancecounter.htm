DmQueryPerformanceCounter

Polls the specified performance counter for its current value.

DMHRAPI DmQueryPerformanceCounter( // Defined in xboxdbg.h
	LPCSTR szName, 
	DWORD dwType, 
	PDM_COUNTDATA pData
	);
DMHRAPI DmQueryPerformanceCounterHandle( // Defined in xbdm.h
	HANDLE hCounter, 
	DWORD dwType, 
	PDM_COUNTDATA pData
	);

Parameters

hCounter
Counter handle. Must be a handle returned by a previous call to DmOpenPerformanceCounter.
szName
Counter name. The XboxDbg version of this function does not use handles because it performs stateless operations.
dwType
Counter type. This parameter must be one of the following values.
ValueMeaning
DMCOUNT_VALUEA value counter.
DMCOUNT_EVENTAn event rate counter.
DMCOUNT_PRATIOA progressive ratio counter.
pData
Pointer to a DM_COUNTDATA structure that receives the counter data.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Querying Performance Counters on the Xbox Development System

On the Xbox development system, users are able to perform two activities:

  1. Enumerate the list of running performance counters on an Xbox target system using DmWalkPerformanceCounters and DmCloseCounters.
  2. Query a running performance counter for its value using DmQueryPerformanceCounter.

Querying Performance Counters on the Xbox Target System

On the Xbox target system, the running application can also enumerate and query the values of running counters. This is useful if a running application would like to keep its own collection of metrics in order to do its own performance analysis. On the target system side, the APIs for enumeration and query are similar, but geared for higher performance (since response times will not be slowed by network latency). Users will use the following API calls to enumerate and query performance counters on the Xbox target system:

  • DmWalkPerformanceCounters
  • DmCloseCounters
  • DmOpenPerformanceCounter
  • DmQueryPerformanceCounter
  • DmClosePerformanceCounter

Requirements

Header: Declared in XbDm.h and XboxDbg.h (see function prototypes above)
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
