DM_USER

The DM_USER structure, used by the DmWalkUserList function, contains information from an entry in the list of authorized users.

typedef struct{
    char UserName[256];
    DWORD AccessPrivileges;
} DM_USER, *DM_USER;

Members

UserName
String containing the username.
AccessPrivileges
DWORD value that describes the access privileges for the user specified by UserName.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.
