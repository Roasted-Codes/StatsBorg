DmLoadSymbolsForAllLoadedModules

Loads debugging symbols for all loaded modules.

DMHRAPI DmLoadSymbolsForAllLoadedModules();

Parameters

None.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

Calling DmLoadSymbolsForAllLoadedModules is equivalent to iterating over the loaded modules with DmWalkLoadedModules and calling DmLoadSymbolsForModule for each module found.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
