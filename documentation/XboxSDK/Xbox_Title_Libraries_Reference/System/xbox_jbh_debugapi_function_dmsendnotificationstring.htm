DmSendNotificationString

Sends a notification string to a registered notification handler.

DMHRAPI DmSendNotificationString(
  LPCSTR sz
  );

Parameters

sz
LPCSTR in the form, "handler!data data data".

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

Code in the XDK sends a notification by calling DmSendNotificationString and passing a string in the form, "handler!data data data", where "handler" is the name in which the notification processor was registered when calling DmRegisterNotificationProcessor.

Requirements

Header: Declared in XbDm.h
Import Library: Use XbDm.lib
