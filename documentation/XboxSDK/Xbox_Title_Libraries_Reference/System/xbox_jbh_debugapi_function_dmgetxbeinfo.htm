DmGetXbeInfo

Retrieves information about an XBE.

DMHRAPI DmGetXbeInfo(
  LPCSTR  szName, 
  PDM_XBE pXbeInfo
  );

Parameters

szName
Buffer containing the file name of the XBE.
pXbeInfo
A pointer to a DM_XBE that contains information about the XBE.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

If an error occurs, the contents of the structure pointed to by pXbeInfo are undefined.

If there is an XBE running on the console, the data that is returned is for the running XBE, not the one specified by szName. In this case, pXbeInfo.LaunchPath is the path of the XBE that was launched.

If there is no XBE already running (the current execution state is DMN_EXEC_PENDING) pXbeInfo.LaunchPath should match szName, and the data that comes back will correspond to that XBE.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
