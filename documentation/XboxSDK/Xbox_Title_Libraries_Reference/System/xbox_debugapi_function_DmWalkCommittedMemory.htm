DmWalkCommittedMemory

Provides a way to examine information about each committed page of memory. Call DmWalkCommittedMemory repeatedly to iterate over the list of memory pages.

DMHRAPI DmWalkCommittedMemory(
    PDM_WALK_MEMORY * ppDmWalkMemory,
    DM_MEMORY_REGION * pMemoryRegion
);

Parameters

ppDmWalkMemory
Pointer to a structure that DmWalkCommittedMemory uses internally to iterate over the list of PDM_MEMORY_REGION structures.
pMemoryRegion
Pointer to a DM_MEMORY_REGION structure that contains information about the next block of committed memory.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

If the return value from this function is anything other than XBDM_NOERR or XBDM_ENDOFLIST, an error has occurred and the walk should be discontinued. After using DmWalkCommittedMemory, your program must free the memory allocated to store the list of memory regions by calling DmCloseCommittedMemory, passing it the value stored in the ppDmWalkMemory parameter.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.

Import Library: Use XboxDbg.lib.
