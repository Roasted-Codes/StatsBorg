DmGetSymbolFromAddress

Retrieves the symbol nearest a specified address.

DMHRAPI DmGetSymbolFromAddress(
    ULONG_PTR Address,
    CHAR *SymbolName,
    SIZE_T BufferSize,
    PULONG Displacement
);

Parameters

Address
[in] Address for which a symbol should be located.
SymbolName
[out] Pointer to a buffer that receives the symbol name.
BufferSize
[in] Size, in bytes, of the SymbolName buffer. If SymbolName is not large enough to contain the entire symbol name, the symbol name will be truncated.
Displacement
[out, optional] Pointer to a ULONG variable that receives the displacement of the specified address, in bytes, from the beginning of the nearest symbol found. If Displacement is NULL, no displacement value is returned.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
