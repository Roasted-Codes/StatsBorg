DmGetModuleLongName

Retrieves the long name of a module given its short name.

DMHRAPI DmGetModuleLongName(
  LPCSTR  szShortName, 
  LPSTR   szLongName, 
  LPDWORD pcch
  );

Parameters

szShortName
Buffer containing the short name. The short name is the name as returned by DmWalkLoadedModules and it is in the form, "filename.exe".
szLongName
Buffer to receive the long name. The long name is the name of the file as it was passed into the ImageBld tool. It is in the form, "c:\mytest\debug\mytest.exe". The long name is available only if the XBE was built by the ImageBld tool using the /debug option.
pcch
On entry, pcch contains the size of the szLongName buffer. On return, pcch contains the number of bytes written to the buffer.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
