Debugger API

The Debugger API provides a means to create a custom remote debugger for the Xbox.

Target System Versus Development System

The Xbox system software contains code that can be called from Xbox executables to assist in debugging Xbox titles. To use this code, your XBE must link with the XbDm.lib static library. On the development system, the Xbox SDK installs a static library called XboxDbg.lib that you can link with your Win32 executables to communicate with the target system remotely. Collectively, these libraries are known as the Debugger API.

Some of the functions in the Debugger API are unique to the target system, some are unique to the development system, and some exist on both. At the end of the documentation for each function and structure, there is a section entitled Requirements that lists which static library you should link against.
