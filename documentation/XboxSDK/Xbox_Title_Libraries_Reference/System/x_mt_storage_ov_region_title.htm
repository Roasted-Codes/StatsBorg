Title-Persistent Data Storage Region

The title data region is used for persistent title data. Examples are updated player rosters, high score lists, new levels, persistent environment deformations, and so on. Persistent storage enables a high level of immersiveness in console games.

As opposed to user data, in which users typically decide when and what is stored (for example, explicitly choosing to save a game), persistent storage is created at various times by the game, either with or without the explicit knowledge of the user.

There is little defined format in this region. Unlike user data, where a defined heirarchy exists for the format and use of the space, this region is left mostly to the title developer to organize as desired. Because of this, the data in this region is opaque to the system and is managed by the user through the Xbox Dashboard in an all-or-nothing manner. The user has the option of deleting the title's entire persistent data region, presumably when they no longer play the game or are shelving it for a while. Otherwise, except as modified by the title, the data in the region remains intact.
