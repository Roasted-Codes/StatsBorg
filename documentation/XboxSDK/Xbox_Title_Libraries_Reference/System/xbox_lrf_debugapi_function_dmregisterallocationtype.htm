DmRegisterAllocationType

Registers a custom memory allocation type for use with the XbMemDump tool.

HRESULT DmRegisterAllocationType(
  LPCSTR   pszName,
  USHORT*  AllocationType,
  );

Parameters

pszName
Pointer to a string to use as a description for your custom allocation type. Because tracking custom data types is dependent on the AllocationType value (rather than on the name), you may assign duplicate string names to different custom allocation types.
AllocationType
Pointer that receives the allocation type number assigned to the new custom data type. Custom allocations are in the range 14–65535, inclusive.

Return Values

If DmRegisterAllocationType successfully registers the custom data type, it returns XBDM_NOERR. If there are no remaining custom data types available, DmRegisterAllocationType returns ERROR_NO_LOG_SPACE.

Requirements

Import Library: Use XbDm.lib
