SwitchToThread

The SwitchToThread function causes the calling thread to yield execution to another thread that is ready to run. The system software selects the thread to yield to.

BOOL SwitchToThread(VOID);

Parameters

This function has no parameters.

Return Values

If calling the SwitchToThread function causes the system software to switch execution to another thread, the return value is nonzero.

If there are no other threads ready to execute, the system software does not switch execution to another thread, and the return value is zero.

Remarks

The yield of execution is in effect for up to one thread-scheduling time slice. After that, the system software reschedules execution for the yielding thread. The rescheduling is determined by the priority of the yielding thread and the status of other threads that are available to run.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
