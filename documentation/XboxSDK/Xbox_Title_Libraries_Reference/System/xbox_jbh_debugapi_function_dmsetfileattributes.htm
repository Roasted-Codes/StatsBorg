DmSetFileAttributes

Sets the file attributes of a file on the Xbox console.

DMHRAPI DmSetFileAttributes(
  LPCSTR szFileName, 
  PDM_FILE_ATTRIBUTES pdmFileAttributes
  );

Parameters

szFileName
Name of the remote file.
pdmFileAttributes
Pointer to a DM_FILE_ATTRIBUTES structure.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

Passing a zero as the file attributes value in the Attributes field of the DM_FILE_ATTRIBUTES structure will not set the file attributes of that file to FILE_ATTRIBUTE_NORMAL. Instead, it signifies that the attributes flag for that file should not be changed. For example, it may be desirable to change the file times or size without changing the attributes flag.
