XSOUNDTRACK_DATA

Describes a given soundtrack.

typedef struct _XSOUNDTRACK_DATA {
  UINT uSoundtrackId;
  UINT uSongCount;
  UINT uSoundtrackLength;
  WCHAR szName[MAX_SOUNDTRACK_NAME];
} XSOUNDTRACK_DATA, *PXSOUNDTRACK_DATA;

Members

uSoundtrackId
The identifier of the soundtrack.
uSongCount
The number of songs in the soundtrack.
uSoundtrackLength
The length, in milliseconds, of the entire soundtrack.
szName[MAX_SOUNDTRACK_NAME]
A null-terminated string that is the name of the soundtrack.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
