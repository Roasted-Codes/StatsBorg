XFindFirstContent

Begins an enumeration of title content offerings that have been installed on the hard disk.

HANDLE XFindFirstContent(
    LPCSTR lpRootPathName,
    DWORD dwFlagFilter,
    PXCONTENT_FIND_DATA pFindContentData
);

Parameters

lpRootPathName
[in] Pointer to a null-terminated string that specifies the root directory of the data storage region to search. The parameter must contain only a drive letter, colon, and trailing backslash; a subdirectory cannot be specified.
dwFlagFilter
[in] Specifies filter flags that determine the offerings that are enumerated. Offerings that contain at least one of the flags in dwFlagFilter will be enumerated. To enumerate all content regardless of flags, specify zero for dwFlagFilter.
pFindContentData
[out] Pointer to the XCONTENT_FIND_DATA structure that receives information about the first content offering found.

Return Values

If the function succeeds, the return value is a search handle that can be used in subsequent calls to XFindNextContent.

If the function fails, the return value is INVALID_HANDLE_VALUE. To get extended error information, call GetLastError.

Remarks

XFindFirstContent opens a search handle and returns information about the first content offering found in the specified title persistent data region (T: or W:). The search handle can then be used in calls to XFindNextContent to continue enumeration of the installed content. When the search handle is no longer needed, close it with the XFindClose function to free up associated resources.

XFindFirstContent enumerates both offline and online offerings that have been installed on the hard disk. To enumerate content available on the online servers, use XOnlineOfferingEnumerate.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
