High-Resolution Timer

A counter is a general term used in programming to refer to an incrementing variable. The high-resolution performance counter provides high-resolution elapsed times.

The QueryPerformanceFrequency function is used to determine the frequency, in counts per second, of the high resolution timer.

The QueryPerformanceCounter function retrieves the current value of the high-resolution performance counter. By calling this function at the beginning and end of an operation to be timed and then adjusting for the frequency of the counter, a title can use this counter as a high-resolution timer.

For example, suppose that QueryPerformanceFrequency indicates that the frequency of the high-resolution performance counter is 50,000 counts per second. If the application calls QueryPerformanceCounter immediately before and immediately after an operation to be timed, the counter values might be 1500 counts and 3500 counts, respectively. These values would indicate that .04 seconds (2000 counts) elapsed while the operation was performed.
