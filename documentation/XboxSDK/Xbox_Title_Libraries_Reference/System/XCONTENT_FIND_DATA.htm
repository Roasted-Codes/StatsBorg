XCONTENT_FIND_DATA

Describes a content offering enumerated by the XFindFirstContent or XFindNextContent function.

typedef struct{
    WIN32_FIND_DATAA wfd;
    XOFFERING_ID qwOfferingId;
    DWORD dwFlags;
    CHAR szContentDirectory[MAX_PATH];
    WCHAR szDisplayName[MAX_CONTENT_DISPLAY_NAME];
} XCONTENT_FIND_DATA, *PXCONTENT_FIND_DATA;

Members

wfd
Specifies file-system information concerning the installation directory of the content offering. For more information, see WIN32_FIND_DATA.
qwOfferingId
The unique offering identifier of the installed content offering.
dwFlags
Title-specific flags describing the offering.
szContentDirectory
A null-terminated string that is the directory path where the content offering is installed.
szDisplayName
A null-terminated Unicode string that is the display name describing the content offering.

Remarks

XFindFirstContent and XFindNextContent enumerate installed content offerings and return, in an XCONTENT_FIND_DATA structure, the information for each offering found.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
