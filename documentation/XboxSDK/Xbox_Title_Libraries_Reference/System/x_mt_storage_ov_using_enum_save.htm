Enumerating Save Games

To enumerate available save games, use the XFindFirstSaveGame function. This function begins enumeration of the title's save games for a specified user data region, either on the hard disk or on a present memory unit. The valid user data regions that can be enumerated are represented by the following logical drives: U:\ for the title's user data region on the hard disk, F:\ through M:\ for the title's user data regions on mounted memory units, and W:\ if another title's user data region has been mounted with the XMountAlternateTitle function.

Enumeration is continued with a call to the XFindNextSaveGame function. The initial call to XFindFirstSaveGame and each additional call to XFindNextSaveGame returns one save game. For each save game enumerated, an XGAME_FIND_DATA structure is returned. This structure contains information detailing the name and location of the save game, as well as information about the save game directory.

Finally, when either the enumeration is complete or the title has found the save game being sought, a call must be made to XFindClose to free the small amount of state data maintained for the search.
