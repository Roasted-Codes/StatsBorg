XTHREAD_NOTIFICATION

The XTHREAD_NOTIFICATION structure is used in conjunction with XRegisterThreadNotifyRoutine to notify static libraries when a new thread is started or stopped.

typedef struct _XTHREAD_NOTIFICATION {
    LIST_ENTRY ListEntry;
    XTHREAD_NOTIFY_PROC pfnNotifyRoutine;
} XTHREAD_NOTIFICATION, *PXTHREAD_NOTIFICATION;

Members

ListEntry
Reserved member.
pfnNotifyRoutine
Pointer to the starting address of the function created for initializing or cleaning up a thread. XTHREAD_NOTIFY_PROC is defined as follows:
typedef VOID (WINAPI *XTHREAD_NOTIFY_PROC)(BOOL fCreate);

Remarks

This structure defines the procedure to call upon the creation or destruction of a program thread.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
