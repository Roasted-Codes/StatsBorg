DmClosePerformanceCounter

Releases a handle that was created with a call to DmOpenPerformanceCounter.

DMHRAPI DmClosePerformanceCounter(
	HANDLE hCounter
	);

Parameters

hCounter
The HANDLE.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h
Import Library: Use XbDm.lib
