TlsFree

The TlsFree function releases a thread local storage (TLS) index, making it available for reuse. 

BOOL TlsFree(
  DWORD dwTlsIndex   // TLS index
);

Parameters

dwTlsIndex
[in] Specifies a TLS index that was allocated by the TlsAlloc function.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero. To get extended error information, call GetLastError.

Remarks

If threads have allocated dynamic storage and used the TLS index to store pointers to this storage, they should free the storage before calling TlsFree. The TlsFree function does not free any dynamic storage that has been associated with the TLS index. It is expected that libraries call this function (if at all) only when they are being unloaded.

For a brief discussion of typical uses of the TLS functions, see the Remarks section of the TlsAlloc function.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
