Nickname Enumeration

To retrieve the list of nicknames, you must first call XFindFirstNickname. This function begins an enumeration of nicknames in the local nickname cache and returns the first nickname found, as well as an enumeration handle that can be used to continue retrieving nicknames. Calling XFindNextNickname with the returned handle will continue the enumeration and return the next nickname in the list. Each subsequent call to XFindNextNickname will return the next nickname from the list, until the entire list has been enumerated.

When beginning the enumeration, the fThisTitleOnly parameter of XFindFirstNickname specifies whether all nicknames in the cache should be returned, or only those nicknames previously added by the calling title should be returned.

The following code example enumerates the entire nickname cache and sends each nickname to the debugger for display:

//------------------------------------------------------------------------------ 
// enumnick.cpp
//
// Example use of the XFindFirstNIckname and XFindNextNickanme functions.
// Enumerate nicknames and send the output to the debugger.
//------------------------------------------------------------------------------ 

#include <xtl.h>


void main( void )
{
    WCHAR wszNick[MAX_NICKNAME];
    HANDLE hFind;
    

    // Start the enumeration and check for failure.
    hFind = XFindFirstNickname( FALSE, wszNick, MAX_NICKNAME );

    if( INVALID_HANDLE_VALUE == hFind )
    {
        OutputDebugStringW( L"XFindFirstNickname failed." );
    }
    else
    {
        // Display each nickname and ask for the next.
        do
        {
            OutputDebugStringW( wszNick );
            OutputDebugStringW( L"\n" );
        } while( XFindNextNickname( hFind, wszNick, MAX_NICKNAME ) );

        // Close the enumeration handle.
        XFindClose( hFind );
    }

    // Don't return from main!
    Sleep( INFINITE );
}
