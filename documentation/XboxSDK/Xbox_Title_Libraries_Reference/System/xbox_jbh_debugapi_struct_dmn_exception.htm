DMN_EXCEPTION

Contains information relevant to a DM_EXCEPTION notification type. For a complete list of notification types, see DmNotify.

typedef struct _DMN_EXCEPTION {
    DWORD ThreadId;
    DWORD Code;
    PVOID Address;
    DWORD Flags;
    DWORD Information[2];
} DMN_EXCEPTION, *PDMN_EXCEPTION;

Members

ThreadId
ID of the thread in which the exception occurred.
Code
Exception code for the exception that was raised.
Address
Address at which the exception occurred.
Flags
Contains DM_EXCEPT_NONCONTINUABLE to indicate a noncontinuable exception, or DM_EXCEPT_FIRSTCHANCE to indicate a first-chance exception.
Information
If Code does not equal STATUS_ACCESS_VIOLATION, then Information is undefined. If Code equals STATUS_ACCESS_VIOLATION, then Information[0] is: zero if the exception was raised due to a read; non-zero if the exception was raised due to a write. And Information[1] is the address that was referenced to cause the exception.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
