DM_D3DSTATE

The DM_D3DSTATE structure, returned by the DmGetD3DState function, contains the current Direct3D render state information.

typedef struct{
    DWORD cbSize;
    DWORD renderState[D3DRS_MAX];
    DWORD textureStageState[D3DTSS_MAXSTAGES*D3DTSS_MAX];
} DM_D3DSTATE, *DM_D3DSTATE;

Members

cbSize
Size, in bytes, of this structure.
renderState
Array of DWORD values that describe the current Direct3D render state.
textureStageState
Array of DWORD values that describe the current Direct3D texture stage state.

Requirements

Header: Declared in XboxDbg.h.
