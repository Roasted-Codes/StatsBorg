File I/O

A file is the basic unit of storage that enables a computer to distinguish one set of information from another. The topics in this section describe the file input and output (I/O) operations and the information provided by the file I/O functions.

These file I/O libraries provide access to individual files. For information about managing file sets and file locations, see the Xbox Storage Reference.
