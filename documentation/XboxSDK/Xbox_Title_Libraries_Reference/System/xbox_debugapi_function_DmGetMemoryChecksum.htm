DmGetMemoryChecksum

Calculates the checksum of a block of memory on the Xbox development kit.

DMHRAPI DmGetMemoryChecksum(
    const LPBYTE lpbAddr,
    DWORD cb,
    DWORD cbBlockSize,
    LPBYTE lpbBuf,
    LPDWORD pcbRet
);

Parameters

lpbAddr
Memory address on the Xbox console of the first byte of memory in the block. This address must be aligned on an 8-byte boundary, and it cannot point to code.
cb
Number of bytes on which to perform the checksum. This value must be a multiple of the value in cbBlockSize.
cbBlockSize
Number of bytes per block. This value must be a multiple of 8.
lpbBuf
Memory address on the Xbox development kit of the byte that immediately follows the last byte checked.
pcbRet
Pointer to a DWORD that receives the number of bytes actually checked.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

DmGetMemoryChecksum computes the checksum(s) on the Xbox, without transferring the contents of memory across the network. Each block of bytes has its own 32-bit CRC calculated separately. If you want to checksum the entire chunk of data as a whole, pass the same value for both cb and cbBlockSize.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.

Import Library: Use XboxDbg.lib.
