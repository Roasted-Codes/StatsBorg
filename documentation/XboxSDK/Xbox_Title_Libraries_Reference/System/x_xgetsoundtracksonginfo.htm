XGetSoundtrackSongInfo

The XGetSoundtrackSongInfo function retrieves information about a specified song within a soundtrack.

BOOL WINAPI XGetSoundtrackSongInfo(
  DWORD dwSoundtrackId,
  UINT uIndex,
  PDWORD pdwSongId,
  PDWORD pdwSongLength,
  PWSTR szNameBuffer,
  UINT uBufferSize
);

Parameters

dwSoundtrackId
[in] Soundtrack identifier of the soundtrack to query.
uIndex
[in] Index within the soundtrack of the song for which information will be retrieved.
pdwSongId
[out] Receives the unique identifier of the specified song.
pdwSongLength
[out] Receives the length, in milliseconds, of the specified song.
szNameBuffer
[out, optional] Pointer to a buffer that receives the name of the song. This parameter can be NULL if the song's name is not required.
uBufferSize
[in] If szNameBuffer is not NULL, specifies the size, in Unicode characters, of the buffer pointed to by szNameBuffer.

Return Values

This function returns TRUE if the data was returned successfully and FALSE otherwise.

Remarks

The XGetSoundtrackSongInfo function allows a title to randomly access songs from within a given soundtrack (to implement shuffle, random play, and so on). However, linearly incrementing the index is the most efficient way to enumerate songs because individual songs are batched in groups of seven within the database. As long as the index is within the same group of seven as the last call to this function, no additional read is necessary.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
