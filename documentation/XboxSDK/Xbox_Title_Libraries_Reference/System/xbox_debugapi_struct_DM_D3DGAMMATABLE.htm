DM_D3DGAMMATABLE

The DM_D3DGAMMATABLE structure, used by the DmGetGammaTable function, contains the current Direct3D render state information.

typedef struct{
    BYTE red[256];
    BYTE green[256];
    BYTE blue[256];
} DM_D3DGAMMATABLE, *DM_D3DGAMMATABLE;

Members

red
Holds the red values in the gamma table.
green
Holds the green values in the gamma table.
blue
Holds the blue values in the gamma table.

Requirements

Header: Declared in XboxDbg.h.
