XGetLanguage

The XGetLanguage function returns the current language for the console, as set by the user in the Xbox Dashboard.

DWORD XGetLanguage();

Parameters

None.

Return Values

The function returns one of the following values:

Value Language
XC_LANGUAGE_ENGLISH English
XC_LANGUAGE_JAPANESE Japanese
XC_LANGUAGE_GERMAN German
XC_LANGUAGE_FRENCH French
XC_LANGUAGE_SPANISH Spanish
XC_LANGUAGE_ITALIAN Italian
XC_LANGUAGE_KOREAN Korean
XC_LANGUAGE_TCHINESE Chinese (Traditional)
XC_LANGUAGE_PORTUGUESE Portuguese

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
