DmScreenShot

Dumps the contents of the Xbox console's screen to a .bmp file on the development system.

DMHRAPI DmScreenShot(
  LPCSTR filename
  );

Parameters

filename
Name of the .bmp file to place the screen capture.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
