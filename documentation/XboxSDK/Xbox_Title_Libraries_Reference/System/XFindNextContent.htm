XFindNextContent

Continues an enumeration of title content offerings that have been installed on the hard disk.

BOOL XFindNextContent(
    HANDLE hFindContent,
    PXCONTENT_FIND_DATA pFindContentData
);

Parameters

hFindContent
[in] A search handle returned by a previous call to XFindFirstContent.
pFindContentData
[out] Pointer to the XCONTENT_FIND_DATA structure that receives information about the next content offering found.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call GetLastError.

Remarks

XFindNextContent returns information about the next content offering found. Continue to enumerate content with repeated calls to XFindNextContent with the search handle and content data structure. When no more content is available, XFindNextContent will return FALSE and GetLastError will return ERROR_NO_MORE_FILES. When the search handle is no longer needed, close it with the XFindClose function to free up associated resources.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
