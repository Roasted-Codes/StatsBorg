DmStopProfiling

Ends collection of profiling data and then flushes any buffered data to the log file specified in the last call to DmStartProfiling.

HRESULT DmStopProfiling(void)

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

The DmStopProfiling function should not be confused with DmStopProfile. DmStopProfiling is used to stop sending profiling information to a log file; DmStopProfile is used to unregister a profiler interrupt handler.

Requirements
