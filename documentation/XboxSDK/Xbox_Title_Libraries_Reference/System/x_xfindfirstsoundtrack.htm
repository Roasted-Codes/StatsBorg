XFindFirstSoundtrack

The XFindFirstSoundtrack function begins an enumeration of the soundtracks on the media and returns the first soundtrack.

HANDLE WINAPI XFindFirstSoundtrack(
  PXSOUNDTRACK_DATA pSoundtrackData 	
);

Parameters

pSoundtrackData
[out] Pointer to an XSOUNDTRACK_DATA structure that receives information about the first soundtrack found.

Return Values

If the function succeeds, the return value is a search handle that can be used in subsequent calls to XFindNextSoundtrack.

If the function fails, the return value is INVALID_HANDLE_VALUE.

Remarks

XFindFirstSoundtrack opens a search handle and returns information about the first soundtrack found. The search handle can then be used in XFindNextSoundtrack to continue enumeration of the available soundtracks. When the search handle is no longer needed, close it using the XFindClose function.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
