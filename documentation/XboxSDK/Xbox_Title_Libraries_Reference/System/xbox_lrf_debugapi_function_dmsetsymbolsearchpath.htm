DmSetSymbolSearchPath

Sets the path that is searched for .pdb symbol files.

DMHRAPI DmSetSymbolSearchPath(
    LPCTSTR lpszPath
);

Parameters

lpszPath
[in] Pointer to a null-terminated string that specifies the new symbol search path.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
