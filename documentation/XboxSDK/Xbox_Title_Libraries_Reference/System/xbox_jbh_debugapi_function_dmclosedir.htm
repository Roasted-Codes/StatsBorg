DmCloseDir

Frees the memory allocated by DmWalkDir.

DMHRAPI DmCloseDir(
  PDM_WALK_DIR pWalkDir
  );

Parameters

pWalkDir
Pointer to the DM_WALK_DIR structure used by DmWalkDir.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
