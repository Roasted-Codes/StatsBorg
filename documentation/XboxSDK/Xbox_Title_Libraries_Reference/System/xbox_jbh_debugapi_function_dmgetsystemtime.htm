DmGetSystemTime

Retrieves the current system time of the console.

DMHRAPI DmGetSystemTime(
  LPSYSTEMTIME lpSysTime
  );

Parameters

lpSysTime
Pointer to an LPSYSTEMTIME structure to receive the time.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
