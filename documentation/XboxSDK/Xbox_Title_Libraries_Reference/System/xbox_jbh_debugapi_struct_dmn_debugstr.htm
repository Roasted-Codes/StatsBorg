DMN_DEBUGSTR

Contains information relevant to a DM_DEBUGSTR notification type. For a complete list of notification types, see DmNotify.

typedef struct _DMN_DEBUGSTR {
    DWORD ThreadId;
    DWORD Length;
    LPCSTR String;
} DMN_DEBUGSTR, *PDMN_DEBUGSTR;

Members

ThreadId
ID of the thread that triggered the notification.
Length
Length of the string containing the message.
String
String containing the message. The string may not be NULL terminated.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
