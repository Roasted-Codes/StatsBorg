DmCrashDump

Sends crash dump information to the development PC.

DMHRAPI DmCrashDump(
);

Return Values

Although the return type of this function is DMHRAPI, it does not return a value.

Remarks

After the crash dump is performed, the Xbox development console resets. Therefore, this function never returns.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.

Import Library: Use XboxDbg.lib.
