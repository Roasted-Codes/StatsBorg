XAutoPowerDownTimeRemaining

The XAutoPowerDownTimeRemaining function indicates whether the automatic power-down feature of the Xbox console has been enabled by the user, and, if so, how much time remains before the console will automatically power-down due to inactivity.

ULONG XAutoPowerDownTimeRemaining(VOID);

Parameters

This function has no parameters.

Return Values

If the automatic power-down feature has been enabled, the return value is the time, in milliseconds, remaining until the console will automatically power-down due to inactivity.

If the automatic power-down feature has not been enabled, the return value is 0xFFFFFFFF.

Remarks

The user must enable the automatic power-down feature by using the Xbox Dashboard. When that feature is enabled, the console will automatically power-down after an amount of time during which no activity occurs through any connected controller.

This function indicates whether the power-down feature has been enabled, and, if so, how much time remains before the console will perform the automatic power-down. User input on a controller currently open by the title (in other words, for which the title has an active handle) resets the power-down count.

This function can be used to detect an imminent power-down in order to automatically save the current game before the power-down occurs.

For example, a player could have been playing a game for a while, successfully fought several fights and performed several shows in different venues, only to fall asleep at some point. The game might be sitting at a prompt for hours until the console finally prepares to automatically power-down. The game could decide to save the current game before the power-down so that the player won't lose the progress they've made.

Note that for some games, it might not make sense to automatically perform a save game under these circumstances. A character in an RPG game, for example, might have died a thousand times while the player was sleeping and have no equipment or abilities remaining; auto-saving at this point would likely be a bad idea.
