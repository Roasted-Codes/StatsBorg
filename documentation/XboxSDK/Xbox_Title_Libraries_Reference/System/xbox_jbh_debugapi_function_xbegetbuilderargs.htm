XbeGetBuilderArgs

Reconstructs the command-line switches used with the ImageBld tool to generate the specified XBE.

DMHRAPI XbeGetBuilderArgs(
  LPCTSTR szXbe, 
  LPSTR   szArgs, 
  LPDWORD pcch
  );

Parameters

szXbe
The XBE file name.
szArgs
Buffer to receive the command-line switches.
pcch
Size of the szArgs buffer.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

The "/OUT" switch is not included in the reconstructed argument list. szXbe contains that information.

Requirements
