XGetAVPack

The XGetAVPack function returns a value indicating the type of AV pack currently connected to the Xbox console.

DWORD XGetAVPack();

Parameters

This function has no parameters.

Return Values

The function returns one of the following values to indicate the currently connected AV pack type:

Value Meaning
XC_AV_PACK_SCART Xbox Advanced SCART AV Pack
XC_AV_PACK_HDTV Xbox High Definition AV Pack
XC_AV_PACK_RFU Xbox RF Adapter
XC_AV_PACK_SVIDEO Xbox Advanced AV Pack
XC_AV_PACK_STANDARD Xbox Standard AV Cable

Remarks

When a new AV pack is inserted, the Xbox console may reboot in order to reset the audio and video signals. If the new and previous AV packs are of the same or similar type, a reboot is not required.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
