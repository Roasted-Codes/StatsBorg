DmSetupFunctionCall

Prepares for a function invocation.

DMHRAPI DmSetupFunctionCall(
    DWORD  tid
);

Parameters

tid
ID of the thread on which to call the function.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.
