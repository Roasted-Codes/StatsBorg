DmSetXboxName

Sets the machine name of the Xbox game system or selects a new console to connect to. (See Remarks below.)

DMHRAPI DmSetXboxName(
  LPCSTR  name
  );

Parameters

name
Pointer to a buffer containing the new name. The name cannot be longer than 255 characters.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

When called from the console, DmSetXboxName registers a new machine name for the console. When called from the development system, DmSetXboxName selects another console to connect to.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
