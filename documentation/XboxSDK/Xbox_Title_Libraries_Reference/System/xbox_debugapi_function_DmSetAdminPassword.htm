DmSetAdminPassword

Sets the password for the Administrator account.

DMHRAPI DmSetAdminPassword(
    LPCSTR szPassword
);

Parameters

szPassword
String that specifies the Administrator password.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.
