XGetVideoFlags

The XGetVideoFlags function returns the global video settings selected in the Dashboard.

DWORD XGetVideoFlags();

Parameters

None.

Return Values

The function will return one or more of the flags below ORed together:

Value Meaning
XC_VIDEO_FLAGS_WIDESCREEN The user selected wide screen.
XC_VIDEO_FLAGS_HDTV_720p The user selected 720p.
XC_VIDEO_FLAGS_HDTV_1080i The user selected 1080i.
XC_VIDEO_FLAGS_HDTV_480p The user selected 480p.
XC_VIDEO_FLAGS_LETTERBOX The user selected letterbox.
XC_VIDEO_FLAGS_PAL_60Hz The user selected 60Hz PAL (rather than the default of 50Hz).

Remarks

The XC_VIDEO_FLAGS_WIDESCREEN and XC_VIDEO_FLAGS_LETTERBOX flags are mutually exclusive.

For PAL output (XGetVideoStandard returns XC_VIDEO_STANDARD_PAL_I), the XC_VIDEO_FLAGS_WIDESCREEN, XC_VIDEO_FLAGS_LETTERBOX, and XC_VIDEO_FLAGS_PAL_60Hz flags are valid and may be set, but none of the HDTV flags will be set. Widescreen and letterbox formats are supported with PAL, but HDTV is not.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
