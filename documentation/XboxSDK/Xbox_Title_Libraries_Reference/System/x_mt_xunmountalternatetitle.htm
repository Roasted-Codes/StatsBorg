XUnmountAlternateTitle

The XUnmountAlternateTitle function is used to unmount another title's data storage region that was previously mounted with the XMountAlternateTitle function.

DWORD XUnmountAlternateTitle(
  CHAR chDrive               
);

Parameters

chDrive
[in] Drive letter to be unmounted. This parameter must the same as the drive letter returned in the pchDrive parameter in a previous call to XMountAlternateTitle.

Return Values

If the function succeeds, the return value is ERROR_SUCCESS.

If the function fails, the return value is an error code defined in winerror.h. The function does not use SetLastError to set the calling thread's last-error code.

Remarks

If another title's user data region on a memory unit has been mounted (with XMountAlternateTitle), XUnmountAlternateTitle must be called before the memory unit itself is unmounted with XUnmountMU.

An alternate title's data region will be unmounted automatically if the mounted user data region is located on a memory unit and that memory unit is removed from the memory unit slot.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
