DmSetMemory

Copies memory from the local PC to the Xbox console.

DMHRAPI DmSetMemory(
  LPVOID  lpbAddr, 
  DWORD   cb, 
  LPCVOID lpbBuf, 
  LPDWORD pcbRet
  );

Parameters

lpbAddr
Address of the first byte of memory on the Xbox console to be written to.
cb
Number of bytes to be copied.
lpbBuf
Address of the buffer on the local PC whose bytes will be copied to the Xbox console.
pcbRet
Pointer to a DWORD that will receive the number of bytes actually written. This value can be NULL. If pcbRet is NULL and the number of bytes requested does not equal the number of bytes received, DmSetMemory will return an error.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
