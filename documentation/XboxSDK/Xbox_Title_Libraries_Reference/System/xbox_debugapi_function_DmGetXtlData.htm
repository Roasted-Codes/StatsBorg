DmGetXtlData

Retrieves error status information about the Xbox Title Libraries.

DMHRAPI DmGetXtlData(
    PDM_XTLDATA pdmxd
);

Parameters

pdmxd
Pointer to a DM_XTLDATA structure. Upon return, this structure contains error information about the XTL.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.

Import Library: Use XboxDbg.lib.
