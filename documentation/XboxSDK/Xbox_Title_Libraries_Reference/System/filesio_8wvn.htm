FlushFileBuffers

The FlushFileBuffers function causes buffered data on the disk to be written to physical media. 

BOOL FlushFileBuffers(
  HANDLE hFile  // handle to file
);

Parameters

hFile
[in] Handle to an open file. The function flushes the disk's cache. The file handle must have GENERIC_WRITE access to the file.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero. To get extended error information, call GetLastError.

Remarks

The WriteFile function always writes data directly to the disk. The FlushFileBuffers function forces that disk to flush its internal cache to physical media.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
