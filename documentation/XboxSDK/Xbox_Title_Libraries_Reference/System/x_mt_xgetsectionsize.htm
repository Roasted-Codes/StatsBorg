XGetSectionSize

The XGetSectionSize function returns the size of the specified XBE image section.

DWORD XGetSectionSize(
  HANDLE hSection
);

Parameters

hSection
[in] Handle to the section to get the size of. This handle can be obtained by calling XGetSectionHandle.

Return Values

If the function succeeds, the return value is the size of the specified section.

If the function fails, the return value is zero. To get extended error information, call GetLastError.

Remarks

The section does not need to be currently loaded in order for this function to succeed.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
