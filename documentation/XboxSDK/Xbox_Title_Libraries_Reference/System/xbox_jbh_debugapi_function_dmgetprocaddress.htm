DmGetProcAddress

Retrieves the address of the specified procedure from the specified debugger extension.

DMHRAPI DmGetProcAddress(
  HANDLE hModule,
  LPCSTR szProcName,
  PVOID  *ppvRet
  );

Parameters

hModule
Handle of the debugger extension.
szProcName
Name of the function to retrieve.
ppvRet
Pointer to a pointer to receive the function address.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in xbdm.h
Import Library: Use xbdm.lib
