DmGetAltAddress

Resolves the currently selected console name to an IP address. The IP address retrieved is the title IP.

DMHRAPI DmGetAltAddress(
  LPDWORD lpdwAddress
  );

Parameters

lpdwAddress
Pointer to a DWORD to receive the IP address of the console. The address is returned in host byte order.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Debug IP versus Title IP

The development console is addressable via two separate IP addresses. When the console starts up, it requests two IP addresses from the DHCP server (or uses the two static IP addresses that you specified in the XDK Launcher). One IP is used for the debugger API's debug channel, and the other is used by the titles to connect multiplayer games.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
