XDeleteSaveGame

The XDeleteSaveGame function deletes a saved game from the specified user data storage region.

DWORD XDeleteSaveGame(
  LPCSTR lpRootPathName,            
  LPCWSTR lpSaveGameName             
);

Parameters

lpRootPathName
[in] Pointer to a null-terminated string that specifies the root directory of the user data storage region where the save game exists. The parameter must contain only a drive letter, colon, and trailing backslash; a subdirectory cannot be specified.
lpSaveGameName
[in] Pointer to a null-terminated string that specifies the name of the save game to be deleted. The name is limited to MAX_GAMENAME characters.

Return Values

If the function succeeds, the return value is ERROR_SUCCESS.

If the function fails, the return value is an error code defined in winerror.h. The function does not use SetLastError to set the calling thread's last-error code.

Remarks

The storage region specified in lpRootPathName can be the title's user data region on the hard disk (U:\), a mapped user data region for an alternate title (X:\), or the title's user data region of a mounted memory unit (F:\ through M:\).

XDeleteSaveGame deletes the directory containing the specified save game and all files inside that directory. Titles should call this function to remove a saved game rather than deleting the save game container directory directly.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
Unicode: Requires both Unicode and ANSI parameters.
