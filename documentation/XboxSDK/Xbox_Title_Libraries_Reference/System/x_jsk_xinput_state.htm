XINPUT_STATE

The XINPUT_STATE structure input data from an Xbox peripheral device. 

typedef struct _XINPUT_STATE
{
    DWORD dwPacketNumber;
    union
    {
        XINPUT_GAMEPAD Gamepad;
#ifdef DEBUG_MOUSE
        XINPUT_MOUSE   DebugMouse;
#endif //DEBUG_MOUSE
    };
} XINPUT_STATE, *PXINPUT_STATE;

Members

dwPacketNumber
The state packet number. The packet number indicates whether there have been any changes in the state of the device. If dwPacketNumber is the same in sequentially returned XINPUT_STATE structures, the device state has not changed.
Gamepad
An XINPUT_GAMEPAD structure containing the current state of a gamepad controller.
DebugMouse
An XINPUT_MOUSE structure containing debug mouse information. The structure contains the current state of the mouse buttons, and the relative mouse and wheel position changes since the last time the mouse state was queried.

Remarks

The dwPacketNumber member will be incremented only if the status of the device has changed since the device was last polled.

The DebugMouse member is available only if DEBUG_MOUSE was #defined before Xtl.h was #included.

See XINPUT_MOUSE for details about the format and use of the debug mouse data.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
