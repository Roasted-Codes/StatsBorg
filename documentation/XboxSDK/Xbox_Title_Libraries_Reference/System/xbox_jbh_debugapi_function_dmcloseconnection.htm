DmCloseConnection

Closes an existing connection to the console.

DMHRAPI DmCloseConnection(
  PDM_CONNECTION pConnection
  );

Parameters

pConnection
Pointer to a DM_CONNECTION structure that contains the connection information.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
