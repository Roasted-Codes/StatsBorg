DmSetXboxNameNoRegister

Selects a new console to connect to.

DMHRAPI DmSetXboxNameNoRegister(
  LPCSTR  name
  );

Parameters

name
Pointer to a buffer containing the new name. The name cannot be longer than 255 characters.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

This function can only be called on the development system. DmSetXboxNameNoRegister selects another console to connect to. This function is identical to DmSetXboxName (when called from the development system), but does not store the new Xbox name in the Microsoft® Windows® registry.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
