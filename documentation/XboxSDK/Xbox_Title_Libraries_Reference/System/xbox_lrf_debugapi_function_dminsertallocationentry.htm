DmInsertAllocationEntry

Adds a memory allocation type to the tracking database. The XbMemDump tool only reports allocation statistics for memory types that are in the tracking database.

HRESULT DmInsertAllocationEntry(
  PVOID   AllocPtr,
  SIZE_T  AllocSize,
  USHORT  AllocType
  );

Parameters

AllocPtr
Pointer to the block of allocated memory that should be tracked.
AllocSize
Size (in bytes) of the allocated memory to be tracked. This parameter is optional, and it allows memory allocation tracking to detect bad deallocations when the deallocation routine requires a size to be specified.
AllocType
Type of memory allocation. This is a number in the range 14-65535, inclusive.

Return Values

If DmInsertAllocationEntry successfully adds the memory type to the tracking database, it returns XBDM_NOERR. If the AllocType specified has not been assigned, DmInsertAllocationEntry returns E_INVALIDARG.

Requirements

Header: Declared in XbDm.h
Import Library: Use XbDm.lib
