XINPUT_FEEDBACK

The XINPUT_FEEDBACK structure describes the data outgoing to a device attached to the system. 

typedef struct _XINPUT_FEEDBACK
{
    XINPUT_FEEDBACK_HEADER Header;
    union
    {
        XINPUT_RUMBLE Rumble;
    };
} XINPUT_FEEDBACK, *PXINPUT_FEEDBACK;

Members

Header
Structure of the type XINPUT_FEEDBACK_HEADER that describes the current status of the instructions sent to the device.
Rumble
Structure of the type XINPUT_RUMBLE that sends rumble feedback information to the device.

Remarks

This structure is used by XInputSetState to provide feedback to a device attached to the system, such as a rumble pack. This structure will most likely change in the future as more devices become available for the system.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
