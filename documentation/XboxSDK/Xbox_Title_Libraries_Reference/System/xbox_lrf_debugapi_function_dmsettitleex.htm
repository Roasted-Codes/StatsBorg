DmSetTitleEx

Tells the system software which title to launch after the Xbox console is rebooted with the DMBOOT_WAIT flag. DmSetTitleEx may also be used to clear the launch title, restoring the Xbox console to its default status of booting to the XDK Launcher or the disc drive.

DMHRAPI DmSetTitleEx(
    LPCSTR szDir,
    LPCSTR szTitle,
    LPCSTR szCmdLine,
    DWORD dwFlags
);

Parameters

szDir
Path that contains the title.
szTitle
File name of the title. The file name muse be relative to the szDir parameter; absolute paths are not allowed.)
szCmdLine
Command line to pass to the title. See XGetLaunchInfo for more information.
dwFlags
Flags that control how reboots occur. Choose from the following two flags:
Flag Description
DMTITLE_PERSIST Makes the directory and title name supplied in the szDir and szTitle parameters persistent across reboots. This causes the indicated title to be launched each time the console reboots.
DMTITLE_UNPERSIST Clears the persistent directory and title name, which causes the Xbox console to boot to either the XDK Launcher or to the disc drive on subsequent reboots. The szDir and szTitle parameters should be set to NULL when using this flag.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

This function is meaningful only if the Xbox console was rebooted with the DMBOOT_WAIT flag. When the Xbox console is rebooted with DMBOOT_WAIT, it will wait 15 seconds after rebooting before launching a title. During this 15-second period (and only during this period), calling DmSetTitleEx will set the title to launch, or it will reset the Xbox console to its default state of starting either the XDK Launcher or the game disc drive.

Requirements

Header: Declared in XbDm.h and XboxDbg.h.

Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code..
