DmAllocatePoolWithTag

Allocates tagged debug memory in the upper 64 MB of RAM. This memory comes from the upper bank regardless of whether the /LIMITMEM ImageBld switch is used to simulate a 64-MB console on a 128-MB development console.

DMAPI PVOID __stdcall DmAllocatePoolWithTag(
  ULONG cb,
  ULONG tag
  );

Parameters

cb
Number of bytes to allocate.
tag
Tag that is associated with this allocation. This parameter is reserved and should always be set to 0.

Return Values

Returns a pointer to the allocated memory if successful. Otherwise, returns NULL.

Requirements

Header: Declared in XbDm.h
Import Library: Use XbDm.lib
