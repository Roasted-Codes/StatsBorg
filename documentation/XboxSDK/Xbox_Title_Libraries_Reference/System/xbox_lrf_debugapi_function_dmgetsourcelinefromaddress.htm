DmGetSourceLineFromAddress

Retrieves the source file and line number of a symbol definition for a specified address.

DMHRAPI DmGetSourceLineFromAddress(
    ULONG_PTR Address,
    CHAR *FileName,
    SIZE_T BufferSize,
    PULONG LineNumber
);

Parameters

Address
[in] Address for which a line number and source file should be determined.
FileName
[out] Pointer to a buffer that receives the string value of the file name where the symbol is located.
BufferSize
[in] Size, in bytes, of the FileName buffer. If FileName is not large enough to contain the entire file name, the file name will be truncated.
LineNumber
[out] Pointer to a ULONG variable that receives the line number on which the symbol is defined.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
