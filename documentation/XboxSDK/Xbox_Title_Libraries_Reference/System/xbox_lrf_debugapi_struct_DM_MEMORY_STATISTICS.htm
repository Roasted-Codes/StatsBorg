DM_MEMORY_STATISTICS

The DM_MEMORY_STATISTICS structure, returned by the DmQueryMemoryStatistics function, contains information about the number of memory pages currently devoted to different types of memory on the Xbox console.

typedef struct{
    DWORD cbSize;
    DWORD TotalPages;
    DWORD AvailablePages;
    DWORD StackPages;
    DWORD VirtualPageTablePages;
    DWORD SystemPageTablePages;
    DWORD PoolPages;
    DWORD VirtualMappedPages;
    DWORD ImagePages;
    DWORD FileCachePages;
    DWORD ContiguousPages;
    DWORD DebuggerPages;
} DM_MEMORY_STATISTICS, *PDM_MEMORY_STATISTICS;

Members

cbSize
Size, in bytes, of the DM_MEMORY_STATISTICS structure.
TotalPages
Total memory pages.
AvailablePages
Number of available memory pages.
StackPages
Number of pages of stack memory.
VirtualPageTablePages
Number of pages of virtual page table memory.
SystemPageTablePages
Number of pages of system page table memory.
PoolPages
Number of pages of pool memory.
VirtualMappedPages
Number of pages of virtual mapped memory.
ImagePages
Number of pages of image memory.
FileCachePages
Number of pages of file cache memory.
ContiguousPages
Number of pages of contiguous memory.
DebuggerPages
Number of pages of debugger memory.
