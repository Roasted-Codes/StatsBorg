DmUnloadSymbolsForModule

Unloads the debugging symbol table for a specified module.

DMHRAPI DmUnloadSymbolsForModule(
    ULONG_PTR BaseAddress
);

Parameters

BaseAddress
[in] Base address of the module.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
