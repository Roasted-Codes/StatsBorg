XbeGetLocalModulePath

Determines the local path of the specified module.

DMHRAPI XbeGetLocalModulePath(
  LPCTSTR szXbe, 
  LPCSTR  szModule, 
  LPSTR   szPath, 
  DWORD   cchPath
  );

Parameters

szXbe
XBE file name.
szModule
Module name.
szPath
Buffer to receive the path to the module.
cchPath
Size of the szPath buffer.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

This function is similar to DmGetModuleLongName, but it deals with xbe files that are located on the development system. szPath is equivalent to the module long name.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
