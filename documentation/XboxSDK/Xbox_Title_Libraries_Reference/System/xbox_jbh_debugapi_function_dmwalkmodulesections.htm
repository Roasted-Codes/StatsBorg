DmWalkModuleSections

Provides a way to examine information about each section of a loaded module. Call DmWalkModuleSections repeatedly to iterate over the list of loaded sections.

DMHRAPI DmWalkModuleSections(
  PDM_WALK_MODSECT *ppWalkModSect, 
  LPCSTR           lzModName
  PDMN_SECTIONLOAD pSecLoad
  );

Parameters

ppWalkModSect
Pointer to a structure that DmWalkModuleSections uses internally to iterate over the list of DMN_SECTIONLOAD structures. Set ppWalkModSect to NULL on the first call to DmWalkModuleSections; the function allocates memory to store a list of loaded module sections and sets the ppWalkModSect pointer to the address of beginning of that list.
lzModName
Name of the module to walk.
pSecLoad
Pointer to a DMN_SECTIONLOAD structure that contains information about the next loaded module section.

Return Values

XBDM_NOERR is returned if the call to DmWalkModuleSections is successful and the end of the list has not been reached. XBDM_ENDOFLIST is returned when the walk is complete. Otherwise, an error value is returned.

Remarks

If the return is not XBDM_NOERR or XBDM_ENDOFLIST, then an error has occurred and the walk should be discontinued. After using DmWalkModuleSections, you must free the memory allocated to store the list of sections by calling DmCloseModuleSections, passing it the value stored in the ppWalkModSect pointer.

The following example illustrates how to use DmWalkModuleSections and DmCloseModuleSections to iterate over the list of sections in a loaded module.

DMHRAPI error;
PDM_WALK_MODSECT *ppWalkModSect = NULL;
DMN_SECTIONLOAD sectLoad;

do
{
    error = DmWalkModuleSections(ppWalkModSect, lzModName, &sectLoad);

    // Examine the contents of sectLoad here.
}
while (error == XBDM_NOERR);

if (error != XBDM_ENDOFLIST)
{
    // Handle error.
}

DmCloseModuleSections(*ppWalkModSect);

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
