XMemFreeDefault

Default memory replacement function that deallocates memory.

VOID WINAPI XMemFreeDefault(
    PVOID pAddress,
    DWORD dwAllocAttributes
);

Parameters

pAddress
[in] Base address of the memory to be freed. This parameter must be the address returned by the XMemAlloc function when the memory was allocated.
dwAllocAttributes
[in] An XALLOC_ATTRIBUTES value that specifies the allocation attributes of the memory to be freed. These attributes should match those that were specified when the memory was allocated.

Return Values

This function does not return a value.

Remarks

XMemFreeDefault is the default memory callback function used by the XTL libraries for deallocations. All memory deallocations performed by the XTL libraries call XMemFree. If the title does not provide an overriden XMemFree, then XMemFreeDefault is used. A title can provide an overriden XMemFree, and within its implementation, call XMemFreeDefault in order to perform the appropriate actions. Thus the title can track calls without needing to fully implement XMemFree functionality.

Requirements

Header: Declared in Xbox.h; for include use xtl.h.
