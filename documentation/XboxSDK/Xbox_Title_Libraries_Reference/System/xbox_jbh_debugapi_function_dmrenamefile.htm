DmRenameFile

Renames or moves a file.

DMHRAPI DmRenameFile(
  LPCSTR szOldName, 
  LPCSTR szNewName
  );

Parameters

szOldName
Name of the file to rename.
szNewName
New name of the file.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
