DmReadFilePartial

Reads a block of data from a file.

DMHRAPI DmReadFilePartial(
    LPCSTR szFileName,
    DWORD dwFileOffset,
    LPBYTE pbDataBuffer,
    DWORD dwDataSize,
    LPDWORD lpdwBytesRead
);

Parameters

szFileName
String that specifies the name of the file.
dwFileOffset
Offset (in bytes) from the beginning of the file where reading should begin.
pbDataBuffer
Address of the destination data buffer.
dwDataSize
Size of the data buffer.
lpdwBytesRead
Address of a DWORD. Upon return, the DWORD contains the number of bytes actually read.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
