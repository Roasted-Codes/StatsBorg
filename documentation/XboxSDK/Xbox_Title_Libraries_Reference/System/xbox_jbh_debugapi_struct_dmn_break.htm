DMN_BREAK

Contains information relevant to a DM_BREAK notification type. For a complete list of notification types, see DmNotify.

typedef struct _DMN_BREAK {
    PVOID Address;
    DWORD ThreadId;
} DMN_BREAK, *PDMN_BREAK;

Members

Address
Address where the break occurred.
ThreadId
Thread in which the break occurred.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
