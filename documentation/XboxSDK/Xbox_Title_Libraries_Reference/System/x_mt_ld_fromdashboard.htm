LD_FROM_DASHBOARD

The LD_FROM_DASHBOARD structure is returned by the XGetLaunchInfo function to retrieve the launch information for a title that was launched by the Xbox Dashboard.

typedef struct _LD_FROM_DASHBOARD {
    DWORD dwContext;
    BYTE  Reserved[MAX_LAUNCH_DATA_SIZE - 4];
} LD_FROM_DASHBOARD, *PLD_FROM_DASHBOARD;

Members

dwContext
The title-provided context value that was passed to the Dashboard in an LD_LAUNCH_DASHBOARD structure when the title launched the Dashboard.
Reserved
Reserved; do not use.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
