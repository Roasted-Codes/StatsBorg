DmContinueThread

Resumes the execution a thread that has been stopped.

DMHRAPI DmContinueThread(
  DWORD dwThreadId, 
  BOOL  fException
  );

Parameters

dwThreadId
Thread ID.
fException
Continue exception processing for the exception that occurred.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

If fException is TRUE, then exception processing is continued for the exception that caused the thread to stop; if fException is FALSE, then the thread is continued as if the exception had never occurred. These behaviors are similar to the Win32 function ContinueDebugEvent, in which fException = TRUE is similar to DBG_EXCEPTION_NOT_HANDLED, and fException = FALSE is similar to DBG_CONTINUE.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
