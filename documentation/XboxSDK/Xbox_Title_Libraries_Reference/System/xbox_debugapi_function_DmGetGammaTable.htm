DmGetGammaTable

Retrieves the red, green, and blue components of the current gamma correction.

DMHRAPI DmGetGammaTable(
    LPDM_D3DGAMMATABLE lpGammaTable
);

Parameters

lpGammaTable
Pointer to a DM_D3DGAMMATABLE structure. Upon return, the structure contains the gamma information.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
