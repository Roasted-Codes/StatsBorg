DmGetThreadInfo

Retrieves information about a given thread.

DMHRAPI DmGetThreadInfo(
  DWORD dwThreadId, 
  PDM_THREADINFO pdmti
  );

Parameters

dwThreadId
Thread ID.
pdmti
Pointer to a DM_THREADINFO structure that receives information about the thread.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
