DmOpenPerformanceCounter

Assigns a HANDLE to a performance counter so that the debugging subsystem won't have to do a string lookup for every call to DmQueryPerformanceCounter on the Xbox console.

DMHRAPI DmOpenPerformanceCounter(
	LPCSTR szName, 
	HANDLE *phCounter
	);

Parameters

szName
Name of the performance counter.
phCounter
Pointer to receive the HANDLE.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h
Import Library: Use XbDm.lib
