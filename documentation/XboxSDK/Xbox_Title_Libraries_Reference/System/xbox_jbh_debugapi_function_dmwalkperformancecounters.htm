DmWalkPerformanceCounters

Provides a way to examine information about each performance counter. Call DmWalkPerformance Counters repeatedly to iterate over the list of performance counters.

DMHRAPI DmWalkPerformanceCounters(
  PDM_WALK_COUNTERS *ppWalkCounter, 
  PDM_COUNTINFO     pCounterInfo
  );

Parameters

ppWalkCounter
Pointer to a structure that DmWalkPerformanceCounters uses internally to iterate over the list of DM_COUNTINFO structures. Set ppWalkCounter to NULL on the first call to DmWalkPerformanceCounters; the function allocates memory to store a list of performance counters and sets the ppWalkCounter pointer to the address of beginning of that list.
pCounterInfo
Pointer to a DM_COUNTINFO structure that contains information about the next performance counter.

Return Values

XBDM_NOERR is returned if the call to DmWalkPerformanceCounters is successful and the end of the list has not been reached. XBDM_ENDOFLIST is returned when the walk is complete. Otherwise, an error value is returned.

Remarks

If the return is not XBDM_NOERR or XBDM_ENDOFLIST, then an error has occurred and the walk should be discontinued. After using DmWalkPerformanceCounters, you must free the memory allocated to store the list of performance counters by calling DmCloseCounters, passing it the value stored in the ppWalkCounter pointer.

The following example illustrates how to use DmWalkPerformanceCounters and DmCloseCounters to iterate over the list of performance counters.

DMHRAPI error;
PDM_WALK_COUNTERS *ppWalkCounter = NULL;
DM_COUNTINFO counter;

do
{
    error = DmWalkPerformanceCounters(ppWalkCounter, &counter);

    // Examine the contents of counter here.
}
while (error == XBDM_NOERR);

if (error != XBDM_ENDOFLIST)
{
    // Handle error.
}

DmCloseCounters(*ppWalkCounter);

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
