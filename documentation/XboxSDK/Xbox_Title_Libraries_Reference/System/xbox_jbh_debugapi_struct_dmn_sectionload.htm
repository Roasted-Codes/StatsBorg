DMN_SECTIONLOAD

Contains information relevant to a DM_SECTIONLOAD or DM_SECTIONUNLOAD notification type. For a complete list of notification types, see DmNotify.

typedef struct _DMN_SECTIONLOAD {
     char Name[MAX_PATH];
     PVOID BaseAddress;
     ULONG Size;
     ULONG Index;
     USHORT Flags;
} DMN_SECTIONLOAD, *PDMN_SECTIONLOAD;

Members

Name
Name of the section that was loaded.
BaseAddress
Address that the section was loaded to.
Size
Size of the section.
Index
Index of the section.
Flags
Additional information concerning the section. May be a combination of the following flags:
ValueMeaning
DMN_SECFLAG_LOADEDThe section is loaded.
DMN_SECFLAG_WRITEABLEThe section is writeable.
DMN_SECFLAG_UNINITIALIZEDThe section is uninitialized.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
