XMUPortFromDriveLetter

The XMUPortFromDriveLetter function allows the title to cull the port of a memory unit (MU) from an assigned drive letter.

DWORD XMUPortFromDriveLetter(
  CHAR chDrive                  
);

Parameters

chDrive
Drive letter currently assigned to the MU about which the title is inquiring.

Return Values

If the function succeeds, the return value is the port of the MU that is assigned to the specified drive letter.

If the function fails, the value will return –1.

Remarks

This function is normally used in conjunction with XMUSlotFromDriveLetter.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
