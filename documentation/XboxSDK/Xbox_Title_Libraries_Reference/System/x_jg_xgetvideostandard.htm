XGetVideoStandard

The XGetVideoStandard function returns the video standard set by the manufacturer.

DWORD XGetVideoStandard();

Parameters

None.

Return Values

The function will return one of the values below:

Value Meaning
XC_VIDEO_STANDARD_NTSC_M The Xbox was set by the manufacturer to use the NTSC standard for North America.
XC_VIDEO_STANDARD_NTSC_J The Xbox was set by the manufacturer to use the NTSC standard for Japan.
XC_VIDEO_STANDARD_PAL_I The Xbox was set by the manufacturer to use the PAL standard.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
