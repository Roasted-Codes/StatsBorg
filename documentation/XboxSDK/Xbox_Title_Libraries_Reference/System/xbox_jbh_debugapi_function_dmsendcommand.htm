DmSendCommand

Sends a command over an existing connection to the debugging subsystem.

DMHRAPI DmSendCommand(
  PDM_CONNECTION pConnection, 
  LPCSTR         szCommand, 
  LPSTR          szResponse, 
  LPDWORD        lpdwResponseSize
  );

Parameters

pConnection
Pointer to a DM_CONNECTION structure that contains information about the connection. If pConnection is NULL, a connection is opened to issue the command and immediately closed when the transaction is complete (regardless of connection sharing settings).
szCommand
The command in the form "handler!data data data", where "handler" is the name under which the command processor was registered (using DmRegisterCommandProcessor).
szResponse
Buffer that receives response data from the handler. An error code is embedded in the first three characters of the response data, followed by the remainder of the data. szResponse can be NULL, in which case no response data will be returned. If NULL, lpdwResponseSize should also be NULL.
lpdwResponseSize
Pointer to a DWORD that, on entry, contains the size of the response buffer specified in szResponse. On return, the DWORD contains the number of bytes written to the response buffer. This parameter can be NULL if and only if szResponse is NULL.

Return Values

If the function succeeds, it returns XBDM_NOERR. Otherwise, an error code defined in XboxDbg.h is returned. See Debugger API Error Codes.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
