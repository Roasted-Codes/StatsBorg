User Nicknames

During gameplay, users are often prompted to give their name or initials. This is done for a variety of reasons: the user may have achieved a high score, created a new save game, found a new character, or reached a specific milestone. In these or any other situations in which users need to provide a name or nickname, many users tend to gravitate toward either one specific nickname (Doughboy) or a small set (SupahKillah, Taximan, BustAMovah, DanceDanceGahd).

Because names or nicknames are needed in various places, and because the text often must be entered laboriously through a letter selector or virtual keyboard, the repeated entry can become frustrating and time consuming. The Xbox provides a mechanism designed to allow titles to guess at a likely set of nicknames and then present those choices to the player. This presentation enables the user to quickly choose a preferred nickname in many situations.

The nickname system on Xbox is implemented using the hard disk as a cooperative local cache of nicknames. Through simple functions, titles enumerate the current list of nicknames in the cache and add new ones to the list.

Note that this system of nicknames differs from online identities. This system behaves as a simple cache of data to aid in entering text. Online identities are separate names used for authentication and tracking (although a user's online name may be the same as the user's local nickname).

The cache of nicknames is stored as a Unicode file on the hard disk. Each nickname can be at most MAX_NICKNAME (32) characters, and there is a limit to the number of nicknames that can be in the cache (currently around 50, but that number may change), so there is a very small upper limit on the amount of hard disk space the cache will use.

Generally, when requesting a nickname from the user, titles should present to the user a list of nicknames from the cache, along with an option to add a new nickname. If the user doesn't find the desired nickname in the list, the user can specify a new nickname, and the title will add that nickname to the cache. Then the next time a name is required, the user's preferred nickname will appear in the list.
