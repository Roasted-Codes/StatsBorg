DmDeleteFile

Deletes a file.

DMHRAPI DmDeleteFile(
  LPCSTR szFileName, 
  BOOL   fIsDirectory
  );

Parameters

szFileName
Name of the file to delete.
fIsDirectory
TRUE if szFileName refers to a directory, FALSE otherwise.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
