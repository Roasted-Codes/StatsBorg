XGetContentInstallLocation

Determines the destination directory where a content offering will be (or was) installed.

BOOL XGetContentInstallLocation(
    DWORD dwTitleID,
    LPCSTR lpSourceMetadataFileName,
    LPSTR lpInstallDirectory
);

Parameters

dwTitleID
[in] Title identifier of the title that will (or did) install the content. If zero is specified for TitleID, the title identifier specified in the content metadata file is used.
lpSourceMetadataFileName
[in] Pointer to a null-terminated string that specifies the full path and file name of the source metadata file that describes the content.
lpInstallDirectory
[out] Pointer to a buffer that receives the installation directory for the content offering. A buffer of size MAX_PATH bytes will always be large enough to contain the returned directory.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call GetLastError.

Remarks

XGetContentInstallLocation is typically used by a title that is manually installing content rather than one that is relying on either a content download installer or a DVD-based package installer.

XGetContentInstallLocationFromIDs returns the same information based solely on the title identifier rather than referencing the content metadata file.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
