FILETIME

The FILETIME structure is a 64-bit value representing the number of 100-nanosecond intervals since January 1, 1601. 

typedef struct _FILETIME { 
    DWORD dwLowDateTime; 
    DWORD dwHighDateTime; 
} FILETIME, *PFILETIME;

Members

dwLowDateTime
Specifies the low-order 32 bits of the file time.
dwHighDateTime
Specifies the high-order 32 bits of the file time.

Remarks

To convert a FILETIME structure into a time that is easy to display to a user, use the FileTimeToSystemTime function.

It is not recommended that you add and subtract values from the FILETIME structure to obtain relative times. Instead, you should:

  • Copy the resulting FILETIME structure to a ULARGE_INTEGER structure.
  • Use normal 64-bit arithmetic on the ULARGE_INTEGER value.

The FATX file system supports the file creation, last access, and last write time values.

Not all file systems can record creation and last access time, and not all file systems record them in the same manner. For example, on FATX, create time, write time, and last access time all have a resolution of 2 seconds. Therefore, GetFileTime may not return the same file time information set using the SetFileTime function. FATX records times on disk using UTC rather than local file time.

The UDFX file system used with game discs (DVDs) stores a single time stamp for the entire volume, with a 100ns resolution (the same resolution used by GetSystemTimeAsFileTime).
