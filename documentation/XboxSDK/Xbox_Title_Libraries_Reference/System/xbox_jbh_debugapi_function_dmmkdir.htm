DmMkdir

Creates a directory on the Xbox.

DMHRAPI DmMkdir(
  LPCSTR szDirectoryName
  );

Parameters

szDirectoryName
String representation of the path to create.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.
