XLocateSignatureByIndex

Retrieves a content signature, by index, from the specified signature data.

BOOL XLocateSignatureByIndex(
    HANDLE hSignature,
    DWORD dwSignatureIndex,
    LPBYTE *ppbSignatureData,
    LPDWORD *pdwSignatureSize
);

Parameters

hSignature
[in] Handle to signature data opened with XLoadContentSignatures.
dwSignatureIndex
[in] Index of the signature to retrieve.
ppbSignatureData
[out] Pointer to an LPBYTE variable that receives the address of the specified signature.
pdwSignatureSize
[out] Pointer to a DWORD variable that receives the size (in bytes) of the signature pointed to by SignatureData.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call GetLastError.

Remarks

XLocateSignatureByIndex retrieves the signature at the specified index from the specified open signature data.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
