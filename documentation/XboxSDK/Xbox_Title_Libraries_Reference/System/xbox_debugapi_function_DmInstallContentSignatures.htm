DmInstallContentSignatures

Installs the file of signed metadata containing the content signature for the title. For more information, see Content Signatures.

DMHRAPI DmInstallContentSignatures(
    LPCSTR szSignatureFileName,
    DWORD dwTitleID
);

Parameters

szSignatureFileName
Name of the file of signed metadata containing the content signature for the title.
dwTitleID
Unique hexadecimal title identifier.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
