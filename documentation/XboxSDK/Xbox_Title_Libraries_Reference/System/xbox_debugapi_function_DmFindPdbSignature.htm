DmFindPdbSignature

Searches for and retrieves the PDB signature, if it exists in the specified module.

DMHRAPI DmFindPdbSignature(
    ULONG_PTR BaseAddress,
    PDM_PDB_SIGNATURE PdbSignature
);

Parameters

BaseAddress
[in] Base address of the module.
PdbSignature
[out] Address of a PDB signature. Upon return, this holds the PDB signature data, if the signature is present.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.

Import Library: Use XboxDbg.lib.
