XRemoveContent

Removes a content offering installed on the local hard disk.

BOOL XRemoveContent(
    LPCSTR lpDirectoryName
);

Parameters

lpDirectoryName
[in] Pointer to a null-terminated string that specifies the full directory path of the content to be removed.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To retrieve extended error information, call GetLastError.

Remarks

XRemoveContent removes all content in the specified directory as well as the directory itself.

Use XGetContentInstallLocationFromIDs to determine the directory where a specified content offering has been installed.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
