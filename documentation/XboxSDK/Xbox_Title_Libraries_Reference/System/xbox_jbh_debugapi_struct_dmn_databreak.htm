DMN_DATABREAK

Contains information relevant to a DM_DATABREAK notification type. For a complete list of notification types, see DmNotify.

typedef struct _DMN_DATABREAK {
    PVOID Address;
    DWORD ThreadId;
    DWORD BreakType;
    PVOID DataAddress;
} DMN_DATABREAK, *PDMN_DATABREAK;

Members

Address
Address where the break occurred.
ThreadId
Thread in which the break occurred.
BreakType
Type of break that occurred.
Value Meaning
DMBREAK_WRITE The address was written to.
DMBREAK_READWRITE The address was written to or read from.
DMBREAK_EXECUTE The address was executed.
DataAddress
Address of the data that triggered the break.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
