DmReceiveBinary

Receives binary data on a development PC from an existing connection to an Xbox.

DMHRAPI DmReceiveBinary(
  PDM_CONNECTION pConnection, 
  LPCVOID        pv, 
  DWORD          cb,
  LPDWORD        pcbRet
  );

Parameters

pConnection
Pointer to a DM_CONNECTION structure that contains information about the connection.
pv
Pointer to the buffer that will receive the binary data.
cb
Number of bytes to receive (the size of the buffer).
pcbRet
Number of bytes actually received. This parameter can be NULL, indicating that you don't care about the size of the returned data.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

DmReceiveBinary is a blocking call. It will wait indefinitely for an incoming packet. For information on how this function is used, see DmSendBinary.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
