Input Device Structures

The Xbox game system uses the following Xbox input device structures:

  • XDEVICE_PREALLOC_TYPE
  • XINPUT_CAPABILITIES
  • XINPUT_DEBUG_KEYQUEUE_PARAMETERS
  • XINPUT_DEBUG_KEYSTROKE
  • XINPUT_DEVICE_DESCRIPTION
  • XINPUT_FEEDBACK
  • XINPUT_FEEDBACK_HEADER
  • XINPUT_GAMEPAD
  • XINPUT_LIGHTGUN_CALIBRATION_OFFSETS
  • XINPUT_MOUSE
  • XINPUT_POLLING_PARAMETERS
  • XINPUT_RUMBLE
  • XINPUT_STATE
