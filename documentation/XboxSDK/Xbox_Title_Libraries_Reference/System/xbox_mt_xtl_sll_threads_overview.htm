Thread Overview

Each title is started with a single thread, often called the primary thread, but can create additional threads from any of its threads as desired. All threads share the title's virtual address space and system resources. In addition, each thread maintains exception handlers, a scheduling priority, and a set of structures the system will use to save the thread context until it is scheduled. The thread context includes the thread's set of machine registers and stack.
