DM_COUNTINFO

The DM_COUNTINFO structure is used by the DmWalkPerformanceCounters function to return information about performance counters.

typedef struct _DM_COUNTINFO {
    char Name[256];
    DWORD Type;
} DM_COUNTINFO, *PDM_COUNTINFO;

Members

Name
Name of the counter.
Type
Counter type. Can be one of the following values.
ValueMeaning
DMCOUNT_VALUEA value counter.
DMCOUNT_EVENTAn event rate counter.
DMCOUNT_PRATIOA progressive ratio counter.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; XboxDbg.lib for development system code.
