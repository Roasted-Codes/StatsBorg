DmIsFastCAPEnabled

Retrieves a flag indicating whether fast call–attribute profiling (FASTCAP) is enabled. or disabled.

DMHRAPI DmIsFastCAPEnabled(
);

Return Values

Returns XBDM_FASTCAPENABLED if FASTCAP is enabled. Otherwise, the function returns XBDM_UNDEFINED.

Requirements

Header: Declared in XbDm.h.

Import Library: Use XboxDbg.lib.
