XLoadSectionByHandle

The XLoadSectionByHandle function loads into memory the XBE image section represented by the specified handle.

PVOID XLoadSectionByHandle(
  HANDLE hSection                       
);

Parameters

hSection
[in] Handle to the section to load. This handle can be obtained by calling XGetSectionHandle.

Return Values

If the function succeeds, the return value is the memory address that the section was loaded at.

If the function fails, the return value is NULL. To get extended error information, call GetLastError.

Remarks

If the function succeeds, the specified image is loaded into memory and the section's reference count is incremented by one.

To unload the section, call XFreeSectionByHandle.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
