LD_FROM_DEBUGGER_CMDLINE

The LD_FROM_DEBUGGER_CMDLINE structure is used in the XGetLaunchInfo function to retrieve the launch information for a title that was launched by the debugger.

typedef struct _LD_FROM_DEBUGGER_CMDLINE {
  CHAR szCmdLine [MAX_LAUNCH_DATA_SIZE];
} LD_FROM_DEBUGGER_CMDLINE, *PLD_FROM_DEBUGGER_CMDLINE;

Members

szCmdLine
A null-terminated string that is the command line passed from the debugger.

Remarks

If a command line is specified in the debugger when launching or debugging a title, it will be available to the title in this structure.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
