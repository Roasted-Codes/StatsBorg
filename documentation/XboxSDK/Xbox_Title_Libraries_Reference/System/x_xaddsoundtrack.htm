XAddSoundtrack

The XAddSoundtrack function adds a new Microsoft® Windows Media® Audio (WMA) soundtrack to the Dashboard Music directory.

DWORD WINAPI XAddSoundtrack(
  LPCWSTR pszSoundtrackName,
  PUINT pdwSoundtrackId
);

Parameters

pszSoundtrackName
[in] Name of the soundtrack to add.
pdwSoundtrackId
[out] Pointer to a UINT variable that receives the soundtrack identifier of the created soundtrack.

Return Values

Returns ERROR_SUCCESS if successful, an error code defined in winerror.h otherwise. The function does not use SetLastError to set the calling thread's last-error code.

Remarks

The XAddSoundtrack function allows applications to add WMA soundtracks to a user's hard disk. Songs (background music) can then be added to the soundtracks and are available for playback by other applications, or directly through the Xbox Dashboard.

Requirements

Header: Declared in xbox.h; for include use xtl.h.
Import Library: Use xsndtrk.lib.
