HeapDestroy

The HeapDestroy function destroys the specified heap object. HeapDestroy decommits and releases all the pages of a private heap object, and it invalidates the handle to the heap. 

BOOL HeapDestroy(
  HANDLE hHeap   // handle to heap
);

Parameters

hHeap
[in] Specifies the heap to be destroyed. This parameter should be a heap handle returned by the HeapCreate function. Do not use the handle to the title's default heap returned by the GetProcessHeap function.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero. To get extended error information, call GetLastError.

Remarks

Titles can call HeapDestroy without first calling the HeapFree function to free memory allocated from the heap.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
