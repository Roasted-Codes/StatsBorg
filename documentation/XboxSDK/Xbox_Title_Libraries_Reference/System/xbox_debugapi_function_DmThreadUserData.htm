DmThreadUserData

Retrieves the programmer-specified data that was passed to a given thread.

DMHRAPI DmThreadUserData(
    DWORD tid,
    LPDWORD * ppdwData
);

Parameters

tid
[in] Unique identifier of a given thread.
ppdwData
[out] Address of a pointer to a DWORD. Upon return, this parameter holds the user data.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h.

Import Library: Use XboxDbg.lib.
