DmReceiveFile

Copies a file from the Xbox console to the local PC.

DMHRAPI DmReceiveFile(
  LPCSTR szLocalName, 
  LPCSTR szRemoteName
  );

Parameters

szLocalName
Path and file name on the local PC where the retrieved file is to be copied.
szRemoteName
Path and file name on the Xbox console of the file that is to be retrieved.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
