XFreeSectionByHandle

The XFreeSectionByHandle function frees from memory the XBE image section represented by the specified handle.

BOOL XFreeSectionByHandle(
  HANDLE hSection                       
);

Parameters

hSection
[in] Handle to the section to free. This handle can be obtained by calling XGetSectionHandle.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call GetLastError.

Remarks

The section must have previously been loaded with a call to XLoadSectionByHandle or XLoadSection.

If the function succeeds, it decrements the specified section's reference count by one. If the reference count is decremented to zero, the section is unloaded from memory.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
