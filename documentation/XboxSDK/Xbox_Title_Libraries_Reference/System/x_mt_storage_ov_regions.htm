Xbox Storage Regions

The Xbox storage system compartmentalizes the total storage space on the console into a number of separate conceptual data storage regions. The system implements appropriate policies for caching, visibility, and space limitations for each specific region.

Note that the concept of storage regions is separate from that of actual partitions on the hard disk. Some regions may span multiple partitions. A single partition may contain several regions. Other regions (such as those defined for the memory units) may be external to the hard disk or outside of the file system altogether.

Titles interact with four data storage regions on the Xbox:

  • User Data Storage Region
  • Title-Persistent Data Storage Region
  • Publisher Common Data Storage Region
  • Utility Data Storage Region
