XUnmountMU

The XUnmountMU function unmounts a memory unit (MU) previously mounted with XMountMU.

DWORD XUnmountMU(
  DWORD dwPort,                  
  DWORD dwSlot                   
);

Parameters

dwPort
[in] Port that the controller containing the MU is connected to on the Xbox console.
dwSlot
[in] Slot on the controller containing the MU.

Return Values

If the function succeeds, the return value is ERROR_SUCCESS.

If the function fails, the return value is an error code defined in Winerror.h. The function does not use SetLastError to set the calling thread's last-error code.

Remarks

This function will succeed only for a memory unit that has been previously mounted with XMountMU.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
