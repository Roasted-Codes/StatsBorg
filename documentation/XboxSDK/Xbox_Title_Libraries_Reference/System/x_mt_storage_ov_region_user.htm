User Data Storage Region

The user data storage region is used to store all user data on the system. This data consists of user save games, customizations, settings, and other user-managed information. User data is stored both on the Xbox hard disk and on the memory units. The user data storage region is the only region that exists on the memory units.

The data in the user region is organized into a specific hierarchy so that the data can be easily managed by the user from the game title or the Xbox Dashboard. Titles use this hierarchy to provide structure and control to the stored save games.

Unlike other data regions that are always present, there will be a varying number of user data regions available during the game session. In addition to the hard disk user data region, up to eight other memory unit user data regions may be available.
