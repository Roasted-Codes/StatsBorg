DmIsSecurityEnabled

Queries whether secure connections are enabled or disabled.

DMHRAPI DmIsSecurityEnabled(
    LPBOOL pfEnabled
);

Parameters

pfEnabled
Address of a BOOL value. Upon return, the BOOL value contains TRUE if security is enabled or FALSE if not.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XbDm.h, XboxDbg.h.

Import Library: Use XboxDbg.lib.
