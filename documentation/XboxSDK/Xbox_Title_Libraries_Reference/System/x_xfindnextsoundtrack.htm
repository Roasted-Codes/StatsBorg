XFindNextSoundtrack

The XFindNextSoundtrack continues the enumeration of soundtracks begun with XFindFirstSoundtrack.

BOOL WINAPI XFindNextSoundtrack(
  HANDLE hFindHandle,
  PXSOUNDTRACK_DATA pSoundtrackData
);

Parameters

hFindHandle
[in] A search handle returned by a previous call to the XFindFirstSoundtrack function.
pSoundtrackData
[out] Pointer to an XSOUNDTRACK_DATA structure that receives data describing the next soundtrack in the list.

Return Values

The function returns TRUE if there are additional soundtracks to enumerate and FALSE otherwise.

Remarks

When the search handle is no longer needed, close it using the XFindClose function.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
