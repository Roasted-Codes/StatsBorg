LocalUnlock

The LocalUnlock function has no effect on memory blocks allocated as fixed; it simply returns TRUE.

Note  All Globalxxx and Localxxx memory management functions use the title's default heap, and provide less flexibility than the heap functions. Titles should use the heap functions instead.

BOOL LocalUnlock(
  HLOCAL hMem   // handle to local memory object
);

Parameters

hMem
[in] Handle to the memory block. This handle is returned by either the LocalAlloc or LocalReAlloc function.

Return Values

The return value is TRUE.

Remarks

Memory allocated with the global or local memory functions is always fixed. Thus LocalUnlock does nothing and always returns TRUE.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
