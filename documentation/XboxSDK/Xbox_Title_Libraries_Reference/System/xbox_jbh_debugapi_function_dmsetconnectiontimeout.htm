DmSetConnectionTimeout

Sets connection and conversation timeouts for a debug connection.

DMHRAPI DmSetConnectionTimeout(
  DWORD dwConnectTimeout, 
  DWORD dwConversationTimeout
  );

Parameters

dwConnectTimeout
Time, in milliseconds, before a connection attempt will timeout with XBDM_CANNOTCONNECT.
dwConversationTimeout
Time, in milliseconds, before an attempt to receive data across an established connection will timeout and close the connection.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
