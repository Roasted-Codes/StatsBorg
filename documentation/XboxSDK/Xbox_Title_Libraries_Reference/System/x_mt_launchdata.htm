LAUNCH_DATA

The LAUNCH_DATA structure is used in the XGetLaunchInfo and XLaunchNewImage functions to retrieve and set a title's launch information.

typedef struct _LAUNCH_DATA {
  BYTE Data [MAX_LAUNCH_DATA_SIZE];
} LAUNCH_DATA, *PLAUNCH_DATA;

Members

Data
Contains launch data for the title. The format of this data can vary; see Remarks below.

Remarks

When XLaunchNewImage is called to launch another Xbox executable, the Data member of the LAUNCH_DATA structure can be filled with data in any format desired by the title. The target .xbe must belong to the same title (have the same title identifier) as the launching .xbe so that they will both know the format of the data that is passed. The size of MAX_LAUNCH_DATA may change in the future, but is currently defined as 3 KB.

When a title is launched by the Dashboard, the title's launch data is an LD_FROM_DASHBOARD structure. When a title is launched by the debugger, the title's launch data is an LD_FROM_DEBUGGER_CMDLINE structure. See XGetLaunchInfo for more information.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
