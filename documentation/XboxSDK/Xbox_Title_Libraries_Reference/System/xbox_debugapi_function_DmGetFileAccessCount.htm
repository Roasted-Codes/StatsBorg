DmGetFileAccessCount

Retrieves a file's access count.

DMHRAPI DmGetFileAccessCount(
    LPDWORD lpdwFileAccessCount
);

Parameters

lpdwFileAccessCount
[out] Address of a DWORD. Upon return, the DWORD holds the file access count.

Return Values

Returns XBDM_NOERR if was successful, an error value otherwise.

Remarks

The debugger keeps a file access count that it increments each time it receives a file, sets its size, sets its attributes, renames it, deletes it, and so forth. This function retrieves the file's access count.

Requirements

Header: Declared in XbDm.h.

Import Library: Use XboxDbg.lib.
