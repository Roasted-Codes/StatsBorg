DmSendFile

Copies a file from the local PC to the Xbox console.

DMHRAPI DmSendFile(
  LPCSTR szLocalName, 
  LPCSTR szRemoteName
  );

Parameters

szLocalName
Path and file name on the local PC where the source file is located.
szRemoteName
Path and file name on the Xbox console where the local file will be copied to.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Requirements

Header: Declared in XboxDbg.h
Import Library: Use XboxDbg.lib
