XFindClose

The XFindClose function closes the specified search handle created by XFindFirstSaveGame, XFindFirstNickname, or XFindFirstSoundtrack.

BOOL XFindClose(
  HANDLE hFind                       
);

Parameters

hFind
[in] Search handle. This handle must have been previously opened by the XFindFirstSaveGame, XFindFirstNickname, or XFindFirstSoundtrack function.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call GetLastError.

Remarks

After the XFindClose function is called, the handle specified by the hFind parameter cannot be used in subsequent calls to XFindNextSaveGame, XFindNextNickname, XFindFirstSoundtrack, or XFindClose.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
