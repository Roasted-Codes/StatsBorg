XGetDeviceEnumerationStatus

The XGetDeviceEnumerationStatus function determines whether the system is in the process of enumerating connected devices.

DWORD XGetDeviceEnumerationStatus();

Parameters

None.

Return Values

If no enumeration is in progress (all connected devices have been enumerated), the return value is XDEVICE_ENUMERATION_IDLE.

If an enumeration is in progress (as a result of device insertions or removals, or a call to XInitDevices), the return value is XDEVICE_ENUMERATION_BUSY.

Remarks

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.

Import Library: Use Xapilib.lib.
