lstrlen

The lstrlen function returns the length in bytes (ANSI version) or characters (Unicode version) of the specified string (not including the terminating null character). 

int lstrlen(
  LPCTSTR lpString   // string to count
);

Parameters

lpString
[in] Pointer to a null-terminated string.

Return Values

The return value specifies the length of the string, in TCHARs. This refers to bytes for ANSI versions of the function or characters for Unicode versions.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
Unicode: Implemented as both Unicode and ANSI versions.
