DmHaltThread

Sends a request to the debugging subsystem that the specified thread break as soon as possible.

DMHRAPI DmHaltThread(
  DWORD dwThreadId, 
  );

Parameters

dwThreadId
ID of the thread to be halted. Send 0 as the thread ID to have the debugging subsystem select a thread to break into.

Return Values

Returns XBDM_NOERR if successful, an error value otherwise.

Remarks

A DM_BREAK debugging event occurs as if the thread had hit a breakpoint.

Requirements

Header: Declared in XbDm.h and XboxDbg.h
Import Library: Use XbDm.lib for target system code; and XboxDbg.lib for development system code.
