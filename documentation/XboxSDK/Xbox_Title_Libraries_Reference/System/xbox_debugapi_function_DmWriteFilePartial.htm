DmWriteFilePartial

Writes a block of data to a file.

DMHRAPI DmWriteFilePartial(
    LPCSTR szFileName,
    DWORD dwFileOffset,
    LPBYTE pbDataBuffer,
    DWORD dwDataSize,
    LPDWORD lpdwBytesWritten
);

Parameters

szFileName
String that specifies the name of the file.
dwFileOffset
Contains the offset (in bytes) from the beginning of the file where writing should begin.
pbDataBuffer
Holds the address of the source data buffer.
dwDataSize
Specifies the size of the data buffer.
lpdwBytesWritten
Contains the address of a DWORD. Upon return, the DWORD contains the number of bytes actually written.

Return Values

Returns XBDM_NOERR if successful. Otherwise, this function returns an error value.

Requirements

Header: Declared in XboxDbg.h.

Import Library: Use XboxDbg.lib.
