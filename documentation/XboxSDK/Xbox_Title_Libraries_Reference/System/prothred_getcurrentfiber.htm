GetCurrentFiber

The GetCurrentFiber macro returns the address of the current fiber.

PVOID GetCurrentFiber(VOID);

Parameters

This macro has no parameters.

Return Values

The return value is the fiber address of the currently running fiber.

Remarks

The CreateFiber and ConvertThreadToFiber functions return the fiber address when a fiber is created. The GetCurrentFiber macro retrieves that address.

Requirements

Header: Declared in Winbase.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
