XDEVICE_PREALLOC_TYPE

The XDEVICE_PREALLOC_TYPE structure is used in the XInitDevices function to indicate a device type and the maximum number of devices of that type that will be used simultaneously by the title.

typedef struct _XDEVICE_PREALLOC_TYPE {
    PXPP_DEVICE_TYPE DeviceType;
    DWORD            dwPreallocCount;
} XDEVICE_PREALLOC_TYPE, *PXDEVICE_PREALLOC_TYPE;

Members

DeviceType
Specifies a device type. DeviceType must be of one the following values:
ValueDescription
XDEVICE_TYPE_GAMEPADAn Xbox Controller.
XDEVICE_TYPE_MEMORY_UNITAn Xbox Memory Unit.
XDEVICE_TYPE_VOICE_MICROPHONEXbox Communicator microphone.
XDEVICE_TYPE_VOICE_HEADPHONEXbox Communicator headphone.
XDEVICE_TYPE_DEBUG_KEYBOARDA debug keyboard. Do not use in shipping titles. See Remarks below.
XDEVICE_TYPE_DEBUG_MOUSEA debug mouse. Do not use in shipping titles. See Remarks below.
dwPreallocCount
Specifies the maximum number of devices of type DeviceType that the title will simultaneously use. The system allocates the resources necessary to allow this number of devices to be used at the same time.

Remarks

To use XDEVICE_TYPE_DEBUG_KEYBOARD, #define DEBUG_KEYBOARD before including Xtl.h. Do not use XDEVICE_TYPE_DEBUG_KEYBOARD in shipping retail game titles.

To use XDEVICE_TYPE_DEBUG_MOUSE, #define DEBUG_MOUSE before including Xtl.h. Do not use XDEVICE_TYPE_DEBUG_MOUSE in shipping retail game titles.

Requirements

Header: Declared in Xbox.h, for include use Xtl.h.
Import Library: Use Xapilib.lib
