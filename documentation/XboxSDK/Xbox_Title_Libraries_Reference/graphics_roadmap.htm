Graphics Libraries

This section provides information about the libraries used for DirectX graphics on Xbox. This section also provides a roadmap describing how to best use those libraries.

  • Libraries
  • Graphics Roadmap

Libraries

Xbox has three types of libraries that can be linked to:

  • Direct3D
  • Direct3DX
  • Xgraphics

Direct3D

Direct3D is the workhorse of DirectX graphics. It provides functions for drawing, manipulating textures, surfaces, push-buffers, lights, resource management, and much more.

There are five import libraries that your title can link to:

  • D3d8.lib
  • D3d8ltcg.lib
  • D3d8d.lib
  • D3d8i.lib
  • D3d8-xbox.lib

For release builds, the only approved libraries are D3d8.lib and D3d8ltcg.lib. To take advantage of Link Time Code Generation, titles need to link to D3d8ltcg.lib.

D3d8d.lib is a debug version of the Direct3D library.

D3d8i.lib provides a version of the Direct3D library that can be used for performance tuning. Software-based performance counters are enabled when you link to this library. For more information about Direct3D performance counters, see Direct3D Performance Counters.

D3d8-xbox.lib is a version of Xbox's Direct3D library that, in conjunction with d3d8-xbox.h, can be used on the PC to create push-buffers at game author time. It functions as the Xbox's Direct3D implementation does when D3D__NullHardware is set to TRUE. This library behaves identically to the D3d8d.lib, except that nothing shows up on the screen and no GPU event callbacks are done. In particular, Direct3DCreate8 and CreateDevice are called as they are on the PC, and push-buffers may be recorded using BeginPushBuffer/EndPushbuffer.

Direct3DX

Direct3DX provides utility functions that allow rapid prototyping of your game. It has functions to load resources from files, manipulate meshes, use fonts, and do many other useful things. Many functions in Direct3DX can cause performance problems. As a result, use of this library in a retail game is not recommended.

There are two import libraries that your title can link to:

  • D3dx8.lib
  • D3dx8d.lib

D3dx8.lib is the only approved version of the library that release versions of your title can link to. D3d8xd.lib is the debug version of the library.

Xgraphics

The Xgraphics library provides you with functions for assembling, splicing, and comparing shader programs, setting resource headers, swizzling and unswizzling textures, and performing many other tasks. It also provides a math library that is optimized for use on the Xbox. In many instances (matrix mulitplying, for example), the Xgraphics math library performs the comparable operation three times faster than Direct3DX.

  • Xgraphics.lib
  • Xgraphicsltcg.lib
  • Xgraphicsd.lib

For release builds, the only approved libraries are Xgraphics.lib and Xgraphicsltcg.lib. To take advantage of Link Time Code Generation, titles need to link to Xgraphicsltcg.lib.

Xgraphicsd.lib is is the debug version of the library.
