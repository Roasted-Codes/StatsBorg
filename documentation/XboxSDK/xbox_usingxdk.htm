Getting Started

The Getting Started section provides instructions for setting up and using the Xbox Development Kit (XDK) and your Xbox console and development system. It discusses how to build, run, and debug Xbox executables (XBEs), and it offers samples you will need to develop Xbox games.

The following topics describe the contents and how to use the XDK:

  • Setting Up Your Console and Development System
  • Working with the Xbox Executable Model
  • XDK Launcher
  • Samples
