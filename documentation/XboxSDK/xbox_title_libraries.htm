Xbox Title Libraries Reference

The Xbox Title Libraries (XTL) is a collection of static libraries containing a wide variety of functions used by game titles to fully utilize the power of the Xbox hardware. The XTL contains the code that titles use to interface with the Xbox console to perform graphics, audio, networking, and system tasks. This section of the XDK provides reference information for XTL functions and services.

The XTL includes the following libraries:

  • Graphics: Services for programming mind-blowing Xbox graphics.
  • Audio: Ear-popping audio support.
  • System: System-level services such as I/O, memory management, and threading.
  • Xbox Live and Networking: Routines for Xbox Live support, the Xbox Secure Network Library, and Microsoft® Windows® Sockets.
  • Streaming Media: The Xbox Media Objects and Xbox Video Object libraries.
  • C Run-Time: The C run-time libraries supported by Xbox.
