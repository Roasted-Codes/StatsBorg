Developing for Xbox

The Developing for Xbox section provides programmatic guidelines for maximizing the potential of the Xbox Development Kit (XDK) to create world-class titles for the Xbox. Rather than discussing how each facet of the XDK works, this section details information on techniques for optimizing performance, understanding the limitations of the hardware and software, and identifying areas where developer solutions and internally developed tools might be of great use to game developers. The Developing for Xbox section will also include suggestions for meeting certification and playtesting requirements from a development standpoint, as opposed to a design standpoint, which is beyond the scope of this documentation.

This section currently covers the following topics:

  • Xbox Software Architecture
  • Reference User Interface
  • Using Xbox Peripherals
  • XBE Sections
  • Developing Audio
  • Xbox Game Certification
  • Using Visual Studio .NET

Note   Some of the content in this section is based on information available previously through downloadable specs, developer conferences, and in-person presentations. As a result, the Developing for Xbox section will continue to grow with each iteration of the SDK.
