The Xbox Development Kit (XDK) for the Xbox® video game system from Microsoft® provides the tools, utilities, samples, and documentation needed to develop Xbox games. This XDK Help describes these tools, utilities, and samples and includes conceptual and reference information for the Xbox Title Libraries (XTL).

The XDK Help documentation is divided into the following additional sections:

Getting Started provides instructions for setting up and using the XDK and your Xbox console and development system, and documentation for the utilities and samples included with the XDK.

Developing for Xbox offers a step-by-step walk-through of the architecture, design, concepts, tools, code, and libraries involved in developing an Xbox game title.

Introduction to Xbox Live provides a quick overview of Xbox Live, an online service created by Microsoft to enhance the functionality of Xbox game titles.

Xbox Title Libraries Reference includes detailed reference material for the Xbox Title Libraries, their functions, structures, and other development concepts.

Xbox Tools gathers information on a wide array of tools available to assist the development process on the Xbox.

White Papers presents additional technical articles with detailed coverage of a variety of Xbox development topics.

Glossary defines common technical terms used in the XDK Help documentation.
