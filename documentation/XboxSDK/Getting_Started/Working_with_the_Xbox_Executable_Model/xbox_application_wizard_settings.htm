Xbox Application Settings

Use this page of the wizard to set options for the Xbox project.

Application type

Creates the specified application type.

Option Description
Xbox Game Creates an Xbox game title.
Xbox Static Library Creates an Xbox static library.

Additional options

Defines the support and options for the application, depending on its type.

Option Description
Empty project Specifies that the project files are blank. If you have a set of source code files (such as .cpp files, header files, and so on) and want to create a project in the Visual C++ development environment, you must first create a blank project, then add the files to the project.

This selection is unavailable for Xbox static library projects.

Precompiled header Specifies that the static library project uses a pre-compiled header.

This selection is unavailable for Xbox game title projects.
