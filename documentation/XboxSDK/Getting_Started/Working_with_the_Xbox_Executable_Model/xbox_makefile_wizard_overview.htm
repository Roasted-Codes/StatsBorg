Xbox Makefile Wizard Overview

The Xbox Makefile Application Wizard enables you to create an Xbox project for use with code that you normally build using command-line tools. With an Xbox makefile project, you can take advantage of useful VS.NET features, such as the debugger, when developing your game, but still compile the project using external command-line tools.

The default makefile project defines only the name of the compiled Xbox executable. To change the executable name and other options, select the Application Settings tab in the left column of the wizard and make the desired changes.
