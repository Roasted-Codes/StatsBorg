Xbox Tutorials

This section provides information regarding the tutorials provided with this release of the XDK.

Building

The tutorials are copied to your development system when you install the XDK. They are located in "Samples\Xbox\Graphics\Tutorials" under the directory in which you installed the XDK. Select a tutorial from that directory by selecting the appropriate subdirectory and opening the Developer Studio Project (*.dsp) file contained within.

The tutorials contain detailed source code comments describing what each section of the code does.

Running

To see the tutorial code in action, you will need to build the project and copy the resulting XBE from the debug directory to the target system. See Building an XBE and Running and Debugging an XBE for more information about building, transferring, and running XBE files.

Media

The Textures and Meshes tutorials rely on media files that must be copied to the target system. When you build the tutorials, these files will be automatically copied over to the target system (via a call to xbcp in the post-build settings) along with your XBE. Be sure to specify your target system (using "xbcp -x [YourXboxName]") before building any samples or tutorials.

The Textures tutorial needs "banana.bmp". The Meshes tutorial needs "tiger.x" and "tiger.bmp".

The Tutorials

The tutorials in this section include:

TutorialDescription
CreateDevice Shows how to create a Microsoft® Direct3D® device and clear the screen to a solid color.
Vertices Shows how to render a single triangle.
Matrices Shows how to rotate 3D geometry using matrices.
Lighting Shows how to add dynamic lighting to your 3D scene.
Textures Shows how to add textures to your 3D geometry to make your objects seem more realistic.
Meshes Shows how to load pre-authored meshes from a file using Direct3DX.
Vertex Shader Shows how to use vertex shaders.
