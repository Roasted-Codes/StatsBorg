Xbox Makefile Wizard Settings

Use this page of the wizard to set options for the Xbox makefile project.

Option Description
Build command-line Specifies the command line used to build the project.
Output Specifies the name of the Xbox executable file produced when the project is built.
Clean commands Specifies the command line to execute for cleaning up intermediate files.
Rebuild command line Specifies the command line to execute when rebuilding the project.
