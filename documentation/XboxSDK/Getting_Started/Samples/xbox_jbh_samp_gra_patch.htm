Patch Sample

The Patch sample illustrates using N-patches in Direct3D. A low-resolution teapot mesh is created and displayed on the screen. The user can then increase the resolution of the teapot using N-patches.

The N-patches technique is the focus of this sample. The Direct3D API is used to turn Bezier input data into a higher-order mesh. The mesh is rendered with a series of calls to DrawRectPatch().
