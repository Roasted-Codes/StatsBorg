Friends Sample

The Friends sample illustrates online Friends lists on Xbox Live.

Overview

This sample shows how to add and remove friends and track friends' status using the XOnlineFriends functions through a FriendsManager wrapper. It also demonstrates using XOnlineFeedbackSend and the XOnlineMutelist functions, which are also accessed through the wrapper.

Path

Source: (XDK Root)\Samples\Xbox\Networking\Friends

Remarks

In a real Xbox application, your title would have a player list that provides the list of accounts to choose from for muting, making friends, and providing feedback. However, since there is no server here and this sample demonstrates only the use of the Friends API functions, it utilizes the user accounts on the Xbox as the potential Friends list and the current Friends list as a proxy player list for muting and feedback.

The XOnlineFeedbackSend function is used to provide feedback for a friend. However, feedback can be provided for any online user, not just friends.
