XACTWMAPlayList Sample

The XACTWMAPlayList sample demonstrates how to use the XACT library to manage both game-provided and user-created soundtracks. This sample can easily be ported for use in your own game, with a few restrictions. Game provided soundtracks must be sets of WMA files in their own directories (no other file types allowed).

As is shown in the sample, it is recommended that one cue in an XACT sound bank be reserved for managing soundtracks. This simplifies the process of managing sound banks for programmers and sound designers.
