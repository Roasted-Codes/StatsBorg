StateShader Sample

This sample illustrates the use of vertex state shaders.

Overview

A vertex shader operates on vertices. Because some shaders can write to vertex shader constants, you can have a vertex state shader that writes only vertex states. This might be useful for concatenating matrices or any other operations that pre-compute constants. Normally, these operations are done on the CPU, but a vertex state shader lets you offload more of that processing to the GPU.

The StateShader sample demonstrates the use of vertex state shaders by altering the contents of the display matrix.

Path

Source: (XDK Root)\Samples\Xbox\Graphics\VertexShader\StateShader

Remarks

When this program executes, it uses a vertex state shader to invert the display matrix. However, before the matrix is used, the program inverts it again to restore it to its original state.
