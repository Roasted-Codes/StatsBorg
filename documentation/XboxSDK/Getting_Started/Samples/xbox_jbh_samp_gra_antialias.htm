AntiAlias Sample

The AntiAlias sample shows how to use the various antialias modes.

The sample allows the antialias modes to be changed on the fly so that the appearance of the objects and the performance penalties for turning each antialias mode on can be seen. By far, the largest win is edge antialiasing, which does not require a larger back-buffer. In many cases edge antialiasing can look as good as 9x sampling. There are caveats to edge antialiasing, which are commented in the code.
