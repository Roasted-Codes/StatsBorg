PointSprites Sample

The PointSprites sample shows how to use the new Direct3D point sprites feature. A point sprite is simply a forward-facing, textured quad that is referenced only by (x, y, z) position coordinates. Point sprites are most often used for particle systems and related effects.

Overview

Without Direct3D support, point sprites can be implemented with four vertices that are oriented with each frame towards the eye point (much like a billboard). With Direct3D, though, you can refer to each point sprite merely by its center position and a radius. This saves heavily on processor computation time and bandwidth uploading vertex information to the graphics processor.

In the PointSprites sample, a particle system is implemented using point sprites. Each particle is implemented using multiple alpha-blended point sprites, giving the particle a motion-blur effect.
