PixelShader Sample

The PixelShader sample renders some geometry that uses pixel shaders.

Overview

The PixelShader sample shows how to both hard-code a pixel shader (by directly programming the combiners), and how to create a pixel shader from a file. The file-based pixel shaders start their life as *.vsh text files, and get assembled into *.xpu files by the Shader Assembler Tool (XSAsm) .

Controls

For a complete list of controller mappings, press the Back button on the Xbox controller while this sample is running.
