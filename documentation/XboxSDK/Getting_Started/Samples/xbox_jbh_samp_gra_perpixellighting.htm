PerPixelLighting Sample

The PerPixelLighting sample uses the D3DTOP_DOTPRODUCT3 texture stage state to achieve per-pixel lighting effects (in this case, bump mapping). This version of the PerPixelLighting sample does not use vertex shaders.

Overview

What you see on the screen when executing this sample is an accumulation of many render passes. A render pass is made for each of the following: the base texture, ambient lighting, directional lights, and point lights. The lighting passes use the D3DTOP_DOTPRODUCT3 texture stage state to perform the lighting calculation. The normal vectors are encoded in a texture that serves as a per-pixel lookup. The light vector is passed in as a vertex component or as a constant texture factor. See the sample source code for more details on the technique.

Note  The PerPixelLighting sample implements the per-pixel lighting technique without the use of vertex shaders and is provided for comparison purposes.
