Swizzle Sample

The Swizzle sample displays a texture that the user can rotate and toggle various options to explore the performances differences between using swizzled and linear textures.

Overview

In almost all cases, textures are "swizzled". That means the hardware (for performance reasons) stores them in such a way that the adressing index (from the tu, tv coordinates) are interleaved. To access the texture data, a Lock() call is made, and special code is needed to access the swizzled texels.

Controls

For a complete list of controller mappings, press the Back button on the Xbox controller while this sample is running.
