Auth Sample

Shows how to implement Xbox Live Authentication.

Overview

This sample illustrates Xbox Live account enumeration, PIN entry, PIN validation, sign on, and authentication. It also shows proper online multitasking using XOnlineTaskContinue, as well as error handling for PIN and sign-on failure.

Path

Source: (XDK Root)\Samples\Xbox\Networking\Auth

Remarks

The Auth sample utilizes multiple state machines for allowing up to four users to select accounts for sign on.
