PersistDisplay Sample

The PersistDisplay sample demonstrates how to keep a constant image on the display while a game is launching. It also shows how to use that persistent image in a transition effect when the game starts.

Overview

Using persistent images enables you to build a game that uses multiple .xbe files and smoothly transition between them. If your game has a given .xbe file in memory and it determines that it needs to load another, the game can place a persistent bitmap image on the display that asks the player to wait. When the new .xbe file loads, it can use the persistent image on the screen in a visual transition effect.
