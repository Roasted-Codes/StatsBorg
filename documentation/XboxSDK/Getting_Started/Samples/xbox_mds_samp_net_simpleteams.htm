SimpleTeams Sample

Overview

The SimpleTeams sample demonstrates how to use the basic team functions to create a team, recruit members, view members and teams, kick members out of the team, and delete teams. A game will also want to use the message functions to receive and process team messages (see SimpleMessaging). The presence functions are also used here to show the current online status of teammates.

Programming Notes

Full technical information on these functions is available in the SDK help.
