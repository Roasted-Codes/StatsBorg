PerformanceCounters Sample

This sample demonstrates how to use performance counters on the Xbox.

Overview

The mechanism by which the XTL collects performance data on various system resources is the performance counter. The XTL contains a pre-defined set of performance counters with which you can interact. Each counter is related to a specific area of system functionality. Examples include counters that monitor a processor's busy time, memory usage, or the efficiency of an audio stream. You use an instance of the PerformanceCounter component to connect directly to existing performance counters and interact with their data in dynamic ways.

For more information on performance monitors, see Monitoring Performance Thresholds in the Visual Studio .NET or MSDN documentation.

Path

Source: (XDK Root)\SamplesSamples\Xbox\Misc\PerformanceCounters

Remarks

The DmWalkPerformanceCounters function enumerates all available performance counters. The counter can then be queried to return a DM_COUNTDATA structure, the interpretation of which depends on the type of counter. Some counters return instantaneous values, some return raw counts, and some return percentages.
