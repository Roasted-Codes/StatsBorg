LowLevelVoiceChat Sample

Overview

The LowLevelVoiceChat sample demonstrates how a title can use the low-level voice chat components (Microphone, Encoder, Voice Queue, Decoder, Headphone) to build a drop-in custom voice engine. All voice code is contained inside the VoiceManager and VoiceCommunicator .cpp and .h files. All interaction between the title code and the voice subsystem is through the public interface of the CVoiceManager class and callback functions specified when the voice subsystem is initialized. In this way, the voice code can be dropped in to any title with little or no changes.

Keep checking this sample for updates in future XDKs, because new functionality will be added, including:

  • Packet coalescing and compression
  • Muting and Mute Lists
  • Additional debug support

Programming Notes

The code in this sample was previously the basis for the SimpleVoice sample. With the addition of the Xbox High Level Voice library (xvoice.lib) in the February 2003 XDK, the SimpleVoice sample has been updated to demonstrate the simplest way to do voice. The voice engine from SimpleVoice was moved to this sample, which is now the LowLevelVoiceChat sample.
