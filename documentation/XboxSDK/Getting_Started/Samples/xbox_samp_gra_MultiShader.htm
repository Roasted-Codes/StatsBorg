MultiShader Sample

This sample demonstrates a technique for keeping multiple small vertex shaders resident in memory at the same time.

Overview

Applications can generally use only one vertex shader at a time. Therefore, they use the SetVertexShader function to load and select a vertex shader. However, because there are 136 instruction slots available, several small vertex shaders can potentially be resident in memory at the same time.

Path

Source: (XDK Root)\Samples\Xbox\Graphics\VertexShader\MultiShader

Remarks

Xbox programs can load several small shaders into memory with the LoadVertexShader function. Next, they can select a particular shader by calling the SelectVertexShader function.

The biggest caveat here is that the fixed pipeline, when used with transformed vertices, will overwrite the first several vertex shader instruction slots. Therefore, it is often advisable to skip the first instruction slots entirely. See the source code for this sample for details.
