ContentDownload Sample

Illustrates Xbox Live content enumeration for the hard disk and online servers.

Overview

The sample demonstrates content download, online content details, content purchase, subscription purchase and cancellation, installation, verification, hard-disk content metadata and content removal.

Path

Source: (XDK Root)\Samples\Xbox\Networking\ContentDownload

Remarks

Copy the media tree to the target machine before running this sample.

If installation is canceled, the title must handle removing the content that was installed. Therefore, this sample demonstrates how to properly handle installation cancellation.

The sample assumes that all subscriptions are needed only once, and attempts to cancel them if their instance count is above 0.

Once a piece of content has been purchased, it can be downloaded multiple times without being repurchased (even if it has been deleted). Note that content ownership is Xbox specific, while subscriptions are user specific. This means that if a user moves to a different Xbox, his subscriptions will be valid but any content must be repurchased for the other Xbox.
