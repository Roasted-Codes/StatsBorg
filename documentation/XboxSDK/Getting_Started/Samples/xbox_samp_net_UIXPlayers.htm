UIXPlayers Sample

The UIXPlayers sample shows how to implement a Players List using the UIX library to handle most of the work.

Overview

This sample illustrates how to create a Players List for titles that are available on Xbox Live. Titles use Players Lists to create lobbies in which players can meet and invite each other to play.

Path

Source: (XDK Root)\Samples\Xbox\Networking\UIXPlayers

Remarks

This sample is integrated with XHV, which demonstrates how to detect when another player is highlighting and talking to the current player.
