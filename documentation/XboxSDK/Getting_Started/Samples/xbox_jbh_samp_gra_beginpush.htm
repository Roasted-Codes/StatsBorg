BeginPush Sample

The BeginPush sample demonstrates the use of the BeginPush and EndPush Direct3D APIs that give the developer tighter control over the push-buffer.

Using BeginPush can result in a performance win, although its use is not trivial. In short, using BeginPush is like exposing the inner workings of the Begin and End APIs, so the game can minimize overhead. See the sample code for details on using BeginPush, EndPush, and the accompanying helper macros.
