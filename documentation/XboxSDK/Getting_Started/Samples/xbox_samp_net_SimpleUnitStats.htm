SimpleUnitStats Sample

Demonstrates the XOnlineStatsUnitXXX API functions. A "unit" is a dynamic team consisting of up to four players.

Path

Source: (XDK Root)\Samples\Xbox\Networking\SimpleUnitStats

Remarks

To run the sample, you will need to run an output viewer such as Visual Studio .NET or XbWatson. Because a unit must have two or more members, at least two valid accounts must be present either on the hard disk or on a memory unit inserted into a controller.
