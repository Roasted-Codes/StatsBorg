SimpleAuth Sample

The SimpleAuth sample illustrates Xbox Live account logon and authentication.

Overview

This sample shows how to enumerate accounts on hard disk and memory units. For brevity, the sample chooses the first account and controller when logging on. It also demonstrates how to specify guest accounts, check for service errors, handle title auto-update, set notification state, and handle duplicate logons. In addition, this sample shows proper online multitasking using the XOnlineTaskContinue function.
