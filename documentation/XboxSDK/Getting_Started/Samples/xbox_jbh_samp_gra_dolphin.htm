Dolphin Sample

The Dolphin sample shows an underwater scene of a dolphin swimming, with caustic effects on the dolphin and seafloor. The dolphin is animated using a technique called "tweening." The underwater effect simply uses fog, and the water caustics use an animated set of textures.

Overview

The Dolphin sample illustrates several rendering methods. Fog is used to give a simple underwater effect.

The water caustics are achieved by animating a set of 32 different textures (caust00.tga through caust31.tga). The caustics can be blended into the scene using multitexturing or multipass blending techniques. This is fairly straightforward except for one tricky situation—because the bottom of the dolphin should not have caustic effects, a separate pass is done in which ambient light is removed and the dolphin is lit from above. Then the diffuse color from that render pass is blended with the caustic texture.

The dolphin is animated using a technique called "tweening" in which the vertices of the dolphin for the current frame are interpolated from three source models of the dolphin (the same model in three different positions). Each set of vertices is passed down to the vertex shader in a separate stream. For each frame, a destination mesh is generated by blending some combination of the positions and normals from these meshes together.

Dolphin vs. DolphinClassic

The DolphinClassic sample is the original version of the Dolphin sample that was included in the October 2000 release of the XDK. The Dolphin sample will continue to mature and gain new features, where the DolphinClassic sample is provided as a benchmark of sorts, so that you can gauge the performance of the XDK from release to release.
