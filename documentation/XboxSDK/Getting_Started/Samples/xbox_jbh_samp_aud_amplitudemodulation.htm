AmplitudeModulation Sample

The AmplitudeModulation DSP sample shows how to write and build a basic DSP effect. This effect takes two inputs: the first is the input signal, the second is an oscillator to be multiplied with the input signal. The single output is the product of this modulation.

You will need to use the DSP assembler (available from http://www.metrowerks.com/embedded/suite56/) to assemble DSP effects.
