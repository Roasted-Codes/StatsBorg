Gamepad Sample

The Gamepad sample lets you examine the run-time behavior of an Xbox gamepad.

Overview

There are several pages of information about the different aspects of the gamepad. There is a control test page that simply enables the user to view values associated with the controls. There is a deadzone calibration page to explore the sensitivity of the thumbstick controls in and around the software-defined deadzone. And there is a control quantization page for seeing which digitized values are read from the analog controls.
