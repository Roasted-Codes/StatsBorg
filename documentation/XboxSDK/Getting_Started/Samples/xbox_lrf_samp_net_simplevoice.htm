SimpleVoice Sample

The SimpleVoice sample demonstrates how a title can use the sample voice code that is provided as a "black box" voice subsystem. All voice code is contained inside the VoiceManager and VoiceCommunicator .cpp and .h files. All interaction between the title code and the voice subsystem occurs through the public interface of the CVoiceManager class and through the callback functions specified when the voice subsystem is initialized. In this way, the voice code can be dropped into any title with little or no change.

Keep checking this sample for updates in future versions of the XDK. New functionality will be added, including:

  • Packet coalescing and compression
  • Voice mask management
  • Muting and Mute Lists
  • Additional debug support
