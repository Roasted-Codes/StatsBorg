XACTInteractiveAudio Sample

The XACTInteractiveAudio sample demonstrates how to play back and control interactive audio using the XACT run-time engine.

Overview

The XACT project for this sample contains just one sound cue, which is an interactive audio cue. The cue responds to changes in a global XACT variable by transitioning between different sounds.

Be sure to copy the media tree to the target machine before running this sample.
