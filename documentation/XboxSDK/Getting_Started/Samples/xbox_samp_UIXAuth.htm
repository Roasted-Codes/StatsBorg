UIXAuth Sample

The UIXAuth sample shows how to implement Xbox Live Authentication using the UIX library to handle most of the work.

Overview

Run copytoxb.bat to copy the required fonts and skin files to the Xbox prior to running this sample.

UIXAuth is designed to be a simple demonstration of how to implement Xbox Live Authentication using the UIX engine.

Path

Source: (XDK Root)\Samples\Xbox\Networking\UIXAuth
