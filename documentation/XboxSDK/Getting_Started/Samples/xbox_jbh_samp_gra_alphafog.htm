AlphaFog Sample

The AlphaFog sample illustrates a fogging technique that modifies the alpha value of far-away objects. This is useful in scene-culling techniques to avoid the visible "pop" that occurs when far-away objects (such as trees and buildings) are added to a scene.

The objects that are alpha-fogged are the columns. See the code (especially the vertex shader) used to render the columns for more information.
