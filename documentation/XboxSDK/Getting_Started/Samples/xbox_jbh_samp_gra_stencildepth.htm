StencilDepth Sample

The StencilDepth sample uses stencil buffers to display the depth complexity of a scene. The depth complexity of a scene is defined as the average number of times each pixel is rendered to.

Using stencil buffers is a depth-buffer technique that can be updated as geometry is rendered and used again as a mask for drawing more geometry. Common effects include mirrors, shadows (an advanced technique), dissolves, and so on.

Overview

Displaying depth complexity is a valuable tool for analyzing the performance of a scene. Scenes with high amounts of overdraw could benefit from some scene optimization, such as sorting the geometry in a front-to-back order.

Controls

For a complete list of controller mappings, press the Back button on the Xbox controller while this sample is running.
