UIXKeyboard Sample

Shows an example of writing a custom UIX feature. In this example, we have wrapped the functionality of the "Virtual Keyboard" sample as a UIX feature.

Overview

UIX has extensibility features that enable a developer to write custom UIX features. This helps to minimize the amount of extra UI code that needs to be written. Within UIX, a feature can load skinnable objects and have convenient access to localized text strings.

Path

Source: (XDK Root)\Samples\Xbox\Networking\UIXKeyboard

Remarks

Copy the media tree to the target machine before running this sample.
