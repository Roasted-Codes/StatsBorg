SimpleMatch Sample

The SimpleMatch sample demonstrates the following:

  • Checking for, and joining, an existing accepted game invitation.
  • Creating and advertising a new game session.
  • Updating an existing game session.
  • Deleting a game session once it is no longer needed.
  • Querying for game sessions.

Overview

On startup, SimpleMatch checks for and joins an accepted game invitation. If no invitation is found, the sample queries for an existing invitation and joins it. Otherwise, the sample creates a new session, updates that session, and then deletes it.

To see the sample in action, you can try a few different scenarios:

  • Game invitations. Build the Friends sample and stamp it with the title ID of the SimpleMatch sample. Invite your friend, and then have your friend run SimpleMatch while you watch for output indicating that an invitation was found.

    Note    Because no session exists, you cannot actually join the session.

  • Querying sessions. Run SimpleMatch on two Xbox consoles, so that one Xbox console will find the session created by the other.
  • Hosting sessions. Run SimpleMatch without any existing sessions. Watch the output as the sample creates, updates, and then deletes a session.

    Note    Sessions created by SimpleMatch last for about 30 seconds.
