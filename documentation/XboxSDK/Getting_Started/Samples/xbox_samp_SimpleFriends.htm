SimpleFriends Sample

The SimpleFriends sample demonstrates the XOnlineFriends API functions.

Overview

In order to run the sample, you should have two accounts on your Xbox, and an Output viewer such as Visual Studio .NET or XbWatson running.

Because no session is actually created, the game invitation API functions are not implemented, but they are described fully with examples in the source code (Search for "Game invitations").

Path

Source: (XDK Root)\Samples\Xbox\Networking\SimpleFriends
