Strip Sample

The Strip sample demonstrates usage of Xbox's tri-stripper. The sample renders a robot (made with 77 individual meshes) 12 times per frame and displays statistics on the render.

Overview

When the Strip sample is launched, it passes the robot mesh to the tris-tripper to evaluate the mesh and generate tri-strips from the vertex data. The Strip sample uses this optimized data when rendering the scene.

Note   You would normally generate the stripped data at authoring time and load the stripped vertices on startup.

Controls

For a complete list of controller mappings, press the Back button on the Xbox controller while this sample is running.
