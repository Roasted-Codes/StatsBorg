Setting Up Your Console and Development System

Before using your XDK, you'll need to setup your Xbox console and your development system.

The Xbox console runs the Xbox games that you develop. Use the console to upload, execute, test, and debug your games. The steps to prepare the console for use are described in Xbox Console Setup.

The development system is the desktop computer you provide that will run the Microsoft® Visual C++® compiler to develop and compile Xbox games, and is where the Xbox SDK is installed. The steps to prepare the development system for XDK installation and use are described in Xbox Development System Setup.

Additionally, if you will be using the Xbox Game Disc Emulation software and hardware to analyze DVD layout and performance, follow the instructions in Xbox Game Disc Emulation System Setup and Upgrade.
